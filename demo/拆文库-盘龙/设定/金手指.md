# 金手指

| 项目 | 内容 |
|------|------|
| 类型 | artifact + bloodline |
| 名称 | 盘龙戒指（大地之戒）+ 龙血战士血脉 |
| 描述 | 盘龙戒指是堪比神器的神秘宝物，通体黝黑、似木似石材质，戒环上刻有模糊的龙形纹路。林雷在后园古屋中偶然发现并命名为"盘龙戒指"。戒指的核心功能是作为圣域魔导师德林柯沃特灵魂的寄居之所。德林柯沃特五千年前被仇敌偷袭后自封灵魂于戒指中，只有通过鲜血滴落认主才能释放其灵魂。认主后主人可看到德林的灵魂形态，与其心灵交流，并享受圣域魔导师级别的魔法指导。德林只能离戒指三米范围活动，其他人均无法感知其存在（除非灵魂层次相当的圣域强者）。龙血战士血脉虽不足以觉醒为龙血战士，但使林雷天生体质优于常人，对血腥战斗有天然的渴望和适应力，且单纯肉体训练可以达到更高境界（曾爷爷曾达七级战士）。两者结合形成完整的金手指体系：血脉提供肉体基础，戒指提供顶级导师。|
| 核心机制 | 鲜血认主→释放魂灵→心灵交流+魔法指导+秘密修炼 |
| 当前能力 | 地系元素亲和力极高（千里挑一）、精神力十倍于常人、德林柯沃特可教授至圣域级别魔法 |
