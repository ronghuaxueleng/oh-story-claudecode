| 项目 | 内容 |
|------|------|
| 标题 | 盘龙之灵与魔法苦修 |
| 类型 | 修炼 |
| 概要 | 因为林雷滴血认主、成功拜五千年前的普昂帝国圣域魔导师德林柯沃特为师，他正式踏上地系魔法师的修炼之路。德林柯沃特说明自己以灵魂形式存在、除林雷外常人皆看不见、可与林雷心灵交流，只有灵魂层次相当的圣域强者才能勉强感知。德林柯沃特系统讲解地系魔法的强大（各系禁忌魔法的毁天灭地威力、地系因陆地战斗占优势、单体防御「大地守护圣铠」的等级进阶）与魔法基础原理（魔法力、精神力、天地元素三者关系）。检测显示林雷的元素亲和力极高（千名魔法师难寻一个）、精神力是同龄人的十倍，德林柯沃特承诺以圣域魔导师之力可助其达到八级魔法师。林雷立誓不让父亲与家族失望，从此过上极度充实的苦修生活——清晨傍晚战士训练、上午贵族课业、下午与晚上修炼地系魔法与冥想，进步飞速，气质变得沉稳，连父亲霍格都惊讶。本剧情是林雷崛起之路的真正起点，节选在他于古屋院子发现速度极快的魔兽幼崽「影鼠」处收尾，留下「收服第一只魔兽」的钩子。 |

| 核心冲突 | 林雷魔法资质的不确定性 vs 圣域魔导师名师加持下的成长可能 |
| 章节范围 | 第20-23章 |

结构分布：
