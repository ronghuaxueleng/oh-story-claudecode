# 剧情：龙血战士家族的秘密

| 项目 | 内容 |
|------|------|
| 标题 | 龙血战士家族的秘密 |
| 类型 | 主线 |
| 概要 | 因为玉兰大陆尚武、男人地位由实力决定，八岁的衰败贵族子弟林雷在护卫队长希尔曼带领下刻苦做战士体能训练，并展露出超越同龄人的惊人天赋。深秋参军季，父亲霍格郑重带林雷夜入宗堂，揭示巴鲁克家族其实是传说四大终极战士之一「龙血战士」的血脉家族，并用龙血针为林雷检测血脉浓度，结果失败。霍格随后吐露真正心愿——找回六百年前因家族衰败被变卖、价值十八万金币的传承之宝战刀「杀戮」，林雷当场立誓要带回传承之宝。因为家族经济拮据，霍格不得不变卖石雕维持家用，林雷提出学石雕改善家境被严厉劝阻。林雷在打扫宗堂后到后园废弃古屋寻宝，偶然在塌落木架的夹层中发现一枚似木似石、毫不起眼的黑色戒指，因戒环雕纹像龙而为它取名「盘龙戒指」。本剧情确立了林雷的崛起野心与「找回传承之宝」的长线目标，并埋下盘龙戒指这一核心机缘。 |
| 核心目标 | 确认家族身份与个人天赋，找到家族复兴的方向 |
| 核心冲突 | 林雷的强者向往与家族复兴渴望 vs 平民阶层局限、家族衰败现实（血脉失传、经济拮据、传承之宝失落） |
| 章节范围 | 第1-7章 |

结构分布：
- 铺垫期：第1章（晨练立人设、世界观铺设）
- 发展期：第2-3章（揭家族龙血身份、血脉检测失败、立誓找回传承之宝）
- 高潮期：第3章（检测失败 + 传承之宝揭秘的情感高潮）
- 收尾期：第4-7章（家境困窘日常、古屋寻宝、发现盘龙戒指）
