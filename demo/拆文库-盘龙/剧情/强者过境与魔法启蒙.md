# 剧情：强者过境与魔法启蒙

| 项目 | 内容 |
|------|------|
| 标题 | 强者过境与魔法启蒙 |
| 类型 | 主线 |
| 概要 | 因为一名八级双系魔法师骑七级魔兽迅猛龙路过乌山镇，与一支为争夺德佩洛影钻的佣兵小队爆发冲突，林雷得以第一次目睹真正的强者战斗。迅猛龙咬碎佣兵战士的血腥场面让林雷初时恐惧，随后体内血液沸腾、生出杀戮战斗的渴望（疑为龙血血脉之故）。神秘魔法师施展八级火系魔法「火蛇之舞」，七条数十米长的火蛇将整支佣兵小队烧成灰烬，乌山镇房屋大量焚毁。这场灾难性的战斗彻底打开了林雷的眼界，他对强大的魔法师生出畏惧与渴望，当场确立了「追求强者巅峰」的人生道路。因为震撼太大，林雷回家后向父亲霍格系统请教魔法师与战士的种种知识——魔法师的分级与地位、强者寿命长的原理、天地元素的属性、收服魔兽的方法。霍格告知林雷今年深秋王都芬莱城有魔法学生招生测试，林雷由此把「成为魔法师」锁定为新的努力方向。本剧情完成林雷从「纯肉体战士路线」向「魔法师路线」的目标转向。 |
| 核心目标 | 打开眼界、确立追求强者巅峰的方向，锁定「成为魔法师」为新目标 |
| 核心冲突 | 林雷井底之蛙的认知 vs 圣域级别力量真相的巨大冲击 |
| 章节范围 | 第8-13章 |

- 收尾期：第12-13章（林雷向霍格请教魔法知识、得知魔法招生测试）
