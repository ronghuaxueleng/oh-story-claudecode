# 霍格

**身份背景**：巴鲁克家族当代族长，传承五千年的龙血战士家族后裔。妻子已故，独自抚养两个儿子。曾学习石雕十年但一无所成。

**核心经历**：作为衰败家族的族长，经济困难到需不断变卖家传物品维持家用。在宗堂为林雷检测龙血血脉浓度。曾学习石雕十年但作品一钱不值。

**性格特质**：严厉但有深厚父爱。对林雷的教育要求堪称严厉（荆条惩罚），但在关键时刻展现出温暖（检测失败后安慰、林雷受伤后流泪）。

**能力特长**：精通玉兰大陆地理政治、魔法体系知识。作为古老家族族长拥有丰富的知识传承。

**关键情节**：
1. 第2-3章：宗堂揭示家族秘密，检测血脉
2. 第5章：变卖石雕维持家用
3. 第11章：紧张应对八级魔法师
4. 第13章：详细讲解魔法师体系
5. 第17章：看到受伤的林雷流泪
