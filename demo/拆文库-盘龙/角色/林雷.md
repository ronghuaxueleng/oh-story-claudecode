# 林雷

> 200-500字档案

**身份背景**：巴鲁克家族长子，传承五千年的龙血战士家族后裔。家族已衰败千年，仅剩父亲霍格和弟弟沃顿三人。母亲在生沃顿时去世，爷爷在战争中阵亡。

**核心经历**：六岁开始在希尔曼指导下进行肉体训练，展现远超同龄人的天赋。六岁半得知家族秘密，龙血血脉检测不达标。八岁目睹八级魔法师的恐怖力量和圣域强者的战斗，确立追求巅峰的决心。在后园古屋发现盘龙戒指。圣域之战中为保护弟弟沃顿受伤，鲜血激活盘龙戒指，释放出五千年前的圣域魔导师德林柯沃特的灵魂。成为德林的秘密弟子，走上地系魔法师道路。

**性格特质**：坚韧执着、聪明早熟、重视亲情。面对挫折（血脉检测失败）不放弃反而更努力；面对血腥（路加被咬碎）从恐惧转为战斗渴望；面对危险（碎石保护沃顿）毫不犹豫牺牲自己。

**能力特长**：天生体质优于常人（龙血血脉）、地系元素亲和力极高（千里挑一）、精神力十倍于常人。肉体训练可赶上大三四岁的少年。

**成长弧线**：从天真少年→得知家族秘密后更加刻苦→目睹强者后确立目标→获得导师后飞速成长。character_arc: 从普通小镇少年到觉醒魔法天赋的崛起者。

**关键情节**：
1. 第3章：龙血血脉检测失败，立誓找回传承之宝
2. 第11章：目睹火蛇之舞，确立追求强者巅峰的道路
3. 第17章：以身体保护沃顿受伤，盘龙戒指激活
4. 第19章：德林柯沃特现身，成为秘密弟子
5. 第23章：天赋测试确认极高天赋，半年苦修飞速进步
