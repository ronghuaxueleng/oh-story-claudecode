# 德林柯沃特

**身份背景**：五千年前普昂帝国的圣域魔导师，地系最强。被仇敌哈姆林和另一位圣域强者偷袭，自封灵魂于盘龙戒指中。普昂帝国在其入戒前已存在近三千年。

**核心经历**：玉兰历4280年被偷袭后自封灵魂于盘龙戒指。沉睡五千多年后，因林雷的鲜血滴落认主而被释放。只能离戒指三米活动，灵魂消散才能彻底脱离束缚。

**性格特质**：和蔼可亲、自信骄傲（尤其谈论地系魔法时）、孤独五千多年后对林雷产生真挚感情。教学循循善诱。

**能力特长**：地系圣域魔导师，可教授七级至圣域级别的魔法。强横精神力可将地系元素浓度提升百倍。

**关键情节**：
1. 第18章：盘龙戒指异变，灵魂现身
2. 第19章：讲述往事，与林雷建立爷孙情感
3. 第20-21章：讲解地系魔法体系
4. 第22-23章：测试林雷天赋，开始秘密教授
