# Codex 题材专用模板

这份文件只做一件事：

**按题材给你现成话术。**

如果你已经知道自己要写哪一类，就不用从通用提示词里再拼。
直接找到对应题材，复制给 `Codex` 就行。

适用范围：

- 女频拉扯
- 男频爽点
- 盐言虐点
- 悬疑反转

完整版入口：

- [CODEX提示词手册.md](./CODEX提示词手册.md)
- [CODEX速用提示词.md](./CODEX速用提示词.md)
- [CODEX情绪词典与场景加压模板.md](./CODEX情绪词典与场景加压模板.md)

---

## 怎么用

最简单的方式：

1. 找到你正在写的题材
2. 复制一段给 `Codex`
3. 如果效果还不够，再叠加一条“场景加压句”

推荐组合：

- 开书时：用“题材开书模板”
- 写某章时：用“单章/单场模板”
- 觉得文平时：叠加“救火模板”

## 一、女频拉扯模板

适合：

- 现言
- 古言
- 婚恋
- 破镜重圆
- 强取豪夺
- 误会虐恋

核心不是“甜”，而是：

**想靠近，又不敢；明明在意，又不肯认；表面有来有回，底下全是旧账和软肋。**

### 1. 开书模板

```text
story-long-write，帮我开一本女频长篇。
平台：番茄
题材：现言拉扯 / 古言虐恋
目标情绪：压抑、试探、嘴硬、后期失控
对标：强拉扯、高压关系开局
要求：
1. 先做题材定位
2. 主打男女主关系张力，不要把重心放到大世界观
3. 先确定两个人各自想要什么、怕失去什么、最不能被碰的软肋是什么
4. 直接给我前三卷卷纲和前30章细纲
5. 每章都要有关系推进或关系恶化，不能空转
直接写入文件。
```

### 2. 感情戏加压

```text
story-long-write，把这一场感情戏重写得更有拉扯。
不要写成直白表白，也不要写成甜言蜜语堆砌。
我要看到：
1. 谁先试探
2. 谁先躲
3. 谁嘴上更硬
4. 谁先露破绽
5. 谁碰到了对方最不能提的点
6. 结尾留一句没说完的话，或者一个让人回想的动作
直接落文件。
```

### 3. 冷战戏

```text
story-long-write，这场冷战太平了。
不要只写两个人不说话。
请重写成：
1. 明明都在意
2. 但谁也不肯先低头
3. 每一句话都带刺
4. 每个动作都在暴露没放下
5. 结尾让局面更僵，不要轻易和解
```

### 4. 误会戏

```text
story-long-write，把这场误会写得更揪心。
不要靠低级信息差硬拧。
我要的是：
1. 两个人其实都不是不在意
2. 但各自都带着旧伤去理解对方
3. 误会成立是因为他们太懂彼此的痛点
4. 吵的时候别把话说透
5. 留一个以后知道真相会更疼的点
```

### 5. 重逢戏

```text
story-long-write，把这场重逢写得更有后劲。
不要一见面就顺。
按这个顺序写：
1. 先认出来
2. 先装不在意
3. 身体反应先露馅
4. 旧账被轻轻碰一下
5. 结尾留一个更难收拾的尾巴
```

### 6. 女频救火句

```text
story-long-write，这段关系戏太平了。
不要只润色词句，直接把“试探、嘴硬、回避、破绽、旧伤、没说出口的话”都补出来。
我要的是拉扯，不是顺滑交流。
```

## 二、男频爽点模板

适合：

- 都市脑洞
- 玄幻升级
- 系统文
- 诸天流
- 末世流
- 重生逆袭

核心不是“赢”，而是：

**先压，再憋，再揭信息差，再硬打回去。**

### 1. 开书模板

```text
story-long-write，帮我开一本男频爽文长篇。
平台：番茄
题材：都市脑洞 / 玄幻升级 / 系统流
目标情绪：持续爽感、强期待感、章尾钩子强
对标：高钩子、高追读开局
要求：
1. 开局必须强冲突，不要慢热
2. 主角优势要早亮，但不要全亮
3. 前3章必须完成人设基点、切入点基点、金手指基点
4. 给我前三卷卷纲和前30章细纲
5. 每3章至少一个明显爽点，每章结尾都要能追下一章
直接写入文件。
```

### 2. 打脸反杀场

```text
story-long-write，把这一场打脸写得更爽。
顺序必须是：
1. 先让对面压着主角
2. 再放大主角这次输不起的代价
3. 再揭开信息差
4. 最后让主角硬打回去
5. 爽点落地后补一个围观反应或后果扩散
不要边爆边解释，不要软收。
```

### 3. 升级场

```text
story-long-write，这场升级太平了。
不要只写“变强了”。
我要看到：
1. 升级前主角卡在哪
2. 这次突破为什么值钱
3. 升级后最先能反压谁
4. 这次升级会带来什么新的麻烦和更大的目标
把升级写成剧情推进，不要写成数据播报。
```

### 4. 装逼场

```text
story-long-write，把这场装逼写得更稳。
不要让主角自己夸自己。
请按这个顺序写：
1. 别人先低估
2. 局面先失控
3. 主角出手
4. 周围人的认知被刷新
5. 结尾留一个更大的麻烦或关注
```

### 5. 章尾钩子

```text
story-long-write，这章结尾不够追读。
不要总结。
请重写章尾，用以下至少一种：
1. 新信息
2. 更大的敌人
3. 主角马上要反压的目标
4. 一个读者比角色先知道的信息差
5. 一句会让人立刻点下一章的话
```

### 6. 男频救火句

```text
story-long-write，这章爽点太软了。
不要改成花里胡哨的文风，直接补：
压制、代价、信息差、硬打回去、外界反应。
我要的是痛快，不是温吞。
```

## 三、盐言虐点模板

适合：

- 婚恋反转
- 家庭错位
- 白月光替身
- 误会错过
- 死后真相
- 女性成长反杀

核心不是“虐”，而是：

**前面一直忍，后面真相一掀，读者回头一看，前面的每个细节都开始疼。**

### 1. 开篇模板

```text
story-short-write，帮我写一个盐言短篇。
题材：婚恋虐文 / 家庭反转 / 女性复仇
目标情绪：意难平、后知后觉更疼、结尾有余波
字数：1万到1.5万
要求：
1. 默认第一人称
2. 开头100字内给足有效信息
3. 先做核心框架、核心反转、3条真铺垫、1条误导线索、小节大纲
4. 反转放在后半段，不要太早揭底
5. 结尾不要总结，只留安静一刀
直接写入文件。
```

### 2. 扎心场

```text
story-short-write，把这场写得更扎心。
不要只把事实说出来。
我要看到：
1. 人物本来想说但没说的话
2. 人物明明想留却只能走
3. 表面没事，身体已经绷不住
4. 一个会在结尾回扣的物件或动作
5. 收尾不要喊，不要哭天抢地，只留后劲
```

### 3. 死后真相 / 迟来的真相

```text
story-short-write，把这个“死后才知道真相”的反转打得更疼。
要求：
1. 前面铺3条能回头对上的真线索
2. 中间给1条误导线索
3. 揭露时不要一次说完，用证物/遗物/旧消息/旁人一句话一点点撕开
4. 让主角或读者意识到：原来最想知道的时候，已经来不及了
5. 结尾给我一个回想起来会二次刺痛的动作或物件
```

### 4. 女性反杀

```text
story-short-write，把这个女性反杀写得更带劲。
前半段压迫感要足，后半段反打要干脆。
不要让反杀来得太轻松。
我要看到：
1. 她之前忍的代价
2. 她被逼到什么程度
3. 她真正记恨的点是什么
4. 她怎么翻盘
5. 翻盘后最痛快的一击是什么
```

### 5. 盐言结尾

```text
story-short-write，这个结尾太像作文收尾了。
重写：
1. 不要升华主题
2. 不要解释人物这一路学会了什么
3. 用一个物件、一个动作、一句很轻的话收尾
4. 让读者看完之后过几秒才开始疼
```

### 6. 盐言救火句

```text
story-short-write，这篇太平了。
不要只改字句，先把“忍耐、错过、迟来的真相、没说出口的话、结尾余波”全部拉起来。
我要的是后劲，不是顺着讲完。
```

## 四、悬疑反转模板

适合：

- 短篇悬疑
- 女频悬疑
- 身份错位
- 现实诡异
- 心理惊悚
- 线索反转

核心不是“谜”，而是：

**读者总觉得哪里不对，但真相掀开那一刻，又发现前面早就埋了。**

### 1. 开书 / 开篇模板

```text
story-short-write，帮我写一个悬疑反转短篇。
题材：身份错位 / 现实惊悚 / 婚恋悬疑
目标情绪：不安、误判、最后一下反转震感
字数：8000到1.2万
要求：
1. 开头前三句必须是事件、对话、动作或信息炸弹
2. 不要靠纯环境描写起头
3. 先做核心反转、真线索、误导线索、小节大纲
4. 反转要能回头对线索
5. 结尾留余震，不要全讲透
```

### 2. 铺线索

```text
story-short-write，重查这篇的线索设计。
我要的不是把真相藏起来，而是把真相摆在眼前但让读者先看偏。
请分别列出：
1. 真线索
2. 误导线索
3. 哪一条线索负责让读者误判
4. 哪一条线索在结尾回看最关键
如果线索太直白，就重做。
```

### 3. 不安感加压

```text
story-short-write，把这场的不安感提起来。
不要直接写“很可怕”“很诡异”。
请补：
1. 一个反常细节
2. 一个读者比角色先隐约察觉的不对劲
3. 一个越想越怪的动作或对话
4. 结尾再推半步，让危险更近
```

### 4. 揭露场

```text
story-short-write，这个反转揭露得太平了。
重写揭露场：
1. 先补一层误导
2. 再让真相被一句话、一个动作、一个证物或一个错位细节掀开
3. 揭开后不要立刻解释完
4. 给读者一点回头对前文的时间
5. 结尾再留一丝余震
```

### 5. 结尾余震

```text
story-short-write，这个悬疑结尾太干净了。
重写结尾，不要把一切讲圆。
请留下：
1. 一点没完全解释完的寒意
2. 一个回头想更不对劲的细节
3. 一个让读者离开页面以后还会想一下的尾巴
```

### 6. 悬疑救火句

```text
story-short-write，这篇悬疑现在只有事件，没有压迫感。
不要堆形容词，直接补反常细节、误判、危险逼近和揭露后的余震。
我要的是越看越不对劲。
```

## 最后给你四个“一眼就能复制”的题材狠句

### 女频

```text
把这场写成拉扯，不要写成顺滑沟通。
我要试探、嘴硬、回避、破绽和没说出口的话。
```

### 男频

```text
把这段爽点打狠一点。
先压、再憋、再揭信息差，最后硬打回来。
```

### 盐言

```text
把这段改成“现在看平静，回头想更疼”。
不要喊苦，要留后劲。
```

### 悬疑

```text
把这段改成“当下说不清哪里不对，回头一看全都不对”。
不要直接讲恐怖，要让危险自己浮出来。
```
