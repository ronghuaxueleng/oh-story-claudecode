# Codex 情绪词典与场景加压模板

这份文件只解决一件事：

**写出来不平。**

它不负责帮你搭世界观，也不负责替你想主线。
它只负责在你已经有故事的前提下，把情绪、压强、拉扯、疼点、爆点往上提。

下面所有内容，都是你可以直接复制到 `Codex` 里说的话。

---

## 先记住一个总公式

情绪不是形容词。

情绪 = **人物想要什么 + 眼下得不到 + 代价越来越大 + 身体先撑不住 + 结尾还留着没完**

所以不要只说：

```text
写得更感人一点。
```

要说：

```text
把这一场的欲望、阻碍、代价、身体反应和余波都拉出来。
```

## 一组万能压场句

这几句几乎什么题材都能用。

### 1. 万能句

```text
这场太平了。
不要只润色文风，直接把场景压强拉高。
给我补足：
1. 人物这一刻最想得到什么
2. 他最怕失去什么
3. 外部阻碍是什么
4. 内部嘴硬是什么
5. 身体先出现什么破绽
6. 这场结束后局面是更糟了，还是更紧了
按这个标准重写，直接落文件。
```

### 2. 针对“只有剧情没有感觉”

```text
这段现在只有事件推进，没有情绪推进。
把“发生了什么”保留，但补上“人物因此失去了什么、忍住了什么、没说出口什么”。
少解释，多写动作、停顿、反常、失控、说一半咽回去的话。
```

### 3. 针对“有情绪词，没有情绪”

```text
不要直接写“他很难过”“她很愤怒”“她很害怕”。
改成身体反应、动作、视线、呼吸、停顿、说话方式和选择变化。
让读者自己感觉到，不要替读者总结。
```

## 情绪词典

下面不是文学定义，是写作上的落地定义。

---

## 1. 爽感爆发

### 读者想感受到什么

- 前面压了很久
- 这一刻终于硬打回去
- 主角不是侥幸赢，是把局面硬翻过来

### 场景必须有的东西

- 压制
- 代价
- 信息差
- 干脆反杀
- 围观反应或后果扩散

### 直接对 Codex 说

```text
把这一段爽点重写得更狠。
顺序必须是：
1. 先压住主角
2. 再放大代价
3. 再揭开信息差
4. 最后让主角硬打回来
要求：
不要边爆边解释，不要拖，不要软收。
爆发后给我补一个外界反应，让爽点落地。
```

---

## 2. 压抑憋屈

### 读者想感受到什么

- 主角现在就是没法立刻翻脸
- 他得忍
- 越忍越替他难受

### 场景必须有的东西

- 实力不对等
- 眼下翻脸的代价太大
- 人物必须硬撑
- 忍耐里要藏着一点反击火种

### 直接对 Codex 说

```text
把这场压得更难受一点。
不要让主角立刻反击。
我要的是“明明被踩了，却只能先忍”的感觉。
请写出：
1. 他为什么现在不能翻脸
2. 他硬撑时最明显的身体反应
3. 他嘴上怎么装没事
4. 他心里真正记下了什么
5. 结尾留一个以后一定会炸回来的钩子
```

---

## 3. 暧昧拉扯

### 读者想感受到什么

- 两个人都动了
- 但谁都不肯直接承认
- 每一句话都像在试探

### 场景必须有的东西

- 想靠近
- 怕失控
- 嘴硬
- 误读或故意装不懂
- 一个留尾巴的小动作

### 直接对 Codex 说

```text
把这一场写得更有拉扯感。
不要写成直白告白，也不要写成平铺暧昧。
请按这个顺序重写：
1. 一方先试探
2. 另一方装没听懂或故意岔开
3. 两个人都露一点破绽
4. 嘴上仍然不承认
5. 结尾留一个会让读者回想的动作或没说出口的话
```

---

## 4. 意难平

### 读者想感受到什么

- 不是大哭大喊
- 是明明有机会，却还是错过去了
- 事后想起来更疼

### 场景必须有的东西

- 来不及
- 没说出口
- 误会或命运错位
- 小物件或旧动作回扣

### 直接对 Codex 说

```text
把这段写成意难平，不要写成单纯煽情。
我要的是“事后回想更疼”。
请补足：
1. 人物本来想说但最终没说的话
2. 他以为还有下次，其实没有了的地方
3. 一个会在结尾回扣的物件或动作
4. 收尾不要升华，只留安静一刀
```

---

## 5. 悬疑不安

### 读者想感受到什么

- 知道不对劲
- 但一时说不清哪里不对
- 越往下越发毛

### 场景必须有的东西

- 反常细节
- 信息缺口
- 角色误判
- 晚一步意识到危险

### 直接对 Codex 说

```text
把这段的不安感提起来。
不要靠大喊大叫和直说恐怖。
请用：
1. 一个反常细节
2. 一个角色没意识到、读者先隐约察觉的危险
3. 一个越想越不对的动作或对话
4. 结尾再往前推半步，让危险更近
```

---

## 6. 反转震撼

### 读者想感受到什么

- 原来是这样
- 回头看前面居然都对得上
- 不是天降，是回看合理

### 场景必须有的东西

- 前面埋过线索
- 读者被误导过
- 揭露时一刀切开
- 揭露后余震持续

### 直接对 Codex 说

```text
这个反转不够震。
不要推翻全篇，先把揭露这一场重打。
要求：
1. 先补一层误导
2. 再让真相在一个瞬间被证物/证人/一句话/一个动作掀开
3. 揭露后不要立刻解释完，留余震
4. 让读者能回想前文，意识到线索早就埋了
```

---

## 7. 共鸣心酸

### 读者想感受到什么

- 不是戏剧化大事
- 是太像真实生活里的某一刻
- 看完会想起自己

### 场景必须有的东西

- 具体生活细节
- 没被照顾到的小委屈
- 人物习惯性忍
- 情绪不是炸，是慢慢发酸

### 直接对 Codex 说

```text
把这场写得更有共鸣，不要写得像空泛鸡汤。
少说大道理，多给生活细节。
我要看到：
1. 一个特别具体的小委屈
2. 人物为什么习惯性忍过去
3. 表面没事，实际心里发酸的那个瞬间
4. 结尾留一点安静的后劲
```

---

## 8. 恐惧压迫

### 读者想感受到什么

- 危险不是突然跳出来
- 是一直在靠近
- 人物发现时已经晚半步

### 场景必须有的东西

- 环境不直接吓人，但不对劲
- 人物判断失误
- 逃不掉或躲不开
- 身体先于理智反应

### 直接对 Codex 说

```text
把这场的压迫感做出来。
不要直接写“很可怕”。
请按这个顺序推进：
1. 先给一个说不上来的不对劲
2. 再让人物做出一次错误判断
3. 再让危险靠近到无法回避
4. 最后先写身体反应，再写人物意识到自己晚了
```

## 场景加压模板

下面这些比情绪词典更直接，适合你拿来现场救火。

---

## 1. 委屈场

```text
重写这场委屈戏。
不要让人物只在心里想。
我要看到：
1. 别人是怎么踩到他最痛的地方
2. 他为什么不能当场翻脸
3. 他表面怎么维持体面
4. 他身体哪个地方先绷不住
5. 结尾埋一个以后会反打回来的点
```

## 2. 摊牌场

```text
把这一场摊牌写得更有压强。
不要一上来就全说透。
按“试探→顶回去→再逼一步→终于戳破”推进。
每一轮都要比上一轮更疼、更近、更难退。
```

## 3. 吵架场

```text
这场吵架太像解释剧情了。
重写时不要让角色把心里话全说出来。
加入：
1. 打断
2. 曲解
3. 翻旧账
4. 说到一半停住
5. 表面攻击别人，实际暴露自己最怕的点
```

## 4. 重逢场

```text
把这场重逢写得更有劲。
不要一见面就顺。
我要的是：
1. 先认出来
2. 先装不在意
3. 身体反应先露馅
4. 旧账或旧情绪被轻轻碰一下
5. 结尾留一个更难收拾的尾巴
```

## 5. 离别场

```text
把这场离别写得更疼。
不要大哭大喊，不要总结感情。
重点写：
1. 两个人真正舍不得的是什么
2. 谁先装没事
3. 谁先露破绽
4. 哪句话本来想说却没说
5. 最后用一个动作或物件收尾
```

## 6. 爽点反杀场

```text
把这一场反杀写得更炸。
前半段把压制和代价写足，后半段硬打回来。
不要让主角靠运气翻盘。
要让读者觉得：前面越憋，这里越痛快。
```

## 7. 亲情扎心场

```text
把这场亲情戏写得更扎心。
不要靠喊口号和回忆杀。
我要的是：
1. 一个具体的小动作
2. 一个多年形成的习惯
3. 一句看似普通、其实一下扎进去的话
4. 人物表面平静，回身以后才露出破绽
```

## 8. 爱情试探场

```text
把这场爱情试探写得更好看。
不要让两个人直接把喜欢说出来。
重点写：
1. 谁先靠近半步
2. 谁先退
3. 谁嘴硬
4. 谁用别的话题掩饰在意
5. 结尾留一个比表白更勾人的破绽
```

## 9. 崩盘场

```text
把这场崩盘写得更有冲击。
不要一句“他崩溃了”带过。
我要看到崩盘是怎么一层层裂开的：
1. 先否认
2. 再硬撑
3. 再失控
4. 最后留下狼狈余波
```

## 10. 结尾余波场

```text
这章结尾太像总结了。
重写结尾，不要升华，不要讲主题。
用一个动作、一个物件、一个眼神、或者一句没说完的话收尾。
我要的是余波，不是总结。
```

## 对话加压模板

“平”经常平在对话。

### 1. 对话太直

```text
这段对话太直白，像在交代设定。
重写时加入潜台词、回避、打断、反问、说一半停住。
不要让角色把真正想说的话完整说出来。
```

### 2. 对话没火花

```text
这段对话没有火花。
请重写成“表面聊这件事，实际在较劲另一件事”。
每一句都要有攻防，不要并排输出信息。
```

### 3. 对话没感情

```text
这段对话有信息，没有感情。
请补上：
1. 角色说这句话时真正想藏住什么
2. 角色为什么不敢直接说
3. 哪一句话明明很轻，却最伤人
4. 哪个停顿比台词更有力
```

## 细纲加压模板

如果你怀疑不是正文平，是大纲就平了，直接用这个。

### 长篇

```text
story-long-write，重查这一卷的细纲。
不要先修文笔，先修情绪曲线。
逐章标出：
1. 目标情绪
2. 冲突级别
3. 代价
4. 人物欲望
5. 本章结尾余波
找出连续两章以上情绪持平的地方，直接重做。
```

### 短篇

```text
story-short-write，重查当前小节大纲。
找出只有信息没有情绪推进的节。
每节补上：欲望、阻碍、代价、误导、余波。
如果反转前铺垫太平，把前半段重新加压。
```
