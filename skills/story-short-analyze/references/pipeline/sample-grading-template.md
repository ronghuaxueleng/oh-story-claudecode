# 样本分级与可学层 模板

这份文件用于 `写作资产/样本分级与可学层.md`。

定位：

- 这是单书样本准入判断
- 不是情节总结
- 不是作者 DNA 本体
- 作用是先判断“这本书能不能提 DNA、能提哪一层”，避免把整本原文默认当正样本

使用要求：

- 必须先看原文，再结合完整拆书结果填写
- 必须按 `A类正样本 / B类骨架样本 / C类负样本` 三选一，不允许模糊表述
- 必须把 `可学层` 和 `禁学层` 分开写
- 必须给判定证据，不能只写“感觉不错”或“整体偏假”
- 必须写清后续调用方式，避免下游误用
