# 《扫黄扫到了我老公》正反例对照

用途：

- 给 `story-short-analyze` 第二种题材壳的 few-shot 参照
- 防止模型把“公权场撞见 + 婚姻清算”误拆成普通抓奸文
- 补充《幼薇》没有覆盖的“执法秩序 / 私域入侵 / 公开反打”类桥段

---

## 1. 故事核

### 正例

```md
### 故事核

**设定**：女警许初在扫黄行动中，亲手抓到隐婚丈夫蒋湛和女学生夏禾出现在暧昧场合。
**主题**：真正毁掉婚姻的不是一场撞见，而是一次次越界后还要求妻子体谅。
**核心行动**：女主从扫黄现场到医院、老房子、民政局、网络公开场，一路取证、离婚、清算、反击。
**一句话**：女警在执勤扫黄时撞见教授丈夫和女学生越界，最终靠证据和公开场彻底切断这段带着债与伪善的婚姻。
```

为什么这是正例：

- 它没有把故事核写成“妻子抓奸后离婚”这种旧壳。
- 它明确保住了 `公权场撞见 -> 私域越界 -> 证据链清算` 三层推进。

### 反例

```md
### 故事核

**设定**：女主发现老公出轨学生。
**主题**：渣男不值得原谅。
**核心行动**：女主和老公吵架，最后离婚。
**一句话**：一个女人发现丈夫出轨后果断离婚。
```

为什么这是反例：

- 这不是错，是严重失真。
- 它把“扫黄执法、公权场拉扯、医院掉位、老房子钥匙、网络公开反打”全部抹掉了。

---

## 2. 题面拆解

### 正例

```md
| 维度 | 分析 |
|------|------|
| 表面题材 | 现言追悔 / 婚姻清算 / 师生越界 |
| 主梗 | 女警在扫黄执法时抓到丈夫和女学生越界，旧婚姻从公权场开始失守 |
| 梗位分工 | 扫黄现场负责开头钩子；医院和老房子负责中段越界升级；网络公开场负责终局清算 |
| 读者最先以为这是什么文 | 以为是抓现行后立刻离婚的标准爽文 |
| 后面哪一步改写了这个预期 | 医院花束、老房子钥匙和朋友圈视频把故事改写成“长期越界 + 私域入侵 + 证据反打”链 |
| 真正特别的地方 | 不是撞见本身，而是丈夫每一次都在更正式、更不该越界的秩序里继续选错 |
```

为什么这是正例：

- 它把“为什么开头炸”和“为什么中段还越来越疼”讲清了。
- 尤其把医院和老房子提到题面层，说明中段不是填空，而是题面的第二抓手。

### 反例

```md
### 题面拆解
- 表面题材：婚姻出轨 / 抓奸 / 离婚
- 深层流派：追悔文
- 题面承诺：女主发现丈夫出轨后清算渣男。
```

为什么这是反例：

- 只有题材标签，没有预期翻转。
- 也没有说明“这本为什么不是普通抓奸文”。

---

## 3. 高敏桥段识别

### 正例

```md
- 桥段名：扫黄现场抓现行
- 原文位置：开头段
- 主要高敏层：公权场 + 场面层
- 原文为什么不像加工稿：不是私下抓奸，而是丈夫在执法现场直接抓她制服、替女学生求情
- 新稿最容易写假的点：只保留“撞见”本身，不保留执法秩序

- 桥段名：医院掉位
- 原文位置：中段
- 主要高敏层：私域越界 + 秩序层
- 原文为什么不像加工稿：花被扔进垃圾桶不是因为不爱花，而是因为女学生花粉过敏，主位是被现实偏向挤掉的
- 新稿最容易写假的点：把它写成单纯争风吃醋，不补“医疗照护优先级”这层

- 桥段名：老房子钥匙失效
- 原文位置：中后段
- 主要高敏层：私域入侵
- 原文为什么不像加工稿：钥匙插不进去、门里饭香、女学生探头叫师母，关系越界一下子实锤
- 新稿最容易写假的点：只写“他们同居了”，不写权限和空间证词
```

为什么这是正例：

- 它把最值钱的中段桥保住了，不是只盯开头抓现行和结尾离婚。
- 尤其保了“医院掉位”和“老房子钥匙失效”两种很典型的私域升级桥。

### 反例

```md
- 桥段名：扫黄抓到老公
- 桥段名：离婚
- 桥段名：公开清算
```

为什么这是反例：

- 只剩桥名，没有“为什么不像加工稿 / 最容易写假在哪里”。
- 中段桥完全消失，后续写作会自然塌成“开头炸一下，中间反复吵”。

---

## 4. 原文细节库：场景 / 物件 / 权限

### 正例

```md
## 医院掉位场
- 具体发生了什么：蒋湛给夏禾照顾、让许初的花被扔掉，只因为夏禾花粉过敏。
- 这个细节为什么有用：不是一句“我更心疼她”，而是照护优先级已经被现实动作改写。
- 它压的是谁、压在哪：压的是许初作为妻子的照护主位，位置在医院这种本该最讲秩序的场域。
- 后续能迁到什么新桥段：能迁去病房、手术单、陪护名单，前提是保留“优先级被正式挪给别人”。
- 它对应的角色 / 情绪 / 反转是什么：对应许初、蒋湛、夏禾；情绪是发冷；反转是婚姻主位不是靠吵掉的，是靠流程慢慢让掉的。

## 老房子钥匙失效
- 具体发生了什么：许初拿钥匙回老房子，门却再也打不开，门内已经是别人做饭、别人探头喊“师母”的场景。
- 这个细节为什么有用：一句“他们住一起了”是说明，钥匙失效才是证词。
- 它压的是谁、压在哪：压的是许初的进入权和旧关系残存位置，位置在门口第一步。
- 后续能迁到什么新桥段：能迁去门禁失效、指纹被删、床位被换，前提是保留“权限先失效，再看到人已经住进去”。
- 它对应的角色 / 情绪 / 反转是什么：对应许初、蒋湛、夏禾；情绪是恶寒；反转是关系崩坏从暧昧怀疑升级成私域实锤。
```

为什么这是正例：

- 细节不是抽象情绪，而是场景、物件、权限三件一起落。
- 它能直接迁到很多“私域越界”题材里。

### 反例

```md
## 医院争吵
- 具体发生了什么：女主在医院和丈夫吵架。
- 这个细节为什么有用：能体现丈夫偏心。
- 后续能迁到什么新桥段：能迁到同类婚姻冲突。
```

为什么这是反例：

- 这已经退回到概括层了。
- 看完以后，写手拿不到任何具体承重件。

---

## 5. 使用方式

正式跑《短篇拆文》时，这本样本主要用来校正 3 类问题：

1. 开头别只写“抓现行很炸”，要保住公权场秩序。
2. 中段别只写吵架，要保住医院、老房子、钥匙、花束这类私域越界证词。
3. 终局别只写离婚，要保住证据链和公开反打。

如果新结果更像这里的反例，而不像正例，默认说明拆文还停在旧桥摘要层。
