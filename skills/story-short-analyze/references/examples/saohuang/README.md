# 《扫黄扫到了我老公》内置样本

这个目录把《扫黄扫到了我老公》作为 `story-short-analyze` 的第二本内置 few-shot 样本。

它和《幼薇》的分工不同：

- 《幼薇》重点约束：中段承重桥别被压掉、题面预期翻转别写薄、人物不同脸别抹平
- 《扫黄扫到了我老公》重点约束：公权场撞见别写成普通抓奸、私域越界别只剩嘴吵、证据链清算别只保终局狠话

包含：

- `扫黄扫到了我老公原文.txt`
  - 原文全文样本，用于回查公权场、医院、老房子、网络公开场这些桥段是不是都真的来自原文
- `正反例对照.md`
  - 展示这本书在 `故事核 / 题面拆解 / 高敏桥段识别 / 细节库` 上的厚拆正例与常见反例

使用口径：

- 不要把警察、教授、女学生这些题材壳当通用模板。
- 重点学习“公权场先撞见，私域再越界，最后证据链公开清算”的推进方式。
- 如果新拆文把这类故事写成“开头撞见很炸，后面就一直吵”，默认说明细节库和桥卡没拆透。
