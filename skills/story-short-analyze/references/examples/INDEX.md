# 内置 Few-Shot 样本索引

这个目录收纳 `story-short-analyze` 的内置样本。

目的不是给模型抄题材壳，而是给模型一个更稳定的“厚拆参照系”：

- 什么叫中段桥不能丢
- 什么叫题面拆解不能只剩摘要
- 什么叫细节库必须保事件并集
- 什么叫高敏桥识别不能只摘头尾亮桥
- 什么叫交流不是补眼神，而是现场压力交换
- 什么叫冲突不是吵架，而是现实权力被争夺

---

## 选用原则

不要把所有样本全文一起塞进当前思路里。

先判断当前原文最像哪一类，再优先取 1-2 本最接近的样本对照：

1. 如果当前失败更像“过检但变薄”
   - 先看《幼薇》
2. 如果当前失败更像“开头撞见很炸，但中段只剩吵架”
   - 先看《扫黄扫到了我老公》
3. 如果当前失败更像“背景说偏心很多，但具体剥夺链不实”
   - 先看《归月学生》

原则：

- 只学拆法，不学壳
- 只学承重顺序，不学现成狠句
- 只学细节并集覆盖，不学角色名和平台感
- 每本都要实际读 `README.md + 相关原文段 + 正反例对照.md`
- 选样和实际对照必须记录到 `_sample_comparison.md`
- 禁止用 bak、上一本文档、旧 profile 或其他拆书目录替代这里的内置样本

## 样本矩阵

| 样本 | 题材壳 | 最适合约束什么问题 | 优先看哪些文件 |
|------|--------|--------------------|----------------|
| 《幼薇》 | 替身婚姻 / 旧伤物件 / 婚礼清算 | 中段承重桥被压掉；题面预期翻转缺失；细节库缩水 | `正反例对照.md`、原文、`动作/场面/对白` 对照 |

新增对照职责：

- 《幼薇》：重点看 `身份物 / 旧物 / 身体后果` 如何让交流和冲突成立，尤其适合校正“只剩对白金句”的问题。
- 《扫黄扫到了我老公》：重点看 `公权秩序 / 医院优先级 / 老房子进入权` 如何让冲突从抓奸升级为权限失效。
- 《归月学生》：重点看 `长期剥夺 / 家族秩序 / 礼制位置` 如何让冲突不是一次吵架，而是连续现实让位。

## 失败类型到样本映射

### 1. 题面拆解太薄

优先看：

- 《幼薇》：预期翻转怎么写
- 《归月学生》：长期剥夺链怎么写进题面

### 2. 高敏桥只剩头尾

优先看：

- 《幼薇》：中段承重桥为什么不能漏
- 《扫黄扫到了我老公》：医院 / 老房子这种中段桥如何保住

### 2.5. 交流只剩眼神和对白

优先看：

- 《幼薇》：抢戒、抡包、拽领带、翻垃圾这类动作如何改变关系位置
- 《扫黄扫到了我老公》：执法现场和医院场如何让外部秩序参与接招

### 2.6. 冲突只剩“他们吵了”

优先看：

- 《扫黄扫到了我老公》：公权场、医院、老房子的争夺权分别是什么
- 《归月学生》：小马驹、婚书、邀帖、花轿如何把偏心变成现实剥夺

### 3. 原文细节库像概括摘要

优先看：

- 《幼薇》：动作 / 对白 / 场面卡怎么写成可迁移证据卡
- 《归月学生》：长期剥夺链如何拆成动作与场面卡

### 4. 终局炸场有了，但中段现实层不够

优先看：

- 《幼薇》：旧伤物件和手术现实伤
- 《扫黄扫到了我老公》：医院优先级和老房子权限失效

### 5. 掉马/身份翻盘像外挂

优先看：

- 《归月学生》：先压低位误判，再翻身份

## 使用建议

正式执行时，推荐这样用：

1. 先读当前原文，不要先入样本。
2. 判断当前书最容易被拆坏的地方。
3. 只挑 1-2 本样本对照对应章节。
4. 先落 `_sample_comparison.md`，写清正例锚点、反例锚点和将影响的正式文件。
5. 写完主报告后，再回看样本反例区，把裁决和实际回写文件写回 `_sample_comparison.md`。

如果结果同时踩中多个反例，默认不要 finalize，先回到：

- `原文资产候选池.md`
- `原文细节库/`
- `高敏桥段识别.md`
- `拆文报告.md` 的 `题面拆解`

做一次定向回补。
