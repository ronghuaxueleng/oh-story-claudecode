# 《归月学生》正反例对照

用途：

- 给 `story-short-analyze` 一个古言 / 换亲 / 身份翻盘向的 few-shot 参照
- 防止模型把“长期剥夺链”误拆成一段背景说明
- 防止模型把“掉马反杀”误拆成纯爽点按钮

---

## 1. 故事核

### 正例

```md
### 故事核

**设定**：从小被家人挤到边角的沈宝珠，在婚书被换、婚嫁被调、身份被低估的连环剥夺里，被宋霁接住并一步步翻回主位。
**主题**：真正伤人的不是一次换亲，而是你从小到大都被训练成“可以让”的那个人。
**核心行动**：女主从被换婚书、被抢邀帖、被调花轿，到借宋霁的身份硬牌和回门清算，把原本属于别人的秩序反压回去。
**一句话**：一个从小连小马驹都分不到的女儿，在婚嫁与身份双重剥夺里，终于用花轿、聘礼和回门局把自己从“嫁谁都一样”翻成唯一不能被替换的人。
```

为什么这是正例：

- 它没有把故事核写成“古言换亲反杀”这种套壳摘要。
- 它明确保住了“长期小剥夺”这条母线，不让换亲变成孤立狗血桥。

### 反例

```md
### 故事核

**设定**：女主被换亲，后来男主掉马救她。
**主题**：坏人会有报应。
**核心行动**：女主经历委屈，最后翻盘。
**一句话**：一个古代女子被欺负后最终逆袭。
```

为什么这是反例：

- 它不算错，但完全失真。
- “小马驹、婚书、邀帖、花轿、回门”这条具体剥夺链全没了。

---

## 2. 题面拆解

### 正例

```md
| 维度 | 分析 |
|------|------|
| 表面题材 | 古言 / 偏心家庭清算 / 换亲反转 / 身份掉马 |
| 主梗 | 从小被排除在家族宠爱之外的女主，在婚书被换后第一次被问“你愿不愿意” |
| 梗位分工 | 小马驹和婚书负责开头剥夺链；百万聘与身份掉马负责中段翻盘；花轿与回门负责终局清算 |
| 读者最先以为这是什么文 | 以为是标准偏心家庭 + 男主救场的古言爽文 |
| 后面哪一步改写了这个预期 | 不是男主一来就翻盘，而是先让小剥夺持续累积，再让掉马和花轿调包把旧秩序整条翻过去 |
| 真正特别的地方 | 女主最痛的不是换亲本身，而是她从小就被训练成“没主张、可以让”的那个孩子 |
```

为什么这是正例：

- 它讲清了这本最值钱的不是“王府掉马”，而是“剥夺链先站稳”。
- 这样后续写作才知道什么该保，什么只是表皮。

### 反例

```md
### 题面拆解
- 表面题材：古言换亲 / 掉马 / 反杀
- 深层流派：爽文
- 题面承诺：女主被换亲后翻盘打脸。
```

为什么这是反例：

- 它只给结果，不给预期翻转。
- 也没写“为什么这本不是普通换亲文”。

## 3. 高敏桥段识别

### 正例

```md
- 桥段名：祠堂改嫁定心
- 主要高敏层：桥段顺序层
- 原文为什么不像加工稿：先多年剥夺，再婚书将换未换，最后才让宋霁在祠堂接住女主
- 新稿最容易写假的点：一上来就让男主英雄救美，不先写剥夺链

- 桥段名：掉马百万聘
- 主要高敏层：场面层
- 原文为什么不像加工稿：先让所有人默认宋霁失势，再用聘礼和身份一层层压回去
- 新稿最容易写假的点：不铺落榜低位，直接宣告贵公子身份

- 桥段名：邀帖与花轿双换
- 主要高敏层：桥段顺序层
- 原文为什么不像加工稿：先邀帖试探，再衣裙卡位，最后才到红盖头和花轿调包
- 新稿最容易写假的点：直接婚礼换人，不做前面试探与升级
```

为什么这是正例：

- 它保住了“前面慢剥夺，中段再翻，后面再清算”的秩序。
- 不是只摘最亮的掉马桥。

### 反例

```md
- 桥段名：换亲
- 桥段名：掉马
- 桥段名：回门打脸
```

为什么这是反例：

- 只有桥名，没有为什么能过、为什么不像加工稿、易假点。
- 写手后续只能学到标签，学不到桥段顺序。

## 4. 原文细节库：动作 / 场面

### 正例

```md
## 眼刀逼退
- 具体发生了什么：定亲抽签时，沈宝珠手指刚碰到写着宋霁的字条，就被阿兄一个眼神逼得缩回手。
- 这个细节为什么有用：比直接拉开更狠，因为没人碰她，却所有人都知道她不能选。
- 它压的是谁、压在哪：压的是沈宝珠的自主动作，位置在当众抽签的瞬间。
- 后续能迁到什么新桥段：能迁到抢话筒、报志愿、签合同；承重件是“一个眼神就足以让主角退让”。
- 它对应的角色 / 情绪 / 反转是什么：对应沈宝珠、阿兄；情绪是胆怯；反转是她表面在选，其实根本没资格选。

## 堂上换亲场
- 具体发生了什么：放榜后前厅里程鸣跪地求娶阿姐，阿姐装憔悴，阿兄与沈父顺水推舟，整个场子都在替沈宝珠决定命运。
- 这个细节为什么有用：不是两个人私下换心，而是一屋子人一起默认主角可被调换，压力更大。
- 它压的是谁、压在哪：压的是沈宝珠的婚嫁主位，位置在家中正式议亲场。
- 后续能迁到什么新桥段：能迁到股东会、家长会、相亲局；承重件是“多人在场时集体替主角做决定”。
- 它对应的角色 / 情绪 / 反转是什么：对应沈家众人、程鸣、沈宝珠；情绪是窒息；反转是她被迫在全员同意的场合独自吞下决定。
```

为什么这是正例：

- 一条写动作权限被压，一条写正式场面里的主位被集体剥夺。
- 这能帮助模型理解“长期剥夺链”不是一句背景，而是一次次具体让位。

### 反例

```md
## 被偏心
- 具体发生了什么：女主一直被家里偏心对待。
- 这个细节为什么有用：能体现她很委屈。
- 后续能迁到什么新桥段：能迁到家庭文。
```

为什么这是反例：

- 这是概括，不是细节卡。
- 没有场面、没有物件、没有动作、没有秩序位置。

## 5. 交流承压拆解 / 冲突载体清单

### 正例

```md
## 交流承压拆解｜眼刀逼退

- 原文位置：定亲抽签场
- 场面锚点：`手指刚碰到写着宋霁的字条`
- 谁先施压：阿兄
- 压力载体：身份 + 空间 + 节奏
- 具体施压动作：阿兄不用开口，只用当众眼神压住她伸出去的手。
- 对方被迫改了什么：沈宝珠把手缩回去，表面还在抽签，实际已经失去选择动作。
- 删掉台词是否仍成立：是。因为压迫来自“伸手 -> 被看住 -> 缩回”的动作链。
- 为什么不是作者替人物接招：女主的退手就是现场反应，不需要旁白说“她知道自己不能选”。

## 冲突载体清单｜邀帖与花轿双换

- 表层冲突：婚嫁安排被人调换。
- 第二层争夺权：婚嫁主位 + 去留决定权 + 礼制公开头位。
- 主载体：object / identity / rhythm
- 谁当场失位：沈宝珠，婚书、邀帖、衣裙、花轿逐步把她从主位挤出去。
- 越界后果：不是一次换亲，而是每个礼制节点都有人替她决定。
- 如果只保留对白会丢掉什么：会丢掉“长期被训练成可以让”的现实链，只剩家人偏心控诉。
```

为什么这是正例：

- 它把长期剥夺拆成连续动作资格变化，不是抽象说“她很委屈”。
- 它特别适合校正古言向“冲突只剩控诉、掉马只剩爽点”的问题。

### 反例

```md
## 家人偏心

- 谁先施压：家人
- 压力载体：态度
- 对方反应：女主很委屈。
- 冲突类型：家庭偏心。
```

为什么这是反例：

- 没有小马驹、婚书、邀帖、花轿，就没有长期剥夺链。
- 写作侧拿到这种材料，只会写控诉句，不会写动作资格如何一点点被剥掉。

## 6. 情绪同构仿写资产

### 正例

```md
## BID｜定亲抽签到堂上换亲

| 拍位 | 原文触发 | 关系位置变化 | 读者体感 | 烈度 | 迁移要求 |
|---|---|---|---|---:|---|
| 情绪进入点 | 女主的手碰到自己想要的字条 | 她表面拥有一次选择机会 | 小心期待 | 5 | 新稿必须先给一个看似属于主角的选择 |
| 受辱/刺痛 | 阿兄用眼神逼她缩手 | 她的自主动作被家族身份压回去 | 憋闷、替她委屈 | 7 | 压迫要落在动作资格，不只写家人偏心 |
| 短暂希望或反抗 | 她仍留在议亲场，等待有人问她 | 她尚未彻底失去被尊重的可能 | 读者等一个人替她开口 | 6 | 希望必须具体，不能空写“她还抱有期待” |
| 反刀 | 满堂人顺水推舟替她决定婚嫁 | 私人退手升级为全员公开剥夺 | 窒息、愤怒 | 9 | 反刀从单人压迫升级为群体秩序，位置必须同位 |
| 场末余痛 | 她被默认成“嫁谁都一样”的人 | 长期被训练成可以让的人再次被确认 | 心酸、等她第一次不让 | 8 | 余痛要连接长期剥夺母线，不能立刻掉马打脸 |
```

### 反例

```md
- 功能：家人偏心并换亲。
- 新稿：股东会把女主项目换给姐姐。
- 烈度：9。
```

为什么这是反例：

- 把“伸手、被眼神压退、等待、全员替她决定”的情绪过程压成了一个结果。
- 总烈度写成 9 也没有意义，因为读者没有经历期待被一点点夺走的过程。

---

## 6. 使用方式

正式跑《短篇拆文》时，这本样本主要用来校正 3 类问题：

1. 别把偏心家庭只写成情绪背景，要写成连续剥夺链。
2. 别把掉马当一锤定音外挂，要保住掉马前的低位误判。
3. 别把回门清算写成长篇控诉，要保住花轿、请帖、回门礼制这些承重件。
4. 写 `交流承压拆解 / 冲突载体清单` 时，必须拆出“谁替女主决定、女主哪一步退手或失位”，不能只写家庭态度。

如果新结果更像这里的反例，而不像正例，默认说明古言向拆法还停在标签层。
