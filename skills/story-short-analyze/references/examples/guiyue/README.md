# 《归月学生》内置样本

这个目录把《归月学生》作为 `story-short-analyze` 的第三本内置 few-shot 样本。

它和前两本的分工不同：

- 《幼薇》重点约束：中段承重桥、题面预期翻转、细节库别缩水
- 《扫黄扫到了我老公》重点约束：公权场撞见、私域越界、证据链清算
- 《归月学生》重点约束：长期小剥夺别被写成空委屈、婚嫁秩序替换别被写成单次狗血、身份翻盘别写成空掉马

包含：

- `归月学生原文.txt`
  - 原文全文样本，用于回查小马驹、婚书、邀帖、花轿、回门这些承重件
- `正反例对照.md`
  - 展示这本书在 `故事核 / 题面拆解 / 高敏桥段识别 / 原文细节库` 上的厚拆正例与常见反例

使用口径：

- 不要把古言、换亲、掉马、王府这些题材壳当通用模板。
- 重点学习“长期小剥夺 -> 婚嫁大剥夺 -> 身份翻盘 -> 回门清算”的顺序。
- 如果新拆文只剩“家里偏心 + 男主救场 + 最后打脸”，默认说明剥夺链和秩序件没拆透。
