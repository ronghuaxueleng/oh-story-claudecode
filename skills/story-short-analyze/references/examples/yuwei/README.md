# 《幼薇》内置样本

这个目录把《幼薇》作为 `story-short-analyze` 的内置 few-shot 样本，专门用于约束“别写成过检但变薄”。

包含：

- `幼薇原文.txt`
  - 原文全文样本。用于重放阅读、校验细节卡是否真来自原文。
- `正反例对照.md`
  - 同一本书的正例 / 反例对照。正例优先展示厚拆该保住的层，反例展示“格式合规但资产变薄”的常见失败。
  - 当前已覆盖：`动作细节库 / 场面细节库 / 对白细节库 / 题面拆解 / 高敏桥段识别 / 桥段施工卡`

使用口径：

- 不要把《幼薇》的题材壳、角色名和现成狠句当通用模板。
- 重点学习它在“中段承重桥、题面预期翻转、原文细节库卡数、人物不同脸”上的拆法。
- 如果新拆文结果更像 `正反例对照.md` 里的反例，而不像正例，默认先回炉，再谈是否进入 finalize。
