### 甜梗 / 虐梗 / 爽梗补充例库

- 甜梗：雨夜送药、嘴硬递伞、替她挡酒、借披风、顺手记住忌口、装不在意却一路跟着
- 甜梗：先护短后训人、表面嫌弃实则收尾、众人面前不偏心私下全偏她
- 虐梗：她最需要他的时候，他去护了别人；信物被退回；同桌却像外人；他说出口的话句句扎在旧情上
- 虐梗：她病着、伤着、怀着委屈，却还得体面退场；所有人都知道，就她最后知道
- 爽梗：公开摆证据、当众切关系、反派互咬、迟到后悔、旧人再也够不着、全场改口、身份翻牌

## 不要乱叠

### 容易打架的叠法

- 追妻题里又硬塞过多家产争夺、身份翻牌、死人文学，容易主线散
- 家庭清算里再硬叠病娇 / 囚禁，容易风格断裂
- 情绪流题里塞太多证据牌公开处刑，容易把细腻情绪打碎
- 病娇救赎题里再强上群体围剿公开打脸，容易失去关系专注度

### 控制口径

- 一篇文只保一个主壳
- 副升级器最多一到两条
- 高潮只保一个最大公开场
- 主情绪母线只保一条

## 组合验收口径

- 主冲突是不是一句话能说清
- 副升级器是不是在抬层，不是在重复
- 关系场是不是天然让读者站队
- 开头钩子是不是服务主卖点
- 高潮公开场是不是打到对方最在意的东西
- 结尾是不是落到了位置、自由、归属或价值感

## 起盘卡口径

### 起盘卡必填

- 平台
- 母结构
- 主冲突
- 副升级器
- 关系场
- 开头钩子
- 高潮公开场
- 硬牌
- 主情绪母线
- 结尾落点
- 题面
- 导语

### 起盘卡判定标准

- 任何一项写不清，说明骨架还没定稳
- 母结构不清，不开写
- 主冲突和副升级器撞车，不开写
- 结尾落点只是“打脸了”，不够

## 回炉顺序条

### 固定顺序

- 先题面
- 再骨架
- 再开头
- 再中段
- 再高潮
- 再情绪
- 最后才修句子

### 为什么这样回

- 题面不对，后面全白干
- 骨架有问题，修句子没意义
- 中段塌了，高潮再好也托不住
- 情绪不顺，句子越修越假

## 常见误配纠偏

### 主冲突误配

- 追妻题却把真正篇幅都花在家产争夺上 -> 主冲突跑偏
- 家庭题却只剩男女拉扯 -> 题核不稳
- 身份翻牌题没有误判过程 -> 反差立不住

### 结尾误配

- 追妻题最后写复合甜甜甜 -> 前面不回头全废
- 情绪流题结尾突然公开打脸 -> 情绪收束断裂
- 病娇题结尾直接普通男二接盘 -> 原关系重心塌掉

### 情绪误配

- 开头就太爽 -> 后面没空间
- 一直委屈不升级 -> 中段发闷
- 高潮讲道理，不炸场 -> 情绪兑现失败

## 各题型失手点

### 追妻题常见失手

- 男主伤害不够实
- 女主撤离不够坚决
- 不回头机制没立住
- 新生活只是口头说说

### 家庭题常见失手

- 只有吵，没有账
- 配角都像背景板
- 群体围剿不够压
- 结尾只和解，不清算

### 身份 / 职场题常见失手

- 误判不够久
- 主角能力线没提前埋
- 高潮翻牌像天降证据
- 位置变化不够明显

### 病娇 / 救赎题常见失手

- 只写疯，不写黏性
- 只写控制，不写依赖
- 失忆只是花活，不改关系逻辑
- 救赎太快，像工具人接盘

## 写前硬闸

- 有没有一个清楚母结构
- 有没有一个清楚第二层冲突
- 有没有一个能公开翻的场
- 有没有主角持续动作线
- 有没有结尾价值落点
- 有没有想清楚最值钱的那一刀是什么

## 写后验收口径

- 开头前三句有没有完成至少两项硬功能
- 前八百字有没有刺激事件
- 每一节有没有新推进
- 中段有没有只解释不推事
- 高潮有没有公开场 / 回收件 / 二次翻刀里的至少两项
- 结尾有没有后效应
- 结尾有没有价值感，不只是处刑

## 平台组合偏好快抽

### 盐选偏好

- 题面要先看懂关系和越界，不要先卖设定
- 开头最好直接进失控现场，不要慢热铺垫
- 女主动作要快，哪怕前期在忍，也得让读者知道她在记账
- 高潮更适合公开翻脸、公开亮牌、公开改口
- 结尾不能只停在处刑，最好再给一格新生活或位置变化

### 小程序偏好

- 开头前三屏要密，几乎每屏都得有新刺激
- 副升级器可以更硬一点，房产、怀孕、旧案、签字局都好用
- 中段不要只压情绪，要不断抬后果
- 高潮适合连翻两次，不然容易显得不够炸
- 结尾可以更直给，重点是“她赢得很实”

### 通用稳妥口径

- 平台不同，主壳也别乱换，还是一篇文只保一个主壳
- 盐选更怕套路感太重，小程序更怕不够猛
- 不确定平台时，先按“题面清楚 + 开头起事 + 中段不断事 + 结尾有后效应”的基本盘来

## 起盘组合补充口径

### 先定结尾再反推前文

- 结尾想落“不回头”，前文就得把伤害写实，把迟到写透
- 结尾想落“新生活落地”，前文就得提早埋新秩序，不要最后凭空出现
- 结尾想落“被坚定选择”，前文就得先把旧关系写到站不住
- 结尾想落“公开翻盘”，前文就得埋硬牌、埋围观者、埋对方最怕丢的脸
- 结尾想落“如释重负”，前文就别堆太多吵架，要多写长期压抑和突然松开的反差

### 常用起盘组合

- 追妻壳 + 财产线 + 家宴公开场 + 不回头收尾
- 家庭清算壳 + 房产线 + 年夜饭公开场 + 自我重建收尾
- 身份误判壳 + 能力线 + 公司大会公开场 + 位置变化收尾
- 病娇束缚壳 + 依赖看清 + 半公开失控场 + 新生收尾
- 情绪流壳 + 旧伤翻开 + 被坚定选择 + 温暖落地收尾

### 小节级事件密度提醒

- 第一小节至少要有一个明确坏事落地
- 第二小节不能只是回忆，最好补一格更具体的后果
- 中段每小节最好承担一种功能：压人、亮相、抬后果、翻旧账、埋硬牌、转立场
- 连续两小节都只在说感受，基本就会发闷
- 高潮前一小节别把话全说完，要留一个“她手里还有东西”的口子

### 结果导向验收

- 这段写完后，关系有没有更坏
- 这段写完后，主角有没有更接近翻盘
- 这段写完后，围观者有没有更明确站队
- 这段写完后，后果有没有比前一段更重
- 如果都没有，这段大概率只是占字数

## 融合写法调用口径

### 融合不是乱拼

- 先定一个母结构，再去借别的升级器和口气
- 主壳只留一个，别一会儿像追妻，一会儿像真假千金
- 能借的是导火索、第二层冲突、公开场、硬牌、结尾落点
- 不能借到把主角的核心处境借没了

### 最稳的融合顺序

- 先抽母结构
- 再抽关系场
- 再抽一个副升级器
- 再抽一个公开场
- 再抽一种结尾落点
- 最后才补导语口气和台词味道

### 常见融合公式

- 追不回母结构 + 公开对账高潮
- 公开对账母结构 + 旧账迟到翻刀
- 身份误判母结构 + 做局反咬中段
- 让位反杀母结构 + 关系错位结尾
- 情绪流母线 + 被坚定选择收束

### 融合时最容易出的问题

- 主冲突和副升级器都想当主线，最后谁都不稳
- 前半段写追妻，后半段突然变职场文
- 中段新加的梗只是热闹，没抬高原冲突
- 结尾换了价值落点，前文却没提前铺

### 融合验收

- 删掉副升级器后，主故事还能不能成立
- 换掉公开场后，这篇是不是还在讲同一个痛点
- 结尾想表达的东西，前面有没有至少两处铺垫
- 新加那条线，是不是把后果抬高了，而不是把题目带偏了

## 起盘底层补充口径

### 短篇默认像最后一章切入

- 不从相识相爱慢慢写起
- 默认关系已经形成，旧账已经攒够，眼下这一刀马上要落
- 开头不是介绍人生，是把人推进最该爆的那个节点
- 如果前八百字还在铺认识过程，基本就已经慢了

### 起盘先写付费期待

- 她怎么翻
- 他怎么后悔
- 她会不会真的走
- 这场公开翻脸要怎么收
- 她怎么把原来的低位换掉

### 使用口径

- 起盘卡里除了题面和导语，还要单独写一句“读者接下来最想看什么”
- 这一句写不出来，说明题还没立住
- 期待最好是结果型，不要写成“看后续发展”

## 中段供氧补充口径

### 中段不是重复受压

- 不能前半段被压，中段还在被压，只是换个人压
- 中段至少要让主角进入一个新状态
- 这个新状态可以是新住处、新同盟、新工作、新计划、新证据线
- 读者要看到她不只是熬，而是开始换活法

### 常见新状态

- 她开始撤离旧生活
- 她开始盘账、留证、做局
- 她有了新的落脚点
- 她和某个人开始建立互信
- 她开始让旧关系失去对她的控制

### 中段验收

- 这一段有没有出现前面没有的新秩序
- 主角是不是比上一段更有主动权
- 旧关系是不是已经开始松动
- 如果只是把前面的委屈再说一遍，这段就不够

## 高潮后效应补充口径

### 高潮不是结束

- 当场翻脸只是大局变化的起点
- 后面还得补谁慌了、谁跑了、谁失位了、谁彻底没资格了
- 女主这边也得补她的新秩序怎么落地

### 后效应至少补一类

- 旧关系崩塌
- 围观者改口
- 反派付现实代价
- 男主进入迟到追悔
- 女主真正开始新生活

### 写后检查

- 高潮后有没有至少一小节不是在重复高潮，而是在写后果
- 旧秩序有没有真的塌
- 新秩序有没有真的立

## 成长线落地补充口径

### 成长不是喊觉醒

- 不要直接写“她变强了”“她觉醒了”
- 要写她怎么偷学、盘资源、攒能力、换位置
- 成长线最值钱的不是情绪变狠，是她有了第二人生出口
- 读者要看见她从旧关系里一点点长出独立活法

### 成长线常见落点

- 她不再靠旧关系吃饭
- 她有了自己的住处、工作、项目、铺子、人脉
- 她从被安排的人，变成能安排自己的人
- 她最后站的位置，不再由旧关系决定

### 成长线验收

- 资源有没有变化
- 位置有没有变化
- 能力有没有被看见
- 如果只有心态变化，没有现实变化，就还不够值钱

## 导火索调用补充口径

### 好导火索的标准

- 一出来就能看懂谁越界了
- 既伤情感，也碰现实利益
- 不是“关系一直不好”，而是“今天坏事真的落地了”
- 最好能顺手带出后面的第二层冲突

### 导火索优先级

- 先选会逼出主角动作的
- 再选能挂住公开场的
- 最后才选设定感强但不一定能推事的

### 常用导火索组合

- 带人上门 + 房本 / 孕检单
- 年夜饭逼让 + 学区房 / 赡养钱
- 领奖换人 + 成果归属 / 审批记录
- 婚约让渡 + 家族偏心 / 账册

### 导火索误区

- 只够让人生气，不够逼出决定
- 只像背景说明，不像当前事件
- 太小，撑不起题面
- 太杂，一开头就堆三四件事抢主线

## 切割动作使用补充口径

### 切割动作要落现实

- 留钥匙
- 断供
- 停共同账户
- 交权限
- 退群
- 搬家
- 当众划线

### 切割动作怎么用

- 最好前中段就先来一次小切割，别都攒到结尾
- 切割动作要和主情绪同步，心寒时切，决绝时断
- 每做一次切割，旧关系就该失去一点控制权

### 切割动作验收

- 这个动作做完，日子有没有真的不一样
- 对方有没有因此慌一下
- 后文有没有顺着这个动作继续出后果

## 公开场选择误区

### 公开场不是越大越好

- 关键不是人多，是对方最怕在哪儿丢脸
- 家庭题优先家宴、病房、祖屋分配，不一定非要婚礼
- 职场题优先复盘会、签约会、发布会，不一定非要全公司大会
- 情绪流细题别硬塞过大的公开场，容易砸碎前面积累

### 公开场前要先埋三样

- 围观者
- 对方最在意的体面
- 主角手里能改写场面的硬牌

### 公开场后要补三样

- 谁先失声
- 谁先改口
- 谁开始承担现实后果

## 硬牌组合快抽补充口径

### 最稳的双硬牌

- 流水 + 录音
- 病历 + 监控
- 合同 + 权限记录
- 身份牌 + 旧物回钩
- 聊天记录 + 第三方公告

### 硬牌怎么亮更值钱

- 第一张先证明确有其事
- 第二张再改写前文理解
- 最好不是一下全倒出来，而是先露边，再当场坐实

### 硬牌误区

- 只证明对方坏，没改写局势
- 全靠天降证据，前文没存在感
- 证据太专业，读者一眼看不懂
- 亮牌以后没人掉位，等于白亮

## 反套路硬闸补充口径

### 开写前必须写清三项

- 题材壳
- 异物
- 第二层冲突

### 这三项怎么理解

- 题材壳是读者第一眼会认出来的那层熟壳
- 异物是这篇最不普通、能把熟壳掰歪的那一下
- 第二层冲突是第一层已经成立以后，再把局势往上抬的一刀

### 过闸标准

- 缺异物，容易写成熟题换皮
- 缺第二层冲突，中段容易只剩重复受气
- 只有异物没有主壳，题面容易不稳

### 常见异物快抽

- 死对头结盟
- 假复合真抽身
- 感情壳底下其实是财产 / 股权 / 旧案
- 公开喜事底下埋着旧伤证据
- 旧关系已经烂透，真正重点是新秩序建立

## 高潮卡点误区补充口径

### 常见失手

- 主角太早崩，高潮提前泄掉
- 证据一股脑全交代完，后面没卡点
- 反派不敢做大局，高潮缺公开疼点
- 高潮只是把真相说出来，没改写前文理解

### 修正口径

- 高潮前让主角更平，不是更吵
- 证据先露边，不先给全貌
- 反派越爱公开，后面翻车越值钱
- 高潮炸完以后，最好还能回译前文一处旧认知

## 一句话梗概发动机补充口径

### 起题先追问这四句

- 主角被谁压着
- 对方在哪个场合越界
- 这一下具体抢走了她什么
- 她后面最值钱的反应会是什么

### 常见发动机

- 被放弃
- 被让位
- 被误判
- 被公开轻贱
- 迟到后悔
- 低位翻牌
- 反派互咬

### 梗概验收

- 一句话里能不能看出极端处境
- 能不能直接看见不公平
- 能不能预感后面会有大后果
- 如果只剩题材名，没有发动机，这题就还没立起来

## 关系重组补充口径

### 不要默认写回抢男人

- 能联手就别强写雌竞
- 能反选旧关系就别只求旧人回头
- 能把重点从爱情换到位置和生活，就更容易不俗

### 常见关系重组方式

- 正宫和白月光发现真正的加害者是同一个人
- 女主不再争旧人，转头去拿更值钱的位置
- 死对头从互咬变结盟，一起拆局
- 男二不是来抢人，是来托住女主的新秩序

### 什么时候适合用

- 熟壳太熟，想加新鲜感时
- 中段只剩男女拉扯，想抬价值层时
- 追妻题不想写成普通后悔求复合时
- 家庭题不想只剩吵架时
