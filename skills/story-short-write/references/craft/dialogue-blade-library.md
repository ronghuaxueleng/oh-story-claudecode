# 对话刀法库

适用时机：

- 写公开场对账
- 写追妻拉扯
- 写心死切割
- 写多人场站队
- 已知这场要“更刺一点”，但还没想清该怎么接

原则：

- 刀法是“怎么接”，不是“堆狠话”
- 先判上句功能，再判下句接法
- 对话的杀伤力常常来自停顿、转称呼、动作垫句，不只来自词更狠

---

## 一、先判上句功能

常见上句功能：

- `逼解释`
- `抢定义`
- `装无辜`
- `压位置`
- `找补`
- `甩锅`
- `装体面`
- `求和`
- `试探`
- `亮底牌`

先判清上句在干什么，再选下句刀法。

---

## 二、常用接法

### 1. 反问接

适合：

- 对方在装无辜
- 对方在逼你讲道理
- 对方想把你拉进解释位

效果：

- 不顺着解释
- 直接把压力退回去

模板口径：

- `我还得……？`
- `你也配问这个？`
- `你是真不知道，还是装不知道？`

### 2. 冷接

适合：

- 主角已经心冷
- 不想再跟对方拉长战线

效果：

- 掉温度
- 掉关系
- 比吵更伤

模板口径：

- `嗯。`
- `随你。`
- `你说完了？`
- `行，记住了。`

### 3. 压回接

适合：

- 对方抢定义
- 对方想把场面带偏

效果：

- 把议程拉回自己这边

模板口径：

- `别扯别的。`
- `我问的不是这个。`
- `你先把刚才那句说完。`
- `先说重点。`

### 4. 装傻接

适合：

- 压迫者、白月光、体面反派

效果：

- 明知故问
- 看似无辜，实则继续压人

模板口径：

- `我有说错吗？`
- `我还以为你不介意。`
- `这也能怪到我头上？`

### 5. 转移接

适合：

- 对方被打到痛处
- 想把焦点挪开

效果：

- 不是赢，是拖

识别口径：

- 突然讲别的旧账
- 突然扯第三人
- 突然讲“大家都难看”

### 6. 停顿后接

适合：

- 亮牌前
- 心死切割前
- 需要让一句话更沉

效果：

- 让下句更重
- 让读者先等半拍

动作垫句例：

- `我看了他一眼。`
- `我把杯子放下。`
- `她嘴唇动了动，最后还是笑了。`

### 7. 改称呼接

适合：

- 关系降级
- 公开场切割
- 从亲密位退到冷位

效果：

- 不用解释，光称呼就能让关系掉位

常见用法：

- 从 `你` 改到 `周总`
- 从 `妈` 改到 `阿姨`
- 从 `既白` 改到 `沈先生`

### 8. 不接话，用动作顶

适合：

- 这时候再说就轻了
- 主角的心已经冷过头

效果：

- 用动作做回答
- 比解释更有杀伤

常见动作：

- 把文件推过去
- 当场离席
- 把门打开
- 把戒指放下

## 三、场景刀法

### 公开场对账

顺序常用：

1. 对方先装体面
2. 主角不顺着解释
3. 主角压回议程
4. 亮硬牌
5. 群体沉默或站队

### 追妻拉扯

顺序常用：

1. 男主先找补
2. 女主冷接
3. 男主再解释
4. 女主改称呼或动作切割
5. 关系掉温

### 心死切割

顺序常用：

1. 对方还想讲道理
2. 主角回得越来越短
3. 关键句前先停一拍
4. 用一句普通人话彻底切断

### 多人场站队

顺序常用：

1. 一个人装体面
2. 一个人放大伤害
3. 一个人装和事佬
4. 主角最后一句改位置

---

## 四、写法硬规则

- 一场对白至少要有 1 次“接法变化”，不要从头到尾全是平推问答
- 一场对白至少要有 1 处动作垫句，防止只剩嘴对嘴
- 一场公开场对白，优先安排 1 次称呼变化或 1 次不接话的动作回答
- 想写得更狠，先改“谁在定义场面”，别先改脏词浓度
- 只会堆狠话，不会接刀，最后就会像互喷，不像值钱对白
