## AI写作指纹（必须避免）

### 高频AI用词

> 完整禁用词表见 [banned-words.md](banned-words.md)

**补充类目**（`banned-words.md` 未覆盖的高阶替换）：

| 类别 | 替代原则 |
|------|---------|
| 抽象升华词（命运、宿命、注定） | 用具体事件代替抽象概念 |
| 万能比喻（像潮水般、如闪电般、仿佛春风） | 要么不用比喻，要么用生活化的比喻 |

### 章末总结体

**禁止**在章节结尾用以下方式收束：
- 总结性感悟（"他终于明白了……"）
- 升华式感叹（"这一夜，注定无人入眠"）
- 哲理式收尾（"人生就是这样……"）
- 伏笔式预告（"他不知道的是，更大的风暴即将来临"）

**正确做法**：章尾用动作、对话或悬念收束，让情节本身制造余韵。

### 非人话表达 / 生造口气句

这类句子不是“太文”，而是“不像人会这么说”。

**高危特征**：
- 动词搭配像临时拼出来的：`往里垫`、`把话塞进去`、`回头摸墙`
- 看着有劲，实际只是陌生说法，不是人物说法
- 换成常用口语后，人物功能完全不变

**处理原则**：
- 先判句子功能：陈述 / 反问 / 嘲讽 / 顶嘴 / 求情 / 敷衍
- 先保住功能，再把怪词退回常用口语层
- 别为了“有味道”硬拧句子

### 代判句 / 作者替人物说透

这类句子往往意思没错，但作者站出来替读者总结了。

这类问题主要依赖上下文语义，关键词脚本只能预警，不能证明“没有代判句”。自动报告零命中后仍必须人工复扫全文；局部回炉还要同时检查新增句和母稿旧句。

**常见表现**：
- `她大概没想到我会这么平`
- `他显然没料到我会这么说`
- `我知道，我这句话进去了`
- `他这才意识到自己输了`

**处理原则**：
- 优先改成现场反应：沉默、停顿、脸色、站位、接不上话
- 能让读者自己看出来的，不要替读者说出来

### 抽象词悬空 / 对象省略过度

这类句子看着短狠，实际要读者自己补关键对象。

**高危例子**：
- `那时候我脑子里全是喜欢`
- `她心里只剩委屈`
- `他最在意的就是体面`

**处理原则**：
- 补对象：喜欢谁、委屈什么、体面给谁看
- 补来源：这种情绪是因为什么起来的
- 不要为了短，把真正承重的信息省掉

### 风险等级落轻 / 结果写虚

这类句子的问题不是不通，而是场上的危险更重，句子却落轻了。

**高危例子**：
- `别让他碰人`
- `别让他乱来`

**处理原则**：
- 先判真风险到底是什么：伤人、砸人、动手、掀桌
- 结果要写实，别写虚

### 叠加式描写（同一动作掰开写三遍）

**检测模式**：一个动作/情绪先写发生，再补感知细节，再补身体反应，分三段依次写完。读者看到的是同一个动作被掰开写了三遍。

**典型特征**：
- 先写一个概括性动作，再展开写同一动作的细节，再写身体反应——三段说的是同一件事
- "发生层→感知层→反应层"按顺序分段出现
- 每个维度独立成段，而不是织入同一段连续正文

**错误示例**：
> 林父低着头，左手把文书压住，右手拿笔，往纸上落——
>
> 手在抖。不是小抖，是从肘到腕的那种。
>
> 笔尖在纸上停了停，写了一横，又停。那个"林"字的撇写歪了。

→ 同一个动作（手抖/写字）分三段写，每段是同一瞬间的不同维度

**正确做法**：发生、感知、反应三个维度织入同一段连续正文，读者读到一个完整瞬间：

> 林父左手压着文书，右手拿笔往纸上落——笔尖一触纸面就偏了，从肘到腕止不住地抖，那一横斜着拖出去。

→ 发生、感知、反应在一段里同时呈现

**处理原则**：不是删除情绪细节，而是合并同一瞬间的重复描写。若合并后明显变薄，优先恢复原文中有功能的信息，或把既有信息改成更自然的动作/对话表达；不要新增原文没有的情节、设定、关系或时间线。

## 核心规则

### 规则 1：段落密度诊断

段落不是越碎越好，也不是越长越有质感。检查重点是朗读和手机阅读是不是卡。

- 一段通常只承载一个动作、一个信息变化或一组紧密相关的反应。
- 逗号串太长、多个完整动作挤在一段里，读起来需要换气时，按动作或信息变化拆开。
- 连续短段碎成提纲时，合并同一镜头内的相邻句，让画面保持连续。
- 连续 3 段以上都是同一动作链里的极短句，默认按 `断段过散` 处理

```
过密：他看着窗外的雨，心中涌起一股说不清的感觉，这些年走过的路和很多已经忘记的事都在这一刻涌上心头。

更自然：他盯着窗外。雨下了很久。
"你还在想她？"老刘问。
他没说话。
```

### 规则 2：动作 + 对话 + 情绪反应

三要素循环推进，不要单写心理活动超过 2 段：

```
动作 -> 对话 -> 情绪反应 -> 动作 -> 对话 -> ……
```

情绪尽量别用“他感到 / 他觉得”，优先用身体反应和行为表现：
- 不写"他很紧张" -> 写"手心全是汗，筷子差点掉了"
- 不写"她很愤怒" -> 写"她把杯子摔在地上，碎片弹到脚背上也没弯腰捡"
- 不写"他很伤心" -> 写"他在车里坐了二十分钟才发动引擎"

### 规则 3：句长交错，不要整页等长短句

短句不是越多越好。**全篇都用极短句，会把正文写成检测器最喜欢抓的“规整短句密排”。**

| 场景 | 建议 | 示例 |
|------|------|------|
| 打斗/紧张 | 可以短句连击，但 3 句左右要换气 | 他侧身。刀光一闪。血溅到墙上。后背也跟着一凉。 |
| 对话 | 口语化，按人物分句法 | "你疯了？" "可能吧。" |
| 日常 | 短句里夹一条正常长度叙述 | 厨房里有炒菜声，闻着像辣椒炒肉。她站门口没进去。 |
| 描写 | 以不卡顿为准，不追求统一长度 | 天快黑了，巷子里没什么人，路口那家店倒是还亮着。 |

**检查重点**：
- 连续 5 段都是差不多长度的短句 = 机械
- 连续多段都是“一句动作一句心理一句台词” = 模板
- 全文几乎没有稍长一点的补充句、偏见句、吐槽句 = 人工感不足

**更自然的做法**：短句负责打点，长句负责挂人物主观、补临场信息、做轻微拐弯。

### 规则 4：口语化表达

- 允许用俚语、粗话（符合角色身份）
- 对话不要书面语（"我认为此事不妥" -> "我觉得不靠谱"）
- 叙述也不要端着（"他目光如炬" -> "他眼珠子一动不动盯着"）
- 短语优先于成语（"无可奈何" -> "没办法"）
- 但口语化不等于乱造句。第一反应不像人话的句子，先回退

## Show Don't Tell 原则

| Tell（告诉） | Show（展示） |
|-------------|-------------|
| 他是个胆小的人 | 他把检查报告在手里翻来覆去看了三遍，还是不敢打开 |
| 这间酒吧很吵 | 酒保凑到他耳边喊了两次他才听见 |
| 她很富有 | 她随手把一张信用卡丢在桌上，卡面上的数字比这顿饭贵十倍 |
| 两人关系很差 | 他把烟掐灭在她刚泡的茶杯里，她面无表情地把杯子推到一边 |
| 他很聪明 | 三秒钟。他看了三秒钟就把文件合上了。"第三页，第二行。" |

**核心方法**：
1. 用行为代替形容词
2. 用细节代替总结
3. 用对话代替旁白说明
4. 用反应代替情绪词

## 质量维度检查

### 1. 核心一致性（权重最高）
- 剧情是否与大纲/前文一致
- 人物行为是否符合人设
- 设定是否有前后矛盾

### 2. 表面改写（防AI指纹）
- 是否包含AI高频用词（见上表）
- 章尾是否有总结/升华
- 是否有大段纯心理描写
- 段落是否超过 3 句
- 是否有生造口气句、代判句、抽象词悬空句
- 是否把脚本零命中误当成人工语义通过

### 3. 格式一致性
- 对话格式统一（按当前目标平台统一；工作稿默认 `""`，知乎投稿版默认 `「」`，如用户或平台后台明确要求 `【】` 则按要求覆盖）
- 场景切换有明显标记
- 时间线清晰可追踪

### 4. 可读性
- 是否有连续 3 句以上的长句导致卡顿
- 是否有连续大量等长短句导致机械
- 对话是否口语化
- 是否有未解释的生僻词/设定术语
- 节奏是否有快有慢（不能全是一种节奏）

### 5. 逻辑连贯性
- 角色动机是否合理
- 事件因果链是否清晰
- 时间线是否对得上
- 角色的知识范围是否合理（不能"开上帝视角"）

## 7 种 AI 写作模式检测

### 模式 1：AI 高频词

| 禁用 | 替换为 |
|------|--------|
| 不禁 | 删掉 |
| 仿佛/宛如 | 删掉或用具体描写 |
| 映入眼帘 | 删掉 |
| 心中暗道 | 用动作展示思考 |
| 沉声道/淡淡地说 | 换成动作标签 |
| 脸色一变 | 用具体表情/动作 |
| 嘴角微扬 | 他笑了/他翘了下嘴 |
| 不由自主 | 删掉 |
| 只见/此时此刻 | 删掉 |
| 目光如炬 | 删掉或具体化 |

### 模式 2：弱化副词泛滥
阈值：每 1000 字超过 3 个 = AI 签名。重点监控：微微、淡淡、缓缓、轻轻。

### 模式 3：意义膨胀
- "意义深远" -> 写具体后果
- "前所未有" -> 给出对比参照
- "可谓" -> 删掉

### 模式 4：万能结论
- "未来可期" -> 用未解决的紧张感结尾
- "前途无量" -> 删
- "充满希望" -> 写具体的下一步动作

### 模式 5：论文体段落结构
小说中出现以下开头句 = AI 入侵：
- "不难看出""由此可见""事实上""综上所述"

### 模式 6：书面语连词泛滥
叙事散文中频繁出现："于是乎""与此同时""从而""因而""诚然" -> 口语化替代或直接删除。

### 模式 7：三连排比癖
AI 喜欢把事情凑成三个以显"完整"。-> 砍到只剩最有力的一条。

### 模式 8：作者代判

`她大概没想到`、`他显然愣住了`、`我知道这话扎进去了` 这类句子，一律优先改成现场反应。

### 模式 9：对象悬空

情绪词、关系词、判断词不带对象，看着狠，实际阅读会打顿。

### 模式 10：非人话口气

读者第一反应是“这句在找说法”，不是“这人真会这么说”，就按高危处理。

## 篇章级 AI 指纹（骨架味，最难查也最致命）

> **前提：检测分数 ≠ 真实 AI 味，是两个问题。** 现有检测器（GPTZero、朱雀等）主要靠"用词可预测性"和"句长波动"做**句/段级**判断，短文本不稳、对打磨过的文风易误报。实测规避实验：只换同义词、改情绪词 → 仍 100% 判 AI；而改句式节奏、改结构 → 可降到 0%。这说明**真正的味在篇章/结构层**。有研究（StoryScope）证明：完全不看用词、句式、标点，仅凭叙事结构特征，就能以 93.2% 准确率区分人类与 AI。
>
> 上面模式 1–10 全是句/段级，能过检测器却过不了读者。下面这几条才是"一页之内被闻出来"的骨架味。**回炉时先查这一节，再回到句词级。** 校准题材：情感 / 都市现实短篇。

### 篇1. 主题过度解释 / 叙述者替读者顿悟

作者或主角把"学到的道理"讲出来，替读者完成顿悟。数据参照：AI 叙述者显式点破主题约 77%，真人约 52%；AI 的说教/道德化整体高约 20%。

**高危特征**：
- 结尾（或每节尾）出现"她终于明白……""原来……"式的意义总结
- 一件事发生后，紧跟一句把它的含义说透的旁白

**反例**：
> 她终于明白，有些告别，是为了更好地重逢。

**改法**：删掉点破句，停在动作上，把意义交给读者——
> 她把他留下的那副碗筷收进柜子最里层，关上柜门。

### 篇2. 冷硬点题短句的高密度堆叠

注意：**这不是滥情升华，而是每节都精准落一个克制的、象征性的短句收尾**（"有些门，关上就不会再开。""右手没有抖。"）。单看每句都高级、克制，但**节节都来一记、密度过高、位置过整齐**，本身就是模板味。真人写作里这种"金句落点"是稀疏、偶发、甚至会错过的。

**高危特征**：
- 连续 3 节以上，每节都收在一句冷冽的象征/点题短句上
- 这些尾句工整得像盖章，可预测下一节还会来一记

**反例**：五个小节，节节收在一句克制象征句，对称得像模板。

**改法**：几节里只留一处点到；其余节收在半截对话、一个无意义动作，或干脆停在半空不收——用节奏破掉"每节精准落点"的对称。

### 篇3. 篇章级结构对称 / 像在填对照表

每节都埋"以前 vs 现在""他说的 vs 他做的""城市 vs 老家"这类对照，段落长度也均匀，每段都绕回主题，因果链过紧、没有闲笔。数据参照：AI 故事约 79% 无支线（真人约 57%）；主角驱动型结局约 69%（真人约 46%）。

**高危特征**：
- 第一节写"如今独居的空房子"，第二节必对照"当年热闹的婚房"；对仗到可预测
- 全篇没有一笔和主线无关的闲事；每个细节都在服务主题
- 段落长度、每节结构高度一致，像同一个模子

**反例**：现在冷掉的饭 ↔ 从前她做的热汤；现在的沉默 ↔ 当年的争吵。节节对照。

**改法**：
- 故意放进**不服务主题的闲笔**（一段跑题的邻居八卦、一个和主线无关的旧物）
- 允许段落长短失衡、某节明显短或长
- 砍掉一半"对照回闪"，让"以前"只在某一处不经意地漏一次，不要节节对齐

### 篇4. 情绪零度过稳 / 过度受控

**过度克制本身也是 AI 味，不只是滥情才露馅。** 主角全程无破绽、体面、滴水不漏；情绪曲线太线性（要么一路稳，要么一路悲），缺少真人的忽然失态、反复、莫名其妙的情绪跳变。研究证实 LLM 故事情绪唤起曲线更平、系统性回避张力、偏向和解与怀旧。

**高危特征**：
- 从头到尾主角没有一次失控、没有一个破绽
- 情绪单调递进或恒定，没有毛刺、没有回跳
- 每句话都恰到好处、成熟得体，像被均匀控制过

**反例**：分手全程，女主始终冷静得体，每句话恰到好处，最后平静祝福对方。

**改法**：给一个**不体面、不受控**的破绽，让曲线长出毛刺——
> 她在便利店对店员莫名发了火，回到家却为一个塑料袋破了哭很久，哭完擦干脸，若无其事回了条工作消息。

（先假装没事、中途突然崩、然后又抽离——比全程克制真实得多。）

### 篇5. 配角功能化 / 对话没有跑题枝节

配角只为推进主线或印证主题而存在，一开口就服务剧情，像人物档案卡；对话过分高效、句句在点上、人物"太善于表达自己的感受"，把潜台词全说破。真人对话有插科打诨、跑题、绕圈、答非所问、欲言又止。

**高危特征**：
- 配角没有一句和情节无关的话，纯功能件
- 对话太 articulate，把该藏的潜台词直接说出来
- 问一句答一句，逻辑严丝合缝，没有留白和错位

**反例**：
> "你还好吗？"
> "不好。我一直没办法接受我们结束了这个事实。"

（太会表达，潜台词全说破。）

**改法**：让对话**跑题、留白、错位**，再给配角一句**只暴露性格、不推进情节**的废话——
> "你还好吗？"
> "……你伞带了没，说下午有雨。"
> "带了。"
> "哦。"

（同事随口抱怨一句食堂涨价那种无用枝节，反而让人物活起来。）

### 篇6. 跨段"同一个人同一种写法"

即使每段单独看都像人写的，只要**段与段之间用力程度、节奏、句法高度一致**，仍会露馅——这是检测器抓不到、读者却能感到"匀速"的味。

**处理原则**：主动制造落差——让某几段明显更碎/更长、更冷/更放，用力不均才像真人在不同情绪下写出来的。

## 系统性去AI三遍法

### Pass 1：去泛化（Strip Generic）
- 抽象情绪总结句 -> 删或替换为具体动作
- 假深度句 -> 删
- 意义膨胀 -> 缩小到具体影响
- 空洞结论 -> 删
- 工整对比句式 -> 打散重写
- 装饰性形容词堆砌 -> 白描
- 过度使用"于是""然而""此刻" -> 删掉一半
- 所有角色说话一样"高级" -> 区分语气
- 生造口气句 -> 回退常用口语层
- 代判句 -> 改成现场反应
- 抽象词悬空 -> 补对象和落点

**原则**：能删就删，不能删就用具体细节替换。这一遍去掉80%的AI味。

### Pass 2：去书面化（Cut Professional Diction）
- 分析性用词（"机制""结构""逻辑""体系"出现在小说中）-> 换成日常表达
- 抽象名词滥用 -> 直接说事
- 体制内用语（"进一步""深入""推进""落实"）-> 删
- 专业术语堆砌 -> 只保留必要的，用白话解释

**例外**：保留专业感的场景（历史题材正式用语、文学向刻意密度、喜剧夸张修辞）。

### Pass 3：回自然感（Restore Natural Presence）
- 具体的感官细节（气味、温度、触感）
- 角色说话方式的区分（不同人不同语气）
- 节奏变化（长短句交错）
- 社会位置感的对话（上级和下属说话方式不同）
- 场景特有的记忆点
- 项目特有的语言习惯（角色的口头禅）
- 允许一点真人会说的偏手、偏见、拐弯，但不能越线成非人话

**原则**：少即是多。每段加 1-2 个具体细节就够了。

### 升级策略

| AI味程度 | 策略 |
|----------|------|
| 轻度 | 只做 Pass 1 |
| 中度 | Pass 1 + Pass 2 |
| 重度 | 完整三遍 + 重点段落重写 |

### 自检清单
- 对话自然度检查：对话是否使用口语化表达，是否避免了书面语/正式腔调
- 删掉任何一句，会影响理解吗？不会 = 可能多余
- 不同角色能通过对话区分吗？
- 有没有一个细节是这个场景特有的？
- 有没有第一眼就不像人会说的话？
- 有没有作者替人物说透的句子？

## 去AI味补充技法

### Show vs Tell

| 告知类型 | AI写法 | 自然写法 |
|----------|--------|----------|
| 告诉期待感 | "他很期待" | 展示期待->情绪->满足的链条 |
| 告诉角色目的 | "她想离婚" | 用行动展示目的 |
| 告诉角色态度 | "她很冷静" | 用对话和反应体现 |
| 告诉剧情走向 | "接下来会发生大事" | 用铺垫->反转->延续展示 |

### 心理描写润物细无声

- 加括号标注内心活动 = 破坏代入感
- 大段内心独白解释动机 = AI签名
- 直接写"她感到""她意识到" = 告知情绪

**自然写法**：心理活动自然融入叙事，用行为暗示心理，用沉默/动作/反常行为表达内心。

### 代入感检查
- 主角行为读者能理解、共鸣、接受吗？
- 反派够强吗？（弱反派 = 读者觉得主角赢了没意义）
- 是否围绕人设写行为？（行为/语言/思维围绕人格展开）
- 读者已知信息是否被有效操控？（信息差制造情绪波动）

## 改写范例库

### 情绪外化范例

**紧张**
- '他感到一阵紧张，心跳不由自主地加快了'
- 他攥紧了手里的纸杯，水洒出来一些

**愤怒**
- '愤怒在他心中燃烧，他不由得握紧了拳头'
- 他把筷子往桌上一拍，碗里的汤溅了出来

**悲伤**
- '一丝悲伤涌上心头，她的眼中闪过泪光'
- 她低头搅着咖啡，搅了很久

**害怕**
- '恐惧瞬间笼罩了他，他感到一阵战栗'
- 他的背贴在墙上，不敢动

**失望**
- '她感到一丝失落，心仿佛被什么东西揪住了'
- "哦。"她把手机锁了屏

**惊讶**
- '他的瞳孔微微收缩，显然没有想到会听到这样的话'
- 他张了张嘴，什么都没说出来

### 场景描写范例

**AI风场景**
- '阳光透过窗帘的缝隙洒进来，在地板上投下斑驳的光影。空气中弥漫着淡淡的花香，仿佛整个世界都沉浸在一片宁静祥和的氛围中。'
- 下午三点，客厅里只有钟在走。

**AI风天气**
- '天空阴沉沉的，乌云密布，仿佛随时都会下起倾盆大雨。凛冽的寒风呼啸而过，带着一丝刺骨的寒意。'
- 要下雨了。风把晾在外面的衣服吹得乱晃。

**AI风打斗**
- '他的拳头犹如疾风骤雨般猛烈，每一击都蕴含着不容置疑的力量。对手的瞳孔微微收缩，显然没有预料到如此凌厉的攻势。'
- 他一拳怼过去，对方没躲开，嘴角破了。

### 结尾改写范例

**升华式结尾** -> '他站在窗前，望着远方的天际线，终于明白了生活的真谛——有时候，放手才是最好的选择。' -> 他把烟掐了，回屋睡觉。

**总结式结尾** -> '这一刻，一切都变了。她知道，从今以后，她的人生将翻开崭新的一页。' -> 她关上了那扇门。没回头。

**感慨式结尾** -> '岁月如流水般悄然流逝……' -> 直接删掉这种段落。

### 节奏调整范例

**排比句**
- '他看着她的眼睛，看着她的嘴唇，看着她微微颤动的睫毛，心中涌起一股难以名状的情感。'
- 他看着她。她没说话。

**长句拆短**
- '当他终于推开那扇沉重的木门时，映入眼帘的是一间昏暗的房间，空气中弥漫着陈旧的气息，墙角堆满了落满灰尘的箱子。'
- 门开了。屋里很暗，角落堆着几个箱子，上头全是灰。

**工整段落打碎**
- '她喜欢春天的花朵，喜欢夏天的阳光，喜欢秋天的落叶，喜欢冬天的白雪。每一个季节都有它独特的美。'
- 她喜欢春天。别的季节也还行。

## 冲突对话改写范例

### AI式温和对话
- '我觉得你这样做不太合适，能不能考虑一下我的感受？' -> "你眼里还有我吗？"

### AI式完美解释
- '其实我这样做是有原因的，因为当时的情况非常复杂……' -> "你能怎么着？"她把茶杯重重放下。

### 对话情绪五级递进范例

同一冲突场景，从弱到强：

1. **客观陈述**："你把我的东西扔了。"
2. **陈述+建议**："你把我的东西扔了，以后能不能先跟我说一声。"
3. **主观指责**："你凭什么动我的东西。"
4. **指责+命令**："你算什么东西，也配碰我的东西？滚出去。"
5. **指责+PUA**："我伺候你吃伺候你穿，你连个东西都放不好。你这辈子也就是这样了，离了我你什么都不是。"

### 强冲突不能只升级台词

强情绪场不能永远是“你问一句、我答一句、最后女主说出正确结论”。逐场人工判断冲突落在哪些载体上：

- `dialogue`：抢话、回避、压程序、说错重点
- `body`：拦路、抓腕、推撞、夺物时碰伤
- `object`：抢走、摔裂、撕坏、扣押珍爱物
- `space`：堵门、逼出原位置、把人隔在安全线或门外
- `identity`：撤权限、换座位、收钥匙、改称呼
- `rhythm`：话头被截断、旁人插入、动作迫使对方改口

不要求每场六项齐全，只要求承重冲突不长期退化成克制答题。身体和物件越界必须留下后果，不能用一句道歉复位。

直接扇打、掐脖、踢踹不是普通“更强冲突”，而是角色伦理变化。追妻文保留此类行为时，必须同步确认男主已不可洗白、女主不复合并承担现实后果；不得把殴打写成深情。

**弱例（只有观点冲突，没有现实争夺）**

- 我说方案有问题，他说先演完再说。
- 我说这很危险，他说你别情绪化。

这两句只说明双方立场不同，不能说明场内谁失位、谁夺权、谁承担后果。

**强例（冲突落到现实权力）**

- 我说停演四十分钟，他当着全组恢复旧版本。
- 我去拿签字本，他先把本子压回报警键下。
- 她刚说“只是借用”，下一秒就把我的旧本扣在怀里不还。

这类句子不是单纯更凶，而是把冲突落到了 `版本控制权 / 记录权 / 物件处置权`。

### 交流感不能用动作词数量代判

“看、盯、停顿、抓腕、递物”都只能帮助模型定位候选场景，不能证明人物已经发生交流。当前模型必须逐场确认：

- 谁先施压，压力具体落在哪里
- 对方是否因此改变动作、站位、物件控制权、回答范围、身份或后果
- 这次反应是否由现场人物完成，而不是作者替人物解释
- 台词删掉后，场内是否仍能看见双方互相改变

只补“他盯着我”“她顿了一下”不算修复。孤立台词、连续答题和作者判词仍然没有真实交流。

**弱例（像交流，其实是修辞装饰）**

- 我要求停演四十分钟，周既明却当着全组的面恢复了旧版本。
- 他盯着我的眼睛，缓缓说道：“程雾等了六年。”
- 他说到这里就停了。

这里只补了视线和语气，人物关系并没有因为这句话立刻发生可见变化。

**强例（有压力交换，现场秩序被改掉）**

- 我要求停演四十分钟，周既明却当着全组的面恢复了旧版本。
- 他没先回我，先把值守员压着的调度本抽走，递回控制台。
- “程雾等了六年。”他说。
- 值守员不再看我，转头等他下令。

这里的交流不靠“盯着”成立，而靠 `先动谁的本子 / 谁不再等女主 / 现场命令链转向谁` 成立。

### 震惊分层改写范例

**AI式一步到位**：所有人都震惊了，不敢相信自己的耳朵。

**自然分层震惊**：
1. 对面的男人手抖了一下，茶杯里的水洒出来。
2. 旁边的人互相看了一眼。有人往后退了一步。角落里有人开始掏手机。
3. 刚才还趾高气扬的女人，脸上的笑僵住了。她张了张嘴，一个字没说出来。

### 代入感修复范例

**被动主角**：她很害怕，不知道该怎么办，只能等着事情过去。

**主动主角**：她锁了门，把手机调成静音，打开了录音。

## 质量检查清单

写完每章后，按此清单逐项扫描。**先扫下面这组骨架级（改词改不掉、最致命），再扫句词级。**

- [ ] **无替读者点题**：结尾/每节尾没有"她终于明白……"式的意义总结（篇1）
- [ ] **点题句不成串**：没有节节都收在冷冽象征短句上的堆叠，金句落点稀疏偶发（篇2）
- [ ] **不填对照表**：没有节节"以前vs现在"式对称；有至少一处不服务主题的闲笔；段落长短不匀（篇3）
- [ ] **情绪曲线有毛刺**：主角至少有一处失控/破绽，情绪不是单调线性，有回跳（篇4）
- [ ] **配角非纯功能件**：有跑题/留白/错位的对话，至少一句只暴露性格不推进情节的废话（篇5）
- [ ] **跨段有落差**：段与段用力程度、节奏不匀速，不像同一模子（篇6）
- [ ] **长窗气口不塌陷**：叙述者气口不只集中在少数短段；连续长窗没有整片匀速铺陈
- [ ] **长窗对白不全勤**：按长窗检查，对话中有打断、错答、回避和生活闲枝，不是句句推进
- [ ] **冲突载体不单一**：当前模型已逐场复核承重冲突，不是全篇只靠克制对话；肢体、物件、空间或身份越界产生了实际后果
- [ ] **每场能答出在抢什么权**：承重冲突至少能落成制止权、解释权、入场权、物件处置权、花钱决定权等现实控制权，不能只写“他们吵了”
- [ ] **直接暴力已裁决**：若存在扇打、掐脖、踢踹，已同步处理角色可追性、女主安全边界与不复合结局；固定词预扫不代替人工判断
- [ ] **交流逐场人工验收**：当前模型已覆盖所有承重对话场，确认压力真正改变了另一个人物；固定词候选没有直接代判
- [ ] **交流有可见变化**：每个承重交流场至少有一项可见变化，落到动作、站位、物件控制权、回答范围、身份、后果或现场秩序，不是只多了眼神和停顿
- [ ] **作者没有替人物接招**：关键交流依靠现场动作和回应完成，不靠作者解释“这句话意味着什么”
- [ ] **灵动感人工验收**：当前模型已全文检查 `rule_evidence_stiffness_and_liveliness`，确认物件、空间和权限不是被写成规则证据清单，而是经由人物失手、错答、回避、生活毛边和不完全顺滑的动作自然发生
- [ ] **规则感过重已回修**：若段落读起来像“门槛证明边界、黄线证明退位、钥匙证明失权”的施工说明，已改为现场人物反应；不能只把固定物件换成另一组固定物件
- [ ] **全文无分镜/施工单写法**：当前模型已全文检查 `full_text_storyboard_construction_list_review`，确认叙述正文、关系场、冲突场、追妻低位、揭示和结尾没有“一句一个动作 / 一句一个证据 / 一句一个反应”的镜头清单，也没有“规则 A 执行、证据 B 展示、边界 C 落地”的验收稿；只有情节内真实清单、报告、日志、合同等文本可作为例外，并已逐条说明情节功能

- [ ] **段落控制**：段落按动作/信息变化断开，读起来不卡
- [ ] **AI高频词扫描**：无不禁/仿佛/映入眼帘/心中暗道/沉声道/嘴角微扬/不由自主/只见
- [ ] **弱化副词计数**：每1000字"微微/淡淡/缓缓/轻轻"不超过3个
- [ ] **无三连排比**：没有AI式的"三个一组"修辞
