# 可直接仿写层资产合同

适用场景：

- `仿写`
- `融合`
- `去原作化后再写`
- `先拆后写`

原则：

- 这些表是接口，不是让 skill 主文件塞进大词库
- 表里记录的是功能、顺序、口气差异、表演层资产，不是堆漂亮话
- 缺关键表时，优先回上游补拆，不要拿概括结论硬写

## 最小字段

### 1. 导语拆解表

| 字段 | 说明 |
|------|------|
| 前20字 | 是否已经起事 |
| 前60字 | 是否完成关系错位 / 事故信息 / 后果预期中的至少 2 项 |
| 前80字 | 是否出现第二推进点 |
| 第一句功能 | 越界现场 / 结果先行 / 关系错位 / 反常反应 |
| 第二句加码 | 抬后果 / 抬危险 / 抬掉位 |
| 继续滑动力 | 读者为什么会继续往下滑 |

### 2. 顺序事件表

| 字段 | 说明 |
|------|------|
| 编号 | `01 / 02 / 03 ...` |
| 谁做了什么 | 必须具体到人 |
| 具体动作 | 不接受空泛概括 |
| 这一拍功能 | 起事 / 压位 / 留钩 / 翻刀 / 抬后果 |
| 可迁移写法 | 换壳后还能怎么用 |

### 3. 物件表

| 字段 | 说明 |
|------|------|
| 物件 | 关键承重件 |
| 首次出现 | 节位 / 场面 |
| 后续回收 | 回收点 |
| 伤害层 | 体面 / 旧伤 / 身份 / 关系位置 / 亲情 |
| 可替换功能件 | 换题材后的同功能件 |

### 4. 动作表

| 字段 | 说明 |
|------|------|
| 动作 | 抢 / 扔 / 拦 / 护 / 留证 / 装无辜等 |
| 谁做 | 动作发起人 |
| 对谁做 | 动作承受人 |
| 当下效果 | 当场怎么压位 |
| 后续效果 | 为后文埋了什么 |

### 5. 对白功能表

| 字段 | 说明 |
|------|------|
| 角色 | 主角 / 压迫者 / 接住者 / 旁观者等 |
| 典型说法类型 | 挑衅 / 找补 / 压人 / 装无辜 / 切割关系 |
| 这类话负责什么 | 拉仇恨 / 洗白自己 / 逼位 / 翻盘 |
| 口吻特征 | 称呼、长短句、先忍还是先刺 |
| 明显不像他说的话 | 用来拦错配口气 |

### 6. 对话衔接表

| 字段 | 说明 |
|------|------|
| 上句功能 | 上一句在逼什么 |
| 下句接法 | 反问 / 冷接 / 装傻 / 转移 / 压回 / 停顿后接 |
| 动作垫句 | 有没有用动作垫一下再说 |
| 称呼变化 | 是否改称呼制造关系变化 |
| 衔接效果 | 为什么这一接更刺人 / 更压人 / 更显心冷 |

### 7. 误判表

| 字段 | 说明 |
|------|------|
| 误判方 | 读者 / 角色 / 群体 |
| 先误判了什么 | 早期认知 |
| 从哪开始翻 | 触发点 |
| 翻完后前文怎么变 | 重读价值 |

### 8. 钩子表

| 字段 | 说明 |
|------|------|
| 位置 | 段末 / 节末 / 场末 |
| 钩子内容 | 留了什么问号或后果 |
| 钩子类型 | 越界现场 / 结果先行 / 信息差 / 掉位后果等 |
| 读者下一步在等什么 | 明确期待 |
| 回收位置 | 哪一拍兑现 |

### 9. 微动作表

| 字段 | 说明 |
|------|------|
| 动作本体 | 捏杯口 / 把话咽回去 / 手指停住等 |
| 对应情绪 | 心冷 / 发紧 / 憋屈 / 防御 |
| 角色状态 | 忍 / 试探 / 硬撑 / 装平静 |
| 替代的解释句 | 替掉了哪类“她很难过”式解释 |
| 可迁移题材 | 适合迁去什么题材 |

### 10. 安静压迫场表

| 字段 | 说明 |
|------|------|
| 场面压力来源 | 谁压谁，靠什么压 |
| 谁没说话 | 沉默者是谁 |
| 环境音 / 物件 | 关门声、筷子声、手机震动等 |
| 未说破结果 | 大家都知道但没说破的后果 |
| 适用公开场 | 饭桌 / 医院 / 婚礼 / 家族群 / 签字场等 |

### 11. 作者DNA指纹

| 字段 | 说明 |
|------|------|
| 切句习惯 | 长短句比例、是否常断句、是否爱留半句 |
| 口气毛边 | 哪种不规整、不抛光的口气最像真人 |
| 情绪落点 | 爱把情绪落在动作、物件、停顿、称呼还是答非所问 |
| 高频小动作 | 哪些动作反复承担压力 |
| 旧伤触发器 | 哪类词、物件、身份关系最容易牵出旧伤 |
| 明显别学错的点 | 最容易被仿成模板腔的位置 |

### 12. 仿写约束_禁写清单

| 字段 | 说明 |
|------|------|
| 禁写类型 | 总结句 / 解释句 / 过渡句 / 假口语 / 补字句 |
| 典型假句 | 这一类最像 AI 的现成句 |
| 为什么假 | 假在太整齐、太会总结、太会替读者判断，还是太会抒情 |
| 替代方式 | 应该改成动作 / 对白 / 物件 / 后果 / 沉默里的哪一种 |
| 使用场景提醒 | 这类错句常在哪类桥段冒出来 |

### 13. 同桥段过检规则

| 字段 | 说明 |
|------|------|
| 桥段名 | 如婚礼让位 / 医院签字 / 饭桌羞辱 / 家长栏掉位 |
| 原文怎么起手 | 第一刀先扎哪，不先解释什么 |
| 原文为什么能过 | 脏感、毛边、动作先后、物件承压、口气不整齐来自哪 |
| 仿稿最易假点 | 最常写成总结句、流程句、漂亮句、标准公开场的地方 |
| 承重件不可丢 | 物件 / 动作 / 掉位 / 迟到惩罚 / 旁观秩序中的哪些不能丢 |
| 禁写句式 | 这个桥段里最该禁的 2-5 类句子 |

## 推荐读取顺序

1. `作者DNA指纹`
2. `仿写约束_禁写清单`
3. `同桥段过检规则`
4. `导语拆解表`
5. `顺序事件表`
6. `物件表`
7. `动作表`
8. `对白功能表`
9. `对话衔接表`
10. `误判表`
11. `钩子表`
12. `微动作表`
13. `安静压迫场表`

规则：

- 先立“学什么 / 禁什么 / 同桥怎么过”，再读导语和事件
- 先读功能，再换壳
- 找到这些表时，先吃表，再读 `拆文报告.md`

---

## 缺库处理

- 缺 `导语拆解表`：不要直接写开头
- 缺 `顺序事件表`：不要直接重组骨架
- 缺 `对白功能表` 或 `对话衔接表`：不要硬写对白
- 缺 `微动作表`：不要只靠情绪词和解释句顶场
- 缺 `安静压迫场表`：高潮和公开场容易只会写吵
- 缺任意 2 张以上：默认还停在概括层，先回拆文阶段补
- 缺 `作者DNA指纹 / 仿写约束_禁写清单 / 同桥段过检规则` 任意 1 份：不要开写，直接退回上游补拆
- 缺 `同桥段过检规则`：不要做同桥段融合、换皮或去AI味回修

## 主文件中该怎么用

- `story-short-analyze`：把这 10 张基础表 + 3 份写作资产当作固定产出合同
- `story-short-write`：把这 10 张基础表 + 3 份写作资产当作固定读取合同
- 需要扩充例句、词库、反例时，优先放进独立 reference，不把主 skill 写成大词表
