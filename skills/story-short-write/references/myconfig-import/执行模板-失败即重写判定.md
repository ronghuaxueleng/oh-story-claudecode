# 执行模板：失败即重写判定

这份模板给模型做第二道闸门。

当模型已经写出正文后，再把这份贴给它，让它自己判这版是不是直接作废。

---

你现在不是继续润色，而是在执行“失败即重写判定”。

你的任务不是解释这版为什么还行，而是尽量找出它为什么不合格。

判定原则：
- 从严，不从宽
- 只要有明显违规，整段作废
- 不准用“基本上”“大体上”“还可以”这种模糊说法

## 总判定

- 如果失败项 >= 1：`本版作废，必须整段重写`
- 如果“场面过于整齐”失败：`本版直接作废`
- 如果“高功能对白”失败且证据句 >= 2：`本版直接作废`
- 如果“漂亮细节”失败且这些细节都不承担伤害事实：`本版直接作废`
- 如果“真相或重大信息闭环链过强”失败：`本版直接作废`
- 如果“同时完成的大任务过多”失败：`本版直接作废`
- 如果“检测切片下会形成完整推进单元”失败：`本版直接作废`
- 如果“这一刀无法概括成一个主任务”失败：`本版直接作废`

## 重写指令

如果作废，必须补：
- 本次最该删掉的3句
- 本次最该拆碎的2处对白
- 本次最该砍掉的1个结尾收口

不准输出安慰，不准输出总结陈词，只输出判定。

## 什么时候用

- 模型第一次写完正文后
- 你怀疑它又开始自作聪明时
- 你要强制它自己承认这稿不合格时
