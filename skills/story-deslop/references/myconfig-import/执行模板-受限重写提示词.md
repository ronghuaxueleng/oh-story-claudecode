# 执行模板：受限重写提示词

把下面整段直接贴给模型，再补你的原文、目标段和额外要求。

---

你现在不是自由创作，你是在执行“受限重写任务”。

任务目标：
1. 严格学习原文这段为什么像人工，不是学习主题，不是学习大意，而是学习它的受力方式。
2. 重写时优先复刻“有效写法”，不是发挥你的文风。
3. 不允许写成示范段、漂亮段、顺滑段、解释段。

本次任务中，“优化”属于违规。
你的目标不是写得更高级，而是更接近原文的人工受力方式。
如果你发现自己在润色、提纯、修整、拔高、压缩、讲清楚，请立刻停下并重写。

你必须先执行下面流程，再给正文：

第1步：列出本段最伤人的3个小事实、最关键的2处偏心顺序、最该保留的1个不体面情绪。

第2步：列出本段必须保留的5个有效受力点。
格式：
- 受力点1：
- 受力点2：
- 受力点3：
- 受力点4：
- 受力点5：

第3步：列出本次绝对禁止犯的10个错误。
必须包含：
- 禁止补漂亮细节
- 禁止写示范腔动作句
- 禁止把对白写成高功能台词
- 禁止提前总结人物关系
- 禁止用作者口吻解释人物心理
- 禁止把场面写得过于整齐
- 禁止磨平原文的俗和脏
- 禁止用抽象判断替代事实伤害
- 禁止让人物说太完整、太会说的话
- 禁止把任务做成“总结后再创作”

第4步：写执行策略。
格式：
- 我这次会保留什么：
- 我这次会删掉什么：
- 我这次会故意写得不那么工整的地方是什么：

第5步：再写正文。

正文要求：
- 只写正文，不解释
- 优先保留脏、俗、直、伤的小事实
- 优先保留互扎型对白
- 优先保留身体反应
- 优先保留偏心发生的先后顺序
- 宁可笨一点，不能示范腔
- 宁可俗一点，不能规整
- 宁可短促一点，不能讲透

正文写完后，必须做证据式自检。
每条自检都必须：
- 给出结论：是 / 否 / 轻度有
- 引用正文原句
- 解释为什么算或不算

自检项至少包含：
1. 有没有补漂亮细节？
2. 有没有出现示范腔动作句？
3. 有没有让对白过于高功能？
4. 有没有提前下关系判断？
5. 有没有把原文的俗和脏磨平？
6. 有没有替人物保留太多体面？
7. 有没有把事实伤害改成抽象感慨？
8. 有没有一不小心写成你自己的文风？
9. 有没有作者在旁边解释？
10. 如果把这段放进原文里，会不会明显更整齐？
11. 有没有把重大信息写成“拿到就看懂、看懂就出手、出手就见效、见效就翻篇”的完整闭环链？
12. 这一段是不是同时完成了三件以上的大任务？
13. 这一段是不是把一个人物/一条旧伤背景一次性补全了？
14. 这一段是不是既给旧事、又给判断、又给决定？
15. 这一段读完会不会太明白、太完整、太像作者交付好的成品？
16. 这次所谓“拆段”，是不是真的拆开了，还是只把同一大段改得更碎？
17. 这一段是不是混了两个以上的时间层？
18. 如果检测器从这句开始切，到下一段结束，这一小块会不会自己就形成一个完整推进单元？
19. 这刀能不能被一句话概括成一个主任务？
20. 这段是不是在办公室/对峙场面里一次性完成了摊牌、问罪、离婚、反杀中的两项以上？
21. 如果这刀主场已经很明确，我是不是还多加了一段“助跑”过渡，导致推进变顺？

如果任意一项结论为“是”，或者“轻度有”超过2项，则当前正文作废，直接重写，不要解释。

附加约束：
- 一次只处理一个高风险段
- 不要顺手改整篇
- 不要擅自补设定
- 不要擅自拔高人物

现在开始执行。

---

## 使用建议

适合这样用：

1. 先贴这份模板
2. 再贴原文目标段
3. 再贴你自己的失败稿对应段
4. 最后补一句：
   `本次只重写 600-1200 字，不准越界`
