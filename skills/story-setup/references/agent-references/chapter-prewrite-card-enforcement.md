# 单章写前卡强制化

## 定位

单章写前卡是正文开写前的硬输入，不是写完后补的说明书。

## 每章必须明确的五项

1. 本章事件责任
2. 本章人物责任
3. 本章压场点
4. 本章群像配合点
5. 本章章尾责任

## 写前卡最小字段

- `事件推进`：这一章实际推进哪件事
- `人物变化`：谁因为什么发生变化
- `爽点/压点`：主角或核心人物在哪一拍压住全场
- `配角抬轿`：谁负责见证、误判、放大、反衬
- `章尾钩子`：本章最后留什么继续点

## 强制口径

- 没有写前卡，不得开正文
- 写前卡只写本章，不写泛化原则
- 写前卡必须能直接映射到正文，不允许空话

## 与参考资料的关系

- 情节库只负责给出可执行剧情条
- 参考原文只负责补表演层方案
- 两者都不能替代写前卡

## 施工摘要词族黑名单

写前卡里不得把以下词直接当成本章任务词、收益词或爽点词：
- `第一刀`
- `第二刀`
- `补刀`
- `追刀`
- `落刀`
- `定顺序`
- `改口`
- `执行次序`
- `解释口`
- `资源去向`
- `口令`
- `旧口`
- `口子`
- `解释权`
- `执行权`
- `决定权`
- `资源位`
- `头位`
- `控盘`
- `抢势`
- `盘面`

推荐替换为：
- 谁先补
- 谁先分
- 谁先放
- 谁被迫认账
- 谁失去先手
- 谁被迫按新的顺序动手
- 谁先吃亏
- 谁先退
- 谁先交东西
- 谁先让地方
