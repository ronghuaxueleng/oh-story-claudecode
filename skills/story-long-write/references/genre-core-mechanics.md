# 主打套路怎么定，微创新怎么加

> 用于梳理主打套路怎么一层层往下落（主题→题材→情绪），并补看微创新、冲突网、金手指怎么配、事业线和爱情线怎么摆。构思和校大纲时可对照。
> 配合文件：题材主线怎么走见 `genre-writing-formulas.md`，读者画像见 `genre-readers.md`。

---

## 你现在写到哪一步

> 构思大纲时可按现在卡住的地方往下走，对着对应那节查。

| 现在写到哪一步 | 要做的事 | 对应章节 |
|---------|---------|---------|
| 定主打套路 | 把主题、题材最能打的点、主情绪这三层拎出来 | 主打套路三层递进设计 |
| 看主打有没有写偏 | 确认没有写偏、没有挂羊头卖狗肉 | 每章循环检查 + 检验清单 |
| 微创新 | 在这类写法能兜住的范围里做点新东西，别超过 3 处 | 微创新与差异化设计 |
| 冲突网怎么铺 | 把上下、平级、交叉这三路矛盾铺起来 | 冲突网络设计 |
| 选金手指 | 金手指与世界观压迫度匹配 | 金手指与世界观匹配 |
| 事业线和爱情线怎么摆 | 把主线和感情线摆好，别让谁压过谁 | 事业线/爱情线设计 |
| 防崩盘 | 看卖点有没有跑偏，题材和写法搭不搭 | 卖点偏移检验 |

## 主打套路怎么一层层往下落（主题→题材→情绪）

这三层不是拿来分门别类的，是一层套一层往下落的。一本文通常会把 2-4 条主打套路揉在一起。

### 一代：主题/中心思想

| 看哪一面 | 怎么看 |
|------|------|
| 这里说的是什么 | 小说的主题和中心思想 |
| 作用 | 给整本书一个奔头，别写着写着散掉 |
| 要紧处 | 没有主题，文章就容易只剩热闹，没劲也没情绪 |

主题在前中后可以有不同表现，不一定死卡成一个样。

### 二代：题材最能抓人的地方

| 看哪一面 | 怎么看 |
|------|------|
| 这里说的是什么 | 该题材独有的最强吸引力 |
| 作用 | 筛选读者、圈出这书大概写哪一口、把氛围带出来 |
| 提取法 | 看同题材火书最能打的卖点和主情绪 |

**先看清这句：写偏题材最能打的地方，就等于把最能抓读者的依仗丢了，在读很容易直线掉。**

偏离案例：

| 作品 | 题材 | 偏离 | 后果 |
|------|------|------|------|
| 《一级一个金词条》 | 都市高武 | 对比对象从地球人跳到异世界神 | 在读直线下降 |
| 《普攻永久加生命》 | 网游文 | 走出新手村后仍和NPC打 | 热度持续下滑 |

### 三代：主情绪

| 看哪一面 | 怎么看 |
|------|------|
| 这里说的是什么 | 一种最主要的情绪体验(扮猪吃虎/弥补遗憾/莫欺少年穷等) |
| 作用 | 筛选读者、提升戏剧性 |

主情绪不等于爽点表现形式。装逼打脸只是形式，主打套路更像一整条情绪链(期待→满足)。

### 剧情构建顺序

**主打卖点 → 情绪套路 → 具体剧情**

先确定主情绪，也就是读者到底想看什么；再想怎么把这种感觉一点点喂给他；最后才落到具体事件。

### 主打套路到底指什么

主打套路 = 金手指最能出效果的用法 + 读者最想看的爽点模式。金手指是工具，主打套路是最有趣的那种用法。主打套路可以平替：女帝→朱元璋→嬴政，背景都能换。

### 每章都要过一遍这件事

每章最好三选一：①有和主打套路相关的期待点 ②有和主打套路相关的爽点 ③停在期待点和爽点之间。

**断期待的判断**：某章既无期待点也无爽点，前一爽点已结束，本身没拉起新期待→读者觉得书失去了"灵魂"开始弃书。

**两种运转模式**：
- 不断循环型：每个大段走法跑一次，再换场景 / 对象 / 条件
- 只运行一次型：围着一个主情绪一段段往前推

### 自己顺手对一遍

- 读者翻开最想看什么？(主打套路)
- 每个剧情单元是否都在展示主打套路？
- 主打套路这套爽点模式能不能循环？
- 有无挂羊头卖狗肉？
- 全文每章是不是最好都满足这三样里的一个？

### 常见误区

| 误区 | 表现 |
|------|------|
| 主打套路太多 | 什么都想写=等于没有主打套路 |
| 立意不明确 | 没有主情绪，文字干巴巴 |
| 纯装逼打脸无主题 | 剧情没有意义 |
| 主打套路用完不升级 | 生硬重复，审美疲劳 |
| 挂羊头卖狗肉 | 书名暗示与实际内容不符 |

### 主打套路写偏的典型案例

| 作品 | 偏离方式 | 后果 | 改善方向 |
|------|----------|------|----------|
| 《完美世界》 | 后期主情绪散了 | 月票直线下降 | 独断万古可以但要有实质收获 |
| 《十方武圣》 | 杀光所有人换地图 | 主情绪崩坏，留存风险显著升高 | 换地图不要清空旧角色资产 |
| 《诡舍》 | 所有角色都只剩剧情作用 | 高开低走 | 身边人遭遇危机后最终克服 |
| 《一人镇守孤城》 | 始终沉浸悲壮无变化 | 读者麻木 | 穿插其他情绪 |

**先看清**：悲情不需要靠身边人死绝来撑，多造挫折就够了。刀人只是手段，不是目的。主角目标得让读者觉得有意义。

### 悲剧命运改写法

1. 选取悲剧原型
2. 选取切入点(主角介入时间节点)
3. 选取装逼方式
4. 敲定贯穿表现力的道具
5. 选取参与配角(不超5人)
6. 重塑事件，构想改写命运画面
7. 处置悲剧变喜剧反转+主角收获+下一故事引子

直接点说，就是把行侠仗义写实，把打怪装逼写得有劲，再把“改命”这件事写得谁都能一下看懂。

## 微创新和差异化怎么拿

创新先收在：**人物、人物关系、情节。最多 3 个微创新点。** 别写到这类路子自己都兜不住。

### 灵感源头

| 这一段灵感主要从哪来 | 来源 | 怎么拿 |
|------|------|------|
| 前期 | 标签系统 | 热门标签交叉组合 |
| 中期 | 热搜/评论区/弹幕 | 高频话题提炼 |
| 后期 | 电影/游戏 | 视觉化叙事、交互式体验 |

### 标签系统

每个标签大概从四面看：

| 看哪一面 | 怎么看 | 权重 |
|------|------|------|
| 兼容度 | 能和多少其他标签组合 | 高 |
| 影响度 | 对故事整体影响深度 | 中 |
| 新鲜度 | 多久没被大量使用 | 高 |
| 喜爱度 | 读者偏好 | 中 |

挑标签一般看这个：**能跟别的东西搭得上 + 影响别太重 + 新鲜感够 = 比较好下手**

### 70/20/10这条配法

- 70%来自过去经历和记忆(共同代际记忆/流行文化)
- 20%来自当前生活状态(工作/爱好/感情)
- 10%来自时事热点话题和趋势

### 五种微创新手法

| 手法 | 最该盯的点 | 示例 |
|------|------|------|
| 精炼法 | 把已有套路做到极致 | 打脸一巴掌→三巴掌层层递进 |
| 升级法 | 原来的路子不变，把东西再往上抬 | "公司"→"跨国集团"，"校花"→"国民校花+学霸+隐藏身份" |
| 加料法 | 原来的路子里再塞新东西 | 重生+规则怪谈，种田+末日 |
| 反套路法 | 反过来写 | 废材逆袭→天才跌落重新站起 |
| 组合法 | 两个不相干套路组合 | 种田+修仙，系统+悬疑 |

### 微创新先过这张表

- [ ] 创新点是否压在 3 个以内？
- [ ] 还在这类写法自己能兜住的范围里吗？
- [ ] 会不会把读者最想看的东西冲淡？
- [ ] 是否有足够素材支撑？
- [ ] 是否增加了创作难度？

### 冲突网怎么织

好的长篇，最好有 3 层矛盾一起在跑。

**矛盾大概这几种**
- 纵向(上下级)：控制与服从/压制与反抗/师徒/君臣
- 横向(平级)：理念冲突/资源争夺/同行/情敌/竞争对手
- 交叉：A和B矛盾，B和C矛盾，A和C因此也产生矛盾

**编织顺序**：定地图→定阵营→定角色→对齐阵营填充纵+横矛盾→推理交叉矛盾

### 换瓶不换酒三条路

| 路子 | 怎么做 |
|------|------|
| 换世界观 | 相同节奏和金手指换到不同题材 |
| 换职业/元素 | 世界观固定，不同职业或问题解法 |
| 加新花样 | 已有写法里再塞一个读者熟悉、但网文里还不算老套的新东西 |

### 题材揉在一起时怎么别散

几种题材往一块揉时，得先找个更大的说法把它们都圈进去：
1. 先找一句能把这些东西都兜住的话
2. 利用这些差别给主角办事（不同题材碰到一起，本身就会带来信息差和优势）
3. 原创替代搬运(经典角色为原型创作，非直接拿)

### 写的时候一直盯住什么

写的时候一直盯着三件事：读者翻书前知道什么，看完设定后又多知道什么，看完这一段剧情后还会改掉什么判断。你能稳稳拿住这些，很多剧情难题就能拆开。

### 金手指和世界观怎么对上

| 世界观是哪种 | 压迫感长什么样 | 更适合什么金手指 |
|------------|----------|------------|
| 极道流(原教旨) | 阶级固化+环境恶劣 | 无条件强行升级(加点流) |
| 极道流(改良) | 压迫感降低 | 熟练度变种(花哨但还是得慢慢肝) |
| 常规升级流 | 换地图升级 | 碎片化/解锁型 |
| 轻松日常流 | 低压迫 | 人际关系/种田型 |

**金手指太单一的风险**：解决矛盾的路数太单一，只适合强压迫世界观，别的题材硬用就得补。

### 认知差异这类主打套路

主角视角“理所当然”vs旁人视角“不可思议”会形成张力。放到无敌流里尤其好用。

| 场景 | 主角这边会怎么想 | 旁人会怎么看 |
|------|----------|----------|
| 无敌流 | 死了复活没啥 | 这人比诡异还诡异 |
| 幕后流 | 随手布置 | 幕后boss深不可测 |
| 迪化流 | 想帮茶农卖茶 | 他在下大棋 |

### 基调和气氛别写散

全文基调得从头撑到尾，变得太狠就像换了一本书。人物会变，不等于书的气口也能乱变。金手指再升级，也别把原来的味道冲没了。

### 故事构思四法

| 这条路 | 最该盯的点 |
|------|------|
| 发散法 | 顺着起点那股劲往外铺出大量爽点 |
| 联系法 | 将两个跨度大的素材关联想象，跨度越大戏剧性越强 |
| 去常态法 | 取人事物特征反常化(恰恰相反/极端化) |
| 点推法 | 把散碎故事一段段接起来，串成完整故事 |

**组合流程**：灵感→联系法/去常态法处理→发散法基于处理后灵感发散→点推法串联

### 爽点得一波比一波高

后一个剧情给到的情绪满足，得高过前一个。

**往上抬一般就看这几面**：影响的人更多了 | 碰到的人物层级更高了 | 拿到的东西更值钱了 | 认知翻面的劲更大了

**递升失效信号**：读者对相似规模爽点不再反应 | 主角升级无差异化展示 | 人际网没随等级变化

### 男女频主打路子的差别

| 这一面 | 女频 | 男频 |
|------|------|------|
| 最主要任务 | 把一个卖点抠深抠透 | 把一套能反复转起来的路子挖出来 |
| 字数支撑 | 短中篇，一个梗写透 | 长篇大长篇，得靠那套走法一直往下转 |
| 创新重点 | 同一卖点换新写法 | 主情绪的新拼法 |
| 拆的时候看什么 | 拆一本书最好看的地方 | 拆一本书到底怎么一直转得动 |

### 模拟文怎么排

**单元**：模拟制造情绪缺口→模拟后得奖励(形成信息差→拉期待)→现实满足

**卷纲**：模拟得知大危机→[模拟+得奖励+现实推进]循环→最大那层危机还没解开→最后吃到总满足

### 同人深层情绪写法

装逼震惊之下加更深层情绪：亲情/救赎/遗憾弥补。原著人物震惊不止"太强了"还有激动/后悔/释然。重新认识熟人有代价，震惊熟人效果远高于震惊陌生人。

## 事业线和爱情线怎么摆

### 先看这几条

- 事业线是长篇主线，爱情线是藤蔓。天花板取决于事业线质量
- 主线是一件事，不是升级。升级只是达成目标的手段，不是主线本身
- 《斗破》主线="被退婚→上云岚宗复仇"，《全职》主线="被辞退后重回巅峰"

### 事业线要点

- 事业线其实就是“主角从弱变强的过程”
- 每一段都得有清楚的目标和对手，打完这一段，后面难度再往上抬
- 没有事业线时抽取有共鸣的人生主线(高中/大学最有效)

### 爱情线四步走

| 走到哪一步 | 内容 | 爽点 |
|------|------|------|
| 萍水相逢 | 展示外貌/性格/他人评价/习惯 | 新鲜感 |
| 怦然心动 | 产生好感注意对方存在 | 期待感 |
| 暧昧升温 | 互动增多关系微妙变化 | 最强拉扯感 |
| 确认关系 | 突破窗户纸 | 满足感 |

### 爱情线推进技巧

- **关系拉扯**：推进和拉开交替，不能只拉近。扯开不是分手是心有芥蒂/暂时冷静/误会
- **英雄救美**：男主事业线行为无意间救女主→女主倒追(非白给)
- **误会**：制造隔阂延缓推进，也是装逼打脸工具
- **窗户纸事件**：好感一下变了，关系也跟着往前走一步。事业线为主的书，一般把它放在大段剧情里的小高潮上

### 狗粮文循环

不能模仿升级文突破模式(牵手/接吻/上床是一次性的)。正确循环：稳定人设组合+变化情境=无限可用套路(复用短视频固定流程模式)。感情升温通过共同困境/互相理解/意外发现，非身体接触升级。

### 事业线与爱情线平衡

- 互相促进：事业为爱情提供场景和冲突，爱情为事业增加牵挂
- 九一开：事业线为主的文，感情线最多占一成，而且得给事业线帮忙
- 读下来的整体感觉得一直对，不然主打套路就会跑偏

### 男频感情线另看这几条

- 不写拉扯式恋爱，不能写男主追女人为女人付出
- 更顺的写法：男主追求事业→无意间英雄救美→女主倒追
- 女主倒追vs白给：区别在于是否有说服力的铺垫
- 读者目的：收获现实中缺失的异性情绪价值补偿

### 后宫与炒股

这类写法中段大多差不多，都是先暧昧着不定关系，最后再落定。后宫最忌讳的是刚和一个定下，又去追下一个，那样会显得男主很渣。修罗场一般三种处理：轻轻带过、拐回事业线里解决、或者拿来做人物反差。炒股文结尾可以留开放口。

### CP行为怎么一层层往上走

好感越高，两个人的行为才越能往前走，别突然跨大步。

| 级别 | 行为 | 表现 |
|------|------|------|
| 1 | 主动亲密 | 肢体亲密接触及以上 |
| 2 | 主权行为 | 争风吃醋、宣示主权 |
| 3 | 信赖行为 | 无条件分享秘密、绝对信任 |
| 4 | 安抚行为 | 敏锐察觉负面情绪并主动安抚 |
| 5 | 特殊待遇 | 对待方式与其他人完全不同 |
| 6 | 先顾着对方 | 做决定时对方是最重要因素 |
| 7 | 默契行为 | 不约而同、无需交流自然配合 |

### 爱情线往下到底靠什么撑

**友情vs爱情**：恋爱双方得一直带着一点低烈度“斗争”底色，在鸡毛蒜皮里互相让步、慢慢磨出共识。爱情能容下的“存异”本来就比友情小很多。

**下面四条顺手对一下**：
1. 人物决策时"男/女人"成分是否超过"人"的成分？超过时有无足够外部条件？
2. 突出性别特质描写是否匹配基础性格/成长经历/身份地位？
3. 避免完全靠外部因素推动，加入更多内生动力的碰撞
4. 不会写就按挚友写再修正(挚友改写法)

## 卖点偏移检验

### 防崩盘先看这三条

1. 能否用一句话说清题材主打卖点想打到什么情绪
2. 卖点能不能拆出 5 个不重样的剧情展开方向
3. 卖点展开所需的关键能力，现有写法能不能撑住

### 题材选择与知识领域匹配

| 这一步 | 怎么看 |
|------|------|
| 1 | 评估自身专业知识储备和了解深度 |
| 2 | 评估能否构建符合的价值观和世界观 |
| 3 | 检验对该题材社会运行规律的了解 |
| 4 | 确认文字能否让读者代入 |

### 金手指一般怎么往上走

| 走到哪一步 | 特征 |
|------|------|
| 1.0 | 纯极道加点(深蓝系统) |
| 2.0 | 加点+探索+装备合成 |
| 2.5 | 熟练度变种+多套走法混着用 |
| 3.0 | 主走法 + 顺手带出副线剧情 |

### 赛道选择

不要直接冲大众赛道。选自带流量但竞争少的赛道做区别化。往外扩时，新作和老作要留点重叠的东西，老读者才跟得上。例子：动画宝可梦→都市宝可梦→都市原创御兽→仙侠御兽→纯仙侠，每一步都留了点能接上的东西。

### 脑洞金手指创新：错位系统

- 时间错位：都无敌了逆袭系统才来
- 环境错位：正常世界觉醒末世系统
- 人员错位：本应寄生皇帝的寄生在了皇后身上
- 错位→落差→戏剧性

### 脑洞大概分哪几路

1. 金手指类脑洞(聊天群/系统/戒指老爷爷)
2. 剧情类脑洞(反转/设定)
3. 设定类脑洞(世界观/修行路子)
4. 角色类脑洞(身份/人设)

一般是这么往上加的：先有金手指，再加角色，再加设定，最后四路一起转。别一上来就全堆满。

### 主打卖点怎么一步步往下做

得到点子→看看它能不能往下做大→放进合适题材→提炼出主打卖点→再把它磨成情绪缺口和读者期待→理出故事雏形→动笔试

## 最后过一遍这张表

> 大纲做完后，把下面一条条对过去。都站住了再算过。

- [ ] **主打套路明确**：能用一句话说出"读者翻开最想看什么"，不是泛泛而谈
- [ ] **三层都在**：主题（这书到底想说什么）→题材最能打的吸引力→主情绪（读者主要吃哪种感觉）都写清了
- [ ] **每章不离梗**：每章最好有期待点或爽点之一，别出现“断期待”的空白章
- [ ] **偏离预警**：对比"偏离案例"表格，确认没有同类偏离
- [ ] **微创新别太多**：创新点落在人、关系、情节里，总数别超过 3 个
- [ ] **冲突三层铺满**：纵向、横向、交叉矛盾最好各有 1 条在运作
- [ ] **金手指和世界观搭得上**：金手指的路数和世界观的压迫感对得住，太单一的话也留了后手
- [ ] **爽点是一波比一波高**：后一个爽点在影响范围、人物层级、收获大小、认知翻面里，最好有一面压过前一个
- [ ] **事业线/爱情线比例**：事业线为主的文爱情线不超过一成，且爱情线为事业线服务
- [ ] **基调贯穿**：全文基调一致，金手指发展没有破坏基调
- [ ] **卖点不偏移**：书名、简介、开头前3章传递的卖点与实际内容一致，没有挂羊头卖狗肉
