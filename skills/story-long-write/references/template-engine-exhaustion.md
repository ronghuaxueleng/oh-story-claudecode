# 这套路子怎么别只剩个壳

这页就是给 `story-long-write` 用的。
- 它专门解决“这套写法有了，但只把名字挂上去、没把戏写透”。

## 什么情况下翻这份

碰上下面任一情况，就翻这份：

- 当前章明确调用事件写法 / 推进主路写法 / 情节库写法
- 当前题材属于 `规则流 / 经营流 / 脑洞玄幻 / 连续翻盘文`
- 当前章主推进靠的是 `公开定输赢 / 资格争夺 / 试错探路 / 第一单开张 / 规则改写`

## 开写前先问这八句

进正文前，先答清：

1. 这套写法，这章到底改什么现实
2. 主角借它先夺到什么
3. 对手借它怎么当场压回来
4. 第三方借它怎么把原话收回去 / 往旁边退 / 把资源改往别处送
5. 主角为此付什么代价
6. 这套写法除了推情节，还显了谁的人物
7. 这套写法能给下一章接出什么新东西
8. 本章最后停在哪个动作上最合适

只要有一项答不稳，就说明这套路子还只是个壳，先别直接进正文。

## 这套路子最少得落出四层

每个主写法都得一起把这些东西落出来：

- 事件层：谁对谁做了什么
- 人物层：谁主动、谁把场子压回来、谁把话缩回去、谁失态
- 代价层：主角 / 对手 / 第三方谁为此付代价
- 章尾层：最后停在哪个动作上

少一层，就说明这套写法还没写透。

## 这套路子最少得同时做到三件事

每个主写法最少得一起做到这几件事：

- 推剧情
- 显人物
- 改局面

补充：

- `改局面` 不等于只把话说得更狠
- 最少得改到下面一项，才算真把局面改了：
  - 说法
  - 先后次序
  - 资格
  - 名单
  - 名分
  - 谁先担责
  - 谁先出手

若只做到“大家更紧张了 / 强者更重视了 / 全场都知道这东西更值钱了”，一般还按没写透看。

劲头更大的写法，一般还会多做一步：

- 开后手

也就是给下一章先喂出一项以上：

- 新物证
- 新资格
- 新名单
- 新争夺
- 新规矩

## 前后别写断了

这套写法别孤立用。

上一套写法的结果，要成为下一套写法的起点。

若前后两套写法之间得靠作者解释才能接上，那就算没接住。

## 别把这套路子的步骤直接喊出来，也别一上来就写满

真实冷启动样本已经反复证明，这套路子最常见的新问题不再是“没用上”，而是：

- `写得太直`
  - 角色一开口就在报这一步要干什么
  - 对话像在替细纲翻译“这一拍该干什么”
  - 旁白直接点名“谁往哪边挪了 / 谁把局面扳过去了 / 谁先吃到后果了”
- `写得太满`
  - 当章正文只做到 `名字先挂上去 / 现场先得按这边的顺序来`
  - 写后检查和追踪已经提前写成 `这份好处已经当场起作用 / 先办事的机会到手 / 现场调人调货都先听主角的`
  - 一章里把“这套写法已经用上”“收益已经有效”“下一章已经接上”全写满
- `写得太快`
  - 对手只回一口嘴
  - 第三方立刻把原话收回去
  - 主角马上就成了现场能拍板的人
  - 中间没有足够的当场压制、争先、被挤开、被迫动作

进正文前，还要再答四个问题：

1. 这一章最容易把哪一句写成“黑话直接露出来”
2. 对手除了嘴硬，还会用哪一手当场压制
3. 第三方不会立刻倒向谁，而是会先迟疑 / 先顾自己 / 先拖哪一步
4. 当前收益最多只能写到哪一层，为什么还不能再往上写

四问里只要有一项答不稳，这章就还没到能写的时候。

## 路子对了，但正文还是薄时怎么补

真实冷启动样本已经反复证明：

- 这套写法里的东西写齐
- 这套写法的效果也基本成立

并不自动等于正文已经够厚。

当前最常见情况是：
- `第一爆点` 对了
- `第二轮收益` 对了
- `两轮外部反应` 也对了
- 但正文仍然偏薄，读感像“高质量提纲扩写”

这时别再往里补：
- 解释句
- 总结句
- 作者判词
- 背景说明
- 把已有收益换句复述

这时先补的不是“信息”，而是“新长出来的动作拍”：

1. 新增一拍受阻
- 谁原本要怎么拿
- 被什么打断
- 只能改成什么打法

2. 新增一拍争先
- 谁先把话说死
- 谁先把先后次序卡死
- 谁先把办事顺序改掉
- 谁因为这一步抢先被迫往后退

3. 新增一拍外部改线
- 第三方怎么往前凑或往后缩
- 办事顺序怎么换到别人手里
- 围观的人 / 协防 / 书吏 / 记簿人怎么改手上的活

4. 新增一拍后果开始起作用
- 哪道名分 / 名单 / 先后次序开始当场起作用
- 谁因此被挡住、被挤掉、被迫退半线

5. 新增一拍迟疑成本
- 第三方原本想怎么观望
- 被谁逼得不能再观望
- 他因此丢掉了什么缓冲空间 / 观望空间 / 退路位置

一句话先记住：

- 这套写法负责“把主线拉对”
- 厚度负责“让主线长肉”
- 若新增内容答不清“谁做了什么 -> 谁被迫怎么改”，那就不算有效补厚度

## 代价别悬空

每个主写法最少得把两边到底赚了什么、丢了什么说清：

- 主角这边疼在哪：折命 / 露底牌 / 背责任 / 失先手
- 对手痛：原话收不住 / 丢名分 / 丢资源 / 压不住场子
- 第三方痛：被逼着选边 / 被迫留档 / 被迫出货 / 被迫得罪人

## 本书反复扣住的那些词得接进来

每个主写法都得明确改动本书反复扣住的那些词里最少一个。

本书里常反复扣住的词，常见有这些：

- 命
- 路
- 货
- 债
- 位
- 入口
- 名额
- 印
- 资格

若这套写法只完成打脸、狠话、震惊，却没改到这些词真正管着的东西，那就还是只做了样子，还没真起作用。

## 第三方得真干活

第三方角色别只会围观和震惊。

最少得承担一个现实上的活：

- 把原话收回去
- 留记录
- 抢资源
- 把前后补顺
- 往前站还是往后退
- 传风向

若第三方承担的是 `强者往前压一步 / 高位人物亲自下场 / 竞争者提前封线`，那还得继续答清：

- 他这一步切走了谁的哪一块利益
- 他这一步逼谁立刻改手
- 他这一步让谁再也没法继续观望

如果答不出来，那就只算“存在感到了”，不算这套路子已经写透。

## 收益到手，大概怎么分档

看这套路子到底起没起作用时，收益先分三档：

1. `开门`
   - 旧规矩被打松
   - 新争夺被抬明
2. `资格`
   - 主角拿到公开试手 / 公开争位 / 公开争夺资格
3. `到手`
   - 主角已经先咬住一格别人不能白拿走的现实利益

`到手` 再细分两档：

- `名字先挂上去 / 单子先落到手里`
  - 新名单
  - 新先后次序
  - 新挂号
  - 新名分
  - 已被公开写入单面 / 簿册 / 票据 / 官面记录
- `好处已经真的落到手里 / 真东西已经吃到手里`
  - 人已经先通过
  - 货已经先到手
  - 路已经先走通
  - 名额已经先兑现
  - 第一步现实收益已经被主角侧实拿

这两者之间，再加一个中间档：

- `这份好处已经当场起作用`
  - 名字或单子已经到手
  - 且该名分当场对现实产生了一道明确作用
  - 例如：
    - 头灯已经拦住活棺
    - 第一挂号已实际压住一道风险
    - 新先后次序已立刻挡住对手一手
    - 新资格已立刻改变谁能动、谁不能动

这道边界就这么看：

- 如果只是 `排在最前的签 / 头号名目 / 先挂上去的名字 / 名单` 写进了簿面，但还没发生现实作用：
  - 仍旧算 `名字先挂上去`
- 如果名字或单子已经到手，而且这道名分立刻在现场起了可见实效：
  - 就算 `这份好处已经当场起作用`
- 如果人、货、路、名额已经被主角这边真正吃到手：
  - 才算 `好处已经真的落到手里 / 真东西已经吃到手里`

把这道边界再说白一点：

- 正例：`名字先挂上去 -> 这份好处已经当场起作用`
  - 若正文不只是写下 `最前那盏灯 / 排在最前的签 / 头号名目 / 最前那一段`
  - 还明确出现：
    - 这道名分当场 `拦住了`
    - 它立刻压住一道风险 / 一个还活着的证人 / 一步先后次序
    - 现场别人因此马上往前往后挪、改挂靠、改守的那一段
  - 则可升判 `这份好处已经当场起作用`
  - 典型形态：
    - `第一盏头灯，拦住了`
    - 头灯一挂，抬棺行立刻改站守门
  - 这不是只有纸面名分，而是这份好处已经开始在现场起作用

- 反例：还停在 `名字先挂上去`
  - 若正文只是：
    - 新说法被写进纸面
    - 新规矩被主角宣布
    - 第三方先落笔背书
    - 大家知道规矩要改了
  - 但仍缺少“某人因此被动被挤开 / 让位 / 换手 / 丢掉已占先手”的硬后果
  - 那就还不能往上提成 `这份好处已经当场起作用`
  - 典型形态：
    - `活货待走`
    - `最先轮到谁改了`
    - `先试命辨灾`
  - 若这些只停在纸面新说法或口头改制，而非现场被动承认，仍判 `名字先挂上去`

一句话看法：

- `名字先挂上去` 看“有没有写进去”
- `这份好处已经当场起作用` 看“写进去之后，现场有没有立刻按它重新运转”

若当前章只做到：

- 排在最前的签写进单面
- 先挂上去的名字写上去
- 先手名分被公开坐实

这时就该写成 `名字先挂上去 / 单子先落到手里`，别误写成“好处已经真的到手”。

若当前章已经做到：

- 第一队真过道
- 第一票真发出
- 第一名额真兑现
- 第一步资源真到主角侧手里

这时才算 `好处已经真的落到手里 / 真东西已经吃到手里`。

冷启动或边界样本复盘时，得写清本章到底停在了哪一层。

## 旧样本说法迁移

若当前在复核旧样本、旧检查表、旧追踪文件，仍看到以下旧说法：

- `这套路子是否已经写透`
- `推剧情 / 显人物 / 改局面 / 开后手`
- `这套东西已经进正文`

这里就按下面处理：

1. 这些内容只允许作为“过程说明”，不再作为最后收口的话
2. 最后统一写回 `收益到了哪一步`
3. 旧表若写“这套路子已写透”，但答不出“到底改掉了什么现实局面”，默认别把收益判得太高
4. 旧表若只证明“这套东西已经挂上”，不能直接等同于“效果已经成立”

迁移时得补答：

- 本章到底改掉了什么现实状态
- 这一改动停在 `开门 / 拿到资格 / 名字先挂上去 / 现场先得按这边来 / 这份好处已经当场起作用 / 好处已经真的落到手里` 哪一层
- 不能夸大成什么

旧说法与新说法的映射提醒：

- `推剧情` 只能证明事件在动，不能证明收益已经到手
- `显人物` 只能证明人物不只是出来办事，不能证明局面已改
- `改局面` 还得继续拆成：
  - 只是 `开门`
  - 已到 `资格`
  - 已到 `名字先挂上去`
  - 已到 `这份好处已经当场起作用`
  - 已到 `好处已经真的落到手里`
- `开后手` 只能证明下一章有料，不等于本章已经拿到高档收益

复盘旧样本时，宁可低判一档，也别把“这套路子写得很足的感觉”直接写成“收益已高档兑现”。

## 先定说法资格词过判防误伤

`最后按谁的话定 / 排在最前 / 排在最前的签 / 头号名目 / 先挂上去的名字 / 第一单 / 最前一段` 这类词，本身不是收益到了哪一步。

它们常常只是以下几类现场东西：

- 公开争夺资格
- 纸面名分
- 待补的半坐实位置
- 即将起作用但还没起作用的先手位

所以这里得拆开看：

1. 若只是把旧私下说法改成公开争先，通常先判 `资格`
2. 若已写进簿面 / 单面 / 名单 / 公开记录，且别人不能当场白抹掉，判 `名字先挂上去`
3. 若这一步 `排在最前 / 排在最前的签 / 头号名目` 当场已经切走别人现实利益、先办事的机会、观望空间或先手权，才可升到 `这份好处已经当场起作用`
4. 只有当第一步货 / 路 / 名额 / 资源 / 办成的结果已实际落到主角侧，才可判 `好处已经真的落到手里`

反过来，这些信号要小心：

- `半认`
- `待补`
- `另记`
- `待定`
- `先挂后验`
- `先把话挂上去了，还没真落下来`

只要正文或追踪里还留着上面任一信号，一般就别把“先定说法的资格”判太高，别直接写成 `这份好处已经当场起作用` 或 `好处已经真的落到手里`。

对旧样本尤其要防：

- 角色状态里写“已经把拍板那一下抢到手”
- 写后检查里写“已经排到最前”
- 情报记录里写“已经被抬到最前面”

若正文实态还只是 `半认 / 待补 / 待定`，最后还是该写回 `拿到资格` 或 `名字先挂上去`，别被措辞带高。

别把：

- 只是重开牌桌
- 只是逼人把原话收回去
- 只是让所有人重新排队

直接写成“主角已经拿到最终收益”。

## 章尾先按这几句看

这套路子的章尾一般先停在：

- 新物证
- 新资格
- 新名单
- 新翻面
- 新争夺

别停在：

- 总结
- 解释
- 站出来替人下结论的感慨话
- 旁白把现场的话先收掉
