# 章节名怎么起才像回事

> 这份专门看章节标题怎么起、怎么预检、怎么回炉。目标不是“起个不重复的名字”，而是让标题真能带点击、带翻页、带情绪预热。

---

## 一、章节名先得顶什么用

章节名最少承担下面 1 项，最好能扛住 2 项：

1. 提前预热本章第一爆点
2. 放大本章最猛的收获
3. 放大本章最冲突的一拍
4. 预埋章尾翻面的情绪方向

如果标题只是“这章写了什么”，一般就不行。
如果标题里连“谁和谁在冲、谁先动了谁的利益、谁先翻脸、谁先失态”都读不出来，也一般不行。

---

## 二、标题先看什么

标题素材按这个顺序挑：

1. 最有冲突感 / 最有翻面感的一句话
2. 本章让读者最想点进去看的冲突动作
3. 最能让局面当场翻过去的物证或规矩变化，但要带冲突
4. 对手最丢脸、最先急、最先失态的一拍
5. 主角最能立威、最能动别人利益的一拍

先别挑这类：

- 普通物件名
- 普通线索名
- 抽象概括词
- 章节说明词

---

## 三、常用标题类型

| 类型 | 一般用在什么地方 | 公式 | 提醒 |
|------|------|------|---------|
| 动作截断型 | 本章先冲再爆 | 一个动作 + 被打断 / 被撞见 | 让读者先看见动作被谁硬截断 |
| 规矩翻面型 | 求生、掌局、官场 | 某条规矩 + 被改写 | 让读者先看见哪条规矩要翻 |
| 对手失态型 | 打脸、验货 | 对手做出最丢脸反应 | 让读者先看见谁会当众失态 |
| 战果到手型 | 经营、资源、情报兑现 | 某收益 + 落到谁手里 | 让读者先看见最大那口收益到底归谁 |
| 物证带刺型 | 情报、悬疑、反杀 | 物证 + 冲突含义 | 让读者先看见某件东西为何会刺痛局面 |
| 台词挑脸型 | 爽文、对峙 | 最狠的一句台词 | 让读者先听见一句能挑脸的话 |
| 身份压人型 | 掉马、验货 | 某人不是你以为的那样 | 让读者先知道身份判断要被改写 |

---

## 四、标题禁区

以下标题一般直接回炉：

1. 过于概括
- 开始
- 反杀
- 线索
- 真相
- 局势
- 博弈
- 翻盘

2. 只有物件说明，没有冲突
- 私印残角
- 那把刀鞘
- 一封信

3. 铺垫腔 / 分析腔
- 局势渐明
- 暗流涌动
- 风雨欲来
- 更深的黑幕

4. 剧透过死
- 某人就是幕后黑手
- 主角成功拿到账本

5. 写作说明 / 过程交代 / 轻度祈使句
- 先把名字挂上
- 把他的封泥揭开
- 这块屯地还挂着营牌
- 先去看看
- 先把账翻出来

6. 一看就像提纲标签或工作话
- 发卖令当场换了说法
- 护营印转到别人手里
- 押路签先后次序被重排
- 旧说法被人当场掀翻
- 名分终于写进簿子里

这类标题的问题不是“字面不通”，而是更像细纲步骤、修改记录或正文里的过渡句，不像点击钩子。
若标题主要功能只是说明“这章准备干什么”，而不是打出“这章主要冲突 / 收获 / 物证 / 谁被挤开”，一般回炉。
这类标题的问题更具体：它们常常知道自己在说什么，但不像角色会说的话，也不像读者会被勾进去的话；看起来像提纲标签、工作备注、收尾总结，而不是章节名。若标题删掉章节号后，更像一条说明话、收口话、进度备注，而不像一个人会点开的冲突事件，一般直接回炉。

---

## 五、怎么粗看一眼

给每个候选标题打三项：

1. 冲突感
2. 画面感
3. 翻页感

只要有两项低，就换。

快速问法：

- 这个标题能不能让人知道“这一章会狠狠干翻一件事”
- 这个标题像不像一章正文，而不是提纲小标签
- 这个标题删掉章节号后，能不能单独勾人点开
- 这个标题是不是更像“本章写作步骤说明”；若像，直接换
- 这个标题是不是更像“提纲标签 / 工作备注 / 收口话”；若像，直接换
- 这个标题里的冲突是谁和谁；若答不出，直接换

---

## 六、与小说类型联动

### 求生爽

标题先往这边挑：
- 规矩翻面型
- 动作截断型
- 台词挑脸型

尽量避开：
- 单纯线索名
- 纯悬疑物件名

### 验货爽

标题先往这边挑：
- 对手失态型
- 身份压人型
- 台词挑脸型

尽量避开：
- 主角自吹型标题

### 经营爽

标题先往这边挑：
- 战果到手型
- 规矩翻面型
- 结果先亮型
- 利益归属型

尽量避开：
- 日常记事型标题
- 工序说明型标题

### 掌局爽

标题先往这边挑：
- 规矩翻面型
- 对手失态型

尽量避开：
- 只会喊大词的标题

---

## 七、细纲里最少要写到哪

细纲阶段最少得写清：

- 标题类型
- 标题主卖点
- 标题禁区
- 备选标题 2-3 个
- 最终采用标题

开书前 `20` 章额外强制要求：

- 每章最少给 `3` 个备选标题，别只留一个“暂定标题”
- 每个备选标题都要打 `冲突感 / 画面感 / 翻页感` 三项分
- 若最终采用标题里，有任意 `2` 项明显偏低，直接回炉
- 若三个备选标题都偏“写作说明 / 过程交代 / 轻祈使句”，说明本章标题思路整体跑偏，不是只换其中一个就算行
- 若三个备选标题都偏“提纲标签句 / 工作话标签句”，例如 `发卖令当场换说法`、`护营印转到别人手里`、`押路签先后次序被重排` 这类只是在总结经过、没人会这么喊出来的标题，说明本章标题思路同样整体跑偏，不是只改短一点就算行
- 标题打分记录应和细纲一起保留，避免后续正文阶段再滑回弱标题

---

## 八、回炉信号

以下任一项碰上，标题回炉：

1. 读完正文后发现标题只是在概括物件
2. 标题和本章第一爆点无关
3. 标题和本章最猛的收获无关
4. 标题过分剧透，削弱章内翻面
5. 标题太虚，放到任意一章都能通用
6. 标题像工作条目、收口备注或流程环节，不像读者会点开的事件句

---

## 九、给“情报 + 流放 + 求生破局”这类书的提醒

以下属于单独提醒，不是所有题材都要照抄。

这类题材标题，先往这边写：

- 谁先把原话吞回去
- 谁先急了
- 谁先动手了
- 谁先碰了不该碰的利益
- 哪条规矩被改掉
- 哪个动作被拦腰截断
- 哪件物证带来的不是信息，而是压制

别先往这边写：

- 单纯物证名字
- 单纯地名
- 单纯情报名词

推荐写法方向：

- 某条规矩要先被改写
- 某个关键动作要先被打断
- 某件关键物得先公开见光
- 某个锅得有人当场先背上

若换题材，保留的是“标题在这里起什么作用”，不是这些表面道具。替换时先对应：

- 规矩被谁改写
- 动作被谁截断
- 哪个物件一露面就让局面当场翻过去
- 谁在公开场合先失态

## 十、标题句式硬禁区

下面这些句式，默认都太像套话标题，直接回炉：

- `最该看的不是……，是……`
- `真正该抢的不是……，是……`
- `更重要的不是……，是……`
- `这不是……，这是……`
- `先把……`
- `把……揭开`
- `这块……还……`
- `先去……`
- `先把……拿回来`
- `发卖令当场换说法`
- `护营印改手`
- `押路签重排`
- `旧说法被改写`
- `名分到手`

标题得让读者先看见动作、规矩、物证、失态或翻面，不能用作者口吻替读者判价。
标题也不能只是把正文顺序提前说一遍。
标题也不能只是把“写完后你会怎么总结这一章”提前说一遍。凡是更像抽象收口话、进度话、过程环节，而不像现场里会炸出来的一句话，默认都按弱标题处理。
标题若不能直接传出冲突，最少也要传出“冲突马上就要炸”的异常点；否则仍按弱标题回炉。
