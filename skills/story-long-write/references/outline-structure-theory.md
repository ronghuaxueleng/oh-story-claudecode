## 一级走法与二级走法

### 一级走法 = 单元剧串联

一条故事线由若干单元剧组成，互相孤立，主角如绳子串起所有单元剧。

- 优势：爽点密集，路子简单，不容易出错
- 劣势：撑不起长篇幅，期待感单一，上限有限

### 二级走法 = 单元剧套娃往上抬

把一级走法里的“单元剧”拉长到整卷，每卷是一个完整故事。最容易出的问题是：一卷二三十万字只展开一个故事，铺垫太长。

**解决办法就是套娃**：大故事里的每一段，本身也得是一个小故事。小故事先给爽点，同时暗中铺垫大故事。一路往上抬，到高潮时再一口气串起来引爆。

### 三级走法

卷与卷之间不再孤立，而是按同一套故事走法统一起来，所有情节彼此咬住、彼此带动。对编织情节能力要求极高。

### 长篇特殊认知

- 最小冲突单元 = 节拍（一个动作对应一个反应）
- 过渡章节的问题往往是信息铺垫太多、冲突太少
- 剧作理论在这儿有两点别硬套：①人物弧——几百万字主角早该成熟，主角可以更多承担推进作用 ②细节打磨——长篇读者不会深研潜台词

## 故事基本走法（主角->障碍->目标）

**最基本的模型**：主角->障碍->目标

从基础到好的三步升级：
1. **加曲折**：多个障碍才叫故事
2. **递增难度**：难度一致就单调，难度递减就乏味，得越走越难
3. **意料之外**：主角即将到达目标时给一个意外——计划失败。解法：灵机一动（临时想到）或神秘一笑（隐藏已久的招）

### 支线与主线的双向对接法

从支线端和主线端同时往下挖，最后接到一起，前后才完整。别让支线变成"孤例"——大政策一层层压下来，主角遭遇只是大现象里切出来的一角。

### 短篇到长篇的连接

长篇 = 一个个短篇连起来，连接物 = 钩子。短篇结束前得留点没处理干净的东西。钩子有"定向"作用。

## 三幕式与五幕式

### 故事到底是什么：事实->事件->故事

- **事实**：静态呈现，不是故事
- **事件**：变化的事实（从X到Y的变化）
- **故事**：对事件进行选择、排序、因果化的产物
- **因果会让这事更有分量**：前后因果一接上，读者就会问"为什么？"，这层意思自然就出来了

### 因果链建立手段

| 手段 | 说明 |
|------|------|
| 排序 | 打乱时间顺序建立因果关系 |
| 铺垫 | 在事件b之前，在事件a旁边埋下因 |
| 伏笔 | 事件a + 事件h的因，隔很多事件后才揭示 |
| 钩子 | 在事件d旁边埋下事件e的因，制造悬念 |

### 五幕式：因果链最顺手的一种走法

| 幕 | 这一幕干什么 | 因果链状态 |
|----|------|-----------|
| 第一幕：开局 | 因必须在此埋下 | 种子 |
| 第二幕：发展 | 果+因，发展为下一幕的因 | 生长 |
| 第三幕：转折 | 果+因，质变为下一幕的因 | 转折 |
| 第四幕：行动 | 果+因，白热化 | 冲刺 |
| 第五幕：结局 | 果，收束 | 完成 |

**先看清五幕式这句**：因果链最好有五环；每一环各管一段事；别乱跳，也别乱序。

### 三幕、起承转合与五幕的关系

- **三幕**（冲突开始->对抗->解决）：世界矛盾足够驱动时使用。写命运悲剧、时代洪流
- **四幕/起承转合**：外部转折介入，主角被动承受。写社会悲剧、宿命、伦理
- **五幕**：外部转折介入，主角主动应对。写爽文、英雄、复仇

**转折直接看**：冲突性质的质变——冲突变得更严重，主角处境更糟。转折的因要偷偷提前埋下。

### 网文第六幕：余波（装逼为什么能打中人）

装逼真正撞上的冲突，是"主角与社会认知的冲突"，不是主角与反派的冲突。

主角实力两个维度：
- **明线**：战力体系里的晋升
- **暗线**：社会对主角的认知定位

明线高于暗线时产生冲突->装逼->暗线拉高到一致。小失衡有小装逼，大失衡有大装逼。

装逼的正确方式：不是主角主动证明"我很强"，而是环境、他人、时间告诉社会"你确实变强了"。

### 八股文套路化写作六大模块

1. **定调**（黄金三章到底在干什么）：回应读者期待。最常用走法：令人不安的处境 + 急待提升的地位 + 充满希望的外挂 + 能被展望的美好未来
2. **金手指设计**：信息挂（最稳，不易失控）> 加速升级挂 > 物品调用挂（容易失控）。先弄清一件事：简单，功能和升级走法要清楚
3. **脸谱化角色**：给角色贴标签，一看到特征就想起角色
4. **升级流节奏**：目标->行动->完成->奖励循环，周期要短
5. **密集爽点轰炸**：不爽->移除不爽=爽
6. **套路化冲突**：冲突本身越简单越好理解，悬念构建越复杂越好吸引人

## 五级大纲一层层往下拆

| 级别 | 比喻 | 说明 |
|------|------|------|
| 1级 | （根） | 统筹全文，一句话概括全书 |
| 2级 | （干） | 一个故事情节 = 一条2级主线 |
| 3级 | （枝） | 给枝上色 = 具体剧情 |
| 4级 | 更细的剧情推演 | 如堂哥矛盾->羞辱->奋发->金手指->干死对手 |
| 5级 | 最细致的具体过程 | 获得金手指的具体过程 |

**先看清**：下一级服务上一级。3级已经够撑长篇，5级太细时反而容易乱。从上往下推不容易卡，从下往上倒推最容易偏主线。

### 暗线设计

- 暗线 = 读者看不出来的线，出其不意
- 暗线统筹全文，从开始到结束不明写但留伏笔
- **一伏千里** = 答案从开局就给了，读者到最后才发现
- 悬疑常用多暗线同时进行->汇合->再汇合其他暗线->最终统筹全文

### 布局八条线

主线、副线、辅线、分线、暗线、支线、感情线、剧情线。主线分5级，下级服务上级。

## 大纲更偏剧情还是更偏人物

### 强主线（二级走法）

典型：升级文、悬疑文。重剧情轻人物，配角更多承担办事作用。

最明显的特点：
- 每卷有完整三幕式，每幕又有小三幕式（嵌套）
- 只有一条主线，所有支线推动主线

**大钩子设计**：主线拆成大支线，每个最终目标 = 大钩子。
- 大钩子别断，一个断了马上接另一个
- 更换大钩子时，前面得先铺垫，不能突兀

### 弱主线（一级走法）

典型：无敌文、日常文。轻剧情重人设。

最明显的特点：
- 单元剧模式，各有完整三幕式
- 公式：主要爽点 + 讨喜人设 + 单元剧舞台
- 不需要明确主线，主角串联一切
- 人设的新颖是重点

### 过渡型

开局弱主线（单元剧日常），后期逐渐过渡为强主线。

### 支线怎么收

支线得跟主线有关，不然就是废线。支线人物身份设定也得自然服务主线。

## 传统文怎么搭与四大留存手段

### 每卷怎么搭（目的+高潮结尾）

大架子是骨相，剧情是皮相。每卷先把两件事弄清：①来到新地方图什么 ②大高潮结尾放在哪。

### 四大留存手段

1. **升级**：不只打斗实力，还包括地位、金钱
2. **资源困境**：修炼需要资源，谋求资源是留住读者的好手段
3. **设置目标**：大目标+小目标+假目标（不一定要实现，用来勾起期待）
4. **解密**：展现冰山一角->层层解密->不断拨开迷雾

## 换地图与地图架构

### 换地图的四种演进方式

| 代际 | 主要路子 | 问题 |
|------|------|------|
| 1.0 飞升流 | 凡间->仙界->神界 | 人际关系全抛弃 |
| 2.0 莲花地图法 | 莲心主地图+花瓣分支 | 空间有局限 |
| 3.0 新路子 | 家族流/模拟器/版本迭代 | 需从中期开始铺垫 |

### 换地图三法

1. **千丝万线法**：两个大地图间不断穿针引线
2. **不换大地图法**：只换小地图，都在一个大地图里转
3. **莲花地图法**：地图像莲花层层展开

### 最上面这层怎么搭（防崩盘）

开书前先设整张地图所有顶尖势力作为"柱子"。从顶层向下延展：区域->小家族->小宗门->功法宝物。每个出场人物都与某个顶层势力挂钩。

**刻度尺思维**：提前设计顶层实力表现力作为标杆。开头boss=5，中期=50，后期才写100。

### 人际关系先行法

去下一个地图前，先把人际关系铺出去。先让人际关系动起来，再让主角跟着动。

### 换地图关键认知

- 大地图转换可能掉三分之一追订
- 损失的读者永远回不来
- 如果后续写得好追订可涨回来

## 多线写长技法

### 多线并进与挖坑填坑

从开头主线就要埋伏笔开支线。主线+两条以上支线同时展开，主线一停->支线接入。不断挖坑填坑 = 剧情不断延展。

### 多线缓解疲劳

长剧情中间插入半章配角视角的无关事件->短暂放松。切换要自然：主线中埋支线伏笔->适当展开->回到主线。

## 西幻创作要点

**选材**：选DND式、巫师、种田、领主文等常青树。避开高武和日式奇幻。

**世界观**：通过人物对话和行为展现，不搞大段设定文旁白。

**人物**：用中文谐音做译名，读起来更顺口；先设计金手指特性，人物性格围绕金手指展开；道德爽感是最高级爽点。

## 质量检查清单

大纲搭完后，逐项检查：

- [ ] **层级匹配**：选择的层级（一级/二级/三级）与题材和篇幅匹配
- [ ] **因果链完整**：五幕式每幕有固定功能，无跳步、无乱序
- [ ] **转折有因**：每个转折的因已提前埋下，不是突兀出现
- [ ] **难度递增**：障碍难度从低到高，没有递减或匀速
- [ ] **钩子不间断**：大钩子一个断了马上接另一个，没有空窗期
- [ ] **支线不废**：每条支线与主线有关，支线人物服务于主线
- [ ] **五级展开合理**：下一级服务上一级，不超过3级（长篇）
- [ ] **暗线有伏笔**：暗线从开局就埋伏笔，不是后期硬塞
- [ ] **换地图有铺垫**：人际关系先行，不是突然跳地图
- [ ] **刻度尺一致**：顶层实力标杆已设，boss难度在合理区间
- [ ] **多线挖填平衡**：挖的坑有计划填，填完有新坑接上
- [ ] **装逼暗线匹配**：明线战力 > 暗线社会认知时触发装逼，装逼后暗线拉平
