# 章节推进常用拼法

这页就是解决“这一章到底怎么把这套路子拼起来”。
- 主要就是管这件事
- 一般先用在细纲和本章短清单这一步

## 这一章常用拼法

- `本章目标 + 主事件 + 对手怎么压回来 + 配角变数 + 本章代价 + 章尾钩子`

## 这六格最好先填上

1. 本章主角想抢什么
2. 谁在拦
3. 对手这章的任务是什么
4. 配角这章具体出来做什么
5. 本章代价是什么
6. 章尾停在哪

## 更容易抓人的几种拼法

### 1. 抢资格这一路怎么拼

- 主角抢名额
- 对手改规则 / 点替死鬼
- 配角承担先挡一道的人 / 先探路的人
- 章尾落更高层人物或更高资格到场

### 2. 抢证据这一路怎么拼

- 主角抢证物
- 对手先抢纸面或先抢人
- 配角承担证人作用 / 护物证的人
- 章尾落在“证据有了，但不能直接用”

### 3. 公开定输赢这一路怎么拼

- 主角要改现场说法
- 对手抢先落印 / 点替死鬼 / 抢先定性
- 配角出来摇摆 / 逼人改说法
- 章尾落在纸面结果或更高权力下场

### 4. 第一单这一路怎么拼

- 主角要做成第一笔交易
- 对手截货 / 抢客户 / 改规则
- 配角承担带路的人 / 抬价的人 / 抢功的人
- 章尾落在分赃失衡或新竞争者到场

### 5. 探路试错这一路怎么拼

- 主角要验证一条路
- 配角先探路或主角主动犯险
- 对手用时间差或新规则压回来
- 章尾落在更深层风险暴露

### 6. 代价续命这一路怎么拼

- 主角眼前赢一点
- 对手不硬拼，改打主角代价
- 配角承担被清算的人 / 被拿捏的人
- 章尾落在“赢了，但后路没了”

## 真用的时候这么拿

- 一章先选 `1` 个主推进路子
- 再搭 `1-2` 个辅助写法就够了
- 同一章要是塞进 `3` 个以上主推进路子，十有八九会空转
- 只写了这一路的名字，没写现实变化，就说明细纲还没写实
