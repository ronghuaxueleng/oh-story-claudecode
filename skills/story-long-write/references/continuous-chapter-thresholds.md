# 连续章节大概到哪该升一级处理

这一页就是把 `story-long-write` 里和连续章节、连续回炉、章组卡触发有关的常用数量线放在一起。
主 skill 只拿“到了这条线就该往上提一档处理”这件事，不在主文件里把数字写死。

## 常用数量线

### 1. 连续多章回炉触发 `story-review full`

- 常用线：`3` 章及以上

### 2. 连续章组使用 `章组收益卡 + 三张单章卡`

- 常用范围：`2-4` 章连续翻面链

### 3. 连续章组补跑 `detect_key_character_promotion.py`

- 常用范围：`2-4` 章连续章组

### 4. 连续章组生成章组汇总

- 常用范围：`2-4` 章同一章组

就记这句：

- 这些数字只是平时先拿来用的线，不是死板规定。
- 若题材、节奏或项目计划明确要求不同数字，以项目内 `追踪/阶段审查计划.md` 与实际章组结构为准。
