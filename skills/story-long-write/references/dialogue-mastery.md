# 对话怎么写才不僵

> 用于对话场景写作时参考，可先判断这场对话更适合哪种路子，再按提醒往下写。

---

## 判断这场对话怎么走

| 你的对话场景是 | 用这种模式 | 怎么写 |
|-------------|---------|------|
| 主角碾压/打脸 | 压制模式 | 对方长篇大论 → 主角一字回应 |
| 主角亮底牌/反转 | 反转模式 | 对方嚣张 → 主角一句话把事实砸出去 → 对方沉默 |
| 关系破裂/心死 | 心死模式 | 对话越回越短：从辩解 → 沉默 → "随意" |
| 日常互动/立人设 | 日常模式 | 让其他人物一起卷进来，别让主角一个人自说自话 |
| 群众震惊/弹幕 | 弹幕模式 | 一层层往上走：普通人震惊 → 懂行的人点破 → 特殊身份者接话 |
| 信息展示/世界观 | 信息嵌入 | 用角色自己的口气把信息带出来，别硬讲设定 |
| 情绪拉扯/虐心 | 情绪推动 | 上一下下一下轮着来，像拉锯一样把期待往上拽 |

### 对话模式选择补充

| 对话类型 | 常见问题 | 修正操作 |
|----------|---------|---------|
| 问答式（通篇一问一答像审讯） | 改为一方主动说，另一方给反应；反应可用动作/表情/心理 |
| 冲突式（太礼貌不够劲） | 递进五级：委婉拒绝→友好人道→命令否定→PUA式→直接侮辱 |
| 日常对话（容易变成没用的水） | 这类话主要拿来立人设；能让别人卷进来的冲突，别让主角一个人独白 |
| 多人对话（混乱/没营养） | 写前先分清每个人进来是干嘛的：谁带信息，谁把情绪顶高，谁专门来搅局 |

## 对话最少得顶一件事

每句对话最少得做到下面一项，不然就删：

1. **把事往前推**：透露新信息、推动事件发展
2. **把人往后钩**：暗示马上要发生什么、把悬念拎起来
3. **把人写活**：通过说话方式把角色性子带出来

### 这些别写

| 禁止 | 理由 |
|------|------|
| 配角无脑夸主角 | 假 |
| 互相解释读者已知信息 | 水字数 |
| 大段说明文式对话 | 闷 |
| 所有角色说话方式一样 | 模糊 |
| 几句对话就把人说服 | 现实里没人这么好劝，用突发状况更像真的 |
| 情绪写完方向变了 | 需回退到情绪拆分 |

## 对话里谁压着谁

### 先看清

对话长度往往跟谁占上风有关。占上风的人话短且冷静，被动的人话多且情绪更乱。

### 压制模式

常见走法：对方长篇大论（3-5 行）→ 主角一字回应。

```
"你以为你是什么东西？我告诉你，这个家轮不到你说话！你嫁进来的那天起就该明白自己的位置——"
"滚。"
```

### 反转模式

常见走法：对方嚣张（2-3 行）→ 主角亮底牌（1 行事实）→ 对方沉默。

```
"你有什么资格管？这是我家的钱，我想怎么花怎么花。"
"你妈的存折，密码是我的生日。"
```

### 心死模式

常见走法：对话越回越短，从辩解到沉默到「随意」。

```
"你听我解释，那天不是你想的那样——"
"嗯。"
"真的，我可以证明——"
"随意。"
```

### 控制要点

- 掌控者/主角亮底牌时：对话 ≤ 10 字，别急着加动作描写
- 被压制方：对话 ≥ 20 字，可加动作描写（攥拳/咬唇/站起来）
- 两人对话时：短句方 = 权力上位，长句方 = 权力下位

## 潜台词与议程

### 潜台词怎么埋

- 角色真正想干什么，别直接写在台词面上
- 现实里人说话都会给自己找个说法，角色也一样
- 每句对白都得同时带着两层：他真想要什么，和他嘴上拿来当理由的是什么

### 进场的人各自想拿什么

- 每个角色一进对话，都有自己想拿到的东西
- 两个人想要的东西撞上了，张力才会起来
- 两边想要的东西都一样，这场对话多半就只剩复述，没什么意思

### 语气主要看三样

关系 × 场合 × 目的 = 语气

| 场合 | 特点 | 适合内容 |
|------|------|----------|
| 私人（单对单） | 深入、感情流露透彻 | 内心剖白、密谋、表白 |
| 公众 | 需考虑体面，措辞收敛 | 需要冲击力时可打破（如当众翻脸） |
| 熟人（朋友/同门） | 深度介于两者之间 | 轻松互动和信息交换 |

私下才会说的话，偏放在大庭广众下说，冲击力才会更狠。对话里说不尽的东西，就让动作、神态、环境去补。

## 情绪推动对话

### 强情绪对话

- 命令式+否定式最能激发读者情绪："我说的还不够清楚吗？"
- 最强话术：打着为你好的幌子，句句不离关心，但句句都是嫌弃、指责、厌恶
- 直接否定比含蓄暗示更伤人

### 情绪连续性

角色情绪是连续的、循序渐进的。从生气到高兴：生气 → 不那么生气 → 不生气 → 高兴，每次转变都得有对应事件触发。别跳步。

### 情绪四步走（少一步都容易空）

1. 遇到事件（失衡状态）
2. 情绪反应（外在表现：动作、表情、语言）
3. 心里怎么转（哪怕是来不及想，也得把那股莽劲写出来）
4. 采取行动（基于前三步结果回应）

### 想让对话别发平，就往这三边想

- 对话本身带来、或者加重某种主要拉力（期待、爽感、悬念）
- 信息交流因某原因受阻碍，阻碍可能导致驱动力到来
- 发展突然脱离读者预期（但得合理、符合人设）
- 把长篇对话塞在期待点和爽点之间，读者为了等那一下会愿意先跟着看

## 信息和设定怎么顺手带出来

- 大量信息全靠对话往外倒，会显得啰嗦；一部分改成情节、心理、旁白、环境、动作去带
- 用角色自己的口气和立场把信息带出来，别机械念设定
- 设定用到哪个，就稍微带出来，不用事事都把前因后果讲全

### 信息拉扯示例（以"主角新书起飞了"为例）

| 角色 | 台词 | 这句是拿来干什么的 |
|------|------|------|
| 甲 | 听说成绩…… | 悬念拉期待 |
| 乙 | 肯定没人看！ | 下行 + 拉期待 |
| 甲 | 听说成绩很不错 | 上行 + 拉期待 |
| 乙 | 首订一万三？ | 亮出最关键的信息 + 达成爽点 |

上行和下行交替，情绪像拉锯不断拉升期待。

## 每个人说话得分得开

每个角色都要有自己的说话路数。对话时老卡住，不知道角色会说什么，多半就是人设还糊，先去想清这类人平时怎么开口。

| 差异化维度 | 操作 |
|------------|------|
| 口癖和惯用语 | 给每个主要角色一个标志性用词 |
| 说话节奏 | 长篇大论 vs 短句连击 |
| 信息偏好 | 技术型带专业术语，江湖人带切口 |
| 立场固定 | 某角色永远从某个角度发言（悲观派/乐观派/务实派） |
| 身份影响措辞 | 老者/少年/贵族/市井，身份不同则措辞、自称、敬谦词不同 |
| 性格影响语气 | 智谋型话里有话；鲁莽型想到什么说什么；冷静型措辞精确，偶尔情感外露反而有冲击 |
| 进度影响态度 | 初见/熟悉/对立/亲密，关系阶段不同则语气与信息量不同 |

## 弹幕/群众对话

这类对话主要干三件事：把事往前推一点、把期待吊起来一点、把情绪烘热一点。它只是加味，不代替主线。

### 设计过的弹幕 vs 没设计的

- 没设计：只有"卧槽好厉害"——单调、浪费
- 设计过：普通人震惊 → 专业人士分析 → 特殊身份者反应 → 情感升华

### 写的时候记这几下

- 语气要分开，别每条都一个味
- 短一点狠一点，一条里塞最关键的那一下就够了
- 善用递进：从最初震惊到逐渐认识全貌
- 可出现"反转"角色：看似路人一句话改变所有人认知
- 不用每章都写，关键爽点/燃点/泪点前后集中使用

## 对话节奏怎么稳

### 对话多的时候怎么别发闷

- 别为了“精简”把能表现人物性格的语气助词一刀删光
- 对话段落中间插点动作、环境变化、心理活动，把节奏松一松
- 紧张段落对话短促，舒缓段落可以长一些
- 关键信息放对话开头或结尾，中间用于拉扯情绪

### 动作和表情怎么配

- 刻意给每句对话都配表情/动作，会让行文机械
- 动作和表情在关键转折处使用效果最好，不需要每句都配
- 语气平淡场景（喝茶、散步）用微小动作和沉默体现氛围

### 对话也得会换气

- 连续多轮对话后要“换气”一下，插入环境描写或角色心理
- 适当停顿（省略号、动作描写）比连续输出更有张力
- "你确定？"比长篇解释更能形成对峙张力

## 对话长短怎么收

### 对话太多时

- 读者已知信息的对话用叙事一句话概括："爱丽丝向安娜讲述了来城里的原因，安娜听后直皱眉头"
- 能用突发状况替代的对话段落直接替换
- 语气词删掉后还是干巴巴，说明对话本身缺信息量，得重写

### 对话太少时

- 能用其他人物对话讲出来的东西，不要让主角旁白平铺直叙
- 引入配角参与冲突和对话，但新人物得安排主线戏份

## 用梗给对话提味

### 梗式 vs 普通

- 普通："兄弟别灰心，你一步步走到今天我是看着你过来的，这点挫折对于你来说算什么事儿？振作起来！"
- 梗式："兄弟别灰心，我相信你总有人头落地，落地人头，头落地上……卧槽那句话怎么说的来着？兄弟你懂我意思是吧……"
- 梗式用“说不出来但意思到了”的状态制造趣味

### 写的时候这样用

- 在对话中融入梗或骚话，有效提升整体趣味性
- 特别是主角或重要配角的突出对话，适合用梗强化记忆点
- 可用某个梗作为高潮点，整段剧情围绕达成这个梗来设计

## 最后自己过一遍

### 先查三样（中一条以上就得回修）

- [ ] 是否存在大量信息都得靠对话来展示
- [ ] 对话是否是问答式的一问一答
- [ ] 是否习惯依赖对话来推动剧情或人物变化

### 再顺一遍

- [ ] 谁压谁：占上风的人对白 <= 10 字 / 被压住的人 >= 20 字，是否有明确的压制/反转/心死模式
- [ ] 潜台词和各自想拿什么：每个角色一进对话都有自己想拿到的东西，真动机别直接写在台词里
- [ ] 人物说话分不分得开：遮住角色名后，能不能听出是谁在说话（7维差异化）
- [ ] 弹幕递进：普通 → 专业 → 特殊身份，是否有层次感
- [ ] 对话推动剧情：每段对话结束时，剧情是否往前推了一步
- [ ] 篇幅控制：单次对话不超过全节 40%，信息密度是否足够

### 再看一句

- 逐句看对话像不像人真会这么说，别像书面问答稿
- 对话结尾能不能顺手带出接下来节奏要怎么变
