# 对话技法

## 核心原则

每句对话必须完成以下至少一项：
1. 推进剧情
2. 暴露角色性格
3. 改变关系压力
4. 隐藏什么（言不由衷）
5. 间接揭示信息

做不到以上任何一项的对话 = 删。

---

## 潜台词技法

### 话题偏移
回答问题时故意跑题。
- "你昨晚去哪了？" → "今天天气不错。"

### 反问回避
用问题回答问题。
- "你是不是喜欢他？" → "你为什么这么关心？"

### 说别的事
通过谈别的传递信息。
- （实际在说分手）"你看见楼下那家店关了吗？开了三年了。"

### 沉默与动作
不回答就是回答。
- "你还爱他吗？" / 她没说话。把杯子放下了。

### 听到更多
无意中听到的话比当面说更有力。
- 门外："她不知道吧？" / "不知道。别告诉她。"

## 对话节奏

### 快节奏（冲突/争吵）
- 短句，15 字以内
- 不用对话标签（"他说"）
- 允许打断
- 不用描写穿插

```
"你疯了。"
"我疯了？你看看你自己。"
"别转移话题。"
"什么话题？你今天又喝了吧。"
```

### 慢节奏（暧昧/深聊）
- 长句，允许停顿
- 用动作和环境做呼吸
- 话题可以绕弯

```
"你还在想那件事？"
她没回答。窗外的梧桐树被风吹得哗哗响，有几片叶子飘到了阳台上。
"有时候，"她终于开口，"我觉得自己一直在等一个不会来的人。"
```

## 权力动态

对话中的权力关系通过以下信号体现：

| 维度 | 高位者 | 低位者 |
|------|--------|--------|
| 谁提问 | 多问少答 | 多答少问 |
| 谁打断 | 随时打断 | 不敢打断 |
| 谁控题 | 控制话题方向 | 跟随话题 |
| 谁沉默 | 用沉默施压 | 用沉默逃避 |
| 句子长度 | 短，果断 | 长，解释多 |
| 直接程度 | 直说 | 绕弯 |

## 角色语气区分

### 6 种核心说话类型

| 类型 | 特征 | 示例 |
|------|------|------|
| 防御幽默型 | 用笑话保护自己 | "没什么，就是手抖了一下。最近咖啡喝多了。" |
| 直球暴力型 | 不拐弯抹角 | "滚。" |
| 过度控制型 | 话说得很完整，像在做报告 | "根据目前的情况来看，我建议我们分两步走。" |
| 自我取消型 | 先说了又收回去 | "我其实……算了，没什么。" |
| 道德绑架型 | 用道理压人 | "你这样做对得起谁？" |
| 柔性诱惑型 | 声音轻，话里有余地 | "你确定？你再想想？" |

### 操作规则
- 一个角色最多用 1-2 种类型
- 不要给每个角色加口头禅（太假）
- 语气来自社会位置和性格，不是"标签"

## 沉默也是对话

| 沉默类型 | 含义 |
|----------|------|
| 抗拒的沉默 | 不想回答，拒绝配合 |
| 羞辱的沉默 | 让对方意识到自己说了蠢话 |
| 算计的沉默 | 在想怎么回答最有利 |
| 吸引的沉默 | 不说话就是在调情 |
| 恐惧的沉默 | 不敢说 |
| 权力的沉默 | 让你等，让你急 |

---

## 对话禁忌

| 禁忌 | 说明 |
|------|------|
| 所有人说话一个味 | 没有区分度 |
| 每句都有"他说""她说" | 用动作替代标签 |
| 角色把自己的动机说清楚 | 真人不会解释自己 |
| 信息量过大的对话 | 一句话交代太多背景 |
| 连续对话超过 15 轮没有动作/描写 | 读者会飘 |
| 对话中大量感叹号 | 降低情绪密度 |
