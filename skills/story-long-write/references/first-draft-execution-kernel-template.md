# 首稿本章短清单

这一页就是把“整套方法”压成“这一章现在最该盯住的短清单”。
每章第一次写正文首稿前先看它。
它不替代 `章组收益卡 / 动作拍清单 / 人物冲突卡 / 章尾现场句预案 / 写前闸门`，只是先把这些东西压成开写前最该盯住的短清单。

---

## 怎么用

- 开写首稿前先生成。
- 长度控制在 `300-600` 字，或同等信息密度的短表。
- 别写成一大段讲解稿。
- 别出现“文风更自然一点”“注意人味”这种光说不落地的提醒。
- 每条都要是“漏掉了这章就容易写歪”的硬提醒。
- 生成后，正文首稿、首稿回压、写后检查都回看这张清单。
