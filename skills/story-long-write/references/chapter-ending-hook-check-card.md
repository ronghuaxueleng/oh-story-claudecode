# 章尾钩子检查卡

这张卡专门拦“章尾像宣言”“连续两章重复同类钩子”。
每章写后检查前，或者连续章回头看之前，都先过它。

---

## 本章章尾必填

- 本章章尾最后一拍是什么：
- 本章章尾类型：
  - `新物到手`
  - `旧说法被当场掀翻`
  - `有人当场被挤开`
  - `新命令落地`
  - `人物到场`
  - `新证据出现`
  - `战略宣言`
  - `等待下一步`
- 本章章尾即时新结果：
- 本章章尾是否含旁白直接把现场说完的句子：`是 / 否`
- 本章章尾是否含长线战略句：`是 / 否`
- 本章章尾是否和上一章撞型：`是 / 否`
- 若撞型，保留哪一章，删改哪一章：

---

## 下面这种章尾别急着说行

- `更大的风暴要来了`
- `真正重要的是后面那条线`
- `接下来整条线都要……`
- `从这一刻起，他要……`
- 连续两章都用同一高位角色入场
- 连续两章都用“有人到了 / 更高位来了 / 真相要开了”

---

## 这几条先照着拦

- 若章尾类型是 `战略宣言`，一般别急着说行
- 若章尾类型是 `人物到场`，连续两章别重复
- 若最后一句删掉后，现场没有任何新结果，一般就按章尾空看
- 章尾只认“已经发生的新结果”，不认“下一章会发生什么”
