# AI味禁用词与句式表

## 一级禁用词（首稿禁止出现，不是写后再删）

怎么用：
- 本表用于起草前避让和写后扫描
- 禁止把本表词面复制进正文、示例正文、情绪描写或动作描写
- 若某个禁用词因“看过黑名单”而在正文中复现，仍按首稿失败处理
- 口头发力类污染不在主流程显式枚举，避免反向提示；如正文出现已知口头发力污染，按一级禁用词处理

### 情态类
仿佛、犹如、宛若、一丝、一抹、些许、几分、隐约

### 动作类
深吸一口气、缓缓、不禁、微微、轻轻、淡淡

### 力度副词污染
口头发力词（不在主流程显式枚举，避免反向提示）

### 表情类
眼中闪过、嘴角勾起、眉头微皱、眉眼低垂、瞳孔微缩

### 心理类
心中一动、心头一震、心下了然、心中暗道、心底泛起、不由得

### 判断类
不容置疑、不易察觉、显而易见、毫无疑问、不可否认

### 形容类
坚定、闪烁着光芒、狡黠、深邃、凛冽

### 过渡类
不由自主、情不自禁、自然而然

## 二级禁用词（高频出现时替换）

### 语境敏感词（仅高频或偷懒时处理）
突然、好像、瞬间（角色口语、真实突发、时间压缩、视角不确定时可保留）
猛地、死死、骤然（单章高频重复或拿来顶替具体动作时处理）

### 总结句式
- "他/她终于明白..."
- "他/她这才意识到..."
- "此刻，他/她..."
- "一切...都..."
- "原来..."
- 包含 `原来` 字面的叙述短语也尽量避开；需要表达旧顺序 / 旧位置时，改写为“旧序 / 先前 / 原先位置”等更具体说法

### 排比句式
- 连续3句以上相同结构的排比
- "有的...有的...有的..."
- "一边...一边...一边..."

### 升华句式
- "这一刻..."
- "他知道..."
- "她明白..."
- "这就是..."

### 提纲术语（正文禁用）
- "第一步现实变化..."
- "第二步现实变化..."
- "下一步该看..."
- "这一段现实变化..."
- "哪一步现实变化..."
- "现实变化这一层..."
- "收益这一层..."
- "资源这一层..."
- "权力这一层..."
- "局面这一层..."
- "活路这一层..."
- "接口物"
- "解释权入口"

就记这句：
- 以上词只允许留在 `卷纲 / 章组卡 / 细纲 / 闸门 / 检查单 / 追踪复盘` 这类工作文件里，拿来给自己记事。
- 正文、示例正文、这类直接拿来就能用的句子示例、章尾收尾那一句里一律不用。
- 真要写进故事，就别写抽象作用，直接写现场东西：`哪张簿子 / 哪块牌 / 哪条命 / 哪班轮值 / 哪批药 / 哪扇库门 / 哪个缺口`。

## 禁用句式

| 句式 | 示例 | 问题 |
|------|------|------|
| "...，带着..." | "他说，带着一丝无奈" | 万能状语 |
| 陈词滥调/万能比喻 | "像刀子一样锋利" | 公式化比喻；生活化、角色化比喻可保留 |
| "他的声音很轻，却像..." | AI最爱声音描写 | 过于文艺 |
| "仿佛能...一般" | "仿佛能穿透一切一般" | 文言腔 |
| 对话标签密度过高/公式化标签 | "好的，他说道" | 普通“说”可保留；高频或公式化时处理 |
| "他/她感到..." | "她感到一丝失落" | 告诉而非展示 |
| "他/她意识到..." | "他意识到事情不对" | 直接告知 |
| "眼中闪过一丝XX" | "眼中闪过一丝悲伤" | 套话味重 |
| "嘴角勾起一抹XX" | "嘴角勾起一抹冷笑" | 套话味重 |
| "心中涌起一股XX" | "心中涌起一股暖流" | 套话味重 |

## 替换策略速查

| 原文类型 | 替换方法 | 示例 |
|----------|----------|------|
| 抽象情绪词 | 具体动作 | "紧张"->"手在抖" |
| "感到XX" | 外化表现 | "感到愤怒"->"攥紧拳头" |
| 形容词堆砌 | 白描手法 | "美丽动人的笑容"->"她笑了" |
| 书面表达 | 口语化 | "不容置疑"->"就是" |
| 解释性描写 | 留白 | "他因为害怕而..."->"他退后一步" |
| 连续排比 | 保留最强一条 | 3句排比留1句 |
| 总结升华句 | 直接删除 | "这一刻，她终于明白了..."->删 |
