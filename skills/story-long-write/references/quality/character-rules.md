# 人物规则

> 本文件负责人物显影、主角声线、配角工具嘴、高价值角色魅力。

---

## 一、首次强出场最低配额

以下角色默认纳入强制显影：

- 主角
- 核心反派
- 核心盟友
- 高价值角色

首次强出场至少落 `2` 项：
- 外形锚点
- 动作锚点
- 说话锚点
- 气场锚点

---

## 二、连续登场要求

- 连续登场章节，每章至少复现 `1` 项稳定锚点
- 删除角色名后，若台词和动作分不出是谁，按显影失败处理
- 连续登场角色不得只重复功能位；每 `2-3` 章至少补出一次 `私欲 / 偏见 / 旧账 / 自保习惯 / 贪心方向` 中的任一项
- 若角色连续多章只会“提供信息 / 执行动作 / 站队表态”，没有个人算盘，默认人物层发空

---

## 三、主角声线

主角组与核心配角必须做台词边界检查：
- 禁止同一种“短句下令 + 发狠 + 收口”说法泛化到多人
- 主角至少要有一个稳定动作签名或说话签名
- 主角不得长期靠“工整金句 / 宣判句 / 结论句”立声线
- 若主角在高压场景里总能把话说得过满、过准、过漂亮，默认不是声线稳，是人味流失

### 三点一、语言指纹硬要求

- 核心角色至少区分以下两项：
  - 句长习惯：短句 / 碎句 / 整句 / 含混句
  - 发力方式：抢断 / 岔答 / 装稳 / 阴压 / 明骂 / 软求
  - 用词范围：粗口、行话、职业口、上位口、含糊口
  - 遇压反应：先护物、先撇责、先沉默、先反咬、先找台阶
- 若删去人名后，多名角色对白仍能互换，默认人物显影不过线
- 若高位敌人和主角都在说同一种“漂亮狠话”，默认人物层不过线
- 若多人都长期使用同一套 `制度口 / 书面口 / 公告口` 说话，默认人物层不过线
- 角色可以有职业口，但职业口不能压过人话；读者先该听懂“他要干什么”，再顺手知道“这行话叫什么”

### 三点二、人物私欲显影硬要求

- 关键角色出场时，默认都要回答：
  - 这人眼下最想保什么
  - 这人最舍不得丢什么
  - 这人愿意先牺牲谁 / 什么来换自己稳住
- 若细纲 / 正文只写“他想赢 / 她想活 / 他要报复”这类总目标，不算私欲显影；必须落到眼前这一场的具体护点
- 关键角色每章至少有 `1` 次动作或台词暴露个人算盘，哪怕与当前阵营口径不完全一致
- 若盟友永远无条件配合、对手永远只为卡主角服务、摇摆位永远按剧情需要摇摆，默认人物层不过线

### 三点三、对话抢势硬要求

- 对话不是信息搬运。关键冲突段里，至少 `1` 轮对白要同时满足：
  - 一方想压人
  - 一方想岔开或反咬
  - 结果真的改了在场动作、站位、口令或资源归属
- 禁止把“解释规则 / 复述局势 / 总结意义”伪装成对话冲突
- 若一段对白删掉后，人物关系、权力高低、场面推进都不变，说明这段对白只是口播，不算有效冲突
- 高压场景对话优先使用：
  - 抢断
  - 岔答
  - 回避关键点
  - 借在场物压人
  - 拿旧账换当前口风
- 不得让所有角色都说完整、准确、漂亮、逻辑自洽的人造台词；有人会嘴硬、漏话、打断、含混、临场改口，这才像人在抢势

### 三点四、对白张嘴测试硬要求

- 关键对白写完后，默认再做一次“张嘴测试”：
  - 念出来是否顺口
  - 是否像人在急场里会说的话
  - 是否一口气塞进了太多概念、术语、判断
- 若一句话需要读者停半拍翻译、改写成白话才能顺，默认不过线
- 若一句对白主要靠制度词、抽象词、术语词支撑，而删掉这些词后角色意图说不清，说明这句不是人话，是口播
- 高压场景里，优先保留：
  - 能直接听懂的短句
  - 骂人、催人、压人、求活时的碎句
  - 半句里直接带动作或代价的说法
- 不得优先保留：
  - 会议纪要腔
  - 制度宣讲腔
  - 书面汇报腔
  - 逻辑总结腔

---

## 四、配角工具嘴

默认至少 `F1` 的情况：
- 只在关键道具 / 关键信息出现时突然出声
- 台词主要负责认出、确认、抬价、解释
- 前文无站位、动作、反应铺垫
- 说完即退场

---

## 五、高价值角色

帅哥 / 美女 / 危险合作对象 / 强对手：
- 不能只在设定里说好看
- 必须在正文里给到可见细节
- 至少一个外形细节
- 至少一个动作或姿态细节
- 至少一个带身份味的说话细节
