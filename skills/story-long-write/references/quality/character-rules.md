# 人物怎么才不发虚

> 这份主要盯人物出场分量、主角声线、配角工具嘴、重要角色魅力。

---

## 一、首次强出场最低配额

以下角色开场都得亮出来：

- 主角
- 主要反派
- 主要盟友
- 重要角色

首次强出场至少落 `2` 项：
- 外形记忆点
- 动作记忆点
- 说话记忆点
- 气场记忆点

补充：

- 若属于重要角色，首次强出场别只满足最低 `2` 项
- 重要角色首出场至少同时落出 `3` 项，且其中至少 `1` 项不能只是外形 / 好看 / 危险标签
- 推荐组合：
  - `动作 + 私欲 + 压住别人后别人怎么变`
  - `外形 + 动作 + 私欲`
  - `动作 + 说话记忆点 + 压住别人后别人怎么变`

---

## 二、连续登场要求

- 连续登场章节，每章至少复现 `1` 项稳定记忆点
- 删除角色名后，若台词和动作分不出是谁，按亮相失败处理
- 连续登场角色不得只重复办事作用；每 `2-3` 章至少补出一次 `私欲 / 偏见 / 旧账 / 自保习惯 / 贪心方向` 中的任一项
- 若角色连续多章只会“提供信息 / 执行动作 / 被逼选边表态”，没有个人算盘，就算人物这一层发空了

---

## 三、主角声线

主角组与重要配角都要做台词边界检查：
- 别把同一种“短句下令 + 发狠 + 收尾”说法铺到多人头上
- 主角至少要有一个稳定动作签名或说话签名
- 主角别长期靠“像提前写好再照着念的书面话 / 宣判句 / 结论句”立声线
- 若主角在高压场景里总能把话说得过满、过准、过漂亮，那通常不是声线出来了，而是人味没了

### 三点一、说话得听得出是谁

- 关键角色至少区分以下两项：
  - 句长习惯：短句 / 碎句 / 整句 / 含混句
  - 发力方式：抢断 / 岔答 / 装稳 / 阴压 / 明骂 / 软求
  - 用词范围：粗口、行话、职业口、上位口、含糊口
  - 遇压反应：先护物、先撇责、先沉默、先反咬、先找台阶
- 若删去人名后，多名角色对白仍能互换，就先别算人物出场分量到位
- 若高位敌人和主角都在说同一种“整齐过头的压人话”，就先别算人物这一层到位
- 若多人都长期使用同一套 `制度口 / 书面口 / 公告口` 说话，就先别算人物这一层到位
- 角色可以有职业口，但职业口不能压过人话；读者先该听懂“他要干什么”，再顺手知道“这行话叫什么”

### 三点二、人物得有自己的那点心思

- 关键角色出场时，都要回答：
  - 这人眼下最想保什么
  - 这人最舍不得丢什么
  - 这人愿意先牺牲谁 / 什么来换自己稳住
- 若细纲 / 正文只写“他想赢 / 她想活 / 他要报复”这类总目标，不算私欲亮相；得落到眼前这一场的具体护点
- 关键角色每章至少有 `1` 次动作或台词暴露个人算盘，哪怕与当前阵营说法不完全一致
- 若盟友永远无条件配合、对手永远只为卡主角服务、摇摆位永远按剧情需要摇摆，就先别算人物这一层到位

### 三点三、对白得真能压到人

- 对话不是信息搬运。关键冲突段里，至少 `1` 轮对白要同时满足：
  - 一方想压人
  - 一方想岔开或反咬
  - 结果真的改了在场动作、谁往前谁往后、口令或资源归属
- 禁止把“解释规则 / 复述局势 / 总结意义”伪装成对话冲突
- 若一段对白删掉后，人物关系、权力高低、场面推进都不变，说明这段对白只是在把意思念出来，不算有效冲突
- 高压场景对话优先使用：
  - 抢断
  - 岔答
  - 回避关键点
  - 借在场物压人
  - 拿旧账换当前说法
- 别让所有角色都说完整、准确、漂亮、前后全圆上的人造台词；有人会嘴硬、漏话、打断、含混、临场把话吞回去，这才像人在场上顶着说

### 三点四、张嘴念一遍别别扭

- 关键对白写完后，再做一次“张嘴测试”：
  - 念出来会不会别扭
  - 是否像人在急场里会说的话
  - 是否一下塞进了太多概念、术语、判断
- 若一句话需要读者停半拍翻译、改写成白话才能顺，就先别算过
- 若一句对白主要靠制度词、抽象词、术语词支撑，而删掉这些词后角色意图说不清，说明这句不是人话，只是在把意思念出来
- 高压场景里，优先保留：
  - 能直接听懂的短句
  - 骂人、催人、压人、求活时的碎句
  - 半句里直接带动作或代价的说法
- 别优先保留：
  - 会议纪要腔
  - 制度宣讲腔
  - 书面汇报腔
  - 前后给你总结完的腔

---

## 四、配角工具嘴

至少记 `F1` 的情况：
- 只在关键道具 / 关键信息出现时突然出声
- 台词主要负责认出、确认、抬价、解释
- 前文没人往前站、往后退，动作和反应也没铺垫
- 说完即退场

---

## 五、重要角色

帅哥 / 美女 / 危险合作对象 / 强对手：
- 不能只在设定里说好看
- 正文里得给到看得见的细节
- 至少一个外形细节
- 至少一个动作或姿态细节
- 至少一个带身份味的说话细节

### 五点一、首出场分量

- 重要角色首出场若只做到：
  - 一个动作
  - 一句功能性改线对白
  - 一次有人往前站、往后退
  这最多只算 `基本成立`
- 想过“读者能顺手想出这人的样子”这条线，首出场还得再多出至少一层：
  - 具体私欲
  - 具体怕失去什么
  - 压住别人后别人怎么变
  - 与在场其他人的非对称信息
- 若角色卡已经写明这些层次，正文首出场却没落下来，先当成“首出场分量不够”，别先算成角色卡不足

### 五点二、首出场声线去同质

- 重要角色首出场不得反复复用同一类压人句法：
  - `既然先X，就别先Y`
  - `乱不乱，等你认稳再说`
  - `先试完，再说谁该闭嘴`
- 这类句法可以偶尔用，但跨角色高频复用，会削弱区分度
- 同一批冷启动首章组里，若多个重要角色都靠同一套压人句法起手，就按“声线同质”看
- 声线去同质时，先改：
  - 句长
  - 发力方式
  - 用词范围
  - 先护的东西
  - 话先落在人、物、账、规矩里的哪一处

### 五点三、看到哪一步才算写出来了

- `基本成立`：
  - 读者能看出这是个重要角色
  - 但还脑补不出更多层
- `读者能顺手想出这个人的样子`：
  - 读者已经能从首出场同时抓到：
    - 这个人护什么
    - 这个人怎么压人
    - 这个人和别人哪里不一样
- 若重要角色走的是“先压自己人 / 自己物 / 自己名分，再开口压住别人”这一路，还得继续往下问：
  - 压进去之后，她/他自己当场失去了什么
  - 这次失去是否具体落成 `实际位置 / 先后次序 / 面子 / 名分` 之一
  - 这次失去是否立刻逼出别人把话咽回去 / 往前站或往后退 / 改先后 / 改权限
- 若只有“先压自己人”而没有真实失去，那最多只算“比只会出来办事厚一点”，别直接算到 `读者能顺手想出这个人的样子`
- 若只剩“设定上很重要”，正文没有额外一层私欲或压住别人后别人怎么变，那就过不了 `读者能顺手想出这个人的样子` 这条线
- 若重要角色已经写到下面这层，别急着直接往高了算：
  - `先压自己边`
  - `本人立刻失去实际位置`
  - `后续稳定权下滑`
  - `自己人开始不再替她/他说圆场话`
  仍得继续往下看，这些后果是不是长在该人物身上，而不是机械套模板：
  - 她/他失去的是否真是“这个人最护、最舍不得丢、最会偏着护的那一口”
  - 那个先不再替她/他说圆场话的人，是否和她/他的既有控制链相关，而不是随便抓一个路人站出来把话收回去
- 若换一个同职能角色也能原样套用，就记为“路子没错，但这个人还没写出自己的样子”
- 这种情况最多只算“刚搭起来”，别顺势把人物抬得太高

## 六、主角自己的味道

- 主角若想稳定脱离“只会办事、没自己味道的主角”，不能只做到：
  - 固定判断顺序
  - 旧伤动作
  - 冷口报后果
- 还得继续往下问：
  - 这套看事办法是不是只属于该题材 / 该场 / 该主角本人
  - 这句往下压的说法里是否同时压住了主角自己的 `私利 / 代价 / 现实改局`
  - 删掉角色名后，读者还能不能认出“这就是他会说的话”
  - 如果把同一句换给同书另一个会办事的人，是否立刻失真
  - 他会不会给“将坏未坏”的状态起一个只属于自己的名字
  - 这个名字是否长在本题材常见的东西和场面上，而不是抽象判断词
- 若只做到“有效推进、报哪口先坏、逼对手接盘”，但答不出上面至少 `2` 项，那仍只算“刚脱离纯办事说话”，别直接写成“主角自己的味道已经出来了”
- 主角自己的味道，至少要同时满足：
  - 固定坏点识别顺序
  - 身体旧伤 / 习惯动作真实回落正文
  - 一句绑在本题材常见东西上的个人判断句
  - 该句同时压住 `私利 / 代价 / 现实改局` 中至少 `2` 项
  - 该句不依赖单一旧走法，换走法后仍能成立
  - 至少 `1` 次给“将坏未坏”的状态起出只属于该主角的题材化名字

## 七、更高层人物上限

- 重要角色若已经写到“先伤自己一下，后面连身边人都不再替她/他说话”这层，不得立刻停在：
  - `撤签`
  - `失权`
  - `身边人开始不再替她/他说圆场话`
- 还得继续往下问：
  - 她/他真正没丢掉的那样东西是什么
  - 这东西是否比表层 `签 / 权 / 名分 / 面子` 更值钱
  - 这东西是否具体落成 `秩序 / 手感 / 她/他真正压得住人的那一层 / 更深一层的实权` 之一
- 若正文只能证明“这个人吃了亏”，却答不出“这个人真正还握着什么”，那仍只算“路子刚搭起来”，别顺势把人物抬得更高
- 若主角已经能拆掉对方表层护口，就继续追问：
  - 他能不能进一步点穿对方更深一层真正护的那口
  - 这句是否比前一层“拆假稳口”更不可替换
