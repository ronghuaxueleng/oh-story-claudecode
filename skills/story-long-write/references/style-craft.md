## 写正文先盯这四下

### 第一件：“然后呢”得接住
每一段都得接住读者心里的“然后呢”。一个信息点写完，马上用下一个信息点接上。

### 第二件：别老正面硬说
正面描写很多时候只是铺垫，侧面反应才更容易出爽点。写配角嚼饼、哄抢，比干巴巴说“这饼宇宙第一”有用得多。

### 第三件：围着这本书最想给人的那股劲写
动笔前先定这本书最想让读者爽在哪、疼在哪、上头在哪（被认可/复仇/恋爱/升级），后面的情节、人设、冲突都别离这股劲太远。

### 第四件：大处小处一起盯
宏观看整体节奏和情绪走向，微观看每段里的细节和张力。

## 基础写法

### 白描
白描就是少说废话，把信息和情绪一把递到位。动词名词越准，画面越容易立起来。

### 五感怎么带
- 尽量带上两到三种感官（视觉/听觉/触觉/嗅觉/味觉）
- 别只写“看到了什么”，也写“人是怎么感到的”
- 五感得给情绪服务——怕的时候写冷、暗、静；兴奋时写亮、快、响

### 动起来通常比干摆着强
- 人物特征尽量用动作和反应带出来，别靠形容词堆
- 示范：众人看到她时，说话的声音不自觉地放轻了
- 环境不要大段铺陈，在角色行动中穿插点染

### 对话先盯这几下
- 每句话最好都顶一件事：推进剧情 / 展示人设 / 制造冲突
- 用语气、动作、省略来暗示潜台词
- 日常用语，别写成书面腔

### 场景切换怎么别生硬
- 用相似物/相似五感/相似情绪做转场
- 时间跳转：动作或物件衔接
- 空间跳转：声音或光影衔接

### 镜头和分镜怎么想
- 每个段落 = 一个镜头，有明确的"拍摄对象"
- 镜头大致这么切：远景（环境/氛围）-> 中景（人物关系）-> 近景（表情/细节）-> 特写（关键物品/情绪）
- 快节奏：短句、短段、密集动作、快速切换
- 慢节奏：长句、环境描写、心理活动、静止镜头

### 代入感怎么立
- 主角的目标清晰且让读者认同
- 主角的行为读者能理解动机
- 叙事立场得站在主角这一边

### 往人物挖，还是往故事里挖
- 向故事深化：复杂阴谋、多层推理——上限高但难度大
- 向人设深化：人物关系和互动中找戏剧性——更容易上手
- 最好让读者觉得“这几个人凑一块就有戏看”

### 聪明人别写成旁白嘴替
- 用行为把聪明写出来，别靠旁白替他说
- 借真实案例时，拿走最值钱的那条走法，再换成你这本书自己的场景和细节

## 行文最后盯两件事

### 第一件：顺

| 卡在哪 | 怎么顺过去 |
|----------|------|
| "了"字卡顿 | 控制不必要的"了"字使用频率 |
| 玩梗卡顿 | 只用目标读者群体熟悉的梗 |
| 信息缺失卡顿 | 适时补上必要信息 |
| 伏笔过远卡顿 | 揭示前简短提示前文 |

### 第二件：准
什么情绪就写什么味，该热血就热血，该温暖就温暖，该恐怖就恐怖。

| 情绪 | 最要紧的点 | 反面信号 |
|------|------|----------|
| 悲壮 | 牺牲要有分量，代价要真实 | 角色死了读者无感 |
| 温暖 | 日常细节中的关心 | 刻意煽情 |
| 恐怖 | 未知和失控感 | 靠血腥描写堆砌 |
| 热血 | 宣言感 | 口号空洞无铺垫 |

## 镜头式写法

先忘掉全知视角。你就是跟在主角身后的摄像机，读者也只该看到主角看到的东西。

### 先看这几句
1. 只写能看到/听到/闻到/摸到的，少写“他心里想”“他感到”
2. 每句对话都要推进剧情、塑造人物、或传递信息
3. 动作代替描述："他很紧张" -> "他的手在抖"
4. 手机上看文为先：多用短句，按动作和信息变化换行，读起来一觉得卡就拆段

### 容易翻车的地方

**这些视角容易写飘**
- 全知叙述："此时的他还不知道，三天后..."
- 上帝视角："如果她知道真相..."
- 心理分析："他内心深处其实..."

**这些词一出来就容易像套模板**
仿佛、犹如、一丝、一抹、深吸一口气、缓缓、不禁、眼中闪过、嘴角勾起、眉头微皱、不容置疑、不易察觉（“好像”按语境判断，不机械替换）

**这些句式也容易写僵**
- "...，带着..."（如：他说，带着一丝无奈）
- 对话标签太密/太像套话 -> 写多了就用动作神态替一下，普通“说”留着也行
- "他/她知道..." -> 直接展示，不告诉
- 陈词滥调/万能比喻 -> 改成生活化比喻或白描

### 可以往这边改

| 禁用 | 替代 |
|------|------|
| 他感到紧张 | 他的手在抖 |
| 她很伤心 | 她转过身，肩膀微微颤动 |
| 高频公式化对话标签 | 用动作、语气或上下文接过去；普通“说”可以留 |
| 他深吸一口气 | 他把烟摁灭在烟灰缸里 |
| 她眼中闪过一丝惊讶 | 她的手停在了半空 |

### 对话别写成套话
每句对话最好都得占一头：推进剧情 / 塑造人物 / 制造张力

**对话格式**
- 用动作打断长对话
- 每次说话不超过3句
- 别老堆高频套话型对话标签；普通“说”可以留，真写得机械了再用动作或上下文替

**对话情绪五级递增**
1. 客观陈述事实
2. 客观陈述 + 提出建议
3. 主观指责
4. 主观指责 + 强制命令
5. 主观指责 + PUA抬升自己（最强）

### 三机位怎么切

| 机位 | 视角 | 主要拍什么 | 这一机主要拿来干什么 |
|------|------|------|------|
| 机位1 | 近景 | 主角动作、表情、心理 | 推进主线，读者代入 |
| 机位2 | 远景 | 配角反应、环境变化、围观者 | 扩展信息面，制造反差 |
| 机位3 | 旁白 | 设定补充、背景交代 | 适时补充必要信息 |

机位切的时候，主要靠 1 和 2 往前顶，3 只拿来补非补不可的东西。设定最好顺着冲突往外带。

## 文本里到底有哪些东西

### 先把这几样分清
- **画面感**：人+环境（静/动），如高清摄像机忠实记录
- **思想**：计划/目标/吐槽/揣测/分析等内心活动
- **情绪**：藏在画面感与思想之中，不直接写出来
- **旁白**：身世背景/设定/人物关系/回忆，尽可能简洁

### 情绪从低到高大概这么写
1. 最低：直接写"xx愤怒/伤心/快乐"——告诉情绪，读者感受不到
2. 低：用思想"xx暗骂"
3. 中：用神态"xx脸色铁青"
4. 高：用动作"xx攥紧拳头，指甲一点点压进掌心"

先记这句：别直接把情绪喊出来，越能从外面看见，读起来越顺。

### 环境描写别空摆着
没有交互的环境描写，通常就不值当写。
- 物理交互：梨花飘到鼻尖打喷嚏 / 雪山瑟瑟发抖 / 河水洗脸
- 精神交互：食物比喻人物关系 / 喂饭=暧昧 / 餐刀切鱼=威胁
- 题材交互：仙侠写遮天蔽日鲲鹏 / 武侠写匹夫一怒血溅五步

## 文笔主要看这三样，再顺手把水挤掉

### 文笔最要命的三样
- **精准**：一个词能说清的不用一句话，动词选择决定画面感
- **节奏**：句式长短交替，连续三句短句后接长句，反之亦然
- **画面感**：调动五感，不只写"看到"

### 哪些地方是在灌水
- 水分 = 不推动剧情也不塑造人物的内容
- 合理的"水"：伏笔铺垫/氛围营造/角色互动——暗流涌动
- 有害的水：重复已知信息/无意义环境描写/角色自言自语
- 检验法：删掉这段后读者会不会困惑；不会，多半就是水

### 长篇网文里文笔怎么取舍
长篇不一定要精致文笔，但一定要可读、要顺。最好让读者忘记自己在看字，直接看到画面。

## 文笔和环境怎么一起发力

### 写到不同档，文笔占的分量也不一样

| 这一步 | 分量大概有多重 | 大概说明 |
|------|--------|------|
| 小学生文笔 | 10分 | 基础不过关，什么都谈不上 |
| 基础达标 | 5分 | 过线以后，短板多半在故事怎么搭和题材怎么选 |
| 精品级别 | 7分 | 精品之间文笔开始拉开差距 |
| 万订 | 10分 | 顶级作品的文笔本身就能拉开差距 |

判断：成绩好但文笔差 -> 提升文笔性价比极高；成绩差但文笔好 -> 补选材和剧情。

### 一笔多用
一段文字最好能同时干几件事：
- 环境交互 + 人设 + 感情线 = 一笔三用
- 设定融入情节：世界观塞进小故事里带出来
- 埋伏笔：表面信息之下藏隐藏信息
- 紧凑节奏：枯燥背景设定藏在有趣小情节中

### 故事一般怎么一层层往下落
所谓“x级结构”，其实就是一层层往下拆：一句话主线 -> 卷纲 -> 章纲 -> 正文。很多时候倒着推，比顺着铺更省力。

## 文笔这事到底看哪几样

### 先看这三样是不是对的
- **题材怎么选**：目标读者是谁，这书大概写哪一口，主打卖点清不清楚。
- **剧情怎么排**：大纲、卷纲、章纲能不能一直把那股情绪往上拱。
- **文笔落没落地**：正文是不是拿动作、对话、物件和节奏把情绪写出来了。

如果正文的问题根子在题材没选准，或者剧情本身没排好，就别只顾着抹句子，回到前面的文件去改。

### 情绪大概可以这么往上写

| 轻重 | 大概会怎么写 |
|------|------|
| 最低 | 直接写"xx愤怒" |
| 低 | 用思想"xx暗骂" |
| 中 | 用神态"脸色铁青" |
| 高 | 用动作"攥紧拳头，指甲一点点压进掌心" |

先记这句：别直接报情绪，去写情绪把人逼成了什么样。

## 视角和叙事怎么选

### 网文里常用的几种视角

| 视角 | 特点 | 一般更适合哪类 |
|------|------|------|
| 第三人称有限 | 跟随主角视角 | 最常用，代入感最强 |
| 第一人称 | "我"叙述 | 悬疑、日常文 |
| 第三人称客观 | 只写外部行为 | 群像 |
| 跳跃视角 | 不同角色间切换 | 制造信息差 |
| 上帝视角 | 全知全能 | 传统文学，网文少用 |

### 用的时候记这几下
- 网文里通常先用第三人称有限视角
- 跳跃视角主要用于制造"读者知道而主角不知道"的信息差
- 切换视角要有明确分隔标志
- 第一人称慎用——限制信息量，后期展开困难

### 抽象和具体怎么拿捏
- 套路是共性，每本书的个性才是读者留下来的理由
- 写意思维：重在神韵不在形似，该留白的时候留白
- 好的网文文笔：粗看大白话，细看有水平

## 容易翻车的地方怎么躲

### 五种常见翻车方向

| 容易翻在哪 | 怎么躲 |
|------|----------|
| 主角塑造 | 叙事立场站在主角一侧；主角做事必有正反馈 |
| 题材雷点 | 了解目标读者群体，避开敏感话题 |
| 氛围代入感 | 氛围感比梗重要一百倍（乐子文除外） |
| 期待破坏 | 把握节奏，适时满足或引入新期待 |
| 题材选择 | 先参考三五千均订的作品，别只盯顶流 |

### 先记这几句
- 主角得是你自己先护着的人：叙事立场就站在主角这边
- 明确读点：毒点随题材变化，讨好目标受众而非所有读者
- 期待这事得顾着点：连续几次都把读者想看的东西打掉，人就不信你了
- 开头简洁：第一章就展现关键要素

## 黄金三章：先把调子和期待立住

黄金三章 = 验证"实物是否与图片一致"，快速回应读者期待。

### 最常见的一条走法
令人不安的处境 + 亟待提升的地位 + 充满希望的外挂 + 能被展望的美好未来

### 定调时一般看三层
1. 情绪氛围
2. 故事走向
3. 价值观倾向

### 常见两种起手法
- 代入感路线：快速代入主角
- 期待感路线：开头创造危机

记住：别为了硬凑黄金三章，把整本书的基调写坏了。

### 想把人留下来，先别让期待断掉
- 主角即将得到但还没得到 -> 期待感最高 -> 追读最多
- 得到之后追读立刻下降 -> 解法：得到之前套上另一个钩子 -> 循环
- 一本书最好随时挂着两条以上期待线

## 最后自己对一遍

每段每章写完后，对一遍：

- [ ] **段落控制**：按动作/信息变化断段，读起来不卡
- [ ] **高危词扫描**：少用"仿佛/犹如/一丝/不禁/眼中闪过/嘴角勾起"这类一看就像套话的表达
- [ ] **高危句式扫描**：公式化对话标签、`他感到`、`带着一丝XX` 这类别用太多
- [ ] **情绪展示**：没有直接写"他愤怒/伤心/紧张"，全部用动作/神态替代
- [ ] **对话有没有用**：每句对话最好都占推进剧情、塑造人物、制造张力里的一个
- [ ] **五感有没有带出来**：每个场景最好带出两种感官
- [ ] **环境交互**：环境描写有角色交互，不是空镜头
- [ ] **镜头感**：每段有明确拍摄对象，远景/中景/近景/特写交替
- [ ] **水分检查**：删掉任意一段后读者是否困惑——不困惑则该段多余
- [ ] **视角一致**：没有全知叙述/上帝视角/心理分析混入
- [ ] **节奏有没有换气**：长短句有交替，连续 3 个短句后最好接长句，快慢场景别写成一个速率
- [ ] **代入感**：主角目标清晰、行为可理解、叙事立场站在主角一侧
