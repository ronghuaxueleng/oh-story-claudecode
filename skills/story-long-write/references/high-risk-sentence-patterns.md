# 高危句型黑名单

这页专门拦“结构还没坏，但句子已经先把人物口条和叙述质感写坏了”的高频句型。  
一般在首稿、写后检查、协同即时返修时用。  
先这么看：碰上不等于一律删除；但如果删掉这类句型后，人物、动作、后果还成立，那就先删。

## 一类：作者下结论型

- `不是……而是……`
- `真正……的是……`
- `最重要的是……`
- `最要命的是……`
- `因为这……`
- `这意味着…… / 这说明…… / 这证明…… / 这就说明……`

这类句子麻烦在哪：
- 先替读者总结，再让人物或场面补票
- 容易把正文写成检查单口气
- 常压在章尾、第二步收益、外部反应位，直接削弱现场感

常见改法：
- 改成动作、物证、失态、把话吞回去、被挤开
- 若该句扛着章尾最后一拍，先把句子前一拍抬成真正收尾点

## 二类：两拍判词句

- `火籍房不怕……怕的是……`
- `他不是来……的，他是来……的`
- `这不是狠，是老`
- `这不是认输，是最后一下反压`

这类句子麻烦在哪：
- 句子自身比人物更会说
- 容易把场面压成作者判词
- 经常伪装成“很短，所以不算解释腔”

常见改法：
- 先删第二拍判词，看前一拍场面是否已经写出来
- 若没写出来，就补动作和对象；若已写出来，直接删判词

## 三类：排比抒情台词

- `带着什么去？带着……还是带着……`
- `因为穷，因为断供，因为没人……`
- 连续两拍以上整齐对称句
- 岗位/对象/苦处列表式台词

这类句子麻烦在哪：
- 角色会被写成专门替作者把话说顺的人
- 老角色、病角色、守旧角色最容易失真
- 台词就算听着顺，也不等于人物像本人

常见改法：
- 收成短句
- 保留对象和怕处，删修辞走法
- 先查是否违背角色卡里的“字少，句重 / 先咽气再吐重话 / 先护什么”

## 四类：旁白把人物该露出来的反应提前说掉

- `他比谁都清楚……`
- `因为一旦……就……`
- `谁都知道……`
- `这话比什么都硬`
- `围观不是刀，可围观会把刀磨快`

这类句子麻烦在哪：
- 旁白提前替人物和场面做判断
- 人物真实反应被作者概述吃掉
- 多发在首章前半和高潮后半拍

常见改法：
- 改成人物的身体反应、停顿、把原话吞回去、视线、动作
- 改成第三方的现实反应

## 首章加严

如果第 1 章前 `500-800` 字里连续撞上下面任意 `2` 项，最少记 `F1`：
- 熟比喻抢戏
- `不是……而是……`
- 两拍判词句
- 排比抒情台词
- 旁白把人物该露出来的反应提前说掉

## 什么情况下可以先不删

下面这些情况可以人工驳回，不按失败处理：
- 角色口条本身就设定为利口、整句成刀，且全文稳定一致
- 句型虽短，但前后动作记忆点扎实，删掉会损失人物独有的压人方式
- 句型未承担解释功能，只承担节奏断拍，且没有替场面下结论

## 碰上了就顺手记一下

碰上本卡时，写后检查别只写“语言还行”。最少把这些记下来：
- 碰上的句型类别
- 该句是 `人物失真`、`作者先替现场做判断`、还是 `章尾下结论`
- 采用 `删句型 / 改动作 / 改物证 / 改角色口条` 中哪一种修法
