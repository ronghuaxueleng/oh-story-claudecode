# 反作者腔写前处理法

这页不是给你写完后挑错的，是让你**写前就把解释腔、作者腔、旁白补缝句尽量压下去**。
每章第一次写正文首稿前，以及首稿写完后的第一轮人工回压，都看它。

先看这一条（只借方法，不借原话）：
- `Scene Card / Story Card`：先把一场戏拆成可执行拍序
- `Goal / Conflict / Disaster`：先写目标、阻断、翻坏
- `MRU`：先写外部刺激，再写人物反应，不让旁白抢跑替读者解释
- `Authorial Intrusion Pass`：专查“作者帮读者翻译”的句子，再改成现场东西

---

## 一、写前预防：先把“解释意图”改写成“场面任务”

如果你脑中先冒出来的是这类想法：
- 这里要体现主角聪明
- 这里要体现韩烈心虚
- 这里要体现人心开始偏了
- 这里要让读者知道事情没那么简单

先别直接进正文。

先改写成以下任一类任务：
- 谁先动手 / 先开口 / 先摆证物
- 谁被逼着把话咽回去 / 往旁边退半步 / 改顺序
- 谁失态 / 谁躲眼神 / 谁不敢接话
- 哪个物证被摆出来
- 哪条规矩当场换了个说法
- 哪一群人开始改看法

先记这个意思：
- 不写“要表达什么”
- 只写“谁做了什么，导致什么变了”

---

## 二、场景最小承重结构

每个高风险场景，最少先写出这 5 项：

1. `起手目标`
- 主角这场想拿什么

2. `首轮阻断`
- 谁拦，怎么拦

3. `局面翻坏`
- 原打法为什么不够了

4. `被迫改手`
- 主角换成什么更狠 / 更险 / 更公开的打法

5. `现实战果`
- 命 / 人 / 名 / 权 / 物证 / 规矩变化 / 谁往前谁往后变了，至少落一项

如果这 5 项写不出来，首稿很容易写成：
- 解释句
- 总结句
- 旁白直接替现场下判断的句子

---

## 三、MRU 简化用法

别写：
- 他很清楚韩烈已经慌了
- 这说明场上所有人都开始偏向他

先写成：

1. `外部刺激`
- 韩烈先看哪样东西
- 谁把手缩回去
- 谁不敢接眼神

2. `即时反应`
- 谁喉头动了
- 谁往前挪了半步
- 谁把册子按住

3. `再做动作`
- 主角顺势补一句
- 证物顺势落账
- 对手顺势把刚才那话收回去

记法：
- **先看见，再反应，后判断**
- 能让动作自己把意思带出来，就别单独说出来

---

## 四、可替代的现场东西清单

当你想写一句解释话时，先问自己能不能换成下面这些东西：

### 1. 动作
- 踢开
- 按住
- 拽回
- 递出
- 收手
- 不接眼神

### 2. 物证
- 牌
- 纸
- 册
- 签
- 箱封
- 血迹

### 3. 群体反应
- 后排有人往前挪
- 人群先静，再炸
- 有人不再看谁，改盯哪样东西

### 4. 人往哪边挪了
- 谁退了半步
- 谁被推到场心
- 谁从墙头被拖到人前

### 5. 规矩变化
- 先押后审也要落账
- 手令公开挂出
- 原本私下的东西被摆上点卯场

只要能用上面任一项把事写出来，就别用旁白解释。

---

## 五、首稿后第一轮回压：专查旁白补缝句

首稿落完后，先扫这几类句子：
- `那不是……那是……`
- `显然……`
- `这一下 / 这句 / 这一刻……`
- `不是冲谁……是冲什么……`
- `这说明 / 这意味着 / 真正的……`

就看这两条：
- 这句是否提供了新事实？
- 如果删掉，读者还能不能看懂？

若答案是：
- 没提供新事实
- 删掉也能看懂

那就不是“必要说明”，而是“作者在帮读者翻译”，得改。

---

## 六、改写顺序

一旦写出作者腔，别直接“删一句就算完”。

正确顺序：
1. 先删解释句
2. 找空出来的信息功能
3. 用 `动作 / 物证 / 群体反应 / 谁往前谁往后变了 / 规矩变化` 之一补回去
4. 再人工重读这一段，看是否还像作者在旁边讲

---

## 七、实用判断

下面这些情况，说明不是“没细纲”，而是“首稿还在习惯性补说明”：
- 三张卡齐了，结构也顺了
- 章尾也停住了
- 但正文里仍零星冒出一两句“意义翻译句”

这种情况别回头无限加长细纲。

这时就这么处理：
- 现有细纲先别再硬拆更细
- 加一轮首稿后 `作者腔残留复检`
- 把漏句固定归类、固定替换
