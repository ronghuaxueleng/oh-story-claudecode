# 最小人物卡模板

> 本模板不是人物百科，不写大而全设定。  
> 本模板只服务正文落地，尤其服务“人物像活人、删名可分辨、冲突能自己露底”。

---

## 一、基础定位

- 角色名：
- 剧情职能：
- 当前所属阵营 / 站位：
- 前 `3` 章首次承担的功能：

## 二、这人这场到底要什么

- 公开目标：
- 私下最怕失去什么：
- 这场真正护的东西：
- 核心矛盾：
  - 例：想掌权，但怕先暴露
  - 例：想保人，但先护账
  - 例：想合作，但先自保

## 三、行为规则

- 平时说话规则：
- 被压迫时说话规则：
- 被威胁时先护什么：
- 会下意识碰什么东西：
- 会怎么撒谎 / 回避 / 装硬 / 甩锅：

## 四、显影锚点

- 外形锚点：
- 动作锚点：
- 口气锚点：
- 压场姿态：
- 删名可分辨依据：

## 五、正文硬要求

- 本章必须回落的锚点：
- 本章至少一次“嘴上说A，动作护B”：
- 本章至少一次“身体先泄露，台词后补位”：
- 本章不得只靠旁白直译的点：

## 六、事实层

- 当前已落地事实：
- 状态变更记录：

规则：
- 本区块只能写正文已发生内容
- 不得提前写未来关系变化 / 身份变化 / 伤情 / 证据口径
