# 读者爱看什么，题材该怎么对上

> 看读者心里到底缺什么、题材现在还有没有得写、不同平台口味差在哪、书名简介该怎么起。做选题和定位时可拿来对照。
> 配合文件：题材主线怎么走见 `genre-writing-formulas.md`，主打套路设计见 `genre-core-mechanics.md`。

## 你现在卡在哪

> 做选题和定位时，可先想清自己眼下到底在定什么，再翻到对应章节。

| 你要做的决定 | 先看哪个章节 | 关键要问什么 |
|-------------|-------------|---------|
| 选题材/判断题材潜力 | 题材生命力规律 | 这个题材现在还有没有得吃？ |
| 确认目标读者想要什么 | 读者心理需求与题材匹配 | 读者心里最缺什么？我的题材给不给得上？ |
| 选平台/调整写法 | 跨网站题材适配 | 同一题材放到目标平台，写法要怎么改？ |
| 判断题材边界/能不能混搭 | 题材边界感 | 我手里的素材、知识储备和篇幅，撑不撑得住这个题材？ |
| 取书名 | 书名命名技法 | 书名能不能在 3 秒里把人抓住？ |
| 写简介/文案 | 简介/文案怎么分 | 简介有没有给足安全感，也顺手挂住钩子？ |
| 检查代入感 | 代入感与塑料感 | 读者能不能把自己代进主角身上？ |

## 读者心理需求与题材匹配

### 从爆款倒着摸读者心思

- 从爆款里先拎出最能打的剧情东西，再框读者画像，再推心理需求，最后两边对着验
- 爆款的情绪 usually 是一根线打到底；不火的常常东一块西一块
- 小情绪(升级/复仇/寻宝)应统一到主情绪之下
- 验的时候，拿提炼出来的情绪需求去对目标读者真实生活状态
- 特别盯住那种“外人觉得剧毒，但那批读者就是上头”的东西

### 题材→读者画像→深层需求

| 题材 | 读者画像 | 三层心理需求 |
|------|----------|-------------|
| 末日重生 | 对阶级跃升绝望的人 | 颠倒秩序(权力渴望)+不安全感(生存焦虑)+捡漏(最小努力最大回报) |
| 战神赘婿 | 中年男人，经济压力大 | 无能的痛苦→不被体谅的愤怒→羞耻的自卑→需要借口合理化隐忍→反转暴爽 |
| 网文通用 | — | 深层需求不是表面"打脸"，而是被认可/被理解/被尊重 |

### 怎么往深里挖

- 往读者那层心窝里扎：他们爱什么、恨什么、渴什么、躲什么
- 不同题材的读者画像不一样，先定位读者，再设计情绪
- 动笔前先问：目标读者在现实里最缺什么？最想要什么？
- 情绪需求挖得越深，作品天花板通常越高，越容易打到共鸣

### 情绪根子一般在哪

情绪来自"不公平"和"为什么"。区别对待→读者理所当然觉得不公平→这就是情绪根源。
都市情绪例：事业小有成就→爸要创业→妈要给弟弟买婚房→不公平感自然产生。

## 题材现在还有没有得写

- 题材还有没有命，先看样本，别把老黄历直接当今天。
- 先用当前 scan/analyze 或用户指定样本确认目标平台仍有读者反馈。
- 判断题材所处阶段：新鲜期 / 成熟期 / 审美疲劳期。
- 新鲜期先拎创意方向；成熟期先稳住读者边界期待；疲劳期就得找新切口。
- 跟风只能借人家那套用法，别照着剧情直接复刻。
- 阶段看不准时，就先按成熟期处理：边界期待先满足，微创新别超过 3 个。

### 常见三段变化

| 时代 | 特征 |
|------|------|
| 自由时代 | 无固定风格，灵感来自传统文学/漫画 |
| 游戏化时代 | 等级+关卡+BOSS+副本+装备，引导消费 |
| 生活化倾向 | 重故事轻升级，有生活细节，贴近时代，反游戏化；是否适用需以当前平台样本验证 |

## 同一题材放到不同平台怎么改

同一个题材丢到不同网站，写法经常差很远，归根到底还是看那边读者爱吃什么。别拿 A 网站的样本直接套 B 网站，先看目标平台样本，那边到底吃什么节奏、烦什么雷点。

### 先看平台口味差在哪

| 平台 | 最能拉人的东西 | 不适配题材 |
|------|-----------|-----------|
| 刺猬猫 | 整活与人设，动漫同人/二游基础 | 全民流、高武流基本行不通 |
| 起点(非轻小说) | 正常剧情推进的节奏与代入感 | 纯整活难以长期维持 |
| 起点轻小说 | 整活+人设+节奏并重 | 需持续整活能力 |
| 番茄 | 强情绪、噱头、爽感直给 | 需慢热铺垫的题材 |

### 各平台读者差在哪

| 平台 | 读者特征 |
|------|----------|
| 飞卢 | 青年少年居多，只想爽，不想看真实苦难 |
| 起点 | 筛选成本高，读者更耐心，慢节奏也能接受 |
| 新媒体(30岁+) | 偏爱接近真实的故事，被打击的生活才是常态 |

**先看清**：同一套写法扔到不同平台，效果可能差很远，得跟着平台改口味。

### 飞卢这边一般怎么吃

**读者画像**：偏宅系热血装逼类，喜二次元/游戏/游戏面板，爱国情绪高涨，偏爱快节奏爽文不虐主，偏爱养成感和收获感。

**四大高风险雷点**：①绿帽 ②圣母 ③虐主 ④无脑。撞上就当高劝退风险，顺手查清有没有足够铺垫、补偿和读者安全感。

**天榜高概率题材**：都市嘘头装逼文/都市娱乐文/都市军工文/游戏制作文/原神文/洪荒文/综武同人/群像曝光文/全民御兽文

**高风险题材候选（需实时验证）**：大唐/大秦历史、西游、原创仙侠、传统末日、老套退婚/废材玄幻这些题材，如果缺少近期有效样本，就先按审美疲劳处理。

**题材怎么挑**：先挑受众广、样本多、边界清楚的题材；在热门写法里做点微创新。热门样本不够，或者用户点了冷门方向，就先把篇幅和复杂度压下来。

**飞卢同人写法**：快节奏直给，简单粗暴爽不等于无脑。最能打的地方是主角熟悉剧情，世界对他没秘密，他像开了挂一样一路兑现爽点。每条单线都得有清晰高潮。

### 刺猬猫这边先看什么

同人先确认目标圈层最近在看什么 | 脑洞要长在次元文化里 | 前后能说通就行，也得看那圈人吃不吃 | 少写打斗多写日常 | 对“梗”要求很高，得看最新榜单和评论在笑什么

### 番茄改名一般怎么试

书名 A/B 方案：金手指相关 / 第一个单元剧情相关 / 主线相关 / 与原书名相近的保守款 / 稍微夸张吸量的款。用目标平台规则和用户要求筛选。

## 这个题材你撑不撑得住

### 有边界和没边界的区别

| 类型 | 特点 | 适合人群 |
|------|------|----------|
| 有边界感 | 已有成熟受众、写法有迹可循 | 样本验证充分，适合稳定去写 |
| 无边界感/创新 | 受众不确定、写法需探索 | 风险高，需降低篇幅和创新数量 |

### 玄幻最常抓的三样

- **代入感**：玄幻王朝比赛博朋克代入感强百倍；肉身成圣拳拳到肉比斗法对波好代入
- **期待感**：期待感不崩贯穿全文，读者流失很慢
- **认同感**：读者代入主角对主角认同=死忠粉极难流失。"小德可缺大义不能失"

### 都市文暧昧这块常见四样

1. **女人**：没有女人的都市文不是都市文(男频)
2. **轻松**：读者图轻松愉悦不写苦大仇深
3. **互动**：角色间互动+读者与主角的"秘密"互动
4. **私密**：隐藏主角能力，读者和主角之间的小秘密

### 金手指怎么放才不飘

最好跟主角生活、职业贴得很近。医生配医术秘籍还算顺，医生配隐身就发飘。金手指最好跟主线挂着，技能能升级，一个技能能衍生出不同效果，别老开新金手指。

### 历史文有些地方别乱碰

- 主要立场得稳住："基于民族主义的圣主平等" 这类方向前后别摇，一摇就容易裂
- 历史网文≠历史，以读者为第一要素不是以真实为第一要素
- 历史人物评判先跟当代权威共识走，尤其是教员相关的评判，别乱越线
- 爽感阈值管理：前期家族/势力升级过快→后续难以为继

## 代入感和塑料感

### 代入感

代入感高度重叠于爽感。代入感低时主角做什么读者都不觉得爽。越代入越爽越爽越想看。

### 塑料感一般从哪来

仙侠不仙、武侠不侠、画风撕裂(仙侠世界搞科研)。去塑料感：让世界像真实存在的世界不是纸糊的。

### 同人为什么开头高追读

世界观够硬，初始代入感够高。例外：搞笑文、梗文最能打的不是爽，而是有趣，更多时候走旁观视角，不一定非得强代入。

### 扫榜时怎么才算看对

扫榜时这么看：1) 跑当前 scan/analyze 流程，拿到目标平台近期样本 2) 排除自带粉丝基础的头部作品 3) 分析可复用样本的开头结构共性。开头最好在 300 字内立起最重的冲突，一万字基本能看出高下。

### 商业化这事怎么想

私人表达别占太高：不属于主情绪的个人表达，尽量别超过全篇 5%，也别把叙事节奏打断。商业化不是故意恶心读者。更稳的做法是让所有私人表达都去服务主打卖点，别自己飘在主线外面。

## 书名怎么起

### 身份法（很常见）

| 类别 | 常见写法 | 示例 |
|------|------|------|
| 高逼格身份 | [仙帝/魔神/财阀]爹妈+[反差行为/目标] | "武神爹，财阀妈，我当躺赢狗怎么了？" |
| 戏剧身份对照 | [低位]替[高位]做[超出预期的事] | "让你当书童，你替少爷科举中状元？" |
| 身份擦边型 | 身份关系制造暧昧暗示 | 有风险但吸量极强 |

### 其他常见起法

| 方法 | 最能抓人的点 | 一般更适合哪边 |
|------|------|----------|
| 掉马甲型 | 隐藏身份逐渐暴露 | 若目标平台近期同质样本过多则慎用 |
| 伪同人类 | 原创中加入同人热词 | 番茄/飞卢 |
| 逼格起名法 | 意境感强的词汇 | 起点 |
| 反义词法 | 正反反差制造好奇 | 通用 |
| 金手指法 | 直接把金手指写书名 | 番茄/飞卢 |
| 主线+爽点法 | 主线剧情主打看点+爽点暗示 | 起点 |
| 乐子文法 | 一句话让人想笑 | 番茄/飞卢 |

### 各平台起名口味差别

| 平台 | 风格 | 提醒 |
|------|------|----------|
| 番茄 | 噱头优先吸量第一 | 书名决定80%流量上限 |
| 起点 | 轻度反差+金手指+主线+爽点 | 内容得贴合书名 |
| 飞卢 | 极致吸量 | 需要拿最新榜单和 scan 样本对着看 |

## 简介和文案怎么排

### 五种常见排法

| 类型 | 怎么排 | 效果 | 一般用在哪 |
|------|------|------|------|
| 四段式成功型 | 情境→冲突→行动→成功 | 安全感，明确会赢 | 最通用 |
| 三段式行动型 | 情境→冲突→行动(截止) | 留悬念 | 事业线突出的文 |
| 二段式悬疑型 | 情境→一个反转/悬念 | 制造好奇 | 悬疑/烧脑/冷门 |
| 四段式失败型 | 情境→冲突→行动→失败 | 虐感和愤怒 | 追妻火葬场 |
| 三段式成功型 | 情境→转折→成功 | 简洁明快 | 甜文/轻松向 |

### 几种常见拼法

- 成功型+悬疑型=安全感+好奇心
- 行动型+成功型=先期待再满足
- 失败型+悬疑型=虐后留悬念

### 文案里先盯这几下

- 得体现女主主体性，主体性太差会把吸引力和留存一起拖坏
- 读者要安全感，最少也得暗示主角会成
- 文案和书名货不对板会劝退
- 悬疑型不宜叠过多——太"谜语人"理解不了且安全感不足

## 最后自己对一遍

> 选题定位做完后逐项对一遍。都说得过去，再往下走。

- [ ] **读者画像明确**：能用一句话说出目标读者是谁、现实生活中最缺什么
- [ ] **深层需求对位**：题材提供的主情绪和读者深层需求匹配，不是自嗨
- [ ] **题材生命力判断**：已用当前样本确认题材阶段，不把历史热度当作当前事实
- [ ] **平台适配**：写法已针对目标平台调整，不是用A平台的经验硬套B平台
- [ ] **边界感确认**：当前素材、知识储备和篇幅能支撑所选题材
- [ ] **书名3秒抓人**：书名在目标平台的命名规则内，3秒内能传递主打卖点或钩子
- [ ] **简介有安全感+钩子**：简介类型选择正确，给了安全感同时留了悬念
- [ ] **代入感无塑料感**：世界观自洽，画风统一，没有"仙侠搞科研"式撕裂
- [ ] **书名简介内容三位一体**：书名暗示的卖点=简介承诺的内容=正文实际写出来的东西
- [ ] **金手指与生活关联**：金手指设定与主角职业/生活紧密相关，不是硬贴的
