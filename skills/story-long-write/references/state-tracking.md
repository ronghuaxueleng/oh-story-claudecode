# 状态怎么记才不乱

> 这份主要讲两件事：“最简记忆包”怎么提，还有“角色状态摘记”怎么写。写作 skill 在准备层筛状态时会读这份。

---

## 最简记忆包

写作每一章、每一节前，从已经读进来的上下文里挑出只和本节有关的信息。目的就是别把一堆无关东西也塞进去。

### 怎么筛

从上下文里提三类信息：

1. **当前状态**：本章涉及角色的最新能力、关系变化、公众形象
2. **历史因果**：与本章事件直接相关的伏笔成因、前史事件
3. **世界约束**：本章涉及的世界观规矩（力量体系、社会规矩、地理限制）

### 怎么判断要不要留

**只保留“如果不知道这个，本章就会写错”的信息。**

具体就按下面看：
- 本章细纲中提到了某个角色 → 保留该角色的当前状态
- 本章有伏笔回收 → 保留伏笔的埋设细节和前文铺垫
- 本章涉及特定地点/能力/规矩 → 保留相关世界观约束
- 纯背景知识（与本章事件无因果关联） → 丢弃

### 最后就按这个记

筛完后输出一个简洁的“最简记忆包”：

```
## 最简记忆包（第{N}章）

### 角色状态
{角色A}：{一句话当前状态，含最近变化}
{角色B}：{一句话当前状态}

### 相关伏笔/前史
{伏笔1}：{埋设细节，{埋设章节}}，本章需要{回收/推进}

### 世界约束
{约束1}：{与本章相关的规矩/设定}
```

**示例（都市文娱第 12 章）：**
```
## 最简记忆包（第12章）

### 角色状态
邵阳：穿越者，前世二线歌星，刚在宋佳伦演唱会一曲惊人，目前"临时女友"薛嘉嘉在身边
薛嘉嘉：女主，金牌制作人隐藏身份，表面是邵阳临时女友，实际已对他产生好感
宋丽：前女友，看到邵阳爆红后心态复杂，与新欢郑松关系出现裂痕

### 相关伏笔/前史
薛嘉嘉隐藏身份：第3章邵阳帮她解围时暗示过，本章需要在公开场合暴露身份的前兆
五万块彩礼：宋丽以练习生名义拿走的钱，第2章埋设，本章邵阳决定要回

### 世界约束
蓝星平行世界：无地球文娱作品，邵阳的前世记忆是降维打击
```

## 角色状态摘记怎么记

> ⚠️ **这节只管长篇。** 短篇一般不用单独拉角色状态追踪文件。

`追踪/角色状态.md` 用来记主要角色的状态变化。第 3 步大纲做完后先建初始状态，第 4 步每章写完后再更新。

### 记的时候用这个格式

```markdown
# 角色状态追踪

> 这是最简记忆包的数据源。写作每章前，从这里挑本章相关的角色状态。
> 更新时机：第 3 步大纲完成后建初始状态；第 4 步每章写完后更新变化。

## {角色名}
- **当前身份**：{最新身份/职业}
- **当前能力**：{最新能力水平}
- **关键关系**：
  - 与{角色B}：{当前关系状态}（第{N}章变化）
  - 与{角色C}：{当前关系状态}
- **公众形象**：{外界如何看待该角色}
- **待回收伏笔**：{与该角色相关的未回收伏笔}
- **状态变更记录**：
  - 第{N}章：{变化描述}
```

### 更新时怎么记

1. 每章写完后，先看本章有没有带出角色状态变化（身份、能力、关系、公众形象）
2. 如果有变化：更新对应角色的对应项，并在“状态变更记录”里追加一行
3. 如果无变化：跳过
4. 只追踪主要角色（出场 ≥3 次或有独立剧情线的角色），不追踪路人角色
5. 变更记录维护：每个角色保留最近 10 条详细变更记录。超过 10 条时，把最早的记录压成摘要写回对应项（如“当前身份：第 1-20 章从学生到歌手”），再删掉已合并的旧记录行

### 和最简记忆包是什么关系

写作每章前的“状态筛选”步骤，会从 `追踪/角色状态.md` 里提取本章涉及角色的当前状态，放进最简记忆包里的“角色状态”部分。不是整份全读，只挑本章细纲提到的角色。
