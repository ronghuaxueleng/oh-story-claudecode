# 常用事件写法一眼看完

这一页用于“当前章要尽快确定主写法”时参考。
这里只留高频、可直接开写的主路。

## 开篇最常用的几种起手

- 第一锤写法：
  - 开篇 `1-3` 段先砸现实危机
- 第一章现实改局写法：
  - 第一章结尾要改掉一件现实状态
- 第二章加压写法：
- 第二章不是重复第一章，而是对手补第二下压力
- 第三章真收益写法：
- 第三章前要给第一轮硬回报
