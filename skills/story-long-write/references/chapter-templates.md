## 三、写作技巧提醒

| 场景 | 技巧 |
|------|------|
| 开篇 500 字 | 必须有钩子，不能从天气/风景开始（除非反差极大） |
| 对话 | 推进剧情或揭示性格，不能只为了凑字数 |
| 打斗 | 不要流水账，写策略和反转，不写“你一拳我一脚” |
| 日常 | 要有人物互动和伏笔，不能只是“吃饭睡觉” |
| 爽点释放 | 铺垫要充分，释放要干脆 |
| 爽点密度 | 每 3000-5000 字至少一处情绪节点 |
| 章尾 | 每章结尾都要有翻页动力 |

---

## 四、字数硬约束

| 节奏 | 最低字数 | 说明 |
|------|----------|------|
| 高速推进 | ≥ 2000 | 每章一个明确事件 |
| 正常节奏 | ≥ 3000 | 主线 + 少量副线 |
| 舒缓铺垫 | ≥ 3000 | 人物互动 + 伏笔 |
| 高潮爆发 | ≥ 2000 | 集中释放，不拖沓 |

默认最低字数：`3000`。

---

## 五、追踪文件归档

每完成 `50` 章或一个卷结束时：
- `追踪/上下文.md` 保留最近 `5` 章详记
- 更早内容压缩到 `追踪/归档/第XXX-YYY章.md`
- 活跃伏笔、时间线、角色状态仍保留在当前文件
