# 章组收益卡

这张卡用在：
- **连续 `2-4` 章进入正文前**
- 解决“三卡写得很认真，但章组还是只在搜证、跑流程、拖兑现”的问题

这张卡先用来定：这一个章节组到底要**改掉什么局面**。
再往下拆到每章的 `动作拍清单 / 人物冲突卡 / 章尾现场句预案`。
这不是全书一次写完的大汇总，就是个**滚动预写窗口**。

---

## 怎么用

- 本卡是正文前要看的卡，不是可选补充。
- 一般只给**当前要写的连续 `2-4` 章**创建，不要求把全书所有章组一次写死。
- 若本章属于某个连续翻面链，就先有对应 `章组收益卡`，再进单章三卡。
- 本卡要回答“这一组章结束后，**谁被迫改说法 / 往前挤或往后退 / 把事情交给别人去办 / 失去先开口定事的资格 / 失去资源 / 失去脸面**”，别只写“查到什么”“发现什么”“得知什么”。
- 若本卡只能写出 `新证据 / 新怀疑 / 新线索`，却写不出 `现实改变量`，这章组就先别进正文。

## 最低能用示例

- 覆盖章节：`第009-010章`
- 这组章写完后，主角要拿到的现实收益：把“并烽呈条”从值房私下就能定的说法，硬拽成场面上当众对账，并逼对手当众承认旧册要重验
- 这组章写完后，对手要失去或被迫改变的东西：韩烈一系失去“先压后拖”的节奏控制，只能跟着主角公开对账
- 旁观者 / 群体外部变化：墙头哨和火队都开始按“挂出来的账”说话，而不是按值房私话说话
- 第一次阻断：对手先把事压成“手续未全，先别乱翻”
- 第二次翻面：主角发现光有呈条不够，还得把旧册和人一起拽到场心
- 真正兑现点：`第010章中后段，对手被逼把原话缩回去，只能承认旧册得重验`
- 为什么不是搜证链：因为结果不是“又翻出一条线索”，而是“场面上的先定说法资格和调度资格已经换手”
