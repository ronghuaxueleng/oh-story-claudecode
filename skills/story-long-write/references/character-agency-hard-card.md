# 人物别写成只会办事

这张卡专门拦“角色设定有了，但正文里只剩下替剧情办事”。
- 用于关键角色首稿前、关键角色首章后、连续章写后检查

---

## 怎么用

- 主角、主要对手、主要盟友、重要角色、软肋角色都要填
- 每个角色最少答出一项“不是为了主线，而是为了自己”
- 别只写 `认数 / 记门 / 见证 / 递簿 / 传话 / 背书`

---

## 角色逐一填写

- 角色名：
- 这章最怕失去什么：
- 这章最想保住什么：
- 这章为自己做的动作：
- 这章被迫付出的代价：
- 这章最露私心的一下：
- 这章最露软肋的一下：
- 这章最容易被写成只剩办事作用的位置：
- 这里要改成什么“人”的动作：
- 这章最可能被旁白替他/她提前说破的位置：
- 若删掉旁白先替读者下判断的句子，这个人还剩什么真实动作：
- 这章最像“配角替作者翻译”的一句：
- 这句要改成什么利益动作 / 代价动作：

---

## 软肋角色加题

- 软肋对象当前状态：
- 这章有没有再碰到身体 / 私人关系 / 拖累感 / 不敢再求：
- 如果没有再碰到，下一章最迟在哪一拍补：

---

## 一看就知道写虚了的地方

- 若一个角色只能答出 `他帮主角做了什么`，答不出 `他为自己做了什么`，多半已经写成了只会替剧情办事的人
- 若软肋角色连续两章没有再碰到这一面，这个人物多半已经发虚
- 若重要角色只能答出“把规矩改了”，答不出“他为此付了什么代价”，那就容易显得只是出来晃一下
- 若重要角色得靠旁白句才能显得“厉害 / 重要 / 能压住场面”，说明旁白把意思说得太多了
- 若配角一句话删掉后，局势仍成立，只是少了“给读者解释”，这类配角多半已经成了专门讲解的人
- 若主角连续两章只剩重复旧话、发命令、解释局势，主角声线就容易开始变机械
