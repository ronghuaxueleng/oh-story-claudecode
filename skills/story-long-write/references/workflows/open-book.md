# 开书流程

> 本文件负责从零开书的最小执行顺序。详细参考仍由各类 reference 承接。

---

## 顺序

1. 确认题材方向
2. 产出核心设定
3. 落题材定位
4. 搭建卷纲
5. 搭建前 `10` 章细纲
6. 建立追踪主表
7. 进入第 `1` 章写前闸门
8. 写第 `1` 章正文
9. 写后验收
10. 同步追踪与角色卡
