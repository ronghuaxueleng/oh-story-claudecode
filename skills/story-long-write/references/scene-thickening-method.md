# 扩场面、补动作拍方法

这页用于给 `story-long-write` 补厚度。
- 解决“主线没错，但正文还是薄”的问题
- 用于冷启动、回炉、首稿补厚度，不是拿来继续堆规则说明的

---

## 一、什么时候该读这份

碰上以下任一情况，先读这份：

- `scene_lint.py --json` 碰上 `字数.warn / 字数.error`
- `解释.error` 已经压住，但正文还是偏薄
- 第一爆点、第二轮收益、两轮外部反应都已成立，读感却仍像“高质量提纲扩写”
- 写后检查能说清“局面哪儿变了”，却说不清“这一章到底多了哪几下动作”

---

## 二、先判：你缺的不是信息，是拍子

正文偏薄时，先别问：
- 还要不要再解释一点
- 要不要再补一段背景
- 要不要再补一点心理

先问：
- 这一章还缺哪几拍动作
- 哪一下能让人往前一步、退半步，或者被挤到外圈
- 哪一拍能让人改先后
- 哪一下能让第三方别光看热闹，得真下场挪位子、改顺序

一句话先分清：

- 主线问题：根本没想清这一章到底要把什么事掰过来
- 厚度问题：大方向知道，但场面里真动起来的几下还是不够

---

## 三、厚度不是“多写”，是“多改”

有效厚度至少得碰上一种：

1. 新增一拍受阻
- 谁原本准备怎么拿
- 被什么打断
- 只能改什么打法

2. 新增一拍争先
- 谁抢先解释
- 谁抢先排顺序
- 谁抢执行次序
- 谁抢现场第一手

建议改写为动作结果问法：
- 谁抢先解释
- 谁抢先排顺序
- 谁抢先下令动手
- 谁抢到现场第一手判断权

3. 新增一拍外部变化
- 协防改栏
- 书吏翻页
- 记簿的人换了写法
- 围观者让线
- 第三方顺手又添了一把火

4. 新增一拍代价落地
- 主角先扛了哪一下风险
- 对手先丢了哪一下先手
- 第三方先被逼着偏到哪一边

5. 新增一拍后果开始起作用
- 哪张名单开始真拦人
- 哪个先后次序开始真压到人头上
- 哪个资格开始真改掉谁能先动手

下面这些不算有效厚度：
- 换句重述已有收益
- 解释这件事为什么重要
- 总结场面变得更危险
- 回忆过去为什么会这样
- 主角心里再想一遍自己赢在哪

---

## 四、动作拍扩写清单

当章节偏薄时，别自由补写，先写这张卡。

至少补出 `3-5` 拍，每拍必须写成：

`谁做了什么 -> 谁被逼得只能怎么变`

模板：

1. 原本哪一拍太短：
- 位置：
- 原来只写了什么：

2. 扩拍后新增动作：
- 新拍 1：
- 新拍 2：
- 新拍 3：
- 若还不够，再补：
- 新拍 4：
- 新拍 5：

3. 每一拍改了谁：
- 新拍 1 动到了谁的动作 / 前后位置 / 顺序：
- 新拍 2 动到了谁的动作 / 前后位置 / 顺序：
- 新拍 3 动到了谁的动作 / 前后位置 / 顺序：

4. 每一拍落在哪段：
- 前 `1/3`：
- 中段：
- 后 `1/3`：
- 章尾前：

5. 哪一拍最重要：
- 哪一拍把“提纲信息”变成了“现场结果”

若写不出“这一下到底动到了谁”，说明这拍还是空的。

---

## 五、三类最常用的补肉方式

### 1. 中段补争先

适用：
- 第一爆点已打出
- 第二轮收益也有了
- 但中段太滑，像直接从 A 走到 C

补法：
- 多一个角色抢先发言
- 多一个现场执行人改顺序
- 多一个第三方借机插刀

目标：
- 让中段不只是“等结果”

### 2. 后段补现场变化

适用：
- 外部反应只有“大家都懂了”
- 但没人真往前挤、往后退，顺序也没变

补法：
- 围观者让半线
- 协防改封锁栏
- 书吏翻页改序
- 记录人改写条目

目标：
- 让“外部反应”别只停在喊两句，得真把现场带动起来

### 3. 章尾前补承压

适用：
- 章尾最后一拍想落新名单 / 新名分 / 新资格
- 但前一拍抬升不够，最后一拍显得太轻

补法：
- 让对手先抢第二手
- 让第三方先犹豫再改手
- 让现场执行人先停顿一下再落笔

目标：
- 让最后一下不是平白掉下来，而是顶着压力砸下来

## 六、不同题材的厚度来源

### 1. 宗门 / 升级 / 资格争夺
- 前后位置变化
- 先后变化
- 谁先担责
- 谁先摸到入口

### 2. 都市异能 / 事故处置
- 封锁线改法
- 执行次序改令
- 旁人抢报现场
- 谁先拿到处置权

### 3. 女频强情绪 / 婚恋拉扯
- 谁先把话咽回去
- 谁先把两个人的位置换了
- 谁先失态
- 谁先让出或抢回情感主动权

### 4. 轻喜剧 / 日常装逼
- 围观的人把刚才那话咽回去
- 排队的人往旁边挤
- 原先那套说法当场就得改
- 谁先从热闹位变成执行位

厚度来源得跟题材走，不能所有书都补成同一种“大群体抢前后”。

---

## 七、怎么判断你补出来的是真肉

补完后，逐条回答：

1. 新增内容里，有没有至少 `3` 个清楚的动作拍
2. 这几拍里，是否每拍都能答出“到底动到了谁”
3. 新增内容删掉后，现场是否明显变空
4. 新增内容是不是仍然能独立承担 `争先 / 现场变化 / 承压 / 开始起作用` 中至少一项

若以上任一答不稳，就不算补厚度成功。

---

## 八、当前阶段的真实边界

基于真实冷启动样本，当前 skill 已经比较擅长：
- 拉对主线
- 把第一爆点写出来
- 把第二轮收益写出来
- 把两口外部反应写出来

但还不稳定的，是：
- 把这些主线自然扩成够厚的场面拍

所以后续优化重点，不是继续加“为什么要厚”的说明，
而是继续研究“哪几拍最值得加、加了谁会改、怎么让中后段更有肉”。

若已经补出基础动作拍，但脚本仍掉厚度，下一步别重复补并列拍，
而要转读 `chain-reaction-scene-thickening.md`，把最重要的 `2-3` 拍继续拉成长连锁。
