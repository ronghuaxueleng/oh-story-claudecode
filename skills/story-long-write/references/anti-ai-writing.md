## 这些 AI 味先避开

## 叙事整层冒出来的 AI 味（先看）

很多“AI 味”不是某个词的问题，而是叙事本身出了问题。下面这些问题，比单个禁用词更该先看。

### 1. 作者代言句

典型特征：
- 不是角色自然说话，而是作者借角色或旁白先替读者下判断
- 常见形式：
  - “这意味着……”
  - “真正可怕的是……”
  - “从这一刻起……”
  - “他现在要的不是……”
  - “这才是……”

这里先按这个改：
- 如果动作、对白、反应已经写出来了，就别再补作者判断
- 把“判断句”改成“现场结果”

### 1.5 旁白直接替现场下判断的句子

典型特征：
- 作者不写眼前发生了什么，而是替读者总结“什么更值钱、更顶用、更重要”
- 常见形式：
  - “真正重要的不是……而是……”
  - “更硬的是……”
  - “现在真正该抢的是……”
  - “最该抢的已经不是……”
  - “名字不顶用，票才顶用”
  - “这一下就把原话逼回去了”

这类句子的问题在于：
- 不是人物在求生、争抢、翻盘
- 是作者站出来替读者算账、下判断、把爽点讲白

这里先按这个改：
- 别用“价值判断句”代替事件结果
- 直接改写成：
  - 谁抢到了什么
  - 谁因为慢半拍被挤掉
  - 哪条规矩当场被掀开
  - 哪个人因此把原话吞回去、让路、被挤开、见血
- 如果一句话删掉后剧情不受影响，说明它大概率就是作者代言句，先删

### 2. 信息推进冒充情绪推进

典型特征：
- 新线索越来越多
- 黑幕越来越深
- 设定越抬越高
- 但读者情绪没有同步抬升

就这么看：
- 读完一章后，若只能回答“知道了什么”，答不出“谁赢了、谁亏了、谁丢脸了”，就是此类问题

这里先按这个改：
- 别再补更多背景解释
- 直接补：
  - 谁被当场掀掉
  - 谁被逼把原话吞回去
  - 谁失去东西
  - 谁手里的事先被别人拿走
  - 谁被抓现行

### 3. 细纲味直接漏进正文

典型特征：
- 正文像在执行“要表达什么”
- 而不是在写“谁做了什么”
- 读起来像剧情讲解，不像事件发生

高危词：
- 体现
- 展现
- 讲清楚
- 暗示
- 推动
- 铺垫
- 表达

这里先按这个改：
- 回到动作层
- 把“这一段要体现 XX”改成“谁做了什么，别人怎么反应”

### 4. 爽点落地后又解释一遍

典型特征：
- 角色已经被打脸/按住/夺权
- 旁白又补一句“这就说明主角已经拿到了权/对方已经怕了”

这里先按这个改：
- 二次解释先删
- 爽感要靠场面本身成立，别靠作者翻译成立

### 5. 章尾别自己出来收话

典型特征：
- 结尾不是停在危险、发现、反扑、动作上
- 而是停在：
  - 总结
  - 升华
  - 定性
  - 哲理

这里先按这个改：
- 章尾先用动作、对白、悬念收束
- 别靠作者出面“收一下”

### 高频AI用词

> 完整禁用词表见 [banned-words.md](banned-words.md)

**补充类目**（`banned-words.md` 还没覆盖到的替换方向）：

| 类别 | 怎么改 |
|------|---------|
| 抽象升华词（命运、宿命、注定） | 用具体事件代替抽象概念 |
| 万能比喻（像潮水般、如闪电般、仿佛春风） | 要么不用比喻，要么用生活化的比喻 |

### 章末别写成总结腔

章节结尾先别这么收：
- 总结性感悟（"他终于明白了……"）
- 升华式感叹（"这一夜，注定无人入眠"）
- 哲理式收尾（"人生就是这样……"）
- 伏笔式预告（"他不知道的是，更大的风暴即将来临"）

**更顺的写法**：章尾用动作、对话或悬念收住，让事情自己留味。

### 固定句式别一批一批地反复上

以下句式不是绝对禁用，但高频出现时，基本就已经带出 AI 味或解释腔了：

- 这不是……而是……
- 这一下……
- 这意味着……
- 从今天起……
- 真正的……
- 他现在要的不是……
- 值得注意的是……
- 换句话说……

这里先按这个改：
- 单句未必有事，连用就容易出问题
- 一旦形成“作者老爱这么转一下”的习惯，就先删改

### 叠加式描写（同一动作掰开写三遍）

**怎么查**：一个动作/情绪先写发生，再补感知细节，再补身体反应，分三段依次写完。读者看到的就是同一个动作被掰开写了三遍。

**典型特征**：
- 先写一个概括性动作，再展开写同一动作的细节，再写身体反应——三段说的是同一件事
- "发生层→感知层→反应层"按顺序分段出现
- 每个维度独立成段，而不是织入同一段连续正文

**错误示例**：
> 林父低着头，左手把文书压住，右手拿笔，往纸上落——
>
> 手在抖。不是小抖，是从肘到腕的那种。
>
> 笔尖在纸上停了停，写了一横，又停。那个"林"字的撇写歪了。

→ 同一个动作（手抖/写字）分三段写，每段是同一瞬间的不同维度

**正确做法**：发生、感知、反应三个维度织入同一段连续正文，读者读到一个完整瞬间：

> 林父左手压着文书，右手拿笔往纸上落——笔尖一触纸面就偏了，从肘到腕止不住地抖，那一横斜着拖出去。

→ 发生、感知、反应在一段里同时呈现

**这里先按这个改**：不是把情绪细节删光，而是把同一瞬间的重复描写并起来。若合并后明显变薄，先恢复原文里有功能的信息，或把既有信息改成更自然的动作/对话表达；别新增原文没有的情节、设定、关系或时间线。

## 先按这几条查

### 1. 段落是不是挤了或者碎了

段落不是越碎越好，也不是越长越有质感。检查重点是朗读和手机阅读会不会卡：

- 一段通常只写一个动作、一个信息变化或一组紧密相关的反应。
- 逗号串太长、多个完整动作挤在一段里，读起来需要换气时，按动作或信息变化拆开。
- 连续短段碎成提纲时，合并同一镜头内的相邻句，让画面保持连续。

```
过密：他看着窗外的雨，心中涌起一股说不清的感觉，这些年走过的路和很多已经忘记的事都在这一刻涌上心头。

更自然：他盯着窗外。雨下了很久。
"你还在想她？"老刘问。
他没说话。
```

### 2. 动作、对话、情绪反应有没有拧在一起

三要素循环推进，别单写心理活动超过 2 段：

```
动作 -> 对话 -> 情绪反应 -> 动作 -> 对话 -> ……
```

情绪不用"他感到/他觉得"，用身体反应和行为表现：
- 不写"他很紧张" -> 写"手心全是汗，筷子差点掉了"
- 不写"她很愤怒" -> 写"她把杯子摔在地上，碎片弹到脚背上也没弯腰捡"
- 不写"他很伤心" -> 写"他在车里坐了二十分钟才发动引擎"

### 3. 短句先走

| 场景 | 句长 | 示例 |
|------|------|------|
| 打斗/紧张 | 3-8 字 | 他侧身。刀光一闪。血溅在墙上。 |
| 对话 | 口语化 | "你疯了？""可能吧。" |
| 日常 | 8-15 字 | 厨房里有炒菜的声音，闻着像辣椒炒肉。 |
| 描写 | 读起来不卡，信息不过载 | 天快黑了，巷子里没什么人。 |

### 4. 口语化表达

- 允许用俚语、粗话（符合角色身份）
- 对话不要书面语（"我认为此事不妥" -> "我觉得不靠谱"）
- 叙述也不要端着（"他目光如炬" -> "他眼珠子一动不动盯着"）
- 短语先于成语（"无可奈何" -> "没办法"）

## 少说结论，多给现场

| Tell（告诉） | Show（展示） |
|-------------|-------------|
| 他是个胆小的人 | 他把检查报告在手里翻来覆去看了三遍，还是不敢打开 |
| 这间酒吧很吵 | 酒保凑到他耳边喊了两次他才听见 |
| 她很富有 | 她随手把一张信用卡丢在桌上，卡面上的数字比这顿饭贵十倍 |
| 两人关系很差 | 他把烟掐灭在她刚泡的茶杯里，她面无表情地把杯子推到一边 |
| 他很聪明 | 三秒钟。他看了三秒钟就把文件合上了。"第三页，第二行。" |

**先按这几句改：**
1. 用行为代替形容词
2. 用细节代替总结
3. 用对话代替旁白往外讲
4. 用反应代替情绪词

## 质量维度检查

### 1. 最先盯的一致性（权重最高）
- 剧情是否与大纲/前文一致
- 人物行为是否符合人设
- 设定是否有前后矛盾

### 2. 表面改写（防AI指纹）
- 是否包含AI高频用词（见上表）
- 章尾是否有总结/升华
- 是否有大段纯心理描写
- 段落是否超过 3 句

### 3. 格式一致性
- 对话格式统一（中文用""，不是「」）
- 场景切换有明显标记
- 时间线清晰可追踪

### 4. 可读性
- 是否有连续 3 句以上的长句（15 字+）
- 对话是否口语化
- 是否有未解释的生僻词/设定术语
- 节奏是否有快有慢（不能全是一种节奏）
- 是否存在“知道了很多，但爽得不够”
- 是否存在“只有信息推进，没有情绪推进”

### 5. 前后得接得上
- 角色动机是否合理
- 事件因果链是否清晰
- 时间线是否对得上
- 角色的知识范围是否合理（不能"开上帝视角"）

## 7 种 AI 写作模式检测

### 模式 1：AI 高频词

| 禁用 | 替换为 |
|------|--------|
| 不禁 | 删掉 |
| 仿佛/宛如 | 删掉或用具体描写 |
| 映入眼帘 | 删掉 |
| 心中暗道 | 用动作展示思考 |
| 沉声道/淡淡地说 | 换成动作标签 |
| 脸色一变 | 用具体表情/动作 |
| 嘴角微扬 | 他笑了/他翘了下嘴 |
| 不由自主 | 删掉 |
| 只见/此时此刻 | 删掉 |
| 目光如炬 | 删掉或具体化 |

### 模式 2：弱化副词泛滥
阈值：每 1000 字超过 3 个 = AI 签名。重点监控：微微、淡淡、缓缓、轻轻。

### 模式 3：意义膨胀
- "意义深远" -> 写具体后果
- "前所未有" -> 给出对比参照
- "可谓" -> 删掉

### 模式 4：万能收口话
- "未来可期" -> 用未解决的紧张感结尾
- "前途无量" -> 删
- "充满希望" -> 写具体的下一步动作

### 模式 5：论文体段落结构
小说中出现以下开头句 = AI 入侵：
- "不难看出""由此可见""事实上""综上所述"

### 模式 6：书面语连词泛滥
叙事散文中频繁出现："于是乎""与此同时""从而""因而""诚然" -> 口语化替代或直接删除。

### 模式 7：三连排比癖
AI 喜欢把事情凑成三个以显"完整"。-> 砍到只剩最有力的一条。

## 篇章级 AI 味（骨架上的味，最难查，也最要命）

> **先说清楚：检测器打的分，跟真实的 AI 味不是一回事。** 检测器主要看用词是不是太好猜、句子长短波不波动，都是句子层面的。实测里，只换同义词、改情绪词，照样 100% 判 AI；改了句式节奏、改了结构，反而能降到 0%。也就是说，**真味在骨架上，不在字词上**。有研究干脆不看用词、不看标点，光看故事结构，就能 93% 分出人写的还是 AI 写的。
>
> 上面模式 1–7 都是字句层面的，能骗过检测器，骗不过读者。下面这几条，才是读者翻一页就能闻出来的骨架味。**回修时先查这一节，再回去抠字词。** 例子都按情感 / 都市现实短篇来。

### 篇1. 作者跳出来替读者悟道理

作者或主角把"这事让我明白了什么"直接讲出来，替读者把顿悟做完。数据上，AI 叙述者爱把主题点破的大概 77%，真人才 52%，AI 的说教味整体高两成。

**先看这几处**：
- 结尾或每节尾冒出"她终于明白……""原来……"
- 一件事刚发生，紧跟一句把它含义说透的旁白

**别这么写**：
> 她终于明白，有些告别，是为了更好地重逢。

**这么改**：那句删掉，停在动作上，让读者自己悟——
> 她把他留下的那副碗筷收进柜子最里层，关上柜门。

### 篇2. 每节都精准落一句冷冷的点题短句

注意，**这跟滥情升华不是一回事**。是每节都稳稳收在一句克制的、带象征的短句上（"有些门，关上就不会再开。""右手没有抖。"）。单看每句都挺高级、挺克制，可**节节都来一记、位置太齐**，这本身就是模板味。真人写东西，这种"金句落点"是稀稀拉拉、碰上了才有、经常还错过的。

**先看这几处**：
- 连着 3 节以上，每节都收在一句冷冽的象征/点题短句上
- 这些尾句齐得像盖章，能猜到下一节还来一记

**这么改**：几节里只留一处点到就行；别的节收在半截对话、一个没意义的动作，或者干脆停半空不收——用节奏把"每节精准落点"这个对称打掉。

### 篇3. 整篇像在填一张对照表

每节都埋"以前 vs 现在""他嘴上说的 vs 他实际做的""大城市 vs 老家"这种对照，段落还一样长，每段都绕回主题，因果咬得死死的、一点闲笔都没有。数据上，AI 故事大概 79% 没有支线（真人才 57%），结局靠主角自己推动的约 69%（真人 46%）。

**先看这几处**：
- 第一节写"如今独居的空房子"，第二节非得对上"当年热闹的婚房"，对仗到能猜
- 全篇没一笔跟主线无关的闲事，每个细节都在服务主题
- 段落长短、每节结构都一个样，像一个模子扣的

**这么改**：
- 故意塞一笔**不服务主题的闲事**（跑题的邻居八卦、一件跟主线没关系的旧物）
- 允许段落长短不匀，某节明显短一截或长一截
- 对照回闪砍掉一半，"以前"只在某处不经意漏一次，别节节对齐

### 篇4. 主角稳过头，一点破绽都没有

**克制过头，本身就是 AI 味，不是只有滥情才露馅。** 主角从头到尾体面、滴水不漏；情绪走得太直（要么一路稳，要么一路悲），没有真人那种突然失态、来回反复、莫名其妙的情绪跳。研究也证实，AI 写的故事情绪起伏更平、专门躲开张力、老往和解和怀旧上靠。

**先看这几处**：
- 从头到尾主角没失控过一次，没一个破绽
- 情绪单调地往一个方向走，没毛刺、没回跳
- 每句话都拿捏得刚刚好，成熟得像被匀速控制过

**这么改**：给个**不体面、控不住**的破绽，让情绪长出毛刺——
> 她在便利店冲店员莫名其妙发了火，回到家却为一个塑料袋哭了很久，哭完擦干脸，没事人一样回了条工作消息。

（先装没事、中间突然崩、然后又抽身——比全程克制真实多了。）

### 篇5. 配角只是个功能件，对话不会跑题

配角只为推情节或印证主题活着，一开口就在办事，像张人物卡；对话太有效率、句句在点上、人都"太会表达自己的感受"，把潜台词全说破了。真人说话是会插科打诨、跑题、绕圈、答非所问、话到嘴边又咽回去的。

**先看这几处**：
- 配角没一句跟情节无关的话，纯工具人
- 对话太"会说"，该藏的潜台词全抖出来
- 一问一答严丝合缝，没有留白、没有错位

**别这么写**：
> "你还好吗？"
> "不好。我一直没办法接受我们结束了这个事实。"

（太会表达，潜台词全说破。）

**这么改**：让对话**跑题、留白、错位**，再给配角一句**只暴露性格、不推情节**的废话——
> "你还好吗？"
> "……你伞带了没，说下午有雨。"
> "带了。"
> "哦。"

（比如同事随口抱怨一句食堂又涨价，什么情节都不推，人反而活了。）

### 篇6. 通篇像同一个人用同一个劲写下来的

哪怕每段单看都像人写的，只要**段跟段之间用的劲、节奏、句法都一个样**，还是会露——这检测器抓不到，读者却能感觉到"匀速"。

**这么办**：主动做出落差——让某几段明显更碎/更长、更冷/更放开，劲使得不匀，才像真人在不同情绪下写出来的。

## 系统性去AI三遍法

### Pass 1：去泛化（Strip Generic）
- 抽象情绪总结句 -> 删或替换为具体动作
- 假深度句 -> 删
- 意义膨胀 -> 缩小到具体影响
- 空洞收口话 -> 删
- 工整对比句式 -> 打散重写
- 装饰性形容词堆砌 -> 白描
- 过度使用"于是""然而""此刻" -> 删掉一半
- 所有角色说话一样"高级" -> 区分语气
- 作者代言句 -> 先删
- 爽点后二次解释 -> 先删
- 章尾下结论句 -> 先删

**这里先这么改**：能删就删，不能删就用具体细节替换。这一遍去掉80%的AI味。

### Pass 2：去书面化（Cut Professional Diction）
- 分析性用词（`机制``结构``逻辑``体系` 这一类分析词出现在小说中）-> 换成平时会说的话
- 抽象名词滥用 -> 直接说事
- 体制内用语（"进一步""深入""推进""落实"）-> 删
- 专业术语堆砌 -> 只保留必要的，用白话解释

**例外**：保留专业感的场景（历史题材正式用语、文学向刻意密度、喜剧夸张修辞）。

### Pass 3：回自然感（Restore Natural Presence）
- 具体的感官细节（气味、温度、触感）
- 角色说话方式的区分（不同人不同语气）
- 节奏变化（长短句交错）
- 社会位置感的对话（上级和下属说话方式不同）
- 场景特有的记忆点
- 项目特有的语言习惯（角色的口头禅）

**这里先这么改**：别贪多。每段加 `1-2` 个具体细节就够了。

### 升级策略

| AI味程度 | 策略 |
|----------|------|
| 轻度 | 只做 Pass 1 |
| 中度 | Pass 1 + Pass 2 |
| 重度 | 完整三遍 + 重点段落重写 |

### 自检清单
- 对话自然度检查：对话是否使用口语化表达，是否避免了书面语/正式腔调
- 删掉任何一句，会影响理解吗？不会 = 可能多余
- 不同角色能通过对话区分吗？
- 有没有一个细节是这个场景特有的？
- 这章现在是更自然了，还是只是更短了？
- 这章有没有“信息很多，但现场太少”？
- 这章有没有“作者在替读者说话”？

## 去AI味补充技法

### Show vs Tell

| 告知类型 | AI写法 | 自然写法 |
|----------|--------|----------|
| 告诉期待感 | "他很期待" | 展示期待->情绪->满足的链条 |
| 告诉角色目的 | "她想离婚" | 用行动展示目的 |
| 告诉角色态度 | "她很冷静" | 用对话和反应体现 |
| 告诉剧情走向 | "接下来会发生大事" | 用铺垫->反转->延续展示 |

### 心理描写润物细无声

- 加括号标注内心活动 = 破坏代入感
- 大段内心独白解释动机 = AI签名
- 直接写"她感到""她意识到" = 告知情绪

**自然写法**：心理活动自然融入叙事，用行为暗示心理，用沉默/动作/反常行为表达内心。

### 代入感检查
- 主角行为读者能理解、共鸣、接受吗？
- 反派够强吗？（弱反派 = 读者觉得主角赢了没意义）
- 是否围绕人设写行为？（行为/语言/思维围绕人格展开）
- 读者已知信息是否被有效操控？（信息差制造情绪波动）

## 改写范例库

### 情绪外化范例

**紧张**
- '他感到一阵紧张，心跳不由自主地加快了'
- 他攥紧了手里的纸杯，水洒出来一些

**愤怒**
- '愤怒在他心中燃烧，他不由得握紧了拳头'
- 他把筷子往桌上一拍，碗里的汤溅了出来

**悲伤**
- '一丝悲伤涌上心头，她的眼中闪过泪光'
- 她低头搅着咖啡，搅了很久

**害怕**
- '恐惧瞬间笼罩了他，他感到一阵战栗'
- 他的背贴在墙上，不敢动

**失望**
- '她感到一丝失落，心仿佛被什么东西揪住了'
- "哦。"她把手机锁了屏

**惊讶**
- '他的瞳孔微微收缩，显然没有想到会听到这样的话'
- 他张了张嘴，什么都没说出来

### 场景描写范例

**AI风场景**
- '阳光透过窗帘的缝隙洒进来，在地板上投下斑驳的光影。空气中弥漫着淡淡的花香，仿佛整个世界都沉浸在一片宁静祥和的氛围中。'
- 下午三点，客厅里只有钟在走。

**AI风天气**
- '天空阴沉沉的，乌云密布，仿佛随时都会下起倾盆大雨。凛冽的寒风呼啸而过，带着一丝刺骨的寒意。'
- 要下雨了。风把晾在外面的衣服吹得乱晃。

**AI风打斗**
- '他的拳头犹如疾风骤雨般猛烈，每一击都蕴含着不容置疑的力量。对手的瞳孔微微收缩，显然没有预料到如此凌厉的攻势。'
- 他一拳怼过去，对方没躲开，嘴角破了。

### 结尾改写范例

**升华式结尾** -> '他站在窗前，望着远方的天际线，终于明白了生活的真谛——有时候，放手才是最好的选择。' -> 他把烟掐了，回屋睡觉。

**总结式结尾** -> '这一刻，一切都变了。她知道，从今以后，她的人生将翻开崭新的一页。' -> 她关上了那扇门。没回头。

**感慨式结尾** -> '岁月如流水般悄然流逝……' -> 直接删掉这种段落。

### 节奏调整范例

**排比句**
- '他看着她的眼睛，看着她的嘴唇，看着她微微颤动的睫毛，心中涌起一股难以名状的情感。'
- 他看着她。她没说话。

**长句拆短**
- '当他终于推开那扇沉重的木门时，映入眼帘的是一间昏暗的房间，空气中弥漫着陈旧的气息，墙角堆满了落满灰尘的箱子。'
- 门开了。屋里很暗，角落堆着几个箱子，上头全是灰。

**工整段落打碎**
- '她喜欢春天的花朵，喜欢夏天的阳光，喜欢秋天的落叶，喜欢冬天的白雪。每一个季节都有它独特的美。'
- 她喜欢春天。别的季节也还行。

## 冲突对话改写范例

### AI式温和对话
- '我觉得你这样做不太合适，能不能考虑一下我的感受？' -> "你眼里还有我吗？"

### AI式完美解释
- '其实我这样做是有原因的，因为当时的情况非常复杂……' -> "你能怎么着？"她把茶杯重重放下。

### 对话情绪五级递进范例

同一冲突场景，从弱到强：

1. **客观陈述**："你把我的东西扔了。"
2. **陈述+建议**："你把我的东西扔了，以后能不能先跟我说一声。"
3. **主观指责**："你凭什么动我的东西。"
4. **指责+命令**："你算什么东西，也配碰我的东西？滚出去。"
5. **指责+PUA**："我伺候你吃伺候你穿，你连个东西都放不好。你这辈子也就是这样了，离了我你什么都不是。"

### 震惊分层改写范例

**AI式一步到位**：所有人都震惊了，不敢相信自己的耳朵。

**自然分层震惊**：
1. 对面的男人手抖了一下，茶杯里的水洒出来。
2. 旁边的人互相看了一眼。有人往后退了一步。角落里有人开始掏手机。
3. 刚才还趾高气扬的女人，脸上的笑僵住了。她张了张嘴，一个字没说出来。

### 代入感修复范例

**被动主角**：她很害怕，不知道该怎么办，只能等着事情过去。

**主动主角**：她锁了门，把手机调成静音，打开了录音。

## 最后自己对一遍

写完每章后，按此清单逐项扫描。**先对下面这组骨架级（换字换不掉、最要命），再对字句级。**

- [ ] **没替读者悟道理**：结尾/每节尾没有"她终于明白……"式的意义总结（篇1）
- [ ] **点题句不成串**：没有节节都收在冷冷的象征短句上，金句落点稀稀拉拉、碰上才有（篇2）
- [ ] **不填对照表**：没有节节"以前vs现在"式对照；有至少一笔不服务主题的闲事；段落长短不匀（篇3）
- [ ] **情绪有毛刺**：主角至少失控/破绽一次，情绪不是一根直线，有回跳（篇4）
- [ ] **配角不是工具人**：有跑题/留白/错位的对话，至少一句只暴露性格、不推情节的废话（篇5）
- [ ] **通篇有落差**：段跟段用的劲、节奏不匀速，不像一个模子（篇6）

- [ ] **段落控制**：段落按动作/信息变化断开，读起来不卡
- [ ] **AI高频词扫描**：无不禁/仿佛/映入眼帘/心中暗道/沉声道/嘴角微扬/不由自主/只见
- [ ] **弱化副词计数**：每1000字"微微/淡淡/缓缓/轻轻"不超过3个
- [ ] **无三连排比**：没有AI式的"三个一组"修辞
- [ ] **无论文体**：无"不难看出/由此可见/事实上/综上所述"
- [ ] **无书面语连词堆砌**：无"于是乎/与此同时/从而/因而/诚然"泛滥
- [ ] **章尾无总结升华**：用动作/对话/悬念收束，无感悟/哲理/预告
- [ ] **无大段心理描写**：心理活动不超过2段，无括号标注内心
- [ ] **情绪用动作展示**：无直接写"愤怒/伤心/紧张"，全用身体反应
- [ ] **对话口语化**：无书面腔，不同角色语气可区分
- [ ] **Show Don't Tell**：用行为代替形容词，用细节代替总结
- [ ] **句长匹配场景**：打斗和冲突段更短，日常和描写段可稍长，但朗读不卡、信息不过载
- [ ] **去AI三遍法有没有真跑**：轻度只做Pass1，中度做Pass1+2，重度完整三遍
- [ ] **对话自然度测试**：无书面语痕迹 = 对得上
