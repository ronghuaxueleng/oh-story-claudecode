# 检查话别漏进正文

这张卡专门拦“写后检查 / 修改表 / 模板节点标签”偷渡回正文。
首稿自检、写后检查、连续章复审都能用。

## 高危回流句型

- `第一次外部反应，落了`
- `第二次外部反应，落了`
- `第一次群体改线`
- `这就是X的压法`
- `这就是第一个即时后果`
- `这就是第二步收益`
- `真正重要的是`
- `现在争的已经不是……`

## 本章必填

- 本章最像检查话的 1 句：
- 这句删掉后事件链是否仍成立：`是 / 否`
- 若仍成立，它属于：
  - `把模板名直接说出来的句子`
  - `旁白先替读者把轻重说出来的句子`
  - `检查话漏回正文`
  - `局势翻译句`
- 这句必须改成哪一个现场东西：
- 本章最像“配角替作者翻译”的 1 句：
- 这句必须改成哪个角色自己的利益动作：

## 硬判规则

- 若删掉后事件仍成立，默认这句就别留在正文
- 若该句压在 `第二步收益 / 第二次外部反应 / 章尾前一拍 / 章尾最后一拍`，默认直接升 `F2`
- 若同一章连续两句都属于这类，默认按“检查话漏回正文”处理
