# 大纲怎么搭最顺手

第 3 阶段搭大纲时可参考这份。主要看大纲怎么搭起来、先后怎么排、关键点卡在哪、细纲再怎么往下接。

| 你现在写到哪一步 | 用什么方法 |
|--------------------|------------|
| 刚开始建大纲 | 五步大纲创建法 |
| 已经有大路要细化 | 节点设计法 + 三层搭法 |
| 设计单卷结构 | 八节点故事走法 |
| 做细纲/章纲 | 细纲与章纲节 |
| 卡文/逆推 | 推演与逆推方法 |

## 故事怎么分层

| 大概是什么走法 | 长什么样 | 一般什么时候用 |
|------|------|------|
| 一级 | 单元剧松散连接，靠时间串联，最后大高潮 | 入门 |
| 1.5 级 | 单元剧之间有线索串着，一路指向主要目标 | 进阶 |
| 二级 | 小高潮→意外下行→上行转折→大高潮链 | 高阶 |

## 五步把大纲搭起来

### 第1步：确定高潮剧情

先把高潮定住。一般就盯三件事：冲突最大、上场的人最多、情绪冲得最高。

先对一遍：
- [ ] 是否本卷最大冲突？
- [ ] 出场人物是否最多？
- [ ] 是否和这一段要打到的目标对得上？
- [ ] 读者情绪波动是否最大？

### 第2步：把一段段剧情先拆出来

每个单元剧 = 主角靠金手指优势拿到爽点。同类型重复 = 剧情套路，前后能接起来地成组出现 = 一整串剧情套路。

先按这几条来：
- 每个单元剧都得让金手指换个用法
- 相邻单元剧别连着用同一种金手指路数
- 单元剧之间得能接得起来，别各演各的

### 第3步：把几条线先排开，再顺手埋点

这里同时看 8 条故事线，每条线都顺手记一下该在哪埋东西：

| 哪条线 | 这一条写什么 | 顺手埋什么 |
|--------|------|----------|
| 地图线 | 层级地点 | 新地图解锁时机 |
| 阵营线 | 势力与地图匹配 | 阵营冲突爆发点 |
| 人物线 | 角色按类型分组 | 新角色登场时机 |
| 金手指线 | 获得的物品/技能 | 能力升级节点 |
| 世界观线 | 世界设定怎么跑 | 设定揭示时机 |
| 矛盾线 | 冲突放入矛盾网络 | 矛盾升级链 |
| 收集线 | 材料/线索收集 | 渐进式收集层级 |
| 感情线 | 感情关系进展 | 亲密度升级节点 |

### 第4步：先把开篇那块地方搭出来

按此顺序构建开篇：

```
情节钩子 → 迅速陷入异常状态 → 获得/介绍金手指 → 矛盾/欲望出现，主角运转金手指 → 目标建立
```

开篇得顺手把这些盼头埋进去：新目标、冲突接下来怎么变大、实力会不会往上跳、资源会不会升级、感情线会不会动、人脉会不会扩、局面会不会更乱、后面还有什么新线索新设定。

### 第5步：确定收尾

收尾前对一遍：
- [ ] 解决主线/副线高潮，展示后续影响
- [ ] 主角拿到的奖励够不够，另外有没有顺手给下段留一口更大的
- [ ] 给本卷角色合适的结局
- [ ] 节奏放缓
- [ ] 为下一卷内容做铺垫

## 剧情别写串了

### 先看看两段剧情像不像一回事

比较两段剧情是不是太像，就看这几样：冲突是什么、金手指怎么用、事情是怎么一路走过去的、最后收在哪里。

### 再看看有没有写歪

| 看哪一块 | 主要看什么 |
|------|----------|
| 进度对不对 | 剧情/感情线进度是否匹配本卷这一段 |
| 场景匹配 | 场景是否在本卷场景列表内 |
| 人物匹配 | 出场人物组合、关系、行为是否一致 |

### 这些别乱

- 时空关联：前后能接起来的推进 >= 1:3（每 `1` 个因果推进，最少配 `3` 个场景推进）
- 后续剧情情绪值不能显著低于前序
- 同一种金手指路数禁止连续使用
- 同类剧情套路间隔 >= 3个不同类型剧情

## 节点设计法

### 先把大路搭出来（起始→终点→关键节点）

- **起始状态**：主角初始状态（废材/穷困/弱小）
- **终末状态**：主角最终状态（无敌/首富/最强）
- **节点**：保证故事不偏离终点的关键成长点

### 这里这样定

- 开始与终末越有戏剧性越好
- 限定词：一个向上，一个向下（好的越好，坏的越坏）
- 流派 = 主角从开始走向终末的手段
- 节点确定后，大纲就是这些节点的排列

### 节点怎么一步步定

1. 确定主角起始状态和最终状态
2. 拆解走到终局需要经过哪几条路（提升境界线 / 提升实力线 / 收集线等）
3. 每条线设定子节点
4. 在节点间填充故事

## 大纲一般分三层

### 三层一般怎么拆

| 这一层 | 主要写什么 | 能动多少 |
|------|------|--------|
| 大纲 | 主体发展方向，几百字 | 尽量别乱动 |
| 剧情纲 | 完整故事线，相对细腻 | 可适度变化 |
| 细节纲 | 创意爆发点、对话、小场景 | 完全灵活 |

剧情要是开始偏，就得能拽回来。前面卡得紧一点，后面反而更好往实里写。

### 三种大纲写法

| 写法 | 一般什么时候用 | 核心怎么搭 |
|------|----------|----------|
| 地图式 | 玄幻 | 按地图更换写进度，标注实力变更/配角/大事件/爽点 |
| 分卷式 | 进阶 | 按主干剧情节点分卷，设起始点+终点+重大节点 |
| 填鸭式 | 创作初期 | 先把大架子搭起来，再往里填内容 |

### 大纲先后顺序

1. 先确定最顶上的创意点→挖掘最顶上的卖点
2. 根据最顶上的卖点定主线，金手指配合主线衍生
3. 金手指最好同时顶两件事：给主角开挂，也不断拉出目标去推主线
4. 金手指一旦脱离最顶上的卖点，整本书就容易写偏，发现了就得拽回来

## 细纲与章纲

### 细纲

- 细纲:正文 = 1:2.5~1:3
- 不展开具体情节，只关注目的和效果
- 每个节点标注"详写"还是"略写"
- 主要就是让后面一看就知道，这一段到底要写成什么样

### 章纲

- 直接看大纲和细纲够用就不必写章纲
- 按关键事件写粗纲（一个关键事件约3-5章）
- 标记钩子、伏笔、写作目的

### 顺写时记住这几句

- 滚动续写时细纲不要写太细，保留可调整空间
- 大纲是动态的，关键节点围着当前主打看点发展就行
- 写完检查：主角当前行动是否和目标有关；新章节是否建立了后续期待？

### 高潮式大纲写作法

- 每10万字设计一个大高潮
- 大高潮提前确定（200-300字写清楚），前面5-6万字用小剧情填充
- 小剧情必须能推进感情线或与大高潮有关
- 极限约40-50万字

## 八节点故事走法

这是长篇网文常用走法，每卷 / 每段大弧线都能套。每个节点可以跨多章。

### 八个节点怎么分

| 节点 | 这一段主要干什么 | 占比 | 章数 | 节奏 |
|------|---------|------|------|------|
| 1.开篇 | 抓住读者，建立期待 | 5-10% | 1-2章 | 快 |
| 2.发展 | 递进事件推进情节，铺设伏笔 | 30-40% | 5-15章 | 中 |
| 3.转折一 | 打破预期，拉升张力 | 5-10% | 1-3章 | 快 |
| 4.转折二 | 升级赌注，更深困境 | 5-10% | 1-2章 | 快 |
| 5.高潮 | 最重的冲突正面对决 | 10-15% | 2-4章 | 极快 |
| 6.矛盾结果 | 解决矛盾，给读者喘息 | 10-15% | 2-3章 | 慢 |
| 7.转折三 | 最终转折，投最后一弹 | 5-10% | 1-2章 | 快 |
| 8.结局 | 收束情感，留有余韵 | 10-15% | 2-5章 | 慢 |

### 每个节点记住什么

**开篇**：直接把人拽进事里，前 500 字就得有钩子。别从背景介绍、天气、风景起手。

**发展**：放 3-5 个一路往上顶的事件，每个都得推主线。每 3000-5000 字给一个爽点，最少埋 2 条伏笔。

**转折一**：丢一个能改局面的消息或事情出来。得同时有意外、说得通、还能把后面往前推。

**转折二**：再来一记更重的转折，赌注要比前一记更高，而且别和前一记是一个路数。

**高潮**：节奏拉快，句子收短，别灌水。主角得把前面积下来的成长打出来。

**矛盾结果**：主要矛盾先收掉，战利品用剧情带出来，别列单子，顺手给下一段埋钩子。

**转折三**：把读者以为的“要收了”再掀一下。前面得先铺过，不然会显硬。

**结局**：把最大的悬念收掉，把情绪往下按稳。别什么都解释光，留一点空更有味。

### 爽点节奏公式

- 每章最少 1 个小爽点
- 每3章解决1个冲突
- 每7章1个大爽点

### 每 3-5 章自己问一遍

1. **压力来源**：主角面临什么具体威胁？
2. **实力展示**：用什么能力解决？方式是否有新意？
3. **认知颠覆**：哪个认知被打破？反转是否合理？
4. **资源增值**：获得了什么？
5. **悬念增殖**：解决后引出什么新疑问？

## 爽文常见五步小循环

| 走到哪一步 | 这一段大概在干什么 | 记住什么 |
|------|------|------|
| 稳定态/危机潜伏 | 主角相对稳定，潜在危机存在 | 不一定是好状态 |
| 危机触发 | 外头把麻烦砸下来 | 一上来就得抓人 |
| 破局行动 | 利用金手指挫败阴谋 | 爽点爆发，可大量铺垫拉期待 |
| 收益结算 | 明确有价值的回报 | 资源/权力/声望/人脉/能力 |
| 新平衡与预埋 | 达到更高稳定态，新挑战出现 | 连接下一个循环 |

## 情绪拉扯五折线

| 段落 | 方向 | 内容 |
|------|------|------|
| 第1段 | 上行 | 铺垫/主角获得有利条件 |
| 第2段 | 拐点+下行 | 不利转折（不能是主角的错）|
| 第3段 | 上行 | 主角采取行动/出现转机 |
| 第4段 | 拐点+下行 | "虚晃一枪"——先放松后意外发生 |
| 第5段 | 爽点爆发 | 金手指发威，目标达成 |

十万字这个量级也能按五折线来排，每一段里还可以再套小的。卡文时，就顺着这条线往下推。

### 情节往下压时怎么写（先按这个走）

- "功都是主角的，锅都是别人的"
- 情绪下行原因绝不能是主角的失误
- 虐主受1点憋屈，报复10倍起步
- 下行后用歪打正着、贵人扶持、好人好报补充情绪缺口

## 推演与逆推方法

### 推演式大纲设计

从最先要定下的设定出发逐步推演：主角起点状态→家庭条件→金手指合理性→冲突来源。每一步推演都可能发现新剧情点。

### 倒着推章纲

顺序就按这个来：爽点→期待点→铺垫。先想清楚读者这一口要吃到什么，再倒着推前面该怎么垫。

六种爽点类型：能力碾压、目标达成、收获盘点、态度转变、隐藏身份/掉马甲、情感圆满度。

### 真相逆推法

1. 先确定那条真相
2. 将真相拆成碎片分散埋在不同章节
3. 每个碎片揭示时触发角色行动或转折
4. 高级变体：叙述性诡计——用信息误导读者

这套一般用在悬疑线、身份线、前世今生线、伏笔很多的故事里。

### 逆推记忆点法

将书分为【世界观】和【剧情】两部分。先定开头和结局记忆点，以记忆点为中心提问，每个问题衍生一条线。

分卷逆推：先想高潮点 → 以高潮为记忆点逆推 → 提问 → 逆推得出铺垫和主线 → 细纲便出。

## 章节名策略

1. 禁止用干瘪概括（"开始""觉醒""升级"）
2. 用最有梗的一句话或最冲突的信息
3. 铺垫章标题也要有噱头
4. 选择章节中最装x的一句话做标题
5. 避免"分析""局势""博弈"等字眼
