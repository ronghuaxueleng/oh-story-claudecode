## 矛盾怎么搭

### 矛盾四重递进

按此顺序递进架设矛盾层次：人与自我 -> 人与自然 -> 人与人 -> 人与世界。

### 先把最底下那层搭稳

从最根上、最掰不开的矛盾开始搭世界。最顶上的卖点要是跑偏了，中段很容易塌。

### 三种动机来源

| 来源 | 一般怎么落下去 |
|------|---------|
| 世界背景 | 设定环境迫使主角变强（如妖魔乱世->没实力就没自保之力） |
| 金手指 | 用金手指把好处和风险拴在一起（如斩妖获得实力） |
| 人物关系 | 通过亲朋遭遇把情绪点燃（如被欺负/被杀->复仇/救人） |

### 动机变化过渡

- 危机一收掉，就让配角或反派把“你还得更强才活得下去”这层意思再压上来，顺着带进下一段。
- 生死危机不适合所有题材--调整困难烈度匹配题材边界。
- 上一个危机刚收掉时，就得顺手埋下一个危机的伏笔。

### 矛盾来源分类

根据题材选择矛盾来源：

| 来源 | 一般什么题材爱用 |
|------|----------|
| 资源/利益 | 通用 |
| 阵营/种族 | 奇幻、玄幻 |
| 超凡途径 | 诡秘类 |
| 信仰/宗教 | 西幻、历史 |
| 政教之争 | 中世纪、权谋 |
| 理念/三观 | 所有题材 |

## 主线怎么往前排

### 高潮逆推法

主线可以按这个顺序往回推：

1. 先确定一句话结局。
2. 逆推各级目标（世界首富<--本州<--本国<--省<--市<--县<--镇<--村<--家庭）。
3. 大主线拆为中主线，中主线拆为小主线，不断细化。
4. 小主线完成后，中主线变为新的小主线，循环推进。

### 主线就是靠冲突往前顶

主线就是一路把麻烦往前推。把大麻烦拆成一串小麻烦，每一段都得有个眼下还啃不动的硬坎。

### 支线 = 完成主线的手段

支线就按这个顺序来：赚钱（主线）-> 怎么赚（支线）-> 用什么赚（金手指确认）。最后还得回到主线上。

### 双线怎么缠

#### 明线与暗线

- **明线**：主角线（读者直接看到的行动和成长）。
- **暗线**：反派线/隐藏元素线（目的：衔接、铺垫、嵌套后续剧情）。
- 暗线后面得和明线撞上，卷内高潮才立得住。

#### 四块怎么摆

| | 明线 | 暗线 |
|--|------|------|
| **主线** | 主角能力升级+那条主要剧情推进 | 反派暗中行动/隐藏信息线 |
| **支线** | 资源线、人脉线、地位线 | 配角暗线 |

**最好用的摆法**：明线有主线+支线（"二带一"），暗线只放主线。

到了高潮那一段，暗线和明线要并到一起，支线要么并回主线，要么先放一放。

### 升级文AB交织法

| 线路 | 这一条主要管什么 | 里头放什么 |
|------|------|------|
| A线（升级线） | 战力升级 | 加点、学技能、装备提升 |
| B线（剧情线） | 矛盾冲突 | 拉出矛盾->卷入矛盾->爆发冲突->解决冲突->收获 |

交织节奏：
```
变强 -> 拉出矛盾 -> 变强 -> 卷入矛盾 -> 变强 -> 引发冲突 -> 变强 -> 解决冲突 -> 收获 -> 变强 -> 拉出矛盾...
```

**先记这句**：让主角一直稍占上风，小赢一轮接一轮，最后再狠狠干大的一下。

丰富代入的方法：
1. A线中丰富：详细设定可掌握的资源、人脉、装备。
2. B线中丰富：插入世界观、环境、情绪、人物互动，给反派/配角贴标签。

## 冲突设计

### 拆成几道关，把剧情往长了拉

把大目标拆成一关一关的小目标，每一关都得有清楚的推力：

| 关卡类型 | 一般怎么卡 |
|----------|---------|
| 系统关卡 | 第一次免费升级，第二次需要资源，后续分普通/超级两档 |
| 目标关卡 | 考进顶级院校需要气血值+灭杀战绩+比赛排名，缺哪样就先补哪样 |
| 收集关卡 | 突破境界需要收集多个物品，一个一个去找 |
| 消耗关卡 | 资源太多时，就把往后走要花掉的东西加重 |

### 用三股劲把剧情往回推

| 驱动力 | 常见写法 |
|--------|---------|
| 戏剧性 | 设计意外收获、阴差阳错 |
| 矛盾 | 设计无法避免的冲突/灾难 |
| 欲望/恐惧 | 利用想得到什么/害怕失去什么 |

**怎么走**：先定主角这一段要长哪一块（金手指升级、武力提升、CP 好感、宝具、大佬人脉、金钱、地位），再拿这三股劲分别倒推能生出什么剧情。

好处：不容易卡文，往下推时也更容易出新东西。

### 冲突起码得写成什么样

#### 冲突最少得像这么回事

- 冲突得一路往上抬：言语->行动->激烈对抗->决定胜负。
- 冲突得有个清楚结果。
- 说到底，就是有人不让主角拿到他想要的东西。

#### 矛盾和冲突不是一回事

- “矛盾”是整本书、整卷在打的那股劲，“冲突”是落到具体场面里的那一下。
- 矛盾决定主线方向，冲突是具体表现。

#### 第一幕陷阱

主角在解决第一个冲突时，顺手把下一个冲突的引线埋进去。这样后面接起来才自然。

#### 超额收获技巧

超额收获是万能技巧：可以是后文伏笔、铺垫、情节钩子，本身出现时也是一个爽点。

经典模式：Boss爆三样东西--1.主角接着就能用的 2.可以送人拉关系的 3.暂时用不到但一看就很厉害的。

### 救赎文冲突设计八法

| 方法 | 一般怎么写 |
|------|---------|
| 救赎目标冲突 | 救赎来自对方牺牲；创伤制造者是对方至亲 |
| 救赎行为冲突 | 救赎靠谎言维持；非得伤到对方，自己这道坎才过得去 |
| 三角关系 | 救赎导致第三方需要被救赎 |
| 救赎倒计时 | 得赶在某件事发生前做完，不然就会彻底掉下去 |
| 增大救赎代价 | 付出代价越来越大 |
| 救赎资源稀缺 | 资源只够救赎一个人 |
| 立场对立 | 立场分歧但彼此需要对方帮助 |
| 性格反差 | 乐观vs悲观、冷血vs富有同情心 |

## 对抗设计

### 压势不压人

反派不针对主角个人，而是针对一个地区/阶层/群体的规划，客观上导致主角陷入困境。

一般就这么走：
1. 给反派设定自己的目的（如打压某地区修行者）。
2. 让主角身处被压迫环境。
3. 主角推翻反派后，下面那群人依旧麻木。
4. 下面那群人也可能顺手占主角便宜->新冲突。

好处：这类矛盾不是硬塞的，能一直往后接。

### 反派视角与信息差技巧

#### 这一段主要拿来干什么

转视角到反派密谋，制造读者知道而主角不知道的信息差：深化反派人设、拉期待感、让读者期待反派被打脸。

#### 写法

- 不要谜语人--大大方方写出反派计划。
- 信息差 = 提前让读者知道部分信息，制造期待。

#### 暗线与嵌套

- 暗线主要就是拿来接后文、垫后文，把后面的事先藏进去。
- 暗线后面得和明线撞上，高潮才会真顶起来。

### 主角行动力设计

主角的行动，一般从这三块去想：

| 看哪一块 | 是什么意思 | 例子 |
|------|------|------|
| 做别人不敢做的事 | 反抗权威，挑战强者 | 斩上司 |
| 做别人做不到的事 | 死里逃生，超越极限 | 必死局面生还 |
| 做别人没想到可以这么做的事 | 阴差阳错，曲线救国 | 用烤肠秘籍凑收藏 |

戏剧性是展现行动力的方法，人设是行动的缘由。不能为了意外崩人设。

## 感情线设计

### 事业线与感情线配合

先判断：看书名简介吸引进来的读者想看什么。

| 模式 | 设计重点 | 读者想看 |
|------|---------|---------|
| 事业线为主 | 金手指展示和升级，恋爱对象是奖励 | 主角变强 |
| 感情线为主 | 关系变化，情节根据对方人设设计 | CP互动 |

### 爱情线常见四步走

按以下顺序推进爱情线：

| 走到哪一步 | 这一段最该写什么 | 下手重点 |
|------|----------|---------|
| 萍水相逢 | 初次相遇，第一印象 | 用差异评价制造张力 |
| 爱情喜剧 | 被迫/偶然频繁接触 | 试探那一段最要紧，占 30-50% 剧情 |
| 爱隔山海 | 外部压力/身份真相/误会分开 | 烈度要高，人设变化基于原人设 |
| 大结局 | 克服最大阻碍 | 不要再设计新矛盾，专注发糖 |

#### 试探那段怎么写

试探那段是爱情线最有味的时候：两个人都不敢把心意说死，就一直来回碰、来回退。每次试探都得让人既觉得有点甜，又觉得发酸。

#### 人设变化对冲法

爱情线高潮后关系升级：**双方都改变**，改变后与之前的对应关系**反转**。人设变化基于原人设部分改变，不是凭空创造新人设。

## 金手指与身份

### 开篇与切入点设计（双身份双金手指）

| 要素 | 这一项要看什么 |
|------|---------|
| 显性身份（社会身份） | 摆在明面上的标签=职业+状态。得一直往前变 |
| 隐性身份（身世） | 流落凡尘的皇子、穿越者等。与全书最重的矛盾联系 |
| 显性金手指 | 系统、重生记忆等。最好是一整套能力，每次都能翻出点新东西 |
| 隐性金手指=主角性格 | 让主角与众不同的就是金手指。中后期提供爽感 |

**四点统一**：社会身份、身世、金手指、性格这四样最好拧成一股绳。

### 金手指设计要点

#### 这五问先过一遍

设计金手指时，把这几问过一遍：

1. 能否凸显主角与众不同？能否不断协助克服困难？
2. 和世界环境、设定之间是否有紧密联系并拥有反差？
3. 是否有喧宾夺主可能？是否可预防？
4. 主角资源提升百倍千倍时，是否还能解决主要矛盾？
5. 能否通过金手指解决矛盾的同时表达思想和主题？

#### 碎片化处理

把复杂的金手指拆开，一片一片塞进剧情里。没收齐之前，它就是后面的盼头；每拿到一片，最少也得顺手解决一个冲突。

## 循环与爽点

### 爽点层次设计与循环写法

#### 循环（套娃）设计

先有一个最亮的点 -> 顺着这个点往外挖 -> 钉主线 -> 让金手指跟着主线长出来。

#### 并列式写法

在剧情A的铺垫中穿插剧情B，B的铺垫中穿插C，保持铺垫和爽点相续不断。

#### 多层爽点

按以下层次叠加爽点：

1. 当场观众震惊
2. 别处观众震惊
3. 专家看到手法后震惊
4. 配角信息差抹平后再震惊

爽点 = 主角装逼 + 配角认知变化。把注意力放在配角的前后变化上。

### 最顶上的梗三分法

| 类型 | 怎么跑 | 一般用在哪 |
|------|----------|------|
| 可重复型 | 每个一级结构里反复跑 | 升级文、系统文、文抄娱乐 |
| 不可重复型 | 贯穿全文但只在最高潮爆发一次 | 追妻、掉马、禁忌之恋 |
| 长期铺垫型 | 以期待点形式贯穿 | 悬疑、复仇、解谜 |

运行节奏：第一个一级结构里就得跑一次，或者先把第一层铺垫放进去。连着好几个一级结构都不动它，读者就容易跑。

双线构思法：布两条线，一条反复跑最顶上的梗，另一条不重复，交替推进。

### 按题材设计循环主轴

根据题材定清循环主轴，并确保能一直跑下去：

| 题材 | 循环主轴 | 持续条件 |
|------|---------|----------|
| 肝熟练度/低位开局 | 地图（训练场/地头蛇/破坏者） | 地图不崩 |
| 长生 | 社会变迁 | 社会一直在变 |
| 无限游戏 | 副本与现实 | 副本现实不崩 |
| 公路文 | 人际交往 | 人物不崩 |
| 无敌流 | 敌人 | 敌人不崩 |
| 种田/签到/囤积 | 外部环境破坏 | 外部持续被破坏 |
| 脑洞文 | 主打卖点循环 | 主打卖点不偏 |

**提醒**：循环一旦偏离主打卖点，这本书就容易没辨识度。

## 动机与困境

### "给人物一个梦想"动机法

| 说的是哪种劲 | 长什么样 | 例子 |
|------|------|------|
| 动机 | 外力推动，趋向物质 | 因为穷->想买大房子 |
| 梦想 | 内在驱动，追求精神 | 想当伟大的作家 |

写的时候，别只有外头逼他、利害推他，再给人物一个“自己真想要”的东西。两股劲拧在一起，人就活了。

人物目标得和人设对得上。杂役弟子张口就要拯救宗门，会发飘；宗门圣女去扛这事，就顺。

### 困境循环怎么转（极道流）

```
困境（环境压迫）-> 行动（运行金手指）-> 转折（更大困境）-> 行动（更强实力突围）
```

先记这句：不停给主角上强度，主角始终主动解决问题。搭配扮猪吃虎、情绪拉扯、装逼打脸。

记住：压迫别压死，得让读者觉得主角还有翻盘的口子；后面的压力也得比前面更重。

### 设定推演剧情法

最该记的一句：**环境 + 人物 = 情节**

一般就这么推：
1. 给出环境（时间/地点/情境）。
2. 给出人物（身份/关系/动机/弱点/梦想/过往）。
3. 每增加一个设定点，就代进去顺着往下推一遍。
4. 筛选最符合目标情绪的设定组合。

### 噱头-主线-最大爽点怎么拴一起

#### 噱头大概分哪几种

| 类型 | 长什么样 | 开篇怎么起 |
|------|------|---------|
| 事件噱头 | 书名就是事件 | 一上来就进入事件 |
| 金手指噱头 | 分布在全文，前期多后期少 | 先描写主角->困境->目标->金手指施展->少许成果 |
| 人设噱头 | 人设直接影响剧情 | 人设本身产生戏剧性 |

**记住**：事件噱头和金手指噱头不是一个写法。拿写金手指那套去写事件，读者很容易跑。

#### 噱头->主线嫁接

事件结束后**立马嫁接**主角目标。也可以在事件进行中就铺垫目标。

#### 最大的爽点，就是卡在主线上的爽点

偏离主线的装逼，只能算跑偏的爽点，不算最大的爽点。

#### 人物摆出来的劲怎么拿

主角被挑衅时，回过去的那一下得和他的实力、阅历对上。实力高的人，更适合轻轻带过去，不适合当场炸。可以这么看：高实力→轻描淡写/一指灭杀；低实力→先忍着攒劲。被骂时的反应可以这样分：
- 毒点：气得要死，面红耳赤
- 不毒：微微一笑--"他什么都不是"
- 歇斯底里->不爽；风轻云淡一指灭杀->爽

## 剧情组织

### 信息怎么一层层往外放

| 类型 | 常见走法 | 一般什么地方用 |
|------|------|------|
| 直白剧情 | 提出疑问->公布答案 | 基础叙事 |
| 探索剧情 | 提出疑问->正常提示->公布答案 | 铺垫段 |
| 意外剧情 | 提出疑问->虚假提示->公布答案 | 反转段 |
| 意外+反转 | 提问->假提示1->假提示2->公布答案 | 高潮段 |

信息只有先变成“疑问”，读者才会盯着看。每个事件最少得带来一个价值变化，也得能往主线上推；什么都没改的事件，就是水。

### 主角生活路线扩展关系网

确定主角日常活动路线->在日常接触点引入新角色->新角色自带关系网->新的故事素材。

### 分步走和缓一口气的地方

- 缓冲区 = 主角和配角间因利益/事件/感情产生的距离，随着距离被攻略产生爽感。
- 目标一段一段拆开（类似练气/筑基/金丹），不只大目标，小目标也能拆。
- 剧情推进三种模式：主动处理推进目标的麻烦 / 主动推进目标遇到麻烦 / 被找上门的麻烦反而推动目标。

### 事件怎么一路把剧情往前带

- 正文章节得靠事件撑起来，事件内容最好占到一半以上。
- 事件是价值改变的契机：没有事件，主角和主线不会改变。
- 设定尽量通过事件演绎，而非旁白强塞。

### 基于设定推演冲突

先后顺序：
1. **先有趣再有用**：先确保事件有戏剧性。
2. **拆出来的情节就是戏剧性**：拆书时有爽点的事件直接用。
3. **不够有趣时做铺垫拉期待**。

## 最后自己对一遍

矛盾和主线搭完后，把这张单子过一遍：

- [ ] 矛盾是否按四重递进（自我->自然->人->世界）有层次推进
- [ ] 最根上那道矛盾是否和世界观绑紧，没有偏离最顶上的梗
- [ ] 主角三种动机来源里，最少有没有占到两种（世界背景/金手指/人物关系）
- [ ] 主线是不是已经用高潮逆推法拆成大/中/小三层
- [ ] 冲突是否持续升级（言语->行动->激烈对抗->决定胜负），且有明确结果
- [ ] 每一段里有没有一个眼下啃不下的硬冲突，能把读者继续往下拽
- [ ] 支线最终是否汇聚回主线
- [ ] 暗线（如有）是否在高潮与明线交汇
- [ ] 感情线（如有）是否按四步往前走，试探那一段是否占 30-50%
- [ ] 金手指是否把上面那五问都过了一遍，能力有没有拆碎了写进剧情
- [ ] 显性身份/隐性身份/显性金手指/隐性金手指四点是否统一
- [ ] 最顶上的那个梗，是否在第一个一级结构里跑过一次，或者先放进第一层铺垫
- [ ] 循环主轴是否匹配题材，是否偏离主打卖点
- [ ] 每个事件最少有没有改掉一处价值高低，并顺手把主线往前推
- [ ] 信息往外放的顺序，是否和场景类型对得上（基础/铺垫/反转/高潮）
- [ ] 动机过渡是否自然（上一危机解决时埋下下一危机伏笔）
