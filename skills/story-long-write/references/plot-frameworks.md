## 双线法与信息差

### 双线法先这么理解

- 主线：让主角在做事的过程中产生情绪
- 副线：让配角在挖掘主角身份/态度的过程中产生情绪
- 把主角的幕后身份、人脉、背后故事先写清，好把缓冲区搭出来
- 配角不断发现、不断被震惊就是一条很好的情绪线

### 信息差怎么用

- 信息差是并列式衔接的关键工具
- 主角知道读者知道但配角不知道 -> 配角发现时的震惊
- 利用信息差可以制造多层爽点
- 切换视角时用信息差做钩子，重新拉起读者兴趣

### 《作家之旅》式的情绪推动

- 先让主角走一条说得通的路（拒斥召唤）
- 通过一系列情绪拉扯、安全感打破，让主角选无可选
- 最终选A，但前后都得说得通，别生拗

## 分步骤与缓冲区

### 缓冲区是什么

主角和配角之间，因为利益、事件、感情，先隔出一段距离。后面一点点把这段距离吃掉，爽感就出来了。要搭这块，先把主角背后的事写清。

### 目标拆成几步走

- 主角始终得有根本目标，再把目标拆成几步往前走
- 不止大的目标，当前小目标也可以这样拆
- 目的明确后行动才不会拖拉

### 剧情常见三种推法

1. 主角主动去处理能推进目标的麻烦事
2. 主角主动推进目标，过程中遇到麻烦
3. 主角被找上门的麻烦事，解决后反而推动了目标

### 计划-阻碍式走法

计划1 -> 意外1 -> 计划2 -> 意外2 -> 高潮

每一步计划都可能被打断，形成新的期待。

## 大架子怎么搭

### 大架子常常就是一条吊着人的主线

大架子就是拿来吊着读者的噱头。每次都快成了，又总差一点，悬念才不会断。像马甲曝光线：每次都被质疑，每次都能打破质疑，但总差最后那一步。

### 起承转合这条大路

| 走到哪一步 | 主要干什么 |
|------|------|
| 起 | 建立世界观、引出主角、给出金手指 |
| 承 | 主角利用金手指推进，填充大量小剧情 |
| 转 | 局势变化，引入更大的敌人或危机 |
| 合 | 大高潮解决那道主要矛盾 |

主要笔墨集中在"承"上。

### 小剧情怎么往里塞

- 大期待前的小剧情越多，追读越稳定
- 小剧情之间不能留空隙
- 并列式写法最适合拿来填满这些空隙

## 开头设计的要点

- 书名、简介、开篇得拧成一个整体
- 金手指出来后，剧情就得跟它死死咬住
- 代入感模糊时，就先把读者看得懂的社会权力关系摆出来，代入感会更稳
- 视角转变太多会导致不流畅
- 开篇场景要有动作、有信息、有画面感
- 从具体动作切入，引出金手指，再切入剧情

## 分镜与画面感

### 分镜先这么写

- 机位1：主角视角（主角看到了什么）
- 机位2：目标视角（主角做了什么反应）
- 不要同时搞太多机位
- 根据信息团要点进行脑内拍摄

### 分镜到底是干嘛的

分镜不是教怎么把一个信息团写长，而是让整块内容更具体更有画面感。先保证戏剧性，再考虑分镜。

### 画面感怎么补

- 规划一章只写主要信息，不要规划写到多少字
- 拉片这块记住：画面和文字中间隔着一道坎，得把视觉感转成文字感

## RPG怎么排奖励

### 升级文说到底就是RPG

要先想清反馈点怎么排：读者在哪一拍被喂到。

### 奖励点怎么穿进去

- 每一段都得给读者一点“奖励”——不只是主角升级，也可以是配角反应、环境变化、新信息揭示
- 奖励不能太密也不能太稀
- 奖励的样子别老一模一样，收获、认可、揭秘这些要混着来

### 飞升换地图怎么循环

- 换地图可以有多种理由：被追杀隐瞒身份、找宗门伪装、从外门弟子开始扮猪吃虎
- 每次重置都要带来新的信息和新的人设展开空间
- 不是简单的重复，而是螺旋上升

## 故事构型与套路运用

### 从原型故事到套路，常见三层

| 这一层 | 主要是什么 |
|------|------|
| 经典故事情节（具象化） | 起因、经过、结果都具体确定 |
| 戏剧单元（戏剧性抽象） | 保留前后怎么接的路子，可换壳重用 |
| 情绪模块（情绪抽象） | 读者想看什么的直接反映 |

### 戏剧单元有什么特点

- 戏剧单元越抽象，可变性越强
- 多个戏剧单元可以组合成"故事构型"
- 戏剧性会磨损：同类走法用多了，就没那么好使了

### 情绪模块有什么特点

- 戏剧性会磨损，但情绪不会磨损
- 当戏剧单元磨损后，可以通过强化或更换情绪模块来重组

### 正确使用套路 vs 错误使用套路

| 做法 | 结果 |
|------|------|
| 直接套用经典故事情节 | 故事雷同、人设崩坏 |
| 重构使用（更换故事构型） | 正确 |
| 微调使用（深挖人设，根据人设调整情节） | 正确，而且能让人物动机和故事前后更顺 |

长篇作品，尤其百万字级，最好会重构用法，不然很容易越写越像别人。

## 人物和剧情怎么咬住

### 人物带着剧情走

- 你是在替已经写出来的人物挑剧情走法，不是反过来为了走法硬造人物
- 如果先拼一个新人物，再倒过来找他该走什么剧情，容易写崩

### 感情线和事业线怎么并着走

- 事业线关键点：阻碍、危机、装逼、探明、取得、义举
- 感情线得有明确的戏，还得知道每一段怎么往前走
- 感情线需要层次感和阶段性，不是堆砌好看的桥段
- 不同题材的感情线写法有区别：战斗类写生死与共，不要硬塞都市约会
- 不同人设的女主感情线要做出区分，避免观感趋同

### 事业线与感情线的绑定

- 帮助女主和事业线目标完全绑定是高效写法
- 更泛用的写法：男主主观上忙事业线，顺便帮助了女主

### 以退为进

- 以退为进也是拉扯，只要情绪上行下行拉扯好，无所谓付出毒点
- 但只在必要的时候用，前提是先有大格局的装逼/帮助情节

## 主线怎么排，几拨人怎么轮着出手

**直接点说**：就是主线、支线、日常这三样怎么穿插着排。

**阵营手牌法**：像三国杀一样，不同阵营按回合出牌，每张牌带来反应后果，排序成主线。

**三个阵营**：主角阵营 / 敌人阵营 / 观众阵营

**线性剧情三层次**：

| 版本 | 走法 |
|------|------|
| 基础版 | 不同立场角色针对同一事件有不同态度 -> 加先抑后扬 |
| 爽点版 | 敌人逐级登场 -> B打败A -> C打败B -> 主角碾压 -> 全场震惊 -> 引出更大BOSS |
| 高阶版 | 主角幕后操纵 -> 提前攻略最终BOSS -> 己方故意认输放走敌人 -> 敌人信念崩塌 |

**别把人写憋坏**：主角吃瘪时，得从别的地方把情绪拉回来（揭开更深一层原因 / 补一个意外收获）。

**往上走时**：单线一直往前推，迟早会发平；真到后面，得让几条线缠着往上拧。

## 热门书长线一般怎么搭

### 单段剧情常见走法（每段10-40章）

1. **起因**：主角面临新矛盾/进入新环境
2. **发展**：金手指运用/结识新角色/获得新资源
3. **铺垫**：为下一段剧情埋线
4. **高潮**：装逼打脸/比武/夺宝的关键场景
5. **转折**：身份暴露/关系变化/真相揭示
6. **结局/收获**：战利品清点

### 多段剧情的排列规律

- 三种最能顶住读感的元素轮换：夺宝 + 美女 + 比武，三种轮着来，别一直写成一个味
- 每段结束后，收获最好明明白白摆出来
- 铺垫线和收获线交替

### 多线穿插的节奏

| 线索 | 频率 |
|------|------|
| 战力提升线 | 每段最少来一次实力跃升 |
| 装备收获线 | 每段最少拿到一件关键道具/技能 |
| 情感线 | 每段和最少一位女性角色把关系再往前推一步 |
| 声望线 | 每段最少给一个“震惊旁观者”的场景 |

四条线不同步推进，错开节奏避免同质化。

### 一卷怎么排

- 100章左右为一卷，卷末同时解决当前矛盾+开启新地图
- 卷的最后一幕同时是下一卷第一幕 -- 无缝衔接

## 九条主线套路法

### 主角三要素

| 要素 | 说明 |
|------|------|
| 目标 | 支撑主角行进的原动力，没有确切目标不是好主角 |
| 性格 | 单一化最佳，根据性格发展情节 = 上乘手法 |
| 修炼体系 | 照抄无压力，改头换面即可 |

### 上乘矛盾冲突写法

| 这是哪种 | 大概是什么意思 |
|------|------|
| 复仇 | 为亲人/门派复仇，关键 = 写出热血感 |
| 理念冲突 | 两方理念对立，最自然 |
| 门派恩怨 | 两大势力恩怨，主角夹其中不得不战 |
| 生存之战 | 为势力生存而战，不得不出力 |
| 上层恩怨 | 师父/上层人物与对方有恩怨，主角牵连其中 |
| 怀璧其罪 | 获宝物 -> 消息泄露 -> 引来追杀 |

### 下乘冲突（避免）

- 富二代故意欺辱 -> 翻来覆去 -> 读者烦闷
- 无故挑衅 -> 流于肤浅

### 套路与反套路

- 掌握套路是创新的前提
- 套路 = 主线，往里填充血肉，变化场景/人物/情绪，旧走法不变，内容变

## 同一路数怎么别写重复

### 通用升级装逼走法

要素固定：男主 + 男反（高逼格）+ 女人（奖励）+ 名额 + 时间限制

1. 烘托：男反高逼格表现，众人追捧
2. 扮猪：主角选择看似最差的选项，被人看低
3. 吃虎：最后一刻爆发，超越所有预期

关键：场景更换，套路不变，金手指/人物/剧情的变化会自然产生不同故事。

### 套路重复的三个层次

| 层次 | 说明 |
|------|------|
| 看点重复 | 市场上已有成功走法，选最好的拿来用 |
| 人物重复 | 给人物一个固定行为特质，在不同场景重复展现 |
| 剧情重复 | 最顶上的看点 + 人物看点结合后自然诞生的套路循环 |

### 避免审美疲劳

- 好的套路和写作，读者看不出有套路，但其实是有的
- 最高境界：看不出套路，但读者已被深陷其中

### 立Flag技巧

让配角说一句flag："这里面不会有怪物吧？""我做完这个任务就能回家娶老婆了"。有效就反复用。

## 玄幻怎么搭

### 整体怎么排

三大要素循环：夺宝+美女+比武 -> 美女+副本+比武 -> 副本+美女+比武，三种变化着来。

### 关键抽取点

- 节奏明快爽点密集，每个事件都有明确高潮和收获
- 女性角色人设立得深入人心
- 装逼舞台搭建层次分明
- 多线穿插：战力提升+装备收获+情感线+声望，紧密安排
- 每个事件结尾都有收获+铺垫下一事件

---

## 双线交替怎么搭

### 交替着怎么排

现实线和副本线交叉：现实线用主角不平凡+灵气复苏展开，副本线提升实力+收获。

### 三压一爆三震技巧

- 一压：友好势力 -> 觉得男主是大佬
- 二压：敌方势力 -> 两次铺垫 + 不服让男主上
- 三压：中立势力
- 一爆：男主出手碾压
- 三震：对三方势力的震惊反应

### 排行榜主要拿来干什么

1. 期待男主排名提升 -> 提供升级动力
2. 通过排行榜介绍新对手 -> 期待碰撞
3. 排行榜出现 -> 以后装逼有余震效果

### 底牌这块怎么留

手里始终保持3个以上底牌没用，每次只出1个，同时获得新技能。

### 伏笔怎么一层层展开

每一个不同之处/天赋都可以成为伏笔，开启一条线。

### 温酒斩华雄式

B打败A -> C打败B -> 主角打败C -> 引出大BOSS。升级文替换："台词"换"战斗特点"，"主持会议"换"登台"。

## 拆书怎么拆到章

### 长篇桥段怎么拆

300万字的书约等于40-50个单元，每个单元6-8万字，包含4-5个桥段，每个桥段约4章。整本书约200-300个桥段。

### 单桥段四章走法

| 章 | 内容 |
|----|------|
| 第一章 | 上半做代入（日常场景），下半做信息差（展示金手指可解决的困境） |
| 第二章 | 通过人物互动增强期待，结尾让主角开始装 |
| 第三章 | 兑现期待，把爽感写透 |
| 第四章 | 上半善后收尾+体现推进，下半开启下一个目标 |

### 先记这句

- 代入可以用 N+1 路子：用熟悉内容让读者接受新内容
- 信息差 = 最常用的拉期待方式
- 拉长 / 缩短 / 升级 / 反转 / 多轮拉扯 -> 可以灵活变化，但底下那条路别乱
- 拆书要用整体视角，将视野拉高俯瞰群山

## 故事说到底和六幕走法

### 故事的定义

- 事实 = 静态的呈现
- 事件 = 变化的事实
- 故事 = 对事件进行选择、排序、因果化的产物

### 因果链

因果产生意义 -- 意义是故事的灵魂。同样的事件顺序，不同因果链 -> 完全不同的主题。

- 铺垫说到底：在事件B发生之前，在事件A旁边埋下因
- 伏笔说到底：提前埋下远期事件的因
- 钩子说到底：在当前事件末尾埋下下一段的因
- 封死主角其他路 -> 他自然会走向你设计的那条剧情

### 四种组织事件的方式

| 方式 | 说明 |
|------|------|
| 筛选 | 筛掉与主线无关的事件 |
| 时间排列 | 顺叙/倒叙/插叙 |
| 因果路子 | 事件之间建立因果关系（网文 `99%` 用这类） |
| 主题/联想路子 | 按主题关联组织 |

### 开篇技巧为什么有效

列出开篇所有事件 -> 找到最有戏剧张力的事件 -> 安排在开头。

### 五幕式说到底

五幕 = 因果链的弯曲走法，把平直的因果链弯出节奏，节奏产生情绪。

| 幕 | 内容 |
|----|------|
| 第一幕 | 开始，埋下因的种子 |
| 第二幕 | 发展，因->果，果成为新因，主角被动承受 |
| 第三幕 | 转折，冲突性质升级（变糟） |
| 第四幕 | 行动，主角被逼到墙角，只能主动行动 |
| 第五幕 | 结局，冲突解决，意义浮现 |

### 三幕/四幕/五幕统一论

三幕、起承转合、五幕，说白了都是同一类走法，只是看冲突有多猛，再决定删哪段、留哪段。

### 转折最该盯的地方

转折的目的 = 增加冲突 -> 增加情绪。不是冲突变大，是冲突性质质变 -- 主角处境变得更糟。要做到"意料之外，情理之中"。

### 第六幕：余波（网文特色）

- 明线（实力）vs 暗线（社会认知）：每次明暗线失衡就是装逼机会
- 装逼 = 文本论证主角社会地位提高的过程

### 爽感来源图谱

| 爽点是哪种 | 它在回答什么 |
|------|-----------|
| 升级 | 我能变强吗 |
| 碾压 | 我够强吗 |
| 装逼 | 我的强被认可了吗 |
| 复仇 | 不公会被纠正吗 |
| 绝境逃亡 | 我能逃脱吗 |
| 种田 | 弱者该如何变强 |
| 苟道 | 弱者该如何生存 |

### 情节的定义

情节，说白了就是把事件前后接起来，再塞进五幕式里。主线是全书最主要那条因果线，支线是围着子题去跑的那几条线。

## 卖点循环和怎么拆细

### 卖点循环是什么

卖点循环 = 一套固定的剧情发展模式，不断换花样循环。排行榜上的书几乎都有这东西。它决定了整本书的观感基调。

### 常见卖点循环案例

| 作品 | 循环模式 |
|------|----------|
| 诡秘之主 | 灰雾空间 -> 迪化脑补 -> 交流情报 -> 交易材料 -> 赞美愚者 |
| 放开那个女巫 | 魔女被唾弃 -> 拯救 -> 现代知识震撼 -> 魔法训练 -> 教会追杀 |
| 无限流 | 主神 -> 轮回 -> 穿越 -> 完成任务 -> 获得奖励 -> 兑换 |
| 阵营法（传统） | 反派打压 -> 友军劝阻渲染 -> 中立观望 -> 打脸 -> 反应 |

### 细化法

从一句话主线到正文，通常这样一层层细化：

```
一句话主线 -> 粗略卷纲 -> 详细卷纲 -> 章纲 -> 正文
```

每一层都是把上一层再写具体一点。天然就会长出多级走法，不用再额外套一层大架子。

选走法时记住：这些都是给故事服务的工具，不是非得照搬的模具。你怎么一层层细化，最后就会长成什么样。

## 冲突怎么一层层往前顶

### 先记这句

冲突+情节+悬念 = 情感上令人满足的体验。悬念最重要。

危机感可用收获感替代：从"不做就死"改为"做了就获得"。

### 三种危机（死亡）走法

| 是哪种 | 例子 |
|------|------|
| 肉体死亡 | 叶凡被抓炼药 |
| 身份/职场死亡 | 萧炎天才陨落、被退婚 |
| 心理死亡 | 道诡异仙开头，"幻觉是不是真的" |

先定一种最主要的类型贯穿全局，其他类型拿来推情节就行，别抢戏。

### 开篇 = 从乱子开始

开局就得让主角遇上不平常的事。一个值得跟下去的主角，一般都是：他是谁 -> 遇到什么乱子 -> 他怎么处理 -> 读者想继续看他还会怎么做。

### “有进无出”这套走法

主角从安逸走到激烈，得有个硬理由。不是说主角表面上被谁逼了，而是**读者也会觉得他这一步非踏不可**。

期待感 = 有进无出 = 铺垫。所有期待都是在说"快看我之后要写这个"。

### "有进无出"的铺垫手法（三个层次）

| 层次 | 手法 |
|------|------|
| 基础 | 渲染敌人和主角的不共戴天 -> 塑造压迫感 |
| 进阶 | 写主角变得前所未有地强 -> 敌人恰好跳出来 -> 主角拿他试刀 |
| 更好 | 铺垫敌人背景，解决他引发什么大地震 -> 多重期待叠加 |

趋势：正向驱动（收获感/升级）比传统仇怨渲染更受欢迎。

### 网文 = 复数个乱子 + 复数个关口

传统文学一个乱子两个关口，网文不断重复关口，不断将大门塑造成有进无出。每个关口之间用收获感和升级感连接。

## 冲突里的死亡赌注和黏结剂怎么搭

### 四种常见黏结剂

| 黏结剂 | 说明 |
|--------|------|
| 杀人理由 | 反派有很硬的理由，非得把主角弄死不可 |
| 工作职责 | 侦探接案就得破案，宗门弟子接了任务就得做完 |
| 道德责任 | 亲人朋友遇险，主角无法袖手旁观 |
| 实体场所 | 特殊场所束缚对抗双方（副本、密室） |

如果读者感觉人物可以随时脱离困境，冲突紧张感荡然无存。

### 常见冲突有哪些

复仇、灾祸、反抗、恋爱阻碍、野心、身份矛盾、环境矛盾、生死危机、阶层矛盾、理念矛盾。

---

## 用矛盾推剧情，常见五种路子

| 是哪种 | 最要紧的点 |
|------|----------|
| 身份矛盾 | 靠身份自带的外部压力创造矛盾，天然明确目标 |
| 环境矛盾 | 外部压力直接创造矛盾，适合大格局小说 |
| 生死危机矛盾 | "打了小的来老的"，始终保持高压，经典推进走法 |
| 阶层矛盾 | 人的欲望一层一层往上抬，每一段想要的都不一样 |
| 理念矛盾 | 理想主义vs现实，高级手法，上限极高 |

### 进阶：序列唯一性设定法

- 扮演法：升级有风险，非得做 xxx -> 这样主角做支线也就有了理由
- 序列唯一性：你升级就触碰既得利益者的利益 -> 同序列超凡者全是敌人 -> 天然矛盾线

---

## 悬念与冲突的深化技巧

### 期待感 > 爽点

- 爽 = 情绪的释放过程，释放完就没了
- 读者的紧张/好奇/担忧/愤怒/憋屈比"爽"更持久
- 恋爱文最吸引人的是暧昧拉扯 -> 确认关系后反变无聊
- 教训：过于完美的情节解决 -> 损害长线期待感

### 悬念vs意外

- 意外 = 桌子底下炸弹突然爆炸
- 悬念 = 观众听到定时炸弹在滴答但不知道何时爆炸 -> 悬念远比意外有力
- 章末断章留悬念 = 每章结尾尚未解决的危险

### 时间锁

限定时间内非得做完某件事，紧张感和节奏感都会一下提起来。要是不知道开篇怎么起，先试着挂一个时间锁。

## 剧情流怎么判断，强在哪

### 什么是剧情流

一本书在后期是否还精彩、"有劲"、有让读者津津乐道的高潮 -- 有就是剧情流。脑洞文高开低走容易崩，剧情流越到后期越精彩。

### 千均写作指南：作品五不崩

1. 目标不缺失：主角始终有明确目标
2. 卖点不减少：最顶上的卖点不能中途消失
3. 主角社会关系别空着：得有人际网络在互动
4. 上层地位不缺失：要有对上位者的追求和接触
5. 地位收获提升不能停：持续的成长感

### 千均写法：三线走法

- 世界线：世界观层面的宏大叙事
- 剧情线：具体事件和矛盾冲突的推进
- 升级线：主角实力/地位的持续成长
- 三条线交织推进，缺一不可

### 千均写作指南：装逼五步法

1. 主角很强但别人不知道
2. 渲染困境难度
3. 配角不看好主角
4. 主角用金手指轻松解决
5. 众人震惊

---

## 补充写法与剧情技巧

### 情绪怎么一层层往上带

成功作品往往是几股东西一起带读者情绪，不是只靠表面上一招鲜。

角色情绪别让人人都抢着带：每个角色都在引导，等于没人真在引导，所以得挑重点角色。

好的反转走法：先引导读者持续讨厌某个角色，再在关键事件里揭开他更高的一面。

### 小说四维自检法

| 看哪一面 | 主要查什么 | 常见翻车样子 |
|------|----------|-------------|
| 思想-形式 | 有无内核？读者能感受到什么？ | 完全没有思想，纯流水账 |
| 题材-风格 | 是否发挥了题材本身的优势？ | 元素化表象化 |
| 组织-走法 | 叙事手法是否多样？人物是否立体？ | 平铺直叙，人物脸谱化 |
| 文本-工艺 | 文字是否流畅？修辞是否得当？ | 白话连篇，缺少修饰 |

小白文可以在某个维度上“白”而拿到好效果，但四项全白就直接扑街。写法上的“白”有时反而是优势，直白冲突比太花的设计更容易带起大众情绪。

## 质量检查清单

每次把大架子搭完，或者写完一段剧情后，就用这张清单过一遍。逐项打勾，哪条没对上就改。

### 走法这一层

- [ ] 主角始终有明确目标（当前小目标 + 长线大目标都清晰）
- [ ] 当前段落有起因→发展→高潮→收获的完整收口
- [ ] 段落结尾同时完成两件事：收获清点 + 铺垫下一段
- [ ] 大架子里有“欺骗式主线”吊着读者（每次接近完成又差一点）
- [ ] 主线和支线错开节奏推进，没有同时爆也没有同时空转

### 冲突与悬念层

- [ ] 冲突有明确的黏结剂（读者认为主角不可能随时退出）
- [ ] 每章结尾有未解决的悬念或危险（断章留钩子）
- [ ] 期待感 > 爽点：铺垫的篇幅不少于释放的篇幅
- [ ] 主角吃瘪时从别的地方把情绪拉回来了（揭开更深一层原因 / 补一个意外收获）

### 爽感与节奏层

- [ ] 升级文：每一段都有明确奖励点（升级/装备/认可/揭秘最少一项）
- [ ] 装逼文：有“三压一爆三震”或“装逼五步法”这种节奏走法
- [ ] 感情线：有拉扯有层次，不是堆砌桥段
- [ ] 套路重复时更换了场景/人物/情绪，读者看不出旧走法
- [ ] 没有连续两段使用相同的主要元素组合（夺宝+美女+比武要轮换）

### 人物层

- [ ] 角色行为由性格驱动，不是由剧情需要倒推
- [ ] 配角有自己的立场和动机，不是单纯出来完成剧情作用
- [ ] 感情线和事业线有绑定或交叉，不是两条独立线
- [ ] 底牌这块：手里始终留着 2-3 个还没掀开的底牌

### 全局不崩检查（五不崩）

- [ ] 目标不缺失：主角始终有明确目标
- [ ] 卖点不减少：最顶上的卖点没有中途消失
- [ ] 社会关系不空白：主角有互动的人际网络
- [ ] 上层地位不缺失：有对上位者的追求和接触
- [ ] 成长不停止：地位/实力的提升持续给出
