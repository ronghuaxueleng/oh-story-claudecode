# `writing-red-lines.md`：长篇正文红线总表

本文件不是灵感参考。

本文件是执行红线。

命中任一红线时：
- 不得继续顺写
- 不得并入主正文
- 不得只做词语替换式修补
- 不得跳过 `scripts/scene_lint.py` 的写后质检

---

## 一、正文禁止生成顺序

### 红线 1：禁止效果词先行

错误顺序：
- 先想“这一段要狠、要炸、要压、要有气势”
- 再用抽象效果词、判断词、气氛词去顶正文

正确顺序：
1. 谁在动
2. 动作打在何处
3. 造成什么可见后果
4. 对方如何失态、受伤、变形、退缩、反扑

以上四项没落地前，不准写效果判断。

判死标准：
- 删掉效果词后，段落立不住
- 说明动作层、后果层、疼感层没有写出来

### 红线 1.5：禁止用口头发力词起稿

错误顺序：
- 先想“这里要狠一点、猛一点、炸一点”
- 再用口头发力词或空力度副词把句子顶过去

正确顺序：
1. 先写谁冲上去
2. 再写打在什么部位
3. 再写人或物怎么偏、怎么断、怎么翻、怎么退
4. 最后才允许极少量补强语气

硬规则：
- `references/banned-words.md` 中的 `一级禁用词` 属于首稿禁止生成项，不是写后再删项
- 口头发力词若承担动作主功能，正文起草时不得出现
- 二级敏感词若承担了动作主功能，也视为失手词

判死标准：
- 去掉该词后，句子只剩“有人动了”而没有受力点、位移、伤害结果
- 同一段或相邻两段连续靠这类词顶动作
- 该词承担了本句主要力度来源

唯一正确修法：
- 把力度拆回 `动作路径 + 受力部位 + 位移结果 + 失态反应`
- 不允许只换成别的近义力度词

### 红线 2：禁止作者先知道“结构”，再把结构话漏进正文

错误顺序：
- 先想“这里要撕开军权线”“这里进入第二阶段”“这里是第一卷节点”
- 再把这种后台判断漏成正文句子

正文只能承载：
- 人物当下看到的
- 人物当下听到的
- 人物当下感觉到的
- 人物当下做出来的

正文不能承载：
- 卷结构判断
- 章节功能判断
- 线索调度判断
- 写作复盘判断

## 二、正文禁止句型

### 红线 3：禁止后台调度语

正文不得出现：
- `这一卷/这一段/到这一步`
- `终于把某条线撕开`
- `真正进入某个阶段`
- `现在的问题是`
- `接下来就是`

这些不是现场，是后台写作语言。

### 红线 4：禁止作者盖章句

正文不得靠这些句子收口或定性：
- `这意味着……`
- `这说明……`
- `表明……`
- `证明……`
- `显示……`
- `从这一刻起……`

如果动作、对白、反应已经成立，就不准再补一句作者结论。

### 红线 5：禁止抽象效果句替代现场

正文不得主要依赖：
- `压迫感很强`
- `杀气逼人`
- `气势惊人`
- `场面可怕`
- `危险至极`
- `爽感炸开`

这些只能极少量辅助，不能承担主体表达功能。

### 红线 5.5：正文叙事句禁止让空效果表达承担主功能

以下表达一旦承担了句子主功能，就判失败：
- 抽象强弱词
- 抽象气势词
- 抽象危险词
- 抽象速度词
- 抽象判断词

判定方式：
- 删掉这些表达后，句子只剩空架子
- 说明动作、后果、失态、物理变化没有写出来

命中后不允许：
- 只删词
- 只换近义词
- 只缩短句子

唯一正确修法：
- 回到动作层
- 回到后果层
- 回到受伤/失态/位移/声响层
- 把偷懒表达替换成可见事件

### 红线 6：禁止提纲扩写句

正文不得出现：
- `真正的……`
- `值钱的是……`
- `不是X，而是Y`
- `原来……`

如果这些句子在一段里承担主要翻面功能，说明正文仍停留在提纲层。

## 三、正文失败后的回退顺序

### 红线 7：禁止只删词，不回句子功能

发现口号腔、解释腔、后台句时，回修顺序固定为：
1. 先看句子功能是不是错了
2. 再看场面层是不是没写够
3. 再看细纲是不是承载失败

不允许：
- 只删单个词
- 只换近义词
- 只把句子缩短

### 红线 7.5：写后必须跑正文场面质检

固定动作：
- 正文落盘
- 运行 `scripts/scene_lint.py <正文文件>`
- 按命中结果整句或整段回修
- 复扫通过后才能并入正文

如果没跑脚本，默认视为未验收。

### 红线 8：句子层、章节层、机制层不能混修

句子层：
- 解释句
- 作者盖章句
- 抽象效果句

章节层：
- 场面不足
- 爽点不落地
- 围观层/阻力层/代价层缺失

机制层：
- 题面滑轨
- 连续多章只查不兑现
- 副驱动反客为主

规则：
- 句子层才允许局改
- 章节层必须补场面和章肉
- 机制层必须回细纲或连续章节组

## 四、执行自问

写正文前，先问：
1. 这段是不是先有动作，再有感受？
2. 这段删掉效果词后还成立吗？
3. 这段有没有作者站出来替读者总结？
4. 这段是不是人物现场，而不是提纲后台？

写正文后，再问：
1. 谁赢了？
2. 谁亏了？
3. 谁被打疼、打丢脸、打失态？
4. 现实盘面改掉了哪一块？

任一问答不上来，默认不过线。
