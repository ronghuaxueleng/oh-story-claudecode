# 细纲设计与卷纲展开

> 本文件承接 `story-long-write` 中与 Phase 3 相关的细纲长说明、正反例、自检和口径反校验。写卷纲、写细纲、回炉细纲时加载。

---

## 一、大纲四检

每卷 / 每章设计前，先答：

1. 本卷交付什么情绪？什么剧情模式能可靠交付？
2. 本卷核心冲突是什么？
3. 卷节奏哪段加速，哪段减速？
4. 本卷需要新埋哪些伏笔？上一卷待回收伏笔怎么处理？

---

## 二、分批建纲硬规则

- 默认先建前 `10` 章细纲，再进入正文
- 每写完 `5` 章，滚动补齐后 `5-10` 章
- 全书章数较少（≤30）可一次建完
- 任一章不过线，停止扩写，先回炉该章或该章节组
- 连续 `3` 章及以上出现同类问题，不得继续往后铺章

---

## 三、单章细纲写法

单章细纲不要从“这一章发生什么”起手，要从“这一章必须交付什么”逆推。默认按以下顺序写：

### 1. 先钉卷任务，不先写场面

先回答：
- 这一章服务哪一个卷级任务 / 章组任务
- 这一章如果删掉，会导致哪一口推进失效
- 这一章最值钱的战果，到底落在命 / 人 / 名 / 权 / 物证 / 站队 / 规矩变化中的哪一类

如果这三句答不出来，先不要写正文级细节。

### 2. 再钉主收益，不先钉过程

细纲不是先写“怎么打”，而是先写：
- 本章主驱动
- 本章题面兑现
- 本章副驱动
- 本章第一爆点
- 本章高潮后续收益
- 本章最后 `1/3` 的现实翻面

先把“要拿到什么”写死，再决定“怎么拿到”。

### 3. 用情节点串起章节骨架

情节点序列至少先写出：
- 谁先动
- 对手怎么拦
- 主角怎么破
- 哪一拍出现外部反应 / 群体站位变化
- 章尾停在哪一拍

情节点的职责是把章节从“概念”写成“能施工的骨架”，不是用来写空泛摘要。

### 4. 最值钱的场面，单独拆卡

章节里最值钱的 `1-2` 个硬场面，不要混在情节点里一笔带过，必须单独拆成 `关键硬场面卡 / 动作拍序`。

推荐拆卡顺序：
- 这场为什么会爆
- 谁先手
- 真正的爆点 / 截堵点在哪
- 主角预判依据是什么
- 中间 `4-8` 拍动作怎么连
- 哪一拍最险
- 这场打完现实盘面改了什么

### 5. 最后才补白名单和禁增项

等章节骨架和场面卡都钉住后，再补：
- `本章新增信息白名单`
- `本章正文禁增项`

这样白名单是在给章节服务，不是在给正文临场发挥开口子。

### 6. 写正文前，先问一句

写完细纲后必须再问：

`如果正文阶段不允许我临时新增任何核心信息，这章还能不能完整跑通？`

如果不能，问题不在正文，在细纲。

### 常见错误顺序

以下顺序默认高风险：
- 先想几个酷画面，再倒推本章作用
- 先写“对手来抢 / 主角反打 / 最后翻车”，再试着补过程
- 先写门口对话、氛围、解释，再临场补动作
- 先把正文写出来，再回头补细纲

### 推荐顺序

推荐固定按：

`卷/章组任务 -> 本章主收益 -> 情节点骨架 -> 关键硬场面卡 -> 白名单/禁增项 -> 正文`

### 7. 世界显影要先写进细纲，不许只留给正文临场补

世界观不是只写在蓝图里的背景板。单章细纲必须提前写清：
- 本章要让读者新看见世界的哪一块
- 这块世界信息靠哪个场景载体落地
- 它服务的是哪条当前冲突，而不是独立说明
- 哪些更高层世界信息本章禁止提前抬

默认分工：
- `开书总蓝图` 管世界总控与投放分期
- `章组控制` 管阶段世界显影任务
- `全书章节总表` 管每章世界显影点
- `单章细纲` 管本章世界显影任务 / 载体 / 白名单 / 禁越权
- `正文` 只负责把这些显影点通过冲突和动作落地

若细纲写不出“本章世界显影点”，正文大概率会出现两种坏结果：
- 要么只推剧情、不扩世界
- 要么临场发明新设定，污染正文

## 五、细纲写作硬要求

细纲必须像剧本，不像提纲。

必须先钉死：
- 谁先动，谁后动
- 每一步动作落在哪个现场
- 阻力是什么
- 最值钱收益是什么
- 对手丢掉什么
- 谁失态、变脸、露底、站队
- 物证 / 命 / 名 / 权 / 规矩变化落在哪一拍
- 章尾停在动作 / 物证 / 失态 / 压制 / 翻面里的哪一拍

如果只写“反杀 / 对质 / 拿证 / 翻盘”这类骨架词，不算过线。

本章世界显影也必须具体，不得只写：
- 世界观继续展开
- 体现世界残酷
- 交代新的设定
- 暗示更高层势力

必须写成：
- 哪个新规则被看见
- 哪个新资源口被看见
- 哪个新区域 / 新机构 / 新阶层接口被看见
- 它是通过谁的动作、哪场冲突、哪一拍结果被看见

本章最值钱的 `1-2` 个硬场面，必须额外拆成 `场面卡 / 动作拍序`，至少写清：
- 谁先发现 / 谁先动
- 截堵点 / 爆点卡在哪
- 主角凭什么预判、若误判会死什么
- 中间 `4-8` 拍动作怎么连
- 哪个跟随者 / 对手在哪一拍出险或掉队
- 这一场打完，现实局势具体改了什么

---

## 六、最低标准

细纲必须达到“正文写出来不会改口径”：

- 物证归谁
- 关键台词由谁说
- 章尾停在哪一拍
- 第二口收益是什么
- 本章是否真的改掉一条现实规矩

## 七、正反例

### 反例

- 主角发现有人要杀他，随后反杀对方，拿到重要物证，最后留下悬念。

问题：
- 只有功能，没有场面
- 没写谁先动、谁被压、谁失态
- 没写物证归谁
- 没写章尾停点
- 没写第二口收益

### 正例

- 动手的人借送水靠近，先用藏着的绳套探手，主角故意放他半个身位，等他半边身子探进安全区再反勒喉咙。
- 动手的人挣扎时，现场负责人带人赶到，想把尸体和信物一起带走。
- 主角当众指出动手的人身上有药和索，逼现场负责人亲自翻衣襟。
- 现场负责人摸出密信和私印残角，周围人开始变脸。
- 主角借众人看见的局面，逼现场负责人改口：今夜不得再单独处理主角。
- 章尾停在主角脑海里刷出第二条情报，指向现场负责人的下一步动作。

## 八、写后自检清单

- [ ] 能否说清谁先动手、谁后反应
- [ ] 能否说清最值钱收益最后在谁手里
- [ ] 能否说清阻力是谁、怎么拦、主角怎么破
- [ ] 能否说清第一爆点后的第二口收益
- [ ] 能否说清谁会失态、露底、变脸、站队或改口
- [ ] 能否说清章尾停在动作 / 物证 / 失态 / 压制 / 翻面哪一拍
- [ ] 能否说清本章最值钱硬场面的 `4-8` 拍动作链，而不是只有起点和终点
- [ ] 删除骨架词后，情节点是否还能单独成立
- [ ] 是否提前钉死物证归属、收益归属、章尾落点、规矩变化

## 九、情节点序列标准

每个情节点都必须至少交代：
- 谁在动
- 动作落在哪
- 造成什么可见变化
- 这一拍之后局势往哪边偏了一寸

### 反例

- 主角发现线索
- 双方对质
- 主角翻盘
- 拿到物证
- 敌人震惊
- 留下悬念

### 正例

- 动手的人把水囊探进遮挡后，左手按住主角肩膀，右手把囊口往他嘴边送。
- 主角继续装昏，等动手的人半个身子探进来，抬头撞鼻骨，同时翻起手边束具套住他脖子。
- 动手的人用膝盖顶开主角腰腹，手往后腰摸绳索，想先勒腕再补刀。
- 主角把脚边硬物勾进手里，从肋下捅进去，逼动手的人吐出“上头”两个字。
- 外头旁人听见动静掀开遮挡，先看见绳索和可疑药物，不敢立刻下手。
- 现场负责人赶到，想把尸体拖走，主角当众逼他翻衣襟，逼出密信和私印残角。
- 周围人都看见物证露面，现场负责人只能先改口，今夜不能再单独处理主角。
- 主角刚把这口规矩钉住，第二条情报刷新，直接指向现场负责人下一步动作。

---

## 十、情节点密度要求

- 前 `1/3` 尽快入局
- 中段持续有阻力变化
- 后 `1/3` 至少再拿一口现实收益

默认判法：
- 连续 `3` 个情节点只是“知道更多” → 不过线
- 第一爆点后只剩“众人震惊 / 对手变脸 / 黑幕更深” → 不过线
- 最后 `2-3` 个情节点只是“准备下一章做什么” → 不过线

---

## 十一、情节点模板

```markdown
1. {谁}在{现场位置}做出{具体动作}，先造成{第一层变化}
2. {对手/阻力方}用{什么手段}反压，局势转向{哪里}
3. {主角}抓住{物件/破绽/人心}，打出{第一爆点}
4. {围观者/第三方}出现{失态/站队/迟疑}，让盘面再偏一次
5. {主角}拿到{物证/名分/规矩变化/站队变化}，形成{高潮后续收益}
6. 章尾停在{动作/物证/失态/翻面/新危险}这一拍
```

## 十二、细纲与写前闸门口径反校验

正文开写前，逐项对照：

- 章节标题
- 核心对手
- 核心物证 / 核心收益
- 本章主驱动
- 本章最值钱战果
- 本章第一爆点
- 本章高潮后续收益
- 本章外显代价类型
- 本章硬场面
- 本章章尾钩子
- 本章目标字数

反校验提问：
- 最值钱的东西到底在谁手里？
- 第一爆点后第二口收益由谁拿走？
- 对手最想拦住的是物证、活口、规矩还是名分？
- 章尾最后一句前，现场已经成立了什么不可逆结果？
- 正文按现在口径写，会不会把“该主角拿的”写成“别人代拿”？

一票否决项：
- 细纲写“拿到”，闸门写“看到”
- 细纲写“规矩变化”，闸门说不清怎么改
- 细纲写“第二口收益成立”，闸门说不清是什么
- 细纲写“强钩子”，闸门却只能写成“下一章继续”
- 同一章分别描述不同对手、不同物证、不同收益归属
