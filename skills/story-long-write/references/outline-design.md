# 细纲设计与卷纲展开

> 本文件承接 `story-long-write` 中与 Phase 3 相关的细纲长说明、正反例、自检和口径反校验。写卷纲、写细纲、回炉细纲时加载。

---

## 一、大纲四检

每卷 / 每章设计前，先答：

1. 本卷交付什么情绪？什么剧情模式能可靠交付？
2. 本卷核心冲突是什么？
3. 卷节奏哪段加速，哪段减速？
4. 本卷需要新埋哪些伏笔？上一卷待回收伏笔怎么处理？

---

## 二、分批建纲硬规则

- 默认先建前 `10` 章细纲，再进入正文
- 每写完 `5` 章，滚动补齐后 `5-10` 章
- 全书章数较少（≤30）可一次建完
- 任一章不过线，停止扩写，先回炉该章或该章节组
- 连续 `3` 章及以上出现同类问题，不得继续往后铺章

---

## 四、细纲写作硬要求

细纲必须像剧本，不像提纲。

必须先钉死：
- 谁先动，谁后动
- 每一步动作落在哪个现场
- 阻力是什么
- 最值钱收益是什么
- 对手丢掉什么
- 谁失态、变脸、露底、站队
- 物证 / 命 / 名 / 权 / 规矩变化落在哪一拍
- 章尾停在动作 / 物证 / 失态 / 压制 / 翻面里的哪一拍

如果只写“反杀 / 对质 / 拿证 / 翻盘”这类骨架词，不算过线。

---

## 五、最低标准

细纲必须达到“正文写出来不会改口径”：

- 物证归谁
- 关键台词由谁说
- 章尾停在哪一拍
- 第二口收益是什么
- 本章是否真的改掉一条现实规矩

## 六、正反例

### 反例

- 林渊发现有人要杀他，随后反杀对方，拿到重要物证，最后留下悬念。

问题：
- 只有功能，没有场面
- 没写谁先动、谁被压、谁失态
- 没写物证归谁
- 没写章尾停点
- 没写第二口收益

### 正例

- 子时前，赵黑牙端水进车，先用牛筋索探手，林渊故意放他半个身位，等他半边身子探进来再反勒喉咙。
- 赵黑牙挣扎时，韩守忠带人赶到，想把尸体和信一起带走。
- 林渊当众指出赵黑牙身上有药和索，逼韩守忠亲自翻衣襟。
- 韩守忠摸出密信和私印残角，周围老卒开始变脸。
- 林渊借众人看见的局面，逼韩守忠改口：今夜不得再单独处理林渊。
- 章尾停在林渊脑海里刷出第二条情报，指向韩守忠的下一步动作。

## 七、写后自检清单

- [ ] 能否说清谁先动手、谁后反应
- [ ] 能否说清最值钱收益最后在谁手里
- [ ] 能否说清阻力是谁、怎么拦、主角怎么破
- [ ] 能否说清第一爆点后的第二口收益
- [ ] 能否说清谁会失态、露底、变脸、站队或改口
- [ ] 能否说清章尾停在动作 / 物证 / 失态 / 压制 / 翻面哪一拍
- [ ] 删除骨架词后，情节点是否还能单独成立
- [ ] 是否提前钉死物证归属、收益归属、章尾落点、规矩变化

---

## 八、情节点序列标准

每个情节点都必须至少交代：
- 谁在动
- 动作落在哪
- 造成什么可见变化
- 这一拍之后局势往哪边偏了一寸

### 反例

- 主角发现线索
- 双方对质
- 主角翻盘
- 拿到物证
- 敌人震惊
- 留下悬念

### 正例

- 赵黑牙把热水囊探进囚车，左手按住林渊肩膀，右手把囊口往他嘴边送。
- 林渊继续装昏，等赵黑牙半个身子探进来，抬头撞鼻骨，同时翻起镣铐套住他脖子。
- 赵黑牙用膝盖顶开林渊腰腹，手往后腰摸牛筋索，想先勒腕再补刀。
- 林渊把脚边木刺勾进手里，从肋下捅进去，逼赵黑牙吐出“上头”两个字。
- 外头老卒听见动静掀帘，先看见牛筋索和药水囊，不敢立刻下枪。
- 韩守忠赶到，想把尸体拖走，林渊当众逼他翻衣襟，逼出密信和私印残角。
- 周围人都看见物证露面，韩守忠只能先改口，今夜不能再单独处理林渊。
- 林渊刚把这口规矩钉住，第二条情报刷新，直接指向韩守忠下一步动作。

---

## 九、情节点密度要求

- 前 `1/3` 尽快入局
- 中段持续有阻力变化
- 后 `1/3` 至少再拿一口现实收益

默认判法：
- 连续 `3` 个情节点只是“知道更多” → 不过线
- 第一爆点后只剩“众人震惊 / 对手变脸 / 黑幕更深” → 不过线
- 最后 `2-3` 个情节点只是“准备下一章做什么” → 不过线

---

## 十、情节点模板

```markdown
1. {谁}在{现场位置}做出{具体动作}，先造成{第一层变化}
2. {对手/阻力方}用{什么手段}反压，局势转向{哪里}
3. {主角}抓住{物件/破绽/人心}，打出{第一爆点}
4. {围观者/第三方}出现{失态/站队/迟疑}，让盘面再偏一次
5. {主角}拿到{物证/名分/规矩变化/站队变化}，形成{高潮后续收益}
6. 章尾停在{动作/物证/失态/翻面/新危险}这一拍
```

## 十一、细纲与写前闸门口径反校验

正文开写前，逐项对照：

- 章节标题
- 核心对手
- 核心物证 / 核心收益
- 本章主驱动
- 本章最值钱战果
- 本章第一爆点
- 本章高潮后续收益
- 本章外显代价类型
- 本章硬场面
- 本章章尾钩子
- 本章目标字数

反校验提问：
- 最值钱的东西到底在谁手里？
- 第一爆点后第二口收益由谁拿走？
- 对手最想拦住的是物证、活口、规矩还是名分？
- 章尾最后一句前，现场已经成立了什么不可逆结果？
- 正文按现在口径写，会不会把“该主角拿的”写成“别人代拿”？

一票否决项：
- 细纲写“拿到”，闸门写“看到”
- 细纲写“规矩变化”，闸门说不清怎么改
- 细纲写“第二口收益成立”，闸门说不清是什么
- 细纲写“强钩子”，闸门却只能写成“下一章继续”
- 同一章分别描述不同对手、不同物证、不同收益归属
