## 人物关系类型

可先查表把关系类型定下来，再往下写。

| 关系类型 | 这里说的是什么 | 这层关系拿来干什么 | 示例 |
|---------|------|------|------|
| 冲突型 | 双方利益/理念对立 | 制造张力，推动情节 | 宿敌、竞争对手 |
| 联盟型 | 双方有共同目标 | 提供助力，制造羁绊 | 战友、师徒 |
| 亲密型 | 情感纽带连接 | 制造软肋，提供情感支点 | 恋人、家人、兄弟 |
| 权威型 | 上下级/支配关系 | 制造压力，限制主角行动 | 师父、老板、监管者 |

**关系规则：**
- 每个重要关系最好都安排一次考验（背叛/牺牲/误解）
- 关系要有变化弧线（敌人变盟友、盟友变对手）
- 别把所有关系都写成“你好我好”的铁板一块
- 关系这层作用得服务情节，别只为了甜或虐而存在

## 感情流人设怎么搭

感情流里，剧情是给人设服务的。每次剧情不是在丰满人物，就是在推动感情，不然就是空转。

### 构建步骤

1. **先定人设最抓人的那一下**：写下最初出现在脑海里的角色特征（如"想登上皇位的皇子""病弱皇子"）
2. **围着这一下发散**：回答——为什么会这样？过去经历？环境影响？性格成因？
3. **顺着这一下往外长剧情线**：这一下自带的剧情得写出来（有目标→目标线；有伤痛→治愈线）

### 写的时候记住

- 俗套剧情配上丰满人设也能脱离套路感
- 两个主角都得有闪光点和独立变化路线，别一个人出彩另一个人发灰
- 高光不等于装逼——所有让读者对角色印象深刻的时刻都是高光，包括痛彻心扉的时刻
- 人不止有恋爱，角色心里可以想着恋爱，但人生里不能只剩恋爱，不然会很薄

## 男频与女频爱情线差在哪

**先定读者是谁，再选对应写法。两边别混，不然读者会觉得“不对味”。**

### 男频爱情线怎么转

| 维度 | 说明 |
|------|------|
| 最抓人的点 | "外在因素展示"——主角和对象在一起体现两人的外在因素多优秀 |
| 三种常见走法 | "她要是我的该多好" → "这么优秀的人是我的了" → "这么优秀的人都喜欢我，我更优秀" |
| 围观者想法 | "能得到这么优秀的女人，我好羡慕"（非"恋爱好甜"） |
| 女主 | 最好有多个女性角色喜欢主角，越多说明主角越优秀 |
| 对象到底是什么 | "奖杯"——一切动机最后都指向胜利 |

写男频感情线时，事件最好围着“展示优秀”来排。

### 女频爱情线怎么转

| 维度 | 说明 |
|------|------|
| 最抓人的点 | "连接"——两人之间唯一、坚不可摧、最优先的情感连接 |
| 连接要求 | "你不是因为我的外貌财富才华才爱我，而是因为我是我" |
| 纯洁性 | 连接形成后应逐渐舍去外在因素影响 |
| 双方 | 最好初恋+双洁——保障连接的纯洁性和唯一性 |
| 围观者想法 | "他们的恋爱好甜，我好羡慕" |

写女频感情线时，事件最好围着“深化连接”来排。

## 穿书/穿游戏角色怎么挑

### 先满足这几条

- 叙事主角最好对原世界足够熟（这是期待感的重要来源）
- 确定穿越进熟悉的世界，一两句话交代清楚，不能超过一章还不知道身处何方
- 最好挑穿越后自带强戏剧性或强矛盾的角色（如穿越成恶毒女配）

### 信息差设计

- 在原世界基础上适当设计信息差，为叙事主角提供探索空间
- 可以用原世界知识卡bug获取超额收益

## 修罗场收场策略

修罗场本身就是矛盾。对抗烈度要控住，别让爱情线对象之间闹到彻底收不回来。

| 收场方法 | 操作 |
|---------|------|
| 关联回事业线 | 让爱情线对象们把争风吃醋转化为"比谁对主角事业线贡献更大" |
| 插入事业线突发事件 | 对抗进入白热化时，用突发事件让所有人从内斗切换成一致对外 |
| 一笔带过 | 最多两人互相看不惯、说两句嘲讽话就过去，非常克制 |

## 傻白甜角色塑造避坑

### 六个最好别踩的坑

1. 女主犯错不能是故意的，不能明知不能做偏要做
2. 女主犯错不能违反道德——傻白甜最重要的是天真小孩子般的高道德感
3. 不能站在道德高地指责他人（道德婊行为），自己却做不好
4. 不能因为道德婊行为和队友发生矛盾后还让队友认错赞美她
5. 更顺的写法：她心里那道坎更高 → 做出来也真是那样 → 所以才显得和别人不一样，甚至有点傻
6. 可以写成长线：坐到被她指责的人的位置上后，发现原来那个做法已是最好的选择

### 反向用法

塑造傻白甜反派时，把以上六点全加在她身上，让读者一见就厌恶。

## 人设改变的双向翻转法

爱情线中关系发生变化时的人设调整方法。

### 先看这个思路

- 最好的改变，是两个人都变，而且变完后刚好和之前那套相对着来
- 改变不是乱改，而是从原人设里长出来的

### 写的时候记住

- 人设变化发生在爱隔山海阶段之后（尤其是面对最大阻碍之后），不要再设计新的矛盾，只要发糖
- 真正的阻碍前面已经解决了，这时候读者最想看的就是多吃糖
- 发糖得靠实在的互动和照顾，不是靠“我爱你”这种工业糖精

## 角色行为自洽检查

### 第一步：排除外部强推

检查角色行为是不是只为了让剧情往某个方向走。如果角色“恰好”做了推进剧情的事却没有自身动机，就标记为“人设偏移”，得给这个行为补上角色层面的理由。

### 第二步：先把这段要打到什么情绪看准

写每段前明确三个要素：
1. 本段要打到什么情绪：这段剧情要让读者产生什么情绪？
2. 角色性格特征是否支持该情绪：角色的设定属性是否自然导向此情绪方向？
3. 角色行为是否符合使命定位：角色在本段的行为是否与其在故事中的功能定位一致？

### 第三步：从人设往下推行为

从角色已设定属性（经历/性格/目标/当前状态）推导其在场景中的行为：
- 列出该角色在此场景下的2-3种可能反应
- 评估每种反应与角色已有行为模式的一致性
- 选择最符合人设且最有戏剧张力的那一种

### 第四步：情绪再对一遍

检查角色表达出来的情绪主路，和这场想打到的情绪是不是一致。不一致就调角色行为或场景情绪，保证写出来别拧巴。

**常见两种走法**：
- 走法A：先定这场要什么情绪 → 按人设推角色怎么做 → 回头看前后有没有拧巴
- 走法B：先按人设推角色怎么做 → 再反推这场该落什么情绪 → 回头看前后有没有拧巴

## 配角攻略缓冲区

配角对主角态度怎么变，本身就是主角攻略配角的过程，这里天然带期待感和爽点。

### 缓冲区类型

线上线下、背后议论、异地相处、地位差距、亲密度差距、信任程度、信息差等。

### 操作步骤

1. 始终保持缓冲区存在
2. 在卷纲中挑出事件拐点（5~7个）
3. 在每个拐点处标注配角状态和对主角的态度变化
4. 每次攻略到关键点位时，配角的态度变化得写清楚
5. 态度变化本身就产生情绪波动和期待感

### 写的时候记住

- 配角别像 NPC 一样站着等主角来触发
- 配角要有自己的行动，由配角观点引出事件
- 正面角色也一样：和主角立场相同的人也应有自己的行动和动机

## 利用角色身份认知差制造冲突

同一个角色在不同人眼里的“声望”是会变的，不是定死的。

| 视角 | 看重什么 | 对男主的评价 |
|------|----------|-------------|
| 世俗视角 | 家境对等，抗风险能力 | 综合条件匹配度 |
| 男主自身 | 赚钱养家+感情 | 相对门当户对 |
| 女主视角 | 感情+专一+爱 | 只要在乎的就门当户对 |
| 富二代 | 外貌家世匹配度 | 我才更门当户对 |
| 路人 | 综合条件对比 | 女主该嫁富二代 |

**这里最该记住的是：**
- 不同人对同一个角色评价不一样，本身就是天然冲突来源
- 恋爱文里很顶用的一种爽点，就是这种评价差

## 亦敌亦友关系

这是最有魅力的一类人物关系。

### 写的时候记住

- 前提是两个人本身都得有魅力，不然就只是硬凑
- 最抓人的那一下，是高度认可和绝对冲突同时在
- 真正的宿敌，得是两个人自己都认，不是旁人替他们封的

---

## 竞争者定位与亲情线用法

### 竞争者（鲶鱼效应）

- 定位：给男主/女主增加紧张感上压力 → 推动攻略进度
- 重点一般放在高潮前那一段，用来铺垫
- 高潮过去以后，这类角色往往撑不了太多后续戏

### 亲情线常走四个方向

| 方向 | 作用 | 使用时机 |
|------|------|---------|
| 男方助力 | 提供金钱地位权力 | 高潮前发挥 |
| 男方阻力 | 设置障碍让主角得到认可 | 高潮前后都能用 |
| 女方助力 | 温柔乡感情补给站 | 高潮前发挥 |
| 女方阻力 | 父母不同意→为了认可去努力 | 高潮前后都能用 |

助力主要放在高潮前，阻力前后都能用，阻力更好发挥。

## 男频恋爱文写法攻略

### 读者为什么看恋爱文

| 价值 | 来源 |
|------|------|
| 情绪价值 | 填补"被需要、被在乎"的情感空缺 |
| 自尊价值 | 女主条件好→带出去有面子→配角羡慕嫉妒 |
| 高身份女主原因 | 更强的装逼打脸作用+更大的自尊满足 |

### 主技法：把恋爱当升级文写

- 暧昧拉扯的过程才是最有吸引力的
- 好感度进度条：每一步推进 = 升级文中的一个"等级"
- 每一个事件都是一次"升级"→ 靠冲突、化解、理解和共鸣把好感往上抬

### 好感度怎么往上走

路人 → 好人（扶老奶奶过马路） → 正直勇敢的好人（挺身而出） → 不错的朋友（理解原生家庭困境） → 喜欢但不知（特殊契机看到彼此不为人知的一面）

### 这里要卡住：主角别主动追求

- 当前男频读者普遍反感"舔狗"人设
- 推动好感靠男主自身优秀品质吸引，不是主动追求
- 男主帮女主是举手之劳不经意为之 → 女主因此记住他

### 感情升级的不对称性

- 两条进度线：表面社会关系（路人→朋友→情侣）+ 实际情感好感度
- 两条线不应齐头并进 → 一条快一条慢 → 带来丰富的矛盾冲突
- 情感线快于社会线 → 辉夜大小姐模式（双方好感拉满但嘴硬不表白）
- 社会线快于情感线 → 赘婿/择天记模式（表面夫妻实际好感为零）

### 写女主对主角好

| 女主类型 | 付出方式 |
|---------|---------|
| 自卑社恐 | 偷偷帮忙塞东西但害怕被发现 |
| 傲娇型 | 用心做便当但嘴硬"做多了顺便带的" |
| 直率泼辣 | 大大方方送礼物或直球表白 |

### 善用对比

| 对比类型 | 操作 |
|---------|------|
| 时间线对比 | 女主刚认识主角时 vs 熟悉后的态度变化 |
| 双标 | 别人不能摸头但主角可以 → 凸显特殊地位 |
| 信息差 | 男女主双重身份（网上vs现实）→ 期待揭穿时反应 |

## 好感度这套常用写法

### 四阶段

萍水相逢 → 爱情喜剧 → 爱隔山海 → 大结局

### 好感度 × 关系阶段矩阵

好感度（负/零/半/满） x 关系阶段（熟悉/试探/暧昧/确认）

阶段匹配看低的一方：**按低的一方计算**。好感度到了但关系阶段没到 = 行为突兀。

### CP行为三类分界线

| 行为类型 | 分界线 | 说明 |
|---------|------|------|
| 需容忍行为 | 半好感+ | 主动亲密、妥协，得给补偿，不然就会写成舔狗 |
| 特殊对待行为 | 半好感+ | 主权、信赖、牺牲、特殊待遇、安抚 |
| 关联行为 | 双方半好感+ | 默契、分享、陪伴 |

好感度不足时写需容忍行为 = 油腻/性骚扰感。

### 5套感情线常见走法

| 走法 | 最抓人的点 |
|------|------|
| 好感度变化 | 可能升/降/已降 → 各自怎么接 → 余韵 |
| 受益 | 享受伴侣带来的好处 = 爱情线里最涨脸的那一下 |
| 争风吃醋 | 多角色为主角争风，不是主角吃别人醋 |
| 发展受阻 | 阻碍→试图解决→解决→余韵 |
| 狗粮 | CP日常互动 |

## 角色目标的独立性与关系线设计

### 主角目标得是他自己的

- 主角的目标得是他自己的 → 不能是"帮别人实现目标" → 不然主角会像在替别人跑腿
- 正确做法：把别人的目标转化为主角自己的（平叛=保护自己的利益/获得认可/获取资源）
- 自检方法：随机看一章 → 主角在主动追求什么 → 如果没有 → 主角沦为别人的棋子

### 感情线的层次设计

- 感情推进不是直线走的，应该一段一段往上抬，每一段都对应一个剧情高潮
- 社会关系线 vs 实质情感线的错位制造张力
- 最佳节奏：感情先慢后快 → 前期铺垫积累好感 → 后期集中爆发 → 匹配盛大仪式

### 配角的功能性定位

| 类型 | 功能 | 使用时机 |
|------|------|---------|
| 竞争者（鲶鱼型） | 制造压力推动主角行动 | 高潮前 |
| 助力型亲友 | 提供资源/情感支持 | 高潮前发挥 |
| 阻力型亲友 | 不认可→设置条件→主角克服→获得认可 | 高潮前后都能用 |

---

## 绿茶/负面角色

- 可以推动剧情但要谨慎使用
- 确认该角色是否有不可替代性
- 主角强势时任何角色都能变正反馈
- 主角弱势时负面角色会被读者恨

---

## 最后自己对一遍

每次完成角色关系/感情线设计后，逐项对一遍：

- [ ] **关系类型明确**：每个重要关系已归类为冲突/联盟/亲密/权威之一
- [ ] **关系有弧线**：每个重要关系最少经历一次考验或变化
- [ ] **人设有主心骨**：主角人设有明确的最抓人特征和发散依据
- [ ] **目标独立性**：主角的目标属于自己的，不是帮别人实现目标
- [ ] **好感度匹配**：CP行为与当前好感度阶段匹配（按低的一方计算）
- [ ] **读者群对味**：男频围绕"展示优秀"设计事件，女频围绕"深化连接"设计事件
- [ ] **配角有行动**：配角不是NPC式站桩等待触发，有自己的行动和动机
- [ ] **缓冲区存在**：配角攻略过程中始终保持缓冲区，拐点处标注态度变化
- [ ] **修罗场可控**：多角关系中对象之间无不可调和的矛盾
- [ ] **发糖时机正确**：爱隔山海之后不再设计新矛盾，只发实在的CP行为糖
- [ ] **角色不止恋爱**：角色生命中有恋爱之外的内容，不能只围着感情线打转
