# 长篇写作先照这几条拦

这份用于先帮 `story-long-write` 拦住几个大坑，免得长篇在大纲、细纲、正文里一路滑向慢节奏、解释腔、兑现太轻。

---

## 零、丑话先放前头

这不是随手翻一眼就完事的东西。

开工前先拿这份把坑挡一遍。

碰上下面这些硬问题时：

1. 先别进入下一步
- 细纲没过，先别写正文
- 正文没过，先别收进主正文
- 追踪/角色卡未同步，就当这次工作还没做完

2. 别靠“先写下去再说”混过去
- 一旦碰上连续慢节奏、解释腔回潮、人物功能化、连续两章只查不当章兑现，就先回炉
- 别靠多写几章把问题冲过去

3. 别拿局部修句冒充整层返修
- 句子层问题才允许局部改
- 章节层、整层问题，要回炉整章、连续章节或细纲

4. 别把“差不多能看”当成真能用
- 够不够，不是看“前后好像能圆上”
- 要看“能发、能留、能让读者继续点下一章”

5. 写作前后一般都得过一遍
- 每次写章、改章、回炉，都要做写前问答、写后问答、连续章节复看
- 任一卡点没过，这次就先别算做完
6. 连续回炉一般先把要照着走的单子定下来
- 若一次回炉涉及连续 3 章及以上，或用户已明确抱怨“反复改很多轮还没清干净”，就先生成固定检查表
- 检查表没生成前，正文先别直接改
- 检查表一旦生成，本批次就按单子走，别中途跳章、换级、改优先级

7. 正常写作顺序最好别乱
- 正常写作一般顺序是：先把设定钉住 -> 卷纲 -> 细纲 -> 细纲能写 -> 正文 -> 写后检查 -> 追踪同步
- 别把正文提到细纲之前
- 也别把“正文写完后再补细纲/追踪/角色卡”当成常规工作法

## 一、总的拦法

这份不聊虚的，就管“碰上什么情况先停一下”。

1. 留存优先于解释完整
- 正文如果大半都在补背景、解释局势、说明风险，而不是让现场继续往下走，一般就先别算过

2. 信息推进不能冒充情绪推进
- 若一章答不出“谁赢了、谁亏了、谁被踩了、情绪有没有抬起来”，一般就先别算过
- “知道更多”不等于“爽点成立”

3. 场面不够就按章节失败看
- 若正文长期只是在说局势、说风险、说后果、说下一步，缺少动作、阻力、失态、代价、局面变化，一般至少 `F2`
- 若读起来像把思路铺开重说、一路照功能往前推、只是拿线索转手，一般至少 `F2`

4. 人物一旦只剩工具活，就按质量没过看
- 重要角色若只负责递信息、解释局势、关键节点顶剧情，却没有出场记忆点，一般就先别算过
- 重要角色若读者顺手都想不出样子，一般也先别算过
- 连续登场后只剩功能、不剩人味，一般回炉补出场分量

5. 章节太薄就按回炉处理
- 若一章主要只是“推进到下一步”，却没有改掉任何一处现实局面，一般就算正文分量不够
- 若动作、外部反应、对手失态、代价、谁前谁后变了、规则变化里几乎都缺，一般至少 `F2`

6. 连续“更接近了”就按整层毛病看
- 连续两章主收益都是“更接近真相 / 更接近幕后 / 更接近下一步”，一般至少 `F3`

7. 书名主打一旦写歪，就按整层毛病看
- 若主要推进持续让位给副线推进，一般至少 `F3`
- 若书名 / 文案 / 金手指 / 复仇 / 身份 / 流放处境只能靠总结句证明还在，一般至少 `F3`

8. 情报卖点得尽快兑现
- 情报若只负责解释更深、指向下一站、证明这段也在推进，一般就先别算过
- 情报落下后，`1-2` 章内要转成现实收益；不然一般至少 `F3`

## 二、几个阶段都得照着看

下面这些会跨细纲、正文、回炉三个阶段一起生效，比单个文件里的零碎提醒更靠前。

1. 细纲准入细则先看：
- `references/gates/outline-gates.md`

2. 正文执行要求先看：
- `references/gates/draft-gates.md`

3. 正文句子与段落红线先看：
- `references/writing-red-lines.md`

4. 追踪同步与未来预案禁区先看：
- `references/gates/tracking-gates.md`

5. 以下任一项还没做完，就别收进主正文：
- 章级写后检查没过
- 连续章节检查没过
- 追踪同步未完成
- 角色卡更新未完成
- 若当前属于 review / 回炉批次，则 review 还没给出能收口的说法

6. review 结果这么看：
- `APPROVE` 或“仅句子层噪音，可局部改后保留”才算能收进来
- `CONCERNS` 若碰上章节层 / 整层问题，就先别收进来
- `REJECT` 直接退回回炉

## 三、连续几章一不对劲就得拉警报

1. 连续慢节奏报警
- 连续两章只靠查信息推进：报警
- 连续三章无外显战果：报警
- 连续两章只有信息升级，没有情绪升级：报警
- 连续两章主要战果都是“拿到下一个新凭借”：报警
- 连续两章章尾都更像“准备去下一章做事”：报警
- 连续两章读感像“高质量提纲扩写”：报警

2. 一报警就别顺着往下写
- 别想着“多写几章就冲过去了”
- 要回炉：
  - 细纲
  - 连续章节
  - 或当前章的兑现结构
- 回炉时优先补的不是解释，不是背景，不是总结，而是：
  - 动作
  - 阻力
  - 对手反应
  - 外部反应
  - 代价
  - 被逼选边变化
  - 关系 / 规矩 / 资源变化

3. 书名主打开始写偏报警
- 碰上以下任一情况，就先别顺着往下写：
  - 连续 `3` 章主收益都不在书名主打上
  - 连续 `2-3` 章情报没有兑现成硬战果
  - 书名主打里的身份 / 最主要的处境只剩名义，不再持续带来限制、羞辱、风险、资源卡口或反压收益
  - 复仇线只剩“以后再查”，当章不再写出撕口收益
  - 书名卖点只能靠追踪文件或作者总结来证明还在
- 这类问题统一按整层毛病看，得回到细纲层把章节组重做

4. 连续章节回炉还得多问几句
- 如果一次回炉涉及连续 3 章及以上，还要额外回答：
  - 每章最主要的收获是什么
  - 哪一章是整组最猛的一下
  - 是否仍有连续两章只查不当章兑现
- 任一条答不清，就说明这次回炉还没收住

5. 连续章节还得再看主打有没有跑偏
- 如果一次回炉涉及连续 3 章及以上，还要再额外回答：
  - 这组的主要推进到底是哪一项书名主打卖点
  - 这组结束后，主角在书名主打这一层到底赢到了什么
  - 这组有没有让副线推进反客为主
- 任一条答不清，就说明这组还在往外偏
- 如果答案只能落在“更接近真相/更接近下一站/更接近幕后”，那也说明还没收住

6. 重要人物连着出场时还得再盯一遍
- 如果一次回炉涉及重要人物连续出场段，还要额外回答：
  - 这几章里读者能记住他/她什么
  - 说话方式和动作习惯是不是还认得出来
  - 是否仍然只剩功能，没有魅力、压迫感或可想象空间
- 任一条答不清，就说明人物还没立住

7. 章节厚不厚还得再看
- 回炉或新写完成后，还要额外回答：
  - 这章像“正式正文”，还是像“提纲扩写”
  - 这章除了主推进，还有哪两层以上的肉
  - 本章哪一方被当场改写了处境
  - 本章改掉了哪一步现实
- 如果回答只能落在“发现更多”“更接近了”“下一步更清楚了”，那就还是没过

8. 连续两章兑现不足即整组回炉
- 若连续两章主战果都只是：
  - 拿到下一条线
  - 更接近真相
  - 逼出下一步地点
  - 发现更高层影子
- 那就别只修后一章
- 要把这两章作为一个回炉单元重做

9. 别一点点缝缝补补地回炉
- 如果问题已经被判断成章节层或整层：
- 别采用“先局部改一点再看”的循环
- 别每修完一章就重新定义整组问题
  - 先做整组检查，再分成：
    - 局部改
    - 中等重修并补分量
    - 重写级扩写
- 未分级完成，别开写

10. 连续回炉最后得把结论说明白
- 每个回炉批次结束时，得把这两句答明白：
  - 当前剩下的是句子噪音、分量问题，还是整层毛病
  - 能不能收进主正文
- 别用以下模糊话顶替说法：
  - “已经好多了”
  - “更接近能过了”
  - “基本可以了”
  - “再看一轮”

## 四、返工前先看毛病在哪一层

发现问题后，先看毛病卡在哪一层：

1. 句子层
- AI 腔
- 重复句式
- 作者解释句
- 章尾替读者收尾的句子

这类就局部改写

2. 章节层
- 场面不足
- 节奏拖
- 战果弱
- 章尾弱
- 提纲扩写感
- 正文分量偏薄
- 连续几章都只是在“更接近了”

这类就回炉整章

3. 整层
- 细纲像在发讲解稿
- 连续慢节奏
- 信息推进代替情绪推进
- 爽点分布失衡
- 角色长期只有功能，没有出场分量

这类就回炉连续章节，或者回第 3 阶段修细纲

这里就记一条：
- 整层问题别装成“改几句就好了”

## 五、连续回炉时这些话得答出来

连续回炉或整组检查时，至少答一下：

1. 这一章最重的现场是什么？
2. 这一章最主要的收获是什么？
3. 这一章谁压谁，谁翻盘？
4. 这一章如果只有信息没有兑现，会不会拖？
5. 章尾靠什么把人拽进下一章？
6. 本章最重要的人，读者能记住他/她什么脸、什么动作、什么口气？
7. 本章谁会先失态、先丢脸、先被逼选边？
8. 本章如果写完只剩“拿到下一条线”，是不是立刻判薄？

任一条答不上来，就说明这组还没收住。

## 六、写完后固定这么记

每次新写/回炉做完后，主线程都得补一份简短写后检查，别跳过。先按这个格式写：

### 单章写后检查
- 本章最重现场：
- 本章最主要收获：
- 本章谁被打疼：
- 本章改掉的现实局面：
- 本章关键角色出场落点：
- 是否碰上硬问题：否/是（若是，停止并回炉）

### 连续章节检查（涉及多章时必填）
- 本组最猛的一章：
- 本组是否存在重复推进：
- 本组是否存在连续两章只查不当章兑现：
- 若存在，停止并回炉哪几章：

### 最后能不能收进主正文
- 可收进主正文 / 不可收进主正文

没有这份写后检查，这次活就先别算做完。
