# 纯个人升级流单独变体

> 用于 `纯个人升级流 / 个人突围流 / 单人登阶流`。这类题材不是不能用模板链，而是不能生搬“群体争先型”模板写法。最重要的变体不是删模板，而是把模板里准备用的现场东西从“大群体改线”切成“强者往前压一步 / 资格官把话收短 / 竞争者提前封线 / 主角多轮试探”。

---

## 一、一般什么时候翻这份

- 纯个人升级流
- 个人天赋突围流
- 单人登阶流
- 单人抢机缘流

碰上上面这些题材时，就额外翻这份。

---

## 二、和通用模板链怎么接

通用链仍然保留：

1. `第一爆点`
2. `第二轮收益`
3. `中段外部反应`
4. `后段第二轮外部反应`
5. `章尾前一拍抬升`

但现场里要拿来用的东西，得换一套。

别顺手又写成：

- 大群体排队
- 大群体抢号
- 大群体推车
- 大群体交易口改线

先改成：

- 强者亲自往前压一步
- 资格官 / 守关者当场把话收短
- 对手势力提前压线
- 主角被迫多试一次 / 多赌一次 / 多让一步

---

## 三、第二轮外部反应要怎么改

纯个人升级流里，`第二轮外部反应` 不一定非得写成一大群人炸锅。

下面这些现场东西都算有效：

1. `竞争者往前压一步`
   - 原本旁观的强者提前下场
   - 原本稳坐前头的人主动往前压一步
2. `资格官把话收短`
   - 守关者、簿手、裁定者临场把原话收回一截
   - 公开记录从旧规矩慢慢滑到新规矩
3. `压线者加码`
   - 强势对手把原本隐藏的真实压迫手段提前亮出来
   - 例如：封锁、禁手、压境、夺签、断退路
4. `主角错判后再换手`
   - 主角原本以为先抢 A，临场改成先保 B
5. `围观者只保留最少量`
   - 若确有围观，只能作为陪衬
   - 别让“围观炸锅”去扛主外部反应

再补一条判断法：

- 只“强者下场了”，还不算完全生效
- 最少还得碰上下面一项，才算 `第二轮外部反应` 真立住：
  - 有人被逼着让位
  - 有人被切走先手
  - 有人被迫改先后
  - 有人被迫加价 / 加命 / 加代价
  - 原本还能观望的人，被迫立刻选边或出手

若只做到：

- 强者走近了
- 强者开口了
- 强者脸色变了
- 全场知道她也要抢了

那就按 `半生效` 看，别把结果判太高，说成模板已经写透。

下面这些不算有效外部反应：

- 只是大家更震惊了
- 只是强者脸色变了
- 只是旁白说“全场都明白了”
- 只是主角心里更稳了

---

## 四、正文厚度往哪补

纯个人升级流的厚度，不该主要靠一大群人争先来补。

先往这些地方补厚度：

1. `多轮试探`
   - 一试不成，再试第二步
   - 每次试探都要改风险
2. `抢手`
   - 主角和竞争者围绕同一个资格 / 签 / 令 / 门同时出手
3. `错判`
   - 主角或对手先判断错，再临场换手
4. `压迫升级`
   - 对手从口头压人升级为真实封手 / 封退路 / 封资格
5. `资格重排`
   - 原有先后次序被重写
   - 但不一定靠大群体，只要资格官或强者当场把说法缩回去即可

别靠下面这些东西硬补厚度：

- 主角长段思考
- 设定说明
- 强弱分析
- 旁白先把“这一下到底算什么”替现场说出来

---

## 五、第二轮收益要怎么判

纯个人升级流里，第二轮收益先看这些地方有没有落住：

- 主角抢到公开试手资格
- 对手失去独占先开口定事的资格
- 资格官被迫认下一口新规矩
- 原本不该碰碑 / 碰门 / 碰令的人，被迫获得公开出手机会

收益再往下分，大概是这几步：

1. `开门收益`
   - 只做到把旧规矩打松
   - 只算半口
2. `资格收益`
   - 主角拿到公开试手、公开入场、公开争先资格
   - 算成立
3. `战果收益`
   - 主角已经先咬住一格别人不能白拿走的先后机会 / 名分 / 资格 / 物证
   - 算写透

`战果收益` 再往下分：

- `名分战果`
  - 先占住公开先后
  - 先占住公开挂号
  - 先占住公开名单
  - 先把资格写进记录
- `实利战果`
  - 先过门
  - 先入场
  - 先拿物
  - 先吃到第一步真实收益

若只做到 `名分战果`，别误判成“这一章已经把最终实利拿到手”。

写的时候和检查的时候，别把 `开门收益` 直接判成 `战果收益`。

若第二轮收益只做到：

- 对手更警惕
- 主角更坚定
- 全场更紧张

那就按未成立处理。

若只做到：

- 旧规矩被打松
- 大家开始重新讨论
- 强者开始下场

那就按 `半生效` 处理，别写成“收益已经全到手了”。

---

## 六、章尾前一拍怎么落

纯个人升级流的 `章尾前一拍`，先看这些落点：

- 资格官追加一句会改先后的话
- 强竞争者提前下场
- 主角刚抢到手的资格被立刻要求公开验证
- 原本封死的门，突然只开半口

轻重再补一句：

- 最佳：`新名单 / 新名分 / 新先后 / 新物证` 当场落档
- 次佳：`资格官追加一句会改下一口归属的话`
- 再次：`强竞争者亲自切走半步位置`

若章尾只做到：

- 场面更紧了
- 竞争者更重视了
- 下一章会更危险

那就按还没写透处理。

别这么写：

- “更大的还在后头”
- “下一章才是真麻烦”
- “他知道自己快赢了”

---

## 七、写前单独再查一遍

写纯个人升级流时，写前额外回答：

1. 本章两口外部反应分别由谁来顶出来
2. 若没有大群体，谁来承担“把线改过去”这一下
3. 本章厚度主要靠哪两种：
   - `多轮试探`
   - `抢手`
   - `错判`
   - `压迫升级`
4. 第二轮收益是否真正改了资格归属
5. 章尾前一拍是否会立刻改下一步现场谁先谁后

任一项答不出，就先别进正文。

---

## 八、写后单独再查一遍

写纯个人升级流时，写后额外复查：

1. 两口外部反应是否只是弱化版围观
2. 正文厚度是否过度依赖主角内心
3. 是否出现大段强弱判断替代现场动作
4. 第二轮收益是否只停在气势层
5. 章尾前一拍是否真改变了当前章的资格先后

若碰上下面任一项，最少记 `F2`：

- 两口外部反应都只是围观惊呼
- 多轮试探没写成动作，只写成判断
- 竞争者没有真实下场挤人位置
- 资格官没有真的把原话收回去
- 主角整章只靠“我知道该怎么做了”推进

---

## 九、目前先这么定边界

目前已验证：

- 通用模板链在纯个人升级流里不会完全失效
- 但若不做变体，最先失真的通常是：
  - `第二轮外部反应`
  - `正文厚度`
  - `解释腔控制`

所以，写纯个人升级流时，不是停用模板链，而是要切到这份单独变体。
