# 角色怎么搭得更像活人

> 角色设计时可参考这份，先判断该走哪条路，再往下做。

---

| 角色反差怎么搭 | 三层标签反差法 + 行为反差进阶 |

## 三层标签反差人设法

### 三层大概怎么拆

| 这一层 | 名称 | 说的是什么 | 示例 |
|------|------|------|------|
| 第一层 | 身份标签 | 外界看到的身份 | 豪门弃妇 |
| 第二层 | 表现标签 | 角色展现给外界的行为 | 隐忍不发、逆来顺受 |
| 第三层 | 内核标签 | 角色真正的内心 | 冷静、有计划、步步为营 |

### 做法

1. 给角色填三层标签，三层之间最好有反差，反差越明显，人越立得住
2. 身份标签和表现标签可以相似（先把刻板印象立住），但内核标签最好翻一下
3. 反差越大，角色“亮牌”的那一下越抓人
4. 每个重要角色最好都有一层反差
5. 用行为对比来体现反差，不要直接描述：

```
身份标签：豪门弃妇
表现行为：被骂不还口，被赶不反抗
内核行为：悄悄录音、收集证据、提前转移财产
```

---

## 三层标签再往前走一步

当关系进入第三层（亲密关系）后，角色在第一、二层关系中能做出的行为反而做不出来了。

| 关系走到哪一步 | 能做什么 | 反而做不出来什么 |
|----------|---------|-----------|
| 身份标签期（陌生） | 调戏、冷漠、保持距离 | — |
| 表现标签期（熟悉） | 继续之前的行为模式 | — |
| 内核标签期（亲密） | 展露真实自我 | 之前的调戏/冷漠反而做不出来了 |

### 直接照着做

1. 先看角色现在走到关系的哪一步了
2. 行为反差本身就是人设的一部分，得真写进剧情里
3. 用“行为退一步 = 感情更深”去拉张力，比如前期爱调戏人的，真靠近后反而不敢说
4. 别让角色到了亲密阶段还完全照着早期行为来，不然反差就没了

## 人设关联怎么分轻重

| 轻重 | 说的是哪类关联 | 主要拿来干什么 | 大概多少合适 |
|------|------|------|---------|
| 强关联 | 直接影响剧情走向和最猛那类爽点的设定 | 推动人物碰撞、剧情推进 | 每个角色最好有 3 个 |
| 中关联 | 配角身上那些也能撑场的设定，但别压过主角 | 让配角有用也好认 | 适量 |
| 弱关联 | 丰富人物厚度的个人偏好设定 | 增加角色真实感和记忆点 | 不限 |

### 直接照着做

1. 把主角的实力、钱财、人脉、背景等影响剧情走向的属性归为强关联
2. 看看强关联这块，每个角色最好能凑到 3 个
3. 把"喜欢吃蛋糕""爱好看美女"等归为弱关联，仅用于丰富人设
4. 弱关联别抢戏，真正撑剧情的还是强关联

## 人设别写成空壳

人设 ≠ 人物设定表。

- 人物设定表 = 身高、外貌、家庭背景等信息堆砌（外在信息）
- 人设 = 围着角色那股最要命的劲去搭出来的东西
- 人设主要是拿来撑戏、撑拉扯的，让这个人带着剧情跑，不是被剧情拖着走

**先这么查**：如果你写出来的只有身高、体重、爱好，没有“这个人到底哪里有劲”的那一层，就得重做。

## 配角怎么写得有用

### 先记这几句

1. 配角的能力和特质得真跟剧情挂得上
2. 配角很多时候是拿来衬主角的——他的能力要能给主角出力，他脱困也最好跟主角有关

### 复合型配角：白手套这一路

| 这类人能干什么 | 一般怎么用 |
|------|------|
| 替主角说话 | 凡是主角不适合说的话，他来说 |
| 替主角发狠 | 凡是主角不应该发的狠，他来发 |
| 绝对维护 | 对侵犯主角利益的人坚决反击 |
| 粗中有细 | 将反派密谋录音再播放，取得大众支持 |
| 适时搞笑 | 不断输出正反馈，缓解紧张情绪 |

**这里这样写**：光环比主角弱一点，别抢风头；出场要亮；别人一质疑主角，他能第一时间顶回去。

### 立体人物和扁平人物怎么取

1. 不是所有角色都得写得很立体，先看他上场到底有没有活
2. 扁平化角色保持神秘性，反而更恐怖/更强大
3. "洗白弱三分，黑化强三倍"——高手被立体化就变弱了
4. 缺什么人就补什么人，别把所有角色都硬拽去走立体路线

### 人设反差常见六个方向

| 反差放在哪两头 | 可以怎么拿 |
|--------|------|
| 表面性格 vs 内在性格 | 笑面虎/刀子嘴豆腐心 |
| 外表形象 vs 表面性格 | 看着凶实际软 |
| 外表形象 vs 内在性格 | 看着柔实际狠 |
| 身份 vs 生活方式 | 豪门子弟住地下室 |
| 身份 vs 爱好特长 | 杀手擅长插花 |
| 身份 vs 内在性格 | 军人内心文艺 |

### 直接照着做

1. 先想清楚这个配角上来是替主角说话、发狠、搞笑，还是递信息
2. 从这六种反差里挑 1-2 种给配角贴上
3. 配角的光别压过主角
4. 这种就是来办事的配角，别硬往复杂里堆

## 配角人设怎么才不发飘

1. "正戏反写"——看似正面的行为背后有反派动机
2. 配角情感保护壳：看似冷漠无情，实则有深层情感驱动
3. 避免崩人设：角色怎么选、怎么做，得符合前面立起来的人设
4. 可以先从题材里已经跑熟的“标签人物”入手，成熟题材的人设路子本来就经得住市场检验

---

## 凸显人设的方法

### 直接照着做

1. 抓亮点——找到角色最突出的那个特质，集中火力展现
2. 用行动展现人设，不要用旁白描述（如出狱后不卑不亢、争取利益、展现老练）
3. 用反差碰撞制造张力
4. 反差能拉张力，但也会筛读者，别一下拉得太猛

---

## 高情商/聪明人桥段怎么写

### 直接照着做

1. 高情商得靠具体场面撑出来，别靠旁白夸
2. 写察言观色、临场应变、控场接话的具体场景
3. 高情商桥段最好找个成熟例子借劲，每个桥段最好都带一个“眼看要尴尬，结果被主角化开”的翻面
4. 简单说，就是别平推，最好让场面先绷一下，再被人稳住

---

## 用配角把期待拴住

### 直接照着做

1. 选一个配角做“任务基地”，让这个人同时接住几个短期和长期期待
2. 主角每次解决事件装完逼后去找该人物，开始新一轮装逼
3. 每个剧情单元结束后利用同一人物展开新剧情
4. 人物下线时带来更大的好处，用"歪打误撞收获更多"转变读者"损失厌恶"心理

---

## 群像怎么写

### 直接照着做

1. 每个角色一出场都得有用，干什么都说不清的先别放出来
2. 角色间的关系要有层次（引路人、对手、伙伴、障碍）
3. 不同角色要有不同的说话风格和行为特征
4. 用行为把角色特点亮出来，别靠旁白往外讲

### 配角脸谱化怎么拿

1. 抓住 1-2 个突出特点做鲜明标签（口头禅/标志性动作/极端性格）
2. 标签化让读者迅速记住，不过多占用笔墨
3. 这种办事型配角要把活说清：推剧情、托主角、递信息

### 角色怎么出，频率怎么拿

1. 高人气角色的出场频率最好卡在每 3-5 章一次，既留期待，也别让人看腻。每次出场最好都带新信息或新作用
2. 读者不感兴趣的角色该舍弃就舍弃，自然"下线"
3. 用"以角色带角色"引入新角色——喜爱延续

### 角色池怎么越织越大

1. 以角色带角色：女主讨喜 → 写女主爸妈/姐姐/闺蜜 → 喜爱延续到新角色
2. 像网一样编织下去 → 很快就有取之不尽的角色池
3. 每个新角色贴一个标签，个个鲜明

## 往人物这边再深挖

### 写的时候记住

先判断当前项目更适合“故事深化”还是“人设深化”：

| 碰上什么情况 | 先往哪边使劲 | 具体怎么动 |
|------|------|------|
| 主线已经复杂、多势力、多反转 | 向人设深化 | 保持故事结构简单，在人物欲望、反差、关系互动上制造新鲜感 |
| 主线过直、人物已经鲜明但事件不足 | 向故事深化 | 增加阻碍、目标变化、信息差或代价，不新增无功能角色 |
| 短篇/轻量篇幅 | 向人设深化 | 用 1-2 个强人设标签驱动冲突，不铺大世界观 |
| 长篇关键卷/高潮卷 | 两者结合 | 故事再往上抬一层，同时让关键人物关系真有变化 |

### 直接照着走

1. 先拿一个简单甚至有点老的故事底子
2. 在人物身上加"戏剧性"——能让目标读者找到乐子的设定
3. 靠几个人物来回碰撞，把剧情一直带着往前走

### 循环写法（一个简单情节可写 5 章以上）

- 第1章：主角装逼 → 师叔更迫切
- 第2章：师叔逼女生 → 恋人帮着威逼利诱
- 第3章：女生靠近主角 → 恋人在暗地暗爽
- 第4章：女生回去哭诉 → 恋人假意安慰一边暗爽
- 第5章：复命领赏 → 女生发现恋人对奖赏没那么高兴

## 人物塑造技巧

### 女性角色塑造（男频）

1. 读者喜欢的是女主的标签（高贵优雅、肤白大长腿），不是外貌描写
2. 利用刻板印象：吸血鬼=高贵优雅+外貌出众，不用特意描述读者就知道
3. 网文里美貌是最廉价的资源——不要花大量笔墨写外貌
4. 性格设定可极端化：傲娇到极致、对主角爱得深沉、不考虑自身利益
5. 保持性格一致性：设定后言行始终符合

### 主角可以犯错，但要把握尺度

| 可原谅 | 不可原谅 |
|--------|---------|
| 因不可抗力或信息差导致的错误 | 因蠢、因圣母心泛滥导致的错误 |
| — | 与实力不匹配的错误（斗帝强者因小人物犯蠢） |

### 主角人设内外反差

1. 可以写表面和心里不一样的反差（如表面色牛，实际内心孤僻）
2. 反差感增加角色深度和趣味，提供丰富的情节冲突可能性

### 反派到底是什么

反派 = 与主角立场不同、利益相悖的人。赋予反派自己的想法和追求，不要写纯粹的恶。

### 渣人设写法

1. 开头就清晰传达主角人设，穿越前后保持一致
2. 避免"又当又立"——前面没描写主角是老色皮后面突然追女读者反感
3. 人设先亮明白，其实就是在告诉读者这本书会来哪一口

## 人物记忆点法

1. 每个人物最好都有一处特别突出的性子
2. 只有让某一特征特别突出，才有记忆点
3. 目标：哪怕剧情不推进，几个人物坐一起聊天，读者都看得津津有味

---

## 金手指绑架人设法

### 直接照着做

1. 金手指不只是给主角超能力，目的是让主角的性格合理化
2. 用金手指本身的走法去逼出行为 → 配角反应自然把人立出来 → 不需要直接描写主角品格
3. 追求道德爽感（最高级爽感）：让主角成为行为层面让读者心服口服的"活圣人"

---

## 以梗为中心塑造人设

### 直接照着做

1. 单一标签"杀伐果断""热血"太模糊 → 用具体梗来建设角色 → 读者产生强记忆点
2. 例：配角每次看到主角成功 → 羡慕嫉妒恨 → "你真该死啊" → 强记忆点+趣味性
3. 好人设+平平无奇的金手指+俗套套路 → 也能发挥意想不到的吸引力

## 九维人设表

| 这一块 | 写什么 | 记住什么 |
|------|------|------|
| 基本信息 | 前身今世/外貌特征 | 从人物下手建立长期期待感 |
| 性格 | 反差人设 | 为成长弧线铺垫 |
| 爱好习惯 | 贪财/好色/杀人后超度 | 做强烈记忆点反复强调 |
| 人际关系 | 关系网越复杂越能制造冲突 | 牵扯的重要人物个个有实力有背景 |
| 背景 | 家族/出身/成长环境 | — |
| 成长弧线 | 见下方三步走 | — |
| 对比反差 | 恶人vs好人/医生救死扶伤→扛起枪杀人 | — |
| 价值观态度 | 不同身份地位看法不同 | 态度=行动目标 |
| 人物小传 | 典型重要故事 | 浓缩的生平经历 |

### 人物成长弧线（三步走）

| 走到哪一步 | 这一段最抓人的点 | 对应 |
|------|------|------|
| 小我（幼年） | 虚构自己，想要成为他人眼中的我 | 动机 |
| 自我（青年） | 真实自己，夹在两者之间痛苦 | 态度 |
| 他我（成熟） | 别人眼中的自己就是追求的目标 | 价值观 |

情绪公式：满足（认同）→ 打击（同情）→ 怀疑（共鸣）→ 心痛（共情）

## 关系四步走

| 走到哪一步 | 熟到什么程度 | 一般会怎么表现 |
|------|---------|------|
| 相遇（陌生人） | 只知道外表 | 警惕/保持距离/设防 |
| 相识（过客） | 初步了解不知底线 | 察言观色/阿谀奉承 |
| 相知（好哥们/闺蜜） | 知道秉性和底线 | 聊骚话/吐露心声/倾诉烦恼 |
| 相恋（爱人） | 了解内心伤疤 | 占有欲强/喋喋不休/任劳任怨 |

## 用“职场那套人情事”看网文关系

### 写的时候别忘了

网文很多时候就是：先在一个“职场”里爽透，再到另一个“职场”里继续爽。

### 哪些地方都差不多，哪些地方不一样

| 大家都差不多的地方 | 说明 |
|---------|------|
| 层级 | 任何环境中都有上下级关系 |
| 互助 | 同级之间的合作和互利 |
| 互害 | 同级之间的竞争和倾轧 |

| 不同地方主要差在哪 | 说明 |
|---------|------|
| 合作方式 | 不同环境合作模式不同 |
| 抢的东西 | 不同环境里大家争的东西不一样 |
| 个体能力 | 主角的独特优势 |

### 直接照着做

1. 先给主角找一个“职场环境”（学校/宗门/公司/军队）
2. 设置层级关系（上级/同级/下级/对手）
3. 利用层级关系自然产生矛盾和冲突
4. 主角在这个环境中爽透后，换到更大的"职场"

## 代入感构建法

### 写的时候记住

- 代入感要守住：节奏和情感链条别断。断了以后，读者就会开始拿现实想法挑你这段
- 一旦读者开始拿现实想法评判 = 节奏出了问题
- 最重要的是情感链条别断

### 直接照着做

| 这招 | 怎么做 | 会出来什么感觉 |
|------|------|------|
| 共情切入点 | 用日常生活常见事件 | 瞬间拉近距离 |
| 细节真实性 | 填充生活化细节 | 强化世界真实性 |
| 救猫咪情节 | 开头一个情节强调主角人设某方面 | 读者快速熟悉并代入 |
| 共情前提 | 让角色暴露真实一面（弱点/困境） | 读者产生保护欲 |
| 合理化虑点 | "缺钱"这类人人都懂的困境 | 天然共情 |

### 共情一般分三层

1. **表层共情**：经历类似
2. **深层共情**：情感类似
3. **认同感**：价值观一致

### 破坏代入感的行为（避免）

- 角色做出读者无法理解的选择
- 大段设定说明打断叙事
- 节奏拖沓导致情感链条断裂
- 主角三观与目标读者严重偏离

---

## 安全感与主情绪把控

1. 不同题材的读者对安全感需求不同，先确认题材
2. 极道流等紧张感题材：安全感拉爆 = 危机感消失 = 主情绪偏移——不要给主角找过强的靠山
3. 别给主角塞那种随时能把危机一把抹平的强力靠山。靠山一旦把紧迫感冲没了，这条线就会发虚
4. 精力有限时，情节始终是第一位的，世界观只是加分项

## 最后自己对一遍

角色搭完后，把这几条顺手过一遍：

- [ ] 角色有三层标签（身份/表现/内核），其中最好有一层带反差
- [ ] 强关联设定是否达到 3 个以上
- [ ] 人设不是只堆外貌资料，是真有那股人味和劲
- [ ] 配角上来干什么很清楚，也没把主角风头压过去
- [ ] 每个角色最好都有一个记忆点
- [ ] 配角每次出场都有活，不是在那儿凑热闹
- [ ] 不同角色是否有不同的说话风格和行为特征
- [ ] 主角的错误是否在可原谅范围内
- [ ] 反派是否有自己的想法和追求，不是纯粹的恶
- [ ] 人设言行是否前后一致，有没有崩人设
- [ ] 代入感情感链条是否连续，有没有被设定说明打断
- [ ] 安全感是否匹配题材要求（紧张题材不能给主角太强靠山）
- [ ] 高情商桥段是否靠具体剧情展现，而不是旁白硬夸
